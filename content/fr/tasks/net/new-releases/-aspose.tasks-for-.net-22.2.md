# Meilleure allocation de mémoire pour lire les données MPP

Cette version offre un nombre réduit d'allocations de mémoire lors de la lecture des données d'affichage à partir du fichier "MPP".

# Début et fin de l'échelle de temps du projet pour MS Project®

Auparavant, dans certains cas, l'échelle de temps du projet Début/Fin n'était pas affichée correctement par Microsoft Project®. Dans la version actuelle de l'API, ce problème a été résolu.

> Pour une liste complète des fonctionnalités, améliorations et corrections de bogues de cette version, veuillez consulter [Aspose.Tasks for .NET 22.2 Release Notes](https://releases.aspose.com/tasks/net/release-notes/2022/aspose-tasks-for-net-22-3-release-notes/).
