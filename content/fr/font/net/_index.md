## Pourquoi télécharger Aspose.Font pour .NET ?

Aspose.Font pour .NET est une bibliothèque qui permet à vos applications .NET de charger, modifier et enregistrer des données de police. Aspose.Font pour .NET permet également à vos applications .NET de dessiner du texte avec la police spécifiée.

## Fonctionnalités de traitement des polices

- Charger les fichiers de police à partir du disque ainsi que le flux.
- Lisez les informations sur les polices et enregistrez les fichiers de police mis à jour sur le disque.
- Lire les informations sur les glyphes et les métriques à partir des fichiers de police.
- Détecter les symboles latins dans les polices.
- Extrayez les informations de licence intégrées des fichiers de polices.
- Rendre le texte à l'aide de glyphes de police.
- Rendu du texte à l'aide d'interfaces personnalisées.

### Formats de police de lecture et d'écriture

TTF

### Lire les formats de police

TTC, OpenType, CFF, Type1, WOFF/WOFF2
