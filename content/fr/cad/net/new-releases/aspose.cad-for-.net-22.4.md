# Meilleure gestion de la mémoire

Une fuite de mémoire a été observée lors de la conversion d'AutoCAD® DXF au format TIFF (graphiques raster). Cette version de l'API offre donc une meilleure gestion de la mémoire pour la conversion DXF vers TIFF.

# Amélioration de l'exportation DWG vers WMF

Amélioration de l'exportation d'AutoCAD® `DWG` vers `WMF` (Windows Metafile).

# Exporter l'image intégrée

Possibilité d'exporter une image à partir de données d'image intégrées.

# Rendu `DWG` amélioré

Il a été observé que le "DWG" était rendu sans couleurs. Ce problème a maintenant été résolu dans cette version de l'API.

> Pour une liste complète des fonctionnalités, améliorations et corrections de bogues de cette version, veuillez consulter [Aspose.CAD for .NET 22.4 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-4-release-notes/).
