### Imaging API Fonctions

Voici les fonctionnalités de base d'Aspose.imagerie:

- Créer des images
- Lire des images
- Écrire des images
- dessiner des images
- Processus d'images (y compris les modifications par pixels)
- Convertir entre les formats
- Cramez, tournez et redimensionnez les images via l'API.
- De-Skew & Transformer des images.
- Définir les propriétés de l'image.

Veuillez visiter la [documentation officielle](https://docs.aspose.com/imaging/python-net/) pour une liste plus détaillée des fonctionnalités.

### Read & Write Image Formats

** Formats raster: ** JPEG2000, JPEG, BMP, TIFF, BIGTIFF, GIF, PNG, APNG \
** Imagerie médicale: ** DICOM \
** Metafiles: ** EMF, WMF, EMZ, WMZ \
** trueVision: ** tga \
** Graphiques vectoriels: ** SVG, SVGZ

### Enregistrer les images sous

** Correction: ** PDF \
** Photoshop: ** PSD \
** Web: ** toile HTML5

### lire les formats d'image

** ebook: ** djvu \
** Caméra numérique RAW: ** DNG \
** OpenOffice: ** ODG, OTG \
** bitmap: ** dib \
** Image Web: ** webp \
** CorelDraw: ** Cdr (x6, x7), CMX (v2.0 32 bits) \
** PostScript: ** EPS

Veuillez visiter [Formats de fichiers pris en charge](https://docs.aspose.com/imaging/python-net/supported-file-formats/) pour plus de détails.

### Configuration requise

ASPOSE.I-IMAGIER POUR PYTHON VIA .NET prend en charge les systèmes d'exploitation Windows 64 bits, Linux 64 bits où Python 3.5 ou ultérieure est installé.

### Systèmes d'exploitation pris en charge

** Microsoft Windows: ** Windows Desktop & Server (`x64`) \
** Linux: ** Ubuntu, OpenSuse, Centos et autres.
