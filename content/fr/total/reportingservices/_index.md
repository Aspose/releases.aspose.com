Aspose.Total pour Reporting Services contient les composants suivants :

* Aspose.BarCode pour Reporting Services : Améliore vos rapports SQL pour afficher plus de 60 types de codes à barres en haute résolution.
* Aspose.Cells pour Reporting Services : Concevez et fournissez des rapports Microsoft Excel dans Microsoft SQL Server Reporting Services.
* Aspose.PDF pour Reporting Services : générez des rapports PDF, PDF/A et XPS de haute qualité à partir de MS SQL Server Reporting Services.
* Aspose.Slides pour Reporting Services : générez facilement des rapports au format PPT et PPS dans Microsoft SQL Server Reporting Services.
* Aspose.Words pour Reporting Services : manipulez et exportez des rapports RDL et RDLC vers des formats de fichiers pris en charge à partir de MS SQL Server Reporting Services.
