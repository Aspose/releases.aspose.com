# Amélioration du support pour Ubuntu Focal

Il a été observé que lors de la conversion de Microsoft Powerpoint & Reg; Le format `PPTX` à« PDF »peu de problèmes apparaissent sur Ubuntu Focal. Ces problèmes ont maintenant été résolus.

Prise en charge des graphiques de colonnes # 3D

Cette version API présente le support des graphiques de colonnes 3D. Vous pouvez maintenant ajouter un graphique avec des données par défaut, puis définir les propriétés «Rotation3D» (telles que l'axe d'angle droit, le pourcentage de profondeur, etc.).

# Amélioration de l'effet de texte hébreu

Auparavant, il a été observé que dans certains cas, l'ombre de texte hébreu était incorrecte lorsque les présentations «pptx» sont converties au format «PDF». Ce problème a maintenant été résolu.
