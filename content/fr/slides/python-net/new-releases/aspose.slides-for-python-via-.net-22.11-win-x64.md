# Exportation PDF 1.6 & PDF 1.7 Prise en charge

Le support a été implémenté pour que PDF soit exporté vers les formats PDF 1.6 et PDF 1.7.

# Exporter PDF avec A2A, niveaux de conformité A2B

Cette version API Python introduit le support de la fonction d'exportation PDF avec de nombreux niveaux de conformité, c'est-à-dire A2A, A2B, A2U, A3A et A3B.

# Gérer le son des hyperliens dans les présentations

Une nouvelle propriété ([`hyperlink.sound`] (https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/)) a été ajoutée qui représente le son joué d'un hyperlien.

# L'effet d'animation arrête le son précédent

Une nouvelle propriété ([`effect.stop_previous_sound`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/effecct/)) a été introduite qui spécifie si l'effet d'animation s'arrête le précédent du son.

> Pour une liste complète des fonctionnalités, des améliorations et des correctifs de bogues dans cette version, veuillez visiter, [Aspose.slides pour Python 22.11 Notes de version] (https://docs.aspose.com/slides/python-net/aspose-slides- pour-python-net-22-11-libéré-notes /).
