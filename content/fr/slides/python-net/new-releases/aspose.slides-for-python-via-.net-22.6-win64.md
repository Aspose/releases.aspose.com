# Comportement de verrouillage de la source de présentation mis à jour

La présentation obsolète bloquant Behavior `Load_and_Release_Legacy_Temporary_Default` Enum Le membre a été supprimée. `Load_and_release` est le comportement de verrouillage de présentation par défaut maintenant.

Ce paramètre par défaut verrouille la source uniquement pour le moment de l'exécution du constructeur «ipresentation».

> Pour une liste complète des fonctionnalités, des améliorations et des corrections de bogues dans cette version, veuillez visiter, [Aspose.slides pour Python 22.6 Notes de version] (https://docs.aspose.com/slides/python-net/aspose-slides- For-Python-Net-22-6-Release-Notes /).
