# Prise en charge de MacOS & Reg d'Apple & Reg;

Cette version de la manipulation de la présentation Python API présente la prise en charge du MacOS & Reg;.

# Obtenir des informations de substitution de polices

Capacité à obtenir les informations sur les polices remplacées lors du rendu de présentation. Une nouvelle méthode, «get_substitutions ()», a été ajoutée pour y parvenir.

# Définir le timing pour l'animation de diapositive

Vous pouvez maintenant définir le timing pour répéter l'animation de diapositive de présentation via les deux nouvelles propriétés suivantes ajoutées au [`timing`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/timing /) classer:

- `repeat_until_next_click`: Cette propriété bool se déroule si l'effet d'animation est répété jusqu'à ce que le bouton de la souris soit ensuite cliqué.
- `repeat_until_end_slide`: Cette propriété bool se déroule si l'effet d'animation de diapositive est répété jusqu'à la fin de la diapositive actuelle.

> Pour une liste complète des fonctionnalités, des améliorations et des correctifs de bogues dans cette version, veuillez visiter, [Aspose.slides pour Python 22.9 Notes de version] (https://docs.aspose.com/slides/python-net/aspose-slides- For-Python-Net-22-9-Release-Notes /).
