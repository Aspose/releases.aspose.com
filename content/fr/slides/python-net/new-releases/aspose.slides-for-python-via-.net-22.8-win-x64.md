# Support complet pour les paramètres de diaporama

Cette version API présente la prise en charge détaillée des paramètres de diapositives de présentation. Certaines des nouvelles classes et propriétés sont les suivantes:

- [`Présentation`] (https://reference.aspose.com/slides/python-net/aspose.slides/presentation/).'Slide_show_settings` Propriété: renvoie les paramètres des diapositives pour la présentation.
- [`SlideshowSettings`] (https://reference.aspose.com/slides/python-net/aspose.slides/slideshowsettings/) Classe: représente les paramètres de diaporama pour la présentation. Certaines des propriétés disponibles sont;
  - Faites fonctionner le diaporama dans une boucle
  - Capacité à afficher / masquer la narration du diaporama
  - Activer / désactiver l'animation dans le diaporama
  - Ajustez la couleur du stylo pour le diaporama
