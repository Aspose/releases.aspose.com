# Amélioration de gros maillage dans l'exportation U3D / PDF

Amélioration du grand maillage dans l'exportation du fichier U3D / PDF.

# Fusion de scène 3D simplifiée

Simplifié les fonctions de fusion des scènes via l'API. Cette nouvelle méthode permet la fusion de tout, d'un autre nœud au nœud actuel.

```csharp
var scene1 = Scene.FromFile("file1");
var scene2 = Scene.FromFile("file2");
scene1.RootNode.Merge(scene2.RootNode);
scene1.Save("output.fbx", FileFormat.FBX7700Binary);
```
