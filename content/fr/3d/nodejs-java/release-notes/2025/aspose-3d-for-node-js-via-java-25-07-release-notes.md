## Modifications de l'API ##

### Classe ajoutée **Aspose.ThreeD.Group**

Cette classe définit les regroupements de relations logiques, distincts de la classe `Node` qui représente les relations hiérarchiques pour le rendu. Actuellement, cette fonctionnalité n'est disponible que lors du travail avec le format IFC.
