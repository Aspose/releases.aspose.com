## Modifications de l'API ##

### Classe ajoutée **com.aspose.threed.CenterLineProfile**

Ce profil vous permet de donner une épaisseur à une courbe, ce qui peut être utilisé avec des installations de modélisation procédurale telles que LinearExtrusion.
