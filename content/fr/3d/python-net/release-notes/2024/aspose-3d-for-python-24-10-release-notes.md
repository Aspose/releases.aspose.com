## Modifications de l'API ##

### Classe ajoutée **aspose.threed.profiles.CenterLineProfile**

Ce profil vous permet de donner une épaisseur à une courbe, ce qui peut être utilisé avec des facilités de modélisation procédurale comme LinearExtrusion.
