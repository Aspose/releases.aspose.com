# Meilleure gestion de l'envoi d'e-mails via SMTP

Lors de l'utilisation de la version 21.12 de l'API, le problème "L'opération a été annulée" s'est produit dans certains cas lors de l'envoi d'e-mails via SMTP. Ce problème est maintenant résolu.

# Amélioration de la gestion des drapeaux vus IMAP

Auparavant, l'indicateur Aspose IMAP Vu était toujours renvoyé comme "faux". Ce problème a maintenant été résolu.

# Améliorations dans la gestion de la boîte de réception MBOX

Dans certains cas, les messages électroniques étaient supprimés après l'ajout d'un nouveau message à la MBOX existante. Ce problème est maintenant résolu.

> Pour une liste complète des fonctionnalités, améliorations et corrections de bogues de cette version, veuillez consulter [Aspose.Email for .NET 22.1 Release Notes](https://releases.aspose.com/email/net/release-notes/2022/aspose-email-for-net-22-1-release-notes/).
