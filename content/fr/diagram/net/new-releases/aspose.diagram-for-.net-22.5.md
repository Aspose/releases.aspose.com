# Amélioration de l'exécution de la méthode asynchrone

Lors de la conversion de Visio® VSDX en PDF, le fichier de sortie n'était pas enregistré dans la méthode Async tant que le programme n'était pas entièrement exécuté. La version actuelle de l'API a résolu ce problème.

# Résolution du problème de formule/valeur

Les champs de formule/valeur n'étaient pas actualisés correctement. Ce problème a été résolu dans l'API.

# Meilleure gestion du texte du tableau dans la conversion VSDX en PDF

Pendant le processus de conversion de VSDX en PDF, dans certains cas, le texte présent dans le tableau était supprimé. Ce problème est maintenant résolu.
