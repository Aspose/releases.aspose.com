# Forme de connecteur dynamique

Possibilité d'ajouter des points à la forme du connecteur dynamique dans les formats Visio®. L'exemple de code C# suivant montre comment sélectionner le réacheminement des formes de connecteur dans les diagrammes VSDX à l'aide de l'API :

```csharp
// The path to the documents directory.
string dataDir = RunExamples.GetDataDir_Shapes();

// Call a Diagram class constructor to load the VSDX diagram
Diagram diagram = new Diagram(dataDir + "Drawing1.vsdx");
// Get page by name
Page page = diagram.Pages.GetPage("Page-3");

// Get a particular connector shape
Shape shape = page.Shapes.GetShape(18);
// Set reroute option
shape.Layout.ConFixedCode.Value = ConFixedCodeValue.NeverReroute;

// Save Visio diagram
diagram.Save(dataDir + "RerouteConnectors_out.vsdx", SaveFileFormat.VSDX);
```

# Fonctionnalité Visio® / Interop améliorée

Auparavant, les formes n'étaient pas mises à jour conformément à Visio®/Interop après l'ajout de texte. Ce problème est maintenant résolu dans l'API.

# Conversions améliorées de Visio® vers divers formats

Divers scénarios de conversion de format de fichier ont été améliorés, tels que Visio vers HTML, VSD vers VSDX, Diagramme vers SVG et Conversion VSDX vers PDF.

L'extrait de code suivant en C# montre comment convertir VSDX en PDF à l'aide de l'API, tout en divisant plusieurs pages :

```csharp
// The path to the documents directory.
string dataDir = RunExamples.GetDataDir_LoadSaveConvert();
// Call the diagram constructor to load diagram from a VSDX file
Diagram diagram = new Diagram(dataDir + "Network Diagram_start.vsdx");
// Initialize PdfSaveOptions
Aspose.Diagram.Saving.PdfSaveOptions options = new Aspose.Diagram.Saving.PdfSaveOptions();
// set SplitMultiPages option
options.SplitMultiPages = true;
// save in PDF format
diagram.Save(dataDir + "SplitMultiPages.pdf", options);
```
