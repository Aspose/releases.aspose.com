## Pourquoi télécharger Aspose.SVG pour .NET ?

Cette API .NET sur site vous aide à utiliser la fonctionnalité [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) dans vos applications C#, VB.NET, ASP.NET et autres applications basées sur .NET.

## Fonctionnalités de traitement des fichiers SVG

- Fichiers au format [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/).
- Convertir SVG en autre [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- Manipulation de l'arbre DOM selon les spécifications SVG officielles.
- Prise en charge de la navigation dans le contenu via les fonctionnalités [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), CSS Selectors, Element et Document Traversal.
- Prise en charge d'un rendu de qualité.
