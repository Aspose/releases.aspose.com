# Prise en charge améliorée des modèles de couleurs CSS

D'autres améliorations ont été apportées et une interface plus étendue a été fournie pour fonctionner avec *[CSS Color Module 4](https://www.w3.org/TR/css-color-4/)* et *[CSS Color Module 5](https://www.w3.org/TR/css-color-5/)* comme décrit dans les spécifications officielles du W3C.

# Exécution plus rapide du code de traitement CSS

Des optimisations ont été apportées pour exécuter le code de traitement CSS de manière plus rapide et plus efficace.

> Pour une liste complète des fonctionnalités, améliorations et corrections de bogues de cette version, veuillez consulter [Aspose.SVG for .NET 22.8 Release Notes](https://releases.aspose.com/svg/net/release-notes/2022/aspose-svg-for-net-22-8-release-notes/).
