 Cette page contient des notes de version pour[Aspose.Cells for .NET 4.9.0](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-4.9.0/)

{{% /alert %}} 

 Nous sommes heureux de vous annoncer Aspose.Cells for .NET v !

 Ce qui a changé dans Aspose.Cells :

- Calcule les formules qui dépassent la limite de lignes/colonnes d'Excel2003
- Prend en charge MS Excel 2010 Sparklines.
- Implémente des effets 3D sur les graphiques
- Supprime les valeurs en double
- Prend en charge Autoriser les utilisateurs à modifier les plages
- Le moteur de calcul de formule est amélioré
- Détecte et supprime les connexions aux données externes
- Définit l'image liée dans le fichier xlsx
- Inclut ICustomTypeDesciptor comme source de données des marqueurs intelligents
- Les données source liées externes dans la fonctionnalité Graphique vers image sont prises en charge
- Les options de configuration de page externe MS Excel 2007 dans le fichier xls sont prises en charge
- Spécifie le contenu HTML comme valeur de cellule ou note de commentaire
- 112 correctifs et autres améliorations.

 (Remarque : la classe Aspose.Cells.Charts est renommée en classe Aspose.Cells.ChartCollection.)

 Ce qui a changé dans Aspose.Cells.GridWeb :

- Ajoute de nouveaux formats de date : "M/j/aa", "MM/jj/aaaa" et "jj.mm.aaaa"
- Définit un message d'erreur dans le titre de la cellule lorsque la validation échoue côté client
- Obtient le nom de la colonne pour une cellule côté client
- Enregistre les fichiers Excel pour diffuser ou charge les fichiers Excel à partir du flux
- Prend en charge de nouvelles fonctions : INDEX, RECHERCHE, MATCH
- 8 correctifs et autres améliorations.

 Qu'est-ce qui a changé dans Aspose.Cells.GridDesktop :

- Définit les formats de date "MM/DD/YYYY" et "M/D/YYYY" avec la boîte de dialogue Format Cells
- Importe le format de fichier CSV
- Prend en charge les formules : LOOKUP, MATCH
- 3 correctifs et autres améliorations.
