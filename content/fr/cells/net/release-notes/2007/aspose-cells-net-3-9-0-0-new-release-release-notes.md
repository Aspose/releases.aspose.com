 Chers clients,

 Nous avons publié Aspose.Cells for .NET v3.9 !

- Pour certaines raisons externes, la classe Excel est désormais renommée Workbook et la classe ExcelDesigner est renommée WorkbookDesigner. Toutes les propriétés et méthodes de ces deux classes ne sont pas modifiées.
- Prend en charge l'ajout d'images EMF dans leur format brut
- Un nouveau marqueur intelligent est ajouté
- Prend en charge le chargement de fichiers texte avec un séparateur personnalisé
- Correction d'un bogue dans la définition des hyperliens vers les fichiers locaux
- Correction de bugs dans la méthode InsertRows
- Correction d'un bug dans les paramètres de validation des données
- Correction d'un bug lors de l'ouverture de certains fichiers Excel
