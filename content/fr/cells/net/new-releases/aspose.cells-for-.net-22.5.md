#  Gestion améliorée des formules de tableaux dynamiques

Possibilité d'initialiser la valeur calculée de la cellule lors de la définition d'une formule de tableau dynamique.

#  Suppression plus rapide des colonnes vides

Amélioration des performances de suppression des colonnes vides.

#  Récupérer les pièces jointes liées `OLE` à partir de `ODS`

Un problème a été observé lors de la récupération des pièces jointes OLE liées dans le fichier `ODS`. Ce problème est maintenant résolu.

#  Exporter la chaîne JSON vers Excel&reg;

Cette version API propose un correctif au problème d'exportation de la chaîne JSON vers Excel®.
