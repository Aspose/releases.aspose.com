Nous sommes heureux d'annoncer la première version publique de Aspose.Cells for PHP via Java. Les API intègrent des fonctionnalités de traitement de feuille de calcul MS Excel en PHP (un langage de script open source largement utilisé). Aspose.Cells for PHP via Java est un composant de sous-ensemble qui comprend toutes les fonctionnalités importantes et utiles présentes dans sa version native.[Aspose.Cells for Java](https://products.aspose.com/cells/java/)Le nouveau API est tout aussi robuste et riche en fonctionnalités. Il prend en charge les conversions de format de fichier haute fidélité vers et depuis XLS, XLSX, XLSM, SpreadsheetML, CSV, Tab Delimited, HTML, MHTML et OpenDocument Spreadsheet en PHP. Veuillez visiter[Page de comparaison des fonctionnalités du produit](https://docs.aspose.com/cells/fr/java/aspose-cells-for-php-via-java-features/)pour votre référence. De plus, veuillez consulter le document sur la façon de[Environnement de configuration et autres directives d'installation](https://docs.aspose.com/cells/fr/java/setup-and-installation-guidelines/)pour votre référence.
