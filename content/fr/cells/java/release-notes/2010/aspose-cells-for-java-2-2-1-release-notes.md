 Nous sommes heureux de vous annoncer Aspose.Cells for Java 2.2.1 !

 Ce qui a changé :

-  Inclut la prise en charge du formatage conditionnel avancé fourni par MS Excel 2007
-  Prend en charge les enregistrements étendus de mise en forme conditionnelle enregistrés par MS Excel 2007 pour le fichier XLS
- Conserve les formules non prises en charge lors de la lecture des fichiers MS Excel 2007
-  Lit les images de l'en-tête/du pied de page dans le fichier de modèle MS Excel 2007
-  Des améliorations sont apportées pour la lecture des fichiers modèles
-  17 correctifs et autres améliorations.
