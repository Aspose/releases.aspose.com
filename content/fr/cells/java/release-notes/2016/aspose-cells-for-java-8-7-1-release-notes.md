## **Noter**
Étant donné que la base de code de Aspose.Cells for Java correspond au code de la version .NET pertinente, la plupart des modifications, améliorations et correctifs inclus dans le Aspose.Cells for .NET v8.7.1 sont également inclus dans ce Aspose.Cells for Java v8.7.1.
