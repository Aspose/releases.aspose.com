 Chers clients,



 Nous avons publié Aspose.Cells.ReportingServices V1.1.0.0.



 Quoi de neuf?



- ` `Nous fournissons un outil complémentaire client - Aspose Cells Report Designer qui permet aux utilisateurs d'utiliser MS Excel comme outil de conception de rapport. Avec Aspose Cells Report Designer, les utilisateurs peuvent concevoir un modèle de rapport dans Excel au lieu de Visual Studio ou d'un autre outil de création de rapports. Le modèle de rapport est intégré au fichier RDL de Reporting Services et peut être publié sur Report Server. Cela fait de la conception de rapports une tâche facile et rapide.
-  Le rendu du serveur Aspose.Cells for Reporting Services est grandement amélioré. En travaillant avec le modèle de rapport, les utilisateurs peuvent utiliser toutes les fonctionnalités avancées d'Excel dans Reporting Services, y compris les graphiques Excel natifs, les formules avancées, les tableaux croisés dynamiques, etc.
