# 自定义 `XFORM` 运算符支持

在 USD / USDZ 导入器中添加了自定义 `xform` 运算符支持。

# 改进了 3D 模型 FBX 到 USDZ 的转换

以前从 FBX 转换为 USDZ 的模型结果是全黑的。此问题已在 API 的当前版本中得到解决。

# 改进的材质属性导出

未附加纹理时未导出材质属性。现在这个问题已经解决了。
