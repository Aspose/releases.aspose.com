# 导入 Maya ASCII 文件

在 3D 操作 API 中添加了导入 Autodesk® Maya ASCII 文件的功能。现在我们的 3D API 支持 Maya ASCII 格式作为输入格式 (.ma)。

> 有关此版本中的功能、增强功能和错误修复的完整列表，请访问 [Aspose.3D for .NET 22.10 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-10-release-notes/)。
