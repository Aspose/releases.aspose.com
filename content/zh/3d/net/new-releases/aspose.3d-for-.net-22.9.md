# FBX 导入器的临时文件支持

为 FBX 导入器添加了内部临时文件系统支持。

# 改进的 GLTF 导出

`GLTF` 导出在以前的版本中没有那么好。然而，随着 3D API 的当前版本发布，`GLTF` 格式文件的导出功能得到了改进。

> 有关此版本中的功能、增强功能和错误修复的完整列表，请访问 [Aspose.3D for .NET 22.9 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-9-release-notes/)。
