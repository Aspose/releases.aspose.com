# 网格三角支持

Mesh triangulate 支持带有映射模式 Polygon / PolygonVertex 的 `VertexElementUserData`。

# TriMesh 中的 `VertexElementUserData` 支持

在 TriMesh 中添加了对 `VertexElementUserData` 的支持。

# `VertexElementUserData` 到 `glTF` 导出

添加了将 `VertexElementUserData` 导出到 `glTF` 的功能。

# `GLTF` 自定义顶点属性

添加了对 GLTF 自定义顶点属性的支持。
