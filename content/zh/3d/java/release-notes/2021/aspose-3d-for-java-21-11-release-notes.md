|**钥匙**|**摘要**|**类别**|
|:- |:- |:- |
|THREEDNET-999 |在USD中添加时间样本支持|改进|
|THREEDNET-998 |一些USDZ文件中的NullReferenceException|错误修复|
|THREEDNET-997 |一些旧的6.0和6.1 FBX文件抛出NullReferenceException|错误修复|

这个版本主要是一个bug修复版本，修复了很多USD/FBX/GLTF的关键bug。
