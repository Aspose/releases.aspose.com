# 改进的 XML 到 MPP 的转换

修复了将特定 XML 文件保存为 MPP 格式时发生的异常。以下 C# 代码片段演示了如何使用 API 将 Primavera P6 XML 文件转换为 Microsoft Project® 文件：

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Primavera XER 资源速率的改进读取功能

修复了从 Primavera XER 文件读取资源速率的问题。

# 改进了从各种格式到 MPP 的转换

修复了将特定文件保存为 MPP 格式时触发的错误。
