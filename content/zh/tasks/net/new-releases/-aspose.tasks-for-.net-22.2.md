# 更好的内存分配来读取 MPP 数据

此版本在从“MPP”文件中读取视图数据时减少了内存分配的数量。

# MS Project® 的项目时间表开始和结束

以前在某些情况下，Microsoft Project® 未正确显示项目时间刻度开始/完成。在当前版本的 API 中，此问题已得到修复。

> 有关此版本中的功能、增强功能和错误修复的完整列表，请访问 [Aspose.Tasks for .NET 22.2 Release Notes](https://releases.aspose.com/tasks/net/release-notes/2022/aspose-tasks-for-net-22-3-release-notes/)。
