# 更好地处理通过 SMTP 发送电子邮件

在使用 API 21.12 版本时，在某些情况下通过 SMTP 发送电子邮件时会出现“操作已取消”问题。这个问题现在已经解决了。

# 改进的 IMAP 看到标志处理

以前，Aspose IMAP Seen 标志总是返回为“false”。此问题现已修复。

# MBOX 收件箱管理的改进

在某些情况下，我们将新消息添加到现有 MBOX 后，邮件消息会被删除。现在这个问题已经解决了。
