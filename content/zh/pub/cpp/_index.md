## 为什么要为 C++ 下载 Aspose.PUB？

Aspose.PUB for C++ 是一个简单的 API，允许您在 C++ 应用程序中以编程方式读取 Microsoft Publisher® (PUB) 文件并将其转换为 PDF 格式。它还提供易于理解的界面来编辑 PUB 文件的元数据。

## PUB 文件处理功能

- 阅读 Microsoft Publisher® (PUB) 文件以转换为 PDF 格式。
- 通过 API 读取和编辑 PUB 文件的元数据。

### 读取 PUB 文件

**Microsoft Publisher®：** PUB

### 将 PUB 保存为

**固定布局：** PDF

### 平台独立性

您可以使用 Aspose.PUB for C++ 构建任何类型的 32 位和 64 位 C++ 应用程序。您可以通过简单地复制程序集在服务器和客户端使用它，而无需担心其他服务或模块。
