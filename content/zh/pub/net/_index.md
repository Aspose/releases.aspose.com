## 为什么要为 .NET 下载 Aspose.PUB？

Aspose.PUB for .NET 是一个简单的 API，允许您读取 Microsoft Publisher® (PUB) 文件并将其转换为 PDF 格式。它还提供易于理解的界面来编辑 PUB 文件的元数据。

- 阅读 Microsoft Publisher® (PUB) 文件以转换为 PDF 格式。
- 通过 API 读取和写入 PUB 文件的元数据。
