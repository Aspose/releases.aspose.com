# 支持 AutoCAD® SHX 格式

此 API 版本支持 AutoCAD® SHX 文件格式。它还提供了有关“SHX”格式的以下改进：

- 更好地显示 SHX 字体
- 文字粗细标准化
- 使用 SHX 符号

# 支持STEP（或STP）格式

CAD API 中添加了对 STP/STEP 文件的支持。 STEP（或 STP）格式通常用于保存 2D 和 3D 几何模型以及设计数据。导出为 STEP 格式功能现在也可用。

> 有关此版本中的功能、增强功能和错误修复的完整列表，请访问 [Aspose.CAD for .NET 22.7 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-7-release-notes/)。
