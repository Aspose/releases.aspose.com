# 支持大字体

在 API 中添加了对大字体的支持。

# 内存高效的 DWG 到 PDF 的转换

以前，当 DWG 转换为 PDF 格式时，它会占用大量内存。这个问题现在已经解决了。现在您可以将 DWG 转换为 PDF 并获得改进的结果。

# 改进了 DAE 到 OBJ 导出

在以前的某些情况下，DAE 到 OBJ 的导出失败。此问题现已在 API 中解决。
