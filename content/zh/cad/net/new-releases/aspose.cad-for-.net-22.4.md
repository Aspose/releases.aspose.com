# 更好的内存管理

将 AutoCAD® DXF 转换为 TIFF（光栅图形）格式时发现内存泄漏。因此，此 API 版本为 DXF 到 TIFF 转换提供了更好的内存管理。

# 改进 DWG 到 WMF 导出

改进了 AutoCAD® `DWG` 到 `WMF`（Windows 图元文件）的导出。

# 导出嵌入图像

能够从嵌入的图像数据中导出图像。

# 改进的 `DWG` 渲染

观察到“DWG”被渲染为没有颜色。此问题现已在此 API 版本中得到解决。
