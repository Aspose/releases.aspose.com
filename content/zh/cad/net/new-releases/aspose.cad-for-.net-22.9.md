# 导出匈牙利特殊字符

以前在某些情况下，匈牙利特殊字符会错误地从 DWG 导出为 PDF。在此 API 版本中，上述问题已得到解决。

# 改进了 DWG 到 PDF 的转换

修复了将“DWG”转换为“PDF”格式时出现的“OutOfMemoryError”。

# 改进的场景移动灵敏度

提高了移动场景的灵敏度。
