## 为什么要下载 Aspose.OCR for C++？

Aspose.OCR for C++ 是一个独立的 OCR API，可增强您的 C++ 应用程序以对 JPEG、PNG 和 BMP 图像执行 OCR 操作以提取英文内容。

## 图像 OCR API 功能

- 从图像中识别字符。
- 计算图像的倾斜角度。
- 目前，它支持英语。
- 自动对齐图像和自动检测文本区域。
- 获取一条线并识别它。

### 读取 OCR 的图像格式

**光栅格式：** JPEG、BMP（不支持 32 位和 16 位图像）、PNG

### 支持的识别字符集

CN

### 先决条件

Aspose OCR 库在系统路径中需要`onnxruntime.dll`。

### 评估版本限制

Aspose.OCR for C++ 的评估版将从图像中提取的字符数限制为 300 个。
