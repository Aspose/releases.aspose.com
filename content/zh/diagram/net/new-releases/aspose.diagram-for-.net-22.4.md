# 图像转换问题解决

将 Microsoft Visio® VSDX 转换为 PDF 时，图像转换不工作。当前版本已修复此问题。

# Visio® 形状背景颜色转换

以前在某些情况下，在将 VSDX 转换为 PDF 过程中，形状的背景颜色会被错误地转换。这个问题现在已经解决了。

# 文本位置处理

在 Visio 到 SVG 格式转换期间更好地处理文本位置。

# Visio 中更好的表格渲染到 PNG 转换

在此 API 版本中，解决了 VSDX 到 PNG 转换过程中表格和单选按钮呈现的格式问题。

> 有关此版本中的功能、增强功能和错误修复的完整列表，请访问 [Aspose.Diagram for .NET 22.4 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-4-release-notes/)。
