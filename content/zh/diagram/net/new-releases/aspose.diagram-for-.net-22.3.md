# Visio® 中的数据图形到 PDF 转换

在以前版本的 API 中，在将 Visio® 转换为 PDF 格式时，数据图形遇到了问题。此问题现已在 API 的当前版本中得到解决。

# 将 Visio® 导出为 HTML

将 Visio® 图表导出为 HTML 格式时引发的异常现已解决。
