# 改进的外部资源嵌入

以前在某些情况下，将 Microsoft Visio® 图表保存到 HTML 流时不会嵌入外部资源。不过，这个问题现在已经解决了。

# 更好地处理形状着色

在将 Visio® VSDX 转换为 PDF 时，会出现形状阴影缺失的问题。此版本解决了该问题并提供了更好的形状阴影处理。

# 将 HTML 保存到单个流

添加了将 HTML 保存到单个流的支持。

# 使用嵌入式资源将图表保存为 HTML

能够将 Visio® 图表保存为 HTML，无论是否包含嵌入式资源。
