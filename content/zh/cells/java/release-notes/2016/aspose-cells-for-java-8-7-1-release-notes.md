## **笔记**
由于 Aspose.Cells for Java 的代码库与相关 .NET 版本的代码匹配，因此 Aspose.Cells for .NET v8.7.1 中包含的大部分更改、增强和修复也包含在此 Aspose.Cells for Java v8.7.1 中。
