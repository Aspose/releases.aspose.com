什么是新的？

- 支持为 Arc、Oval、Rectangle、Comment、TextBox 等形状添加富文本
支持所有类型的形状放置
支持从模板中读取所有形状
支持在 SmartMarker 中设置数组

有什么改进？

- 添加新方法来操纵形状的位置和大小，并弃用了一些旧方法
为新的 API 添加一些演示代码和模板
在演示中用新的 API 替换弃用的 API 调用

什么是固定的？

- 修正了一些计算公式的错误
修复导出Excel2007 XLSX格式的bug
修复了复制样式的错误
修复了复制公式的错误
修正了保存图像大小的错误

注释

- 对于从模板文件中导入的图表，不支持对其数据进行操作，目前仅支持对形状属性进行操作，例如位置、位置和大小
