亲爱的客户，

我们已经发布了 Aspose.Cells for Java V1.7.2！包括以下增强和修复：

- 支持在页面设置的页眉/页脚添加图片
支持从 SpreadsheetML 文件导入页面设置、名称、保护、数据验证和文档属性
支持将页面设置、名称、保护、数据验证和文档属性导出到 SpreadsheetML 文件
支持计算更多内置函数
修复了图表中设置字体自动颜色的错误。
