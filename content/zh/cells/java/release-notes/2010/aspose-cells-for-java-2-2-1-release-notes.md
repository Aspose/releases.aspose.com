我们很高兴地宣布 Aspose.Cells for Java 2.2.1！

改变了什么：

- 包括对 MS Excel 2007 提供的高级条件格式的支持
- 支持MS Excel 2007为XLS文件保存的条件格式的扩展记录
- 在读取 MS Excel 2007 文件时保留不受支持的公式
- 从 MS Excel 2007 模板文件的页眉/页脚读取图片
- 对读取模板文件进行了增强
- 17 个修复和其他增强功能。
