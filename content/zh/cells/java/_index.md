[Aspose.Cells for Java](https://products.aspose.com/cells/java)是屡获殊荣的Excel电子表格处理API，可让Java开发人员嵌入阅读，编写和操纵Excel＆Reg的能力；电子表格（XLS，XLSX，XLSM，XLSB，XLTX，电子表格ML，CSV，ODS），HTML，MHTML，PDF和Image File Formats在其自己的Java应用程序中，而无需依赖Microsoft Excel＆Reg;。

## Excel文件处理功能

### 文档功能
 - 从不同来源打开普通或加密的Excel文件（Excel97，Excel2007/2010/2013）。
 - 以各种支持格式保存Excel文件（Excel97- Excel2007/2010/2013）。
 - 将Excel文件和电子表格转换为各种支持格式。
 - 转换为标记的图像文件格式（`tiff'）。
 - 读写Opentocument电子表格（“ ODS”）格式。
 - 修改Excel文件的文档属性。

### 工作表功能
 - 使工作表可见或隐藏。
 - 能够显示或隐藏工作表选项卡，滚动条，网格线和标题。
 - 应用工作表缩放级别。
 - 在冻结窗格中滚动时，将选定的数据可见。
 - 能够预览工作表页面中断的能力。
 - 对工作表内容，对象和方案的保护支持。
 - 执行并将页面设置配置应用于工作表。
 - 对单个或一组行和列执行各种操作。

### 数据管理功能
 - 在运行时插入特定单元格中的数据。
 - 从各种数据酸中获取数据，并导入工作表。
 - 根据其数据类型从单元格中检索数据。
 - 从工作表单元格获取数据并导出到数组。
 - 应用条件格式。
 - 对数据（例如字体设置）执行许多格式操作。

### 图表和图形功能
 - 支持创建各种图表。
 - 将自定义图表添加到工作表中。
 - 在运行时将图片添加到工作表中。
 - 打印工作表的能力。

### 高级功能
 - 使用强大的公式计算引擎来支持公式计算。
 - 操纵VBA代码或宏。
 - 创建枢轴表并在运行时更改其源数据。

## 读写电子表格格式
** Microsoft Excel：** XLS，XLSX，XLSB，XLT，XLTX，XLTM，XLSM，XML \
** OpenOffice：** ODS \
**文本：** csv，tsv \
**网络：** html，mhtml \
**号码：**苹果的IWork Office Suite Numbers应用程序文档

## 将Excel文件保存为
**固定布局：** pdf，pdf/a，xps \
**数据互换：** dif \
**图像：** JPEG，PNG，BMP，SVG，TIFF，EMF，GIF

## 支持环境
 -  ** Microsoft Windows：** Windows Desktop＆Server（X86，X64）
 -  ** macOS：** Mac OS X
 -  ** Linux：** Ubuntu，Opensuse，Centos等
 -  ** java版本：**```J2SE 7.0（1.7）`或更高版本

## 开始

Aspose.cells Java API托管在[Aspose存储库](https://releases.aspose.com/cells/java/)。您可以在Maven项目中轻松地使用Java API的aspose.cells，并具有简单的配置。有关详细说明，请访问[Installing Aspose.Cells for Java from Maven Repository](https://docs.aspose.com/cells/java/installation/)文档页面。

## 使用Java的选项将表转换为范围

```java
// For complete examples and data files, please go to https://github.com/aspose-cells/Aspose.Cells-for-Java
// The path to the documents directory.
String dataDir = Utils.getSharedDataDir(ConvertTableToRangeWithOptions.class) + "Tables/";
// Open an existing file that contains a table/list object in it
Workbook workbook = new Workbook(dataDir + "book1.xlsx");

TableToRangeOptions options = new TableToRangeOptions();
options.setLastRow(5);

// Convert the first table/list object (from the first worksheet) to normal range
workbook.getWorksheets().get(0).getListObjects().get(0).convertToRange(options);

// Save the file
workbook.save(dataDir + "ConvertTableToRangeWithOptions_out.xlsx");
```
