我们很高兴地宣布 Aspose.Cells for .NET 4.7.0！

改变了什么：

- 支持直接将Excel文件转换为PDF。
打印工作表时支持页面设置设置的网格线选项。
支持插入 WAV 文件作为 Ole 对象。
支持 Excel 2007 文件中的 R/W 复选框。
支持在智能标记中使用匿名类型或自定义对象作为数据源
支持设置显示零值选项。
支持合并和相交两个范围。
支持保存工作簿后保存数据。
支持选择是否在数据透视表中插入空白行。
增强计算引擎并支持更多功能。
 45 个修复和改进。
