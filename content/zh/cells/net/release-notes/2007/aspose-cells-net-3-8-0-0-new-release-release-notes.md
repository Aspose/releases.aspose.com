亲爱的客户，

我们发布了Aspose.Cells for .NET v3.8！

- 支持在运行时创建复选框
- 支持在运行时创建 OLE 对象
- 支持保护/取消保护工作簿
- 允许更改外部链接的数据源
- 增强的智能标记处理
- 增强的公式计算引擎
- 修复了 DeleteRows 和 InsertRows 中的错误
- 修复了使用评估版重复打开和保存文件的错误
- 修复了复制某些公式时的 AddCopy 错误
- 自 v3.7 以来的其他改进和修复
