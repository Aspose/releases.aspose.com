亲爱的客户：

我们发布了Aspose.Cells for .NET v4.2！

在这个新版本中，增加了一些重要的新特性：

- Aspose.Cells 现在完全支持在模板文件中操作图表、图像和其他绘图对象
- 支持 Chart2Image 功能。目前仅支持柱形图、折线图、饼图和条形图。并且仅支持 2D 图表。我们将在未来的版本中支持更多的图表类型。
- Xls2Pdf 转换得到增强和优化。
- 公式计算引擎得到增强和优化。
