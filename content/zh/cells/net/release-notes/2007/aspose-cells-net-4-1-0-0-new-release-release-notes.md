亲爱的客户，

我们发布了Aspose.Cells for .NET v4.1！

- 提供了从头开始创建条件格式的 API
- 提供了从头开始创建数据透视表的 API
- 为 CheckBox 类添加了 LinkedCell 属性
- 添加了新的 C# 快速入门演示
- 修复了设置公式中的 2 个小错误
- 修正了计算公式的错误
- 修复了在 Windows Vista RC2 上使用 Aspose.Cells 的问题。这个问题是由混淆器引起的。
- 修复了转换为 pdf 文件时的页面方向错误
- 修复了使用共享公式复制行的错误
