介绍

此版本包含自 v4.2 以来的新功能、功能增强和错误修复。请查看以下说明以获取更多详细信息。

什么是新的？

3136 - 支持导入/导出 Excel2007 数据/格式化

有什么改进？

3155 - 支持 xls2pdf 转换中的页眉和页脚设置设置

3170 - 公式计算引擎支持CELL功能

3175 - 二维条形图、柱形图、折线图和饼图中的形状 Chart2image 功能

什么是固定的？

3064 - Xls2Pdf 转换异常错误

3065 - 复选框设置问题

3066 - 条件格式错误
