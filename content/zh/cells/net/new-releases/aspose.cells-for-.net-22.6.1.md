#  `.NUMBERS` 数据存储5.0支持

新增对`.numbers`数据存储5.0的支持。

# 改进了 XLSX 至 PDF 渲染

据观察，在将 XLSX 转换为 PDF 时，条形图未正确呈现。该问题现已解决。

# 改进的 Excel®图表标签处理

此 API 版本提供了一些有关 Excel® 的修复程序。图表标签。具体如下：

- 图表标签不适合盒子。
- 图表标签是垂直对齐的，而不是水平对齐的。

# 将表格公式保存到 XLSB

添加了将表格公式保存到`XLSB`的支持。

# 改进的 Linux 支持

据观察，在 Linux 中，形状和文本框等对象上书写的字符或标题出现乱码。该问题现已解决。
