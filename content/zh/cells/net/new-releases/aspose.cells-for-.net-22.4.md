# 改进了艺术字形状的渲染

据观察，Microsoft Excel®在 Excel 到 PDF 转换期间，艺术字效果和形状未正确呈现。该问题现已解决。

# 改进了 XLS 到 PDF 的转换

之前在某些情况下，当我们将 `XLS` 转换为 `PDF` 时，文本框中的空白文本行大小无法正确呈现。该问题现已解决。

# 从 CSV 中删除尾随空白 Cells

此 API 版本引入了在保存逗号分隔值 (CSV) 文件时删除尾部空白单元格的功能。
