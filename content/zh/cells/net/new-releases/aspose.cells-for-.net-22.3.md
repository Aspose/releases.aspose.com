# 支持电子表格模板格式

此 API 版本引入了对 Excel&reg; 的电子表格模板格式的支持。 (XLT) & OpenOffice Calc® （OTS）程序。通过 `SaveFormat.Ots` 和 `SaveFormat.Xlt` 支持这些输出格式。

# 设置内边框

能够设置内部边框。

# 按选定行或列对数据透视字段进行排序

添加了对数据透视字段值的排序支持。可以根据所需的行或列完成数据透视字段排序。
