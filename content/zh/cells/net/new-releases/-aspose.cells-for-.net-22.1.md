# 设置打印机作业名称

添加了在将电子表格渲染到打印机时使用 `PrinterSettings` 提供和设置打印作业名称的支持。

# 自动检测`CSV`和`TSV`文件格式

此 API 版本引入了使用 API 自动检测以逗号分隔的 `CSV` 和制表符分隔的 `TSV` 文件格式的文本文件的功能。

# 从范围中排除隐藏的行/列

range 属性覆盖了单元格集合中的所有单元格。从这个版本开始，此功能已得到改进，可以在获取 `Cells.MaxDisplayRange` 时忽略/排除隐藏的行和列。

 > 有关此版本中的功能、增强功能和错误修复的完整列表，请访问：[Aspose.Cells for .NET 22.1 发行说明](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-1-release-notes/).
