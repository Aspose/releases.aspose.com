我们很高兴地宣布 Aspose.Cells for JasperReports v1.0.0.0！

什么是新的：

 l 提供Microsoft Excel格式的可编辑图表，而非静态图片。

 l 支持将Aspose.Cells for JasperReports集成到JasperServer。

 去除POI库的依赖。

 l 支持多种图表，包括条形图、面积图、饼图、折线图等。
