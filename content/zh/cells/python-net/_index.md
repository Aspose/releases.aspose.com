### 电子表格 Python 通过 .NET 本地 API 功能

- 通过 API 生成和操作电子表格。
- 高质量的文件格式转换和渲染。
- 将 Microsoft Excel® 文件打印到物理或虚拟打印机。
- 合并、修改、保护或解析 Excel® 工作表。
- 应用工作表格式。
- 为工作表配置和应用页面设置。
- 创建和自定义 Excel® 图表、数据透视表、条件
  格式化规则、切片器、表格和迷你图。
- 将 Excel® 图表转换为图像和 PDF。
- 将 Excel® 文件转换为各种其他格式。
- 支持所有基本和高级 Excel® 功能的公式计算引擎。

请访问 [official documentation](https://docs.aspose.com/cells/python-net/) 以获取更详细的功能列表。

### 读写电子表格文件格式

**Microsoft Excel®：** XLS、XLSX、XLSB、XLSM、XLT、XLTX、XLTM、CSV、TSV、TabDelimited、SpreadsheetML\
**OpenOffice：** ODS、SXC、FODS\
**文本：** TXT\
**网页：** HTML、MHTML\
**iWork®：** 数字\
**其他：** SXC、FODS

### 将电子表格文件另存为

**Microsoft Word®：** DOCX\
**Microsoft PowerPoint®：** PPTX\
**Microsoft Excel®：** XLAM\
**固定布局：** PDF、XPS\
**数据交换：** DIF\
**矢量图形：** SVG\
**图像：** TIFF、PNG、BMP、JPEG、GIF\
**元文件：** EMF\
**降价：** MD

请访问 [Supported File Formats](https://docs.aspose.com/cells/python-net/supported-file-formats/) 了解更多详情。

＃＃＃ 系统要求

您的机器不需要安装 Microsoft Excel® 或 OpenOffice® 软件。

### 支持的操作系统

**Microsoft Windows®：** Windows 桌面和服务器（`x64`、`x86`）\
**Linux：** Ubuntu、OpenSUSE、CentOS 等\
**其他：** 任何可以安装 Mono（.NET 4.0 框架支持）或使用 .NET 核心的操作系统 (OS)。
