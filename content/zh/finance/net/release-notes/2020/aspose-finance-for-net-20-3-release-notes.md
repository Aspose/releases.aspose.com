我们很高兴地宣布 Aspose.Finance for .NET. Aspose.Finance for .NET 的首次公开发布，作为一个纯 .NET 库提供了更好的性能和易用性来操作财务相关格式，例如 XBRL、iXBRL。财务 API 是可扩展的，易于使用紧凑并提供所有常用功能，因此开发人员可以编写更少的代码来执行常用操作。
Aspose.Finance for .NET 具有丰富的功能

- 从头开始创建 XBRL 实例
- 读取 XBRL 格式
- 读取 iXBRL 格式
- XBRL 验证
- iXBRL 验证
