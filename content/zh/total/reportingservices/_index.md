Aspose.Total for Reporting Services 包含以下组件：

* Aspose.BarCode for Reporting Services：增强您的 SQL 报告，以高分辨率显示 60 多种类型的条码。
* Aspose.Cells for Reporting Services：在 Microsoft SQL Server Reporting Services 中设计和交付 Microsoft Excel 报表。
* Aspose.PDF for Reporting Services：从 MS SQL Server Reporting Services 生成高质量的 PDF、PDF/A 和 XPS 报告。
* Aspose.Slides for Reporting Services：在 Microsoft SQL Server Reporting Services 中轻松生成 PPT 和 PPS 格式的报告。
* Aspose.Words for Reporting Services：从 MS SQL Server Reporting Services 中处理 RDL 和 RDLC 报告并将其导出为支持的文件格式。
