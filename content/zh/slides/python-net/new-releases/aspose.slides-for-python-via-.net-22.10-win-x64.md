# 设定框架渲染和旋转行为

能够指定和控制框架旋转和框架大小的行为，同时从“ PPTX”和其他支持的演示格式呈现形状。

# 压缩嵌入式字体以降低显示大小

如果您的演示文稿包含嵌入式字体，则我们的演示文稿处理Python API的此版本可以执行所有此类嵌入式字体的压缩，从而减小了演示文件的大小，例如 `PPTX`。

# 设置演示文稿线条线颜色

此API发行版介绍了在您的演示幻灯片中设置Leader Lines颜色的能力。
