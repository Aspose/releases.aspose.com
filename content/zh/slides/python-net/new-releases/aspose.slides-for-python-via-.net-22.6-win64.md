# 更新的演示源锁定行为

已过时的表现锁定bohavior  `LOAD_AND_RELEASE_LEGACY_TEMPORARY_DEFAULT` 枚举成员已被删除。  `LOAD_AND_RELEASE` 现在是默认的演示锁定行为。

此默认设置仅在  `IPresentation` 构造函数执行时间内锁定源。
