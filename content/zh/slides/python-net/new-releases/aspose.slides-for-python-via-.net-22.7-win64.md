# 改善了对Ubuntu焦点的支持

据观察，在转换Microsoft PowerPoint＆Reg时； `PPTX` to `PDF` 格式化了几个问题。这些问题现在已经解决。

# 3D列图支持

此API版本介绍了3D列图表的支持。现在，您可以添加具有默认数据的图表，然后设置 `Rotation3D` 属性（例如直角轴，深度百分比等）。

# 改进希伯来文本效果支持

以前据观察，在某些情况下，当 `PPTX` 演示文稿转换为  `PDF` 格式时，希伯来文本阴影都是不正确的。现在已经解决了这个问题。
