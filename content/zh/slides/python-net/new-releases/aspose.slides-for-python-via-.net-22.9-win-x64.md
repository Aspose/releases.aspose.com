# 支持Apple＆Reg;的Macos＆Reg;

本演示的操纵Python API版本引入了对MacOS＆Reg;的支持。

# 获取字体替代信息

能够获取有关在演示渲染过程中要替换字体的信息。已添加了一种新方法 `get_substitutions()` 是为了实现这一目标。

# 设置幻灯片动画的时机

现在，您可以通过以下两个新属性添加到[`Timing`]（https://reference.aspose.com/slides.com/slides/python-net/aspose.slides.animation.animation.animation.timing/timing），通过以下两个新属性来重复演示文稿幻灯片动画。 /） 班级：

- `repeat_until_next_click`:此Bool属性设置如果重复动画效果，直到接下来单击鼠标按钮为止。
- `repeat_until_end_slide`: 此Bool属性设置如果重复幻灯片动画效果，直到当前幻灯片的结束为止。
