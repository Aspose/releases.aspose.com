## Photoshop&reg; 文件处理的 .NET API

一个独立的 .NET API，用于读取、写入、处理和转换 Adobe Photoshop&reg; PSD 和 PSB 格式，而无需安装 Adobe Photoshop&reg;。Aspose.PSD for .NET API 允许您创建和编辑 Photoshop&reg; 文件，并提供更新图层属性、添加水印、执行图形操作以及在各种格式之间转换文件的功能。

## 为什么选择 Aspose.PSD for .NET？

- **无需 Adobe Photoshop®:** Aspose.PSD for .NET 独立于 Adobe Photoshop&reg; 运行，使您可以直接在 .NET 应用程序中处理 PSD 和 PSB 文件。
- **高性能:** 该 API 专为速度和效率而设计，即使处理复杂的图层和效果，也能快速处理大型 PSD 文件。
- **跨平台兼容性:** 支持 .NET Framework、.NET Core 和 Mono，适用于 Windows、macOS 和 Linux 环境。
- **全面的格式支持:** 无缝读取、写入和转换 Photoshop&reg; 和 Illustrator&reg; 格式。
- **易于集成:** 可以直接集成到现有的 .NET 应用程序中，并提供全面的文档和支持。

## Photoshop&reg; 文件处理功能

- **创建和编辑 Photoshop&reg; 文件:** 程序化地生成新的 PSD 和 PSB 文件，或编辑现有文件。
- **高级图层管理:** 更新图层属性，应用混合模式，添加蒙版并管理智能对象。
- **应用滤镜和效果:** 实现中值滤镜和维纳滤镜，应用色彩平衡，反转调整图层等。
- **支持光栅和矢量图形:** 绘制基本形状，管理矢量路径并应用变换。
- **XMP 数据处理:** 在 PSD 文件中创建、读取和写入 XMP 元数据。
- **全面的图像操作:** 精确调整图像的大小、裁剪、旋转和调整。
- **导出为多种格式:** 将您的工作保存为多种光栅和矢量格式，包括 TIFF、JPEG、PNG、GIF、BMP、JPEG2000 和 PDF。
- **批处理:** 在单个批处理操作中高效地处理多个文件。
- **内存优化:** 通过优化内存使用来处理大型文件，确保您的应用程序即使在处理复杂的 PSD 时仍然保持响应。
- **智能滤镜管理:** 应用和管理无损的智能滤镜，保持原始图像数据的完整性。
- **色彩管理:** 调整亮度、对比度、伽马，并应用各种色彩校正技术。
- **文本处理:** 处理文本图层，包括字体管理和文本效果。
- **水印添加:** 轻松将水印添加到图像中以保护版权。

## 裁剪 Photoshop&reg; PSD 并将结果保存为 PNG 格式

您可以运行以下代码片段，查看 Aspose.PSD API 在您的开发环境中的工作方式，或查看 [GitHub 仓库](https://github.com/aspose) 以获取其他常见的使用场景。

```csharp
// 为 PSD 文件实现正确的裁剪方法。
using (RasterImage image = Image.Load(dir + "template.psd") as RasterImage)
{
    image.Crop(new Rectangle(10, 30, 100, 100));
    image.Save(dir + "output.png", new PngOptions() { ColorType = PngColorType.TruecolorWithAlpha });
}
```

## 在 PSD 图像中绘制矩形

```csharp
// 创建 Image 实例
using (Image image = new PsdImage(100, 100))
{
    Graphics graphic = new Graphics(image);
    graphic.Clear(Color.Yellow);
    // 使用钢笔工具绘制矩形
    graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
    // 使用蓝色的实心画笔绘制另一个矩形
    graphic.DrawRectangle(new Pen(new SolidBrush(Color.Blue)), new Rectangle(10, 30, 80, 40));
}
```
