## 为什么要为 .NET 下载 Aspose.SVG？

这个 .NET 本地 API 可帮助您将 [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) 功能融入您的 C#、VB.NET、ASP.NET 和其他基于 .NET 的应用程序中。

## SVG 文件处理功能

- [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/) 格式文件。
- 将 SVG 转换为其他 [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/)。
- 根据官方 SVG 规范进行 DOM 树操作。
- 通过 [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/)、CSS 选择器、元素和文档遍历功能支持内容导航。
- 支持质量渲染。
