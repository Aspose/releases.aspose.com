### PHP BarCode 本地 API 功能

- 支持大多数已建立的条码标准和条码规范。
- 能够读取和导出多种图像格式的条形码，包括 BMP、GIF、JPEG 和 PNG。
- 提供对条形码标签的完全控制。
- 支持二维条码的 X 维和 Y 维。
