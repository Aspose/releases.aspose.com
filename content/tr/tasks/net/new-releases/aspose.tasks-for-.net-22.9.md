# Geliştirilmiş XML'den MPP'ye Dönüştürme

Belirli bir XML dosyasını MPP biçimine kaydederken oluşan istisna düzeltildi. Aşağıdaki C# kod parçacığı, bir Primavera P6 XML dosyasının API kullanılarak bir Microsoft Project® dosyasına nasıl dönüştürülebileceğini gösterir:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Primavera XER Kaynak Oranlarının Geliştirilmiş Okuma Özelliği

Primavera XER dosyalarından okuma kaynak oranları sorunu düzeltildi.

# Çeşitli Formatlardan MPP'ye İyileştirilmiş Dönüşümler

Belirli dosyaları MPP formatına kaydederken tetiklenen hatalar düzeltildi.
