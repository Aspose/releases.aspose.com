Bu sayfa, Aspose.3D for Python via .NET 23.6 için yayın notları bilgilerini içerir.

## **Geliştirmeler ve Değişiklikler**

|**Anahtar**|**Özet**|**Kategori**|
| :- | :- | :- |
| THREEDNET-588 | IFC destek isteği | Yeni Özellik |
| THREEDNET-1395 | CascadiaCode font desteklenmiyor | Hata düzeltme |
| THREEDNET-1397 | IFC 2x3 Dönüşümü yanlış | Hata düzeltme |
| THREEDNET-1403 | Mesh, STL'ye kaydedildi ve ondan tekrar yüklendiğinde çokgenler kayboluyor | Hata düzeltme |


Aspose.3D for Python via .NET 23.6, IFC(2.3 ve 4) için deneysel desteği ekledi; daha fazla IFC dosyası desteği gelecekte eklenecektir.
