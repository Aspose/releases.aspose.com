# Mesh Üçgen Desteği

Mesh üçgenleme, Polygon / PolygonVertex eşleme moduyla 'VertexElementUserData'yı destekler.

# TriMesh'te `VertexElementUserData` Desteği

TriMesh'te `VertexElementUserData` desteği eklendi.

# `VertexElementUserData` - `glTF` Dışa Aktarma

"VertexElementUserData"yı "glTF" olarak dışa aktarma özelliği eklendi.

# `GLTF` Özel Köşe Nitelikleri

GLTF Özel Vertex Nitelikleri desteği eklendi.

> Bu sürümdeki özelliklerin, geliştirmelerin ve hata düzeltmelerinin tam listesi için lütfen [Aspose.3D for .NET 22.5 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-5-release-notes/) adresini ziyaret edin.
