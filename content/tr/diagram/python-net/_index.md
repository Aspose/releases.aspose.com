### .NET Şirket İçi API Özellikleri aracılığıyla Visio Python

- API aracılığıyla visio dosyaları oluşturun.
- Yüksek kaliteli dosya formatı dönüştürme ve oluşturma.
- Microsoft Visio® dosyalarını fiziksel veya sanal yazıcılara yazdırın.
- Visio® sayfalarını birleştirin, değiştirin, koruyun veya ayrıştırın.
- Şekilleri resimlere veya PDF'ye dönüştürün.
- Sayfalar için sayfa kurulumunu yapılandırın ve uygulayın.
- Visio® dosyalarını diğer çeşitli biçimlere dönüştürün.

Daha ayrıntılı özellikler listesi için lütfen [official documentation](https://docs.aspose.com/diagram/python-net/) sayfasını ziyaret edin.

### Visio Dosya Biçimlerini Okuyun

**Microsoft Visio**: VSD, VDW,VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM

### Visio Dosyalarını Farklı Olarak Kaydet

**Microsoft Visio**: VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM
**Sabit Düzen**: PDF, XPS
**Görüntüler**: JPEG, PNG, BMP, SVG, TIFF, GIF, EMF
**Web**: HTML
### Sistem gereksinimleri

Makinenizde Microsoft Visio® yazılımının kurulu olması gerekmez.

### Desteklenen İşletim Sistemleri

**Microsoft Windows®:** Windows Masaüstü ve Sunucusu (`x64`, `x86`)\
**Linux:** Ubuntu, OpenSUSE, CentOS ve diğerleri\
**Diğer:** Mono(.NET 4.0 Framework desteği) yükleyebilen veya .NET çekirdeğini kullanabilen herhangi bir işletim sistemi (OS).
