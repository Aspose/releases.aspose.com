# Geliştirilmiş 'PUB' - 'PDF' Dönüşümü

Microsoft Publisher® PUB dosyasını .NET API aracılığıyla C# kullanarak Adobe Acrobat® PDF belgesine dönüştürme yeteneği geliştirildi.

```csharp
// For complete examples and data files, please go to https://github.com/aspose-pub/Aspose.PUB-for-.NET
// The path to the documents directory.
string dataDir = RunExamples.GetDataDir_Data();

string fileName = dataDir + "halloween-flyer.pub";

var parser = PubFactory.CreateParser(fileName);

var doc = parser.Parse();

Aspose.Pub.PubFactory.CreatePdfConverter().ConvertToPdf(doc, dataDir + "result_out.pdf");
```

> Bu sürümde düzeltilen özelliklerin, geliştirmelerin ve hataların tam listesi için lütfen [Aspose.PUB for .NET 22.6 Release Notes](https://releases.aspose.com/pub/net/release-notes/2022/aspose-pub-for-net-22-6-release-notes/) adresini ziyaret edin.
