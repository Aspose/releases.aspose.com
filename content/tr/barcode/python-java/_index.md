### Java Yerinde API Özellikleri aracılığıyla Python'u Barkod İşleme

- Popüler barkod özelliklerinin ve standartlarının çoğu için destek sunar.
- JPEG, PNG, GIF ve BMP gibi birden fazla görüntü formatındaki barkodları okuyabilir ve dışa aktarabilirsiniz.
- Barkod etiketleri ile çalışabilme.
- 2D barkodlar için X boyutu ve Y boyutu için destek.
