Aspose.Cells for PHP via Java'in ilk genel yayınını duyurmaktan mutluluk duyuyoruz. API'ler, PHP'de (yaygın olarak kullanılan bir açık kaynak betik dili) MS Excel elektronik tablo işleme işlevlerini içerir. Aspose.Cells for PHP via Java, yerel sürümünde bulunan tüm önemli ve yararlı özellikleri içeren bir alt küme bileşenidir.[Aspose.Cells for Java](https://products.aspose.com/cells/java/)Yeni API, eşit derecede sağlam ve zengin özelliklere sahip bir bileşendir. PHP'de XLS, XLSX, XLSM, SpreadsheetML, CSV, Sekmeyle Ayrılmış, HTML, MHTML ve OpenDocument Elektronik Tablo arasında yüksek kaliteli dosya biçimi dönüştürmelerini destekler. Lütfen ziyaret edin[Ürün özellikleri karşılaştırma sayfası](https://docs.aspose.com/cells/tr/java/aspose-cells-for-php-via-java-features/)senin referansın için. Ayrıca, lütfen nasıl yapılacağına ilişkin belgeye bakın.[Kurulum ortamı ve diğer Kurulum Yönergeleri](https://docs.aspose.com/cells/tr/java/setup-and-installation-guidelines/)senin referansın için.
