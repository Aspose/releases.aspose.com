Herkese Açık API ve Geriye Dönük Uyumsuz Değişiklikler

Aşağıda, API numaralı telefon numarasına eklenen, yeniden adlandırılan, kaldırılan veya kullanımdan kaldırılan üyeler gibi genele açık olarak yapılan tüm değişikliklerin ve Aspose.Cells for Java numaralı telefona yapılan geriye dönük uyumlu olmayan değişikliklerin bir listesi bulunmaktadır. Listelenen herhangi bir değişiklikle ilgili endişeleriniz varsa lütfen şu adrese bildirin: Aspose.Cells destek forumu.

 TxtLoadOptions.KeepExactFormat özelliğini ekler
Dize değeri sayıya veya tarih saatine dönüştürülürken hücre için tam biçimlendirmenin korunması gerekip gerekmediğini gösterir.

Güncellemeler Aspose.Cells.Pivot.PivotItemPosition enum
Güncellemeler: Önceki, Sonraki ve Özel.

PlotArea'nın SetPositionAuto() yöntemini ekler.
Çizim alanının konumunu otomatik olarak ayarlar.

Shape.Id özelliği ekler
Şeklin kimliğini almak için kullanılır.

GridDesktop.SheetTabWidth özelliğini ekler
Sayfa Sekmesinin genişliğini ayarlar /alır.

Aspose.Cells for Java kod tabanı, ilgili .NET sürümünün koduyla eşleştiğinden, Aspose.Cells for .NET v8.3.0'da bulunan değişikliklerin, geliştirmelerin ve düzeltmelerin çoğu bu Aspose.Cells for Java v8.3.0'da da yer almaktadır.
