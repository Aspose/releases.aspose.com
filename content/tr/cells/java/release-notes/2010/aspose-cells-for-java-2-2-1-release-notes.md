 Aspose.Cells for Java 2.2.1'i duyurmaktan mutluluk duyuyoruz!

 Neler değişti:

-  MS Excel 2007 tarafından sağlanan Gelişmiş Koşullu biçimlendirme desteğini içerir
-  XLS dosyası için MS Excel 2007 tarafından kaydedilen Genişletilmiş Koşullu biçimlendirme kayıtlarını destekler
- MS Excel 2007 dosyalarını okurken desteklenmeyen formülleri korur
-  MS Excel 2007 şablon dosyasındaki Üstbilgi/Altbilgideki resimleri okur
-  Şablon dosyalarını okumak için geliştirmeler yapıldı
-  17 düzeltme ve diğer geliştirmeler.
