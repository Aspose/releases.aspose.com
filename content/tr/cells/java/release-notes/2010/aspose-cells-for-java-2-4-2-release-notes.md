 Aspose.Cells for Java 2.4.2'yi duyurmaktan mutluluk duyuyoruz!

 Neler değişti:

-  Dizi parametrelerini destekler (

Bilinmeyen makro: {...}

) formülleri ayarlamak için.

- Bağlantılı resimler eklemek için destek sağlar.
- Bellekte tam veri modeli oluşturmadan hücre verilerini doğrudan akışa veya dosyaya kaydetmeyi destekler.
- Lisansın ayarlanıp ayarlanmadığını kontrol edin, aboneliğinizin sona erme tarihini almak için destek içerir.
- Excel 2007 dosyalarını okuma/kaydetme işlemi geliştirildi.
- Grafikten Görüntüye özelliği geliştirildi.
- Doğrudan Excel'den PDF'e özelliği geliştirildi.
- Cell değerleri biçimlendirme özelliği geliştirildi.
- Excel 97-2003 dosyalarını Okuma/Kaydetme işlemi geliştirildi.
- 21 düzeltme ve geliştirme.
