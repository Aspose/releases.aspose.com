 Sayın müşteriler,

 Aspose.Cells for Java V1.7.1 sürümünü yayınladık! Aşağıdaki geliştirmeler ve düzeltmeler dahildir:

- SpreadsheetML dosyasından hücrelerin içeriğini ve stillerini içe aktarmayı destekler
 Hücrelerin içeriğini ve stillerini SpreadsheetML dosyasına aktarmayı destekler
 Formülün hesaplama motorunu destekler
 R1C1 formülünü ayarlamayı destekler
 Diğer iki PivotTable veri kaynağı türünü destekler
 PivotTable eklemeyle ilgili bazı hataları düzeltir.

Not: hücrenin formül içerip içermediğine karar vermek için const CellValueType.FORMULA'yı kaldırdık ve Cell.isFormula() yöntemini ekledik.
