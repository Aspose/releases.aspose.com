 Sayın müşteriler,

 Aspose.Cells for Java V1.7.3 sürümünü yayınladık! Aşağıdaki geliştirmeler ve düzeltmeler dahildir:

- Bir aralık eklemeyi/silmeyi destekler.
 Şifreli Excel dosyasını açmayı ve kaydetmeyi destekler.
 Daha fazla tarih/sayı biçimini desteklemek için Cell.getStringValue() yöntemini geliştirir.
 Web demoları güncellendi.
 JDK1.4 ile formülleri hesaplamayı destekler.
 License.setLicense() yöntemindeki bir hatayı düzeltir.
 Özel sayı formatını ayarlama hatasını düzeltir.
Çalışma kitabından bir çalışma sayfasını kaldırma hatasını düzeltir.
