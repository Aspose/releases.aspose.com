 Aspose.Cells for Java 2.5.1'i duyurmaktan mutluluk duyuyoruz!

 Neler değişti:

· Büyük dosyalarla çalışırken açma/kaydetme sürecini kesmeyi destekler.

 · Daha fazla formül ayarlamak ve hesaplamak için destek sağlar.

 · Şeklin mutlak konumunu almayı/ayarlamayı destekler.

 · Excel 2007 dosyalarını okuma/kaydetme hatalarını giderir.

 · ODS dosyalarını okuma/kaydetme hatalarını düzeltir.

 · Excel 97-2003 dosyalarını okuma/kaydetme hatalarını düzeltir.

 · Chart to Image, Sheet to Image ve Excel to PDF özelliklerinde geliştirme yapıldı.

 · Formülleri ayrıştırmak için iyileştirmeler yapıldı.

 · Formülleri hesaplamak için iyileştirmeler yapıldı.

 · 22 düzeltme ve geliştirme.
