Biz
 Aspose.Cells for Java v7.2.2'yi duyurmaktan mutluluk duyuyoruz!

 Yeni özellikler

- Cells.Find() yöntemi için RegEx Metin Araması

 Geliştirmeler

- Aspose.Cells'i Woodstox kavanozlarının eski sürümleriyle uyumlu hale getirin
- XLS'deki OLE gömülü OOXML dosyaları, paketlenmemiş dosyalar yerine OLE paketli dosyalar olarak çıkıyor
- HTML dosyalarını kaydetmek için ExportObjectListener'ı destekleyin
- Satırları kopyalarken Koşullu Biçimlendirmeyi Kopyala

 İstisnalar

- Picture inPageSetup için Picture.isPrintable(), NullPointerException'a neden oluyor
- Pivot Tablo ile şifrelenmiş çalışma kitabının kaydedilmesi java.lang.NegativeArraySizeException'a neden olur

 Hatalar

- Cells.importCustomObjects(), belirtilen DateTime biçimiyle çalışmıyor
- Yanlış GrafikDağılım grafiğinin türü
- CSV şablon dosyasından okunurken çift değer hassasiyeti kaybeder
- Grafik Serisi, bir görüntüye dönüştürülürken ters çevrildi
- Yeniden kaydedilen XLSX dosyası "Excelfound unreadable content..." hatasına neden oluyor
- Kaydedilen Pivot Tablo, MS Excel'de açıldığında "ProtectionView"a neden oldu
- Aspose.Cellscreates ve XLS dosyası tarafından Liste doğrulamanın kullanılması, Sistemin liste ayırıcısını değiştirdikten sonra çalışmayan
