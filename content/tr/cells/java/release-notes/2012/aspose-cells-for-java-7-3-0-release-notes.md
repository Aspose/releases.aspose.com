Biz
 Aspose.Cells for Java v7.3.0'ı duyurmaktan mutluluk duyuyoruz!

 Ne var ne yok

- Okumak
 ve MHT dosyalarını yaz
- destekler
 XML haritaları
- Uygulamak
 MS Excel 2007/2010 temalarından grafiklere



 Diğer önemli özellikler, geliştirmeler ve düzeltmeler aşağıda verilmiştir.

 Yeni özellikler

 – Geçiş Formülü Değerlendirmesi için ayarları destekler

 Geliştirmeler

- Hücre değerlerini kullanıcı tarafından belirtilen yerel ayar ile biçimlendirin

 İstisnalar

- Desteklenmeyen yazı tipi dosyaları, istisna dışında PDF'ye kaydetmenin başarısız olmasına neden olabilir

 Hatalar

- Cell.setR1C1Formula() yöntemi, sütun ofseti olmadan doğru çalışmadı
- Adlandırılmış aralık iki kez oluşturuldu
- sortNames() yöntemi, bir XLSM dosyasını kaydederken istisnaya neden oldu
- Sayı, kesir için doğru biçimlendirilmemiş
- Boş PDF, PrintOrderType.OVER için oluşturuldu_O ZAMANLAR_AŞAĞI
- Oluşturulan PDF'de arka plan rengi ve ızgaralar yanlış
- Kesişim ve Eğim fonksiyonları doğru hesaplanmadı
- Excel 2007 dosya biçimleri için Pivot alan öğelerinin 33 bin sınırını kaldırın
- Notasyon 1:1, EĞER işlevinde desteklenmez
- DATEDIF formülü yanlış hesaplandı
- Adlandırılmış satır olması durumunda hücrelerin yanlış aranması
