# Ubuntu odak için geliştirilmiş destek

Microsoft PowerPoint & Reg; `Pptx` to` pdf` format Ubuntu focal birkaç sorun ortaya çıktı. Bu sorunlar şimdi çözüldü.

# 3D Sütun Grafikleri Desteği

Bu API sürümü 3D sütun grafiklerinin desteğini tanıtmaktadır. Şimdi varsayılan verilerle grafik ekleyebilir ve ardından `rotation3D 'özelliklerini (dik açılı eksen, yüzde derinlik, vb.) Ayarlayabilirsiniz.

# Geliştirilmiş İbranice Metin Etki Desteği

Daha önce, bazı durumlarda `` pptx '' sunumları `pdf` formatına dönüştürüldüğünde İbranice metin gölgesinin yanlış olduğu gözlenmiştir. Bu sorun şimdi çözüldü.
