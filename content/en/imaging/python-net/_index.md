**Raster Formats:** JPEG2000, JPEG, BMP, TIFF, BIGTIFF, GIF, PNG, APNG\
**Medical Imaging:** DICOM\
**Metafiles:** EMF, WMF, EMZ, WMZ\
**Truevision:** TGA\
**Vector Graphics:** SVG, SVGZ

### Save Images As

**Fixed:** PDF\
**Photoshop:** PSD\
**Web:** HTML5 Canvas

**eBook:** DjVu\
**Digital Camera Raw:** DNG\
**OpenOffice:** ODG, OTG\
**Bitmap:** DIB\
**Web Image:** WebP\

**PostScript:** EPS

### System Requirements

Aspose.Imaging for Python via .NET supports Windows 32,64-bit, Linux 64-bit, macOS 64-bit operating systems where Python 3.5 or later is installed.

### Supported Operating Systems

**Microsoft Windows:** Windows Desktop & Server (`x64`)\
**Linux:** Ubuntu, OpenSUSE, CentOS, and others.
**macOS x64:** since macOS 10.14 (Mojave)
**macOS arm64:** since macOS 11 (Big Sur)
