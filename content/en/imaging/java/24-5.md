# What's new in the Aspose.Imaging for Java 24.5

## Release Highlights

Aspose.Imaging for Java 24.5 focuses on strengthening support for EPS and GIF formats, resolving numerous rendering issues and improving compatibility with complex EPS file structures. This release also optimizes performance in multi-threaded environments when detecting image file formats.

## Enhancements

- **Robust EPS to PNG and SVG Conversion:**
  Multiple enhancements ensure stable conversion of EPS files, addressing previously unimplemented functions, rendering exceptions (`eexec` errors), and distorted output. EPS files with complex PostScript content are now reliably handled.

- **Improved EmfPlus Rendering:**
  Corrected output generation when converting EmfPlus images, ensuring visual consistency with the original content.

- **GIF Export Fix:**
  Fixed issues that previously caused distortions when saving GIF images, particularly under specific encoding conditions.

- **Thread-Safe File Format Detection:**
  Enhanced the `Image.getFileFormat()` method to ensure proper operation in multi-threaded scenarios, removing internal locks that caused potential bottlenecks.
