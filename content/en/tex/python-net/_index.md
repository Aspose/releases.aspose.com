## Why download Aspose.TeX for Python?

Aspose.TeX for Python is a software library that offers an extension for the TeX engine known as ObjectTeX. This library enables the typesetting of documents represented by TeX files. The term "Object" implies that the intermediate result of typesetting is represented as a distinct object model, which can be effortlessly converted into various final formats.

## TypeSetting File Processing Features

- Typesetting of TeX files
- Create custom TeX formats
- Provide input data in various ways
- Fetch output data in various ways

### Supported Input Formats

TeX

### Save TeX As

XPS, PDF, SVG, JPEG, PNG, TIFF and BMP
