## Aspose.TeX 24.7 for Python via .NET has been released!

 * The new *LaTeX Repairer* feature has been implemented.
 * The *LaTeX* kernel is updated to *LaTeX3e*.

| TEXNET-351 | Migrate to *LaTeX3e* | Enhancement |
