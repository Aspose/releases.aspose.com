## Aspose.TeX 25.6 for Python via .NET has been released!

- Enhanced Core engine to support new PDF graphics primitives.
    - \pdfsave
    - \pdfrestore
    - \pdfmatrix

It includes all changes from the equivalent .NET version and further improves the overall stability of the API in terms of functionality.
