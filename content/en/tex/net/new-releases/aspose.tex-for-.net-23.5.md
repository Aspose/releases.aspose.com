## Support for amsfonts Package

This release adds the enahcenement to embed fonts required by the amsfonts package. 

The [amsfonts](https://www.ams.org/arc/resources/amsfonts-about.html) package in LaTeX is a pivotal collection of high-quality fonts tailored for mathematical typesetting. For C# and .NET API developers, integrating this package through the TeX API opens up a realm of possibilities for creating mathematically rich and visually appealing documents. These fonts, meticulously designed by the American Mathematical Society (AMS), guarantee precision and clarity in mathematical notation. Seamlessly embedding amsfonts through the TeX API in your C# .NET applications ensures a professional touch, giving your mathematical documents a refined elegance and visual excellence. Elevate your mathematical typesetting capabilities with this seamless integration.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.TeX for .NET 23.5 Release Notes](https://releases.aspose.com/tex/net/release-notes/2023/aspose-tex-for-net-23-5-release-notes/).
