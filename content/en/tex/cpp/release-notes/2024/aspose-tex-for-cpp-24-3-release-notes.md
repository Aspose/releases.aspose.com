## Improvements and Changes

Aspose.TeX for C++ 24.3 is based on [Aspose.TeX for .NET 24.3](https://releases.aspose.com/tex/net/release-notes/2024/aspose-tex-for-net-24-3-release-notes/) and ready for use with Aspose C++ products ver 24.3.

 * *eTeX*'s page building optimization has been implemented.

You can consult Aspose.TeX for C++ [API reference](https://reference.aspose.com/tex/cpp/) for documentation.
