## Improvements and Changes

Aspose.TeX for C++ 23.10 is based on [Aspose.TeX for .NET 23.10](https://releases.aspose.com/tex/net/release-notes/2023/aspose-tex-for-net-23-10-release-notes/) and ready for use with Aspose C++ products ver 23.10.

Implemented:
 * the mixed direction typesetting as part of the eTeX extension.
 * the testing for errors occurred along a TeX file processing
 * MathRenderer and FigureRenderer plugins

You can consult Aspose.TeX for C++ [API reference](https://reference.aspose.com/tex/cpp/) for documentation.
