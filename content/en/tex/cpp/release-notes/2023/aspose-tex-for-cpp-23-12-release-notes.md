## Improvements and Changes

Aspose.TeX for C++ 23.12 is ready for use with Aspose C++ products ver 23.12.

* The support for fonts provided by external packages was implemented.
* The `\middle` delimiter primitive from the *eTeX* extension was added.
* The conversion from *TeX/LaTeX* to multi-frame *TIFF*s presents now.

For more details, see [Aspose.TeX for .NET 23.12](https://releases.aspose.com/tex/net/release-notes/2023/aspose-tex-for-net-23-12-release-notes/).

You can consult Aspose.TeX for C++ [API reference](https://reference.aspose.com/tex/cpp/) for documentation.
