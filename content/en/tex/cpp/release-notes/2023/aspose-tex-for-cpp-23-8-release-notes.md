## Improvements and Changes

- Enhance eTeX extension support

Aspose.TeX for C++ 23.8 is based on [Aspose.TeX for .NET 23.8](https://releases.aspose.com/tex/net/release-notes/2023/aspose-tex-for-net-23-8-release-notes/) and ready for use with Aspose C++ products ver 23.8.

The support macOS with ARM64 architecture was added.

You can consult Aspose.TeX for C++ [API reference](https://reference.aspose.com/tex/cpp/) for documentation.
