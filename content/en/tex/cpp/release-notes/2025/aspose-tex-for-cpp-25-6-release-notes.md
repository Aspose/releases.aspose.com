## Improvements and Changes

Aspose.TeX for C++ 25.6 is based on latest [Aspose.TeX for .NET 25.6](https://releases.aspose.com/tex/net/release-notes/2025/aspose-tex-for-net-25-6-release-notes/) and ready for use with Aspose C++ products ver 25.6.

- Enhanced Core engine to support new PDF graphics primitives.
    - \pdfsave
    - \pdfrestore
    - \pdfmatrix


You can consult Aspose.TeX for C++ [API reference](https://reference.aspose.com/tex/cpp/) for documentation.
