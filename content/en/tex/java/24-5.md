# What's new in the Aspose.TeX for Java 24.5

## Release Highlights

Aspose.TeX for Java 24.5 introduces a powerful new feature â€” the **LaTeX Repairer**, designed to automatically detect and fix common LaTeX issues. This innovation helps streamline the document preparation process, especially when dealing with incomplete or inconsistent LaTeX source files. Developers now gain access to enhanced flexibility via user-defined callbacks and directory configuration options.

- **LaTeX Repairer Functionality:**
