## Aspose.TeX 25.4 for Java has been released!

- Enhanced Core engine to support new PDF graphics primitives.
    - \pdfsave
    - \pdfrestore
    - \pdfmatrix

This release of Aspose.TeX for Java is a maintenance release which has been migrated from the latest version of Aspose.TeX for .NET. It includes all changes from the equivalent .NET version and further improves the overall stability of the API in terms of functionality.
