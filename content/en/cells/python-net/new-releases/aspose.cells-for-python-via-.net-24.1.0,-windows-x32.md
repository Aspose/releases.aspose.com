### Enhanced .NET Compatibility

Enjoy improved support for .NET environments with .NET 8 support, ensuring seamless integration with your existing projects.

### GridDesktop Autofill 

Experience the efficiency gains of autofill within `GridDesktop` using version 24.1.0 of the Python spreadsheets library, simplifying data entry tasks.

### Personalized HTML Export Options

Customize exported HTML styles on Windows x32 to tailor the appearance of your reports according to your preferences.

### Streamlined Development Workflow

You can benefit from a collection of bug fixes and enhancements related to conversion, image export, `PivotTable`, and more that refine the overall functionality of Aspose.Cells for Python.

### Additional Updates
- Autofill enhancements for merge areas and right-to-left/bottom-to-up data filling.
- Enhanced rendering of active sheet background and support for throwing exceptions when opening corrupted files.
