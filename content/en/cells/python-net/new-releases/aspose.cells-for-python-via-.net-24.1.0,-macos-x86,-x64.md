### CSV to Excel Conversion

Preserve header rows in each worksheet when converting CSV files to Excel with the Excel file processing Python API, maintaining data structure.

### Upgraded GridDesktop Autofill

You can boost data entry efficiency on macOS x64 with the powerful autofill features for `GridDesktop`.

### .NET 8 Support

Python developers can effortlessly incorporate the latest .NET 8 advancements into their applications with the improved compatibility now available in Aspose.Cells for Python via .NET 24.1.0.

### Additional Updates
- Better management of large dynamic array formulas and manipulation of pivot fields.
- Enhanced autofill functionality for merged areas and data filling from right to left or bottom to up.
- This update also includes bug fixes covering various API features related to image orientation, SVG conversion, comment text color, textbox auto-sizing, formula calculations, shape resizing, file opening, picture width, pivot table filtering, and more.
