### Enhanced PDF/A Compliance

You can generate PDF documents compliant with various PDF/A ISO standards (PDF/A-2b, PDF/A-2u, PDF/A-2a, PDF/A-3b, PDF/A-3u, and PDF/A-3a) for superior archival and long-term preservation using the Excel Python API on Windows x64.

### Improved PivotTable Functionality

Maintain data integrity during `PivotTable` refresh and calculations, calculate discrete groupings, and export generic lists, empowering comprehensive data analysis.

### Streamlined Data Handling

Leverage automatic data validation setting clearance in the paste area and refined handling of missing data during `PivotTable` refresh for a more efficient workflow with Aspose.Cells for Python via .NET 23.12.0.

### Optimized Rendering

Ensure optimal rendering of sheet names on mobile devices and address formula bar-related issues for a smoother user experience.

### Bug Fixes and Improvements

This release incorporates a comprehensive range of additional enhancements, bug fixes, and improvements to elevate your spreadsheet processing experience.
