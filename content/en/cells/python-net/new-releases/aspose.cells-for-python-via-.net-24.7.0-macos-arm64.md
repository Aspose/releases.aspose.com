Take your macOS M1-powered spreadsheet manipulation apps to the next level with Aspose.Cells for Python via .NET 24.7.0 release. This version boasts powerful formula support, enhanced chart functionalities, and improved performance.

### New Features

- **Use `DROP` Function for Spreadsheet Management:** You can leverage the `DROP` function to eliminate unwanted rows or columns from your spreadsheets.
- **Chart Outlines with Star Characters on Apple Mac:** Boost the aesthetics of your charts by adding data point outlines with star characters on macOS ARM64/M1 systems.
- **Easily Process Date-only Objects:** Handle `DateOnly` objects while importing custom objects into your Excel spreadsheets.
- **Add Decorative Objects to XLSX Files:** Specify objects as decorative within XLSX spreadsheets using Aspose.Cells for Python via .NET API.

### Enhancements

- **Fine-tuned `LET` Function:** Update your spreadsheet applications with lightning-fast calculations for complex `LET` functions.
- **Series Swap Feature for Python Apps:** Modify the order of chart series for data visualization comfortably within your apps running on macOS ARM64.
- **Learn to Rename Chart Data Table Method:** Rename the chart data table method with ease in your apps and provide a great developer experience.
- **Simplify Calculations with `MAKEARRAY` Function:** The `MAKEARRAY` function helps you simplify calculations to augment the performance of your macOS M1 apps.

### Bug Fixes

Numerous bugs with Excel to PDF conversion, formula calculations, conditional formatting, and more have been addressed in version 24.7.0 of the Python library.
