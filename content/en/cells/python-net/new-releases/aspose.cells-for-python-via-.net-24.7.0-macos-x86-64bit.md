MacOS developers can supercharge their Python applications and easily process spreadsheets using Aspose.Cells for Python via .NET 24.7.0 (macOS x64). Incorporate formula support, enhanced chart functionalities, and elevated performance with this release. 

### New Features

- **Manipulate Spreadsheets with The `DROP` Function:** Developers can use the `DROP` function to remove unwanted rows or columns from their spreadsheets.
- **Create Chart Outlines with Star Characters:** Level up the aesthetics of your charts by outlining data points with star characters on macOS x64 machines.
- **Work with Date-only Objects:** Efficiently manage `DateOnly` objects when importing custom objects into your Excel spreadsheets.
- **Decorative Objects in XLSX Files:** The latest release of the Aspose.Cells for Python via .NET API allows marking objects as decorative.

### Enhancements

- **Advanced `LET` Function:** Upgrade your spreadsheets with faster calculations for complex `LET` function usage.
- **Fine-tuned Series Swap Feature:** Handling the order of chart series for data visualization is easier than ever within your Linux apps.
- **Easily Rename Chart Data Table Method:** Improve clarity by renaming the chart data table method in your apps and deliver a great developer experience.
- **Manage `MAKEARRAY` Function:** With the `MAKEARRAY` function, you can augment the performance of your macOS x64-powered apps with simplified calculations.

### Bug Fixes

Numerous bugs with Excel to PDF conversion, formula calculations, conditional formatting, and more have been addressed in version 24.7.0 of the Python library.
