### PDF/A Compliance for Enhanced PDF Archiving

Ensure long-term accessibility and archiving of your spreadsheets with support for various PDF/A (ISO 19005-2 and ISO 19005-3) standards on macOS x64 using the Excel processing Python .NET API.

### Precise Pivot Table Calculations

Leverage version 23.12.0 of Aspose.Cells for Python via .NET to perform accurate calculations on discrete groupings within pivot tables.

### Grid Enhancements

Improved data validation handling in `GridWeb` and enhanced sheet name rendering on mobiles enable an optimized user experience for Python developers.

### Excel Conversion Accuracy

Various issues related to Excel conversions to various formats like Word, HTML, and PPTX have been resolved in this update.

### Additional Bug Fixes

We have implemented several other enhancements and bug fixes for cross-platform rendering, file handling, and formula evaluation.
