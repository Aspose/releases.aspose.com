Level up your macOS Excel spreadsheet processing applications with high-performance image editing, charts, and enhanced HTML conversion capabilities using the latest Aspose.Cells for Python via .NET release (v24.6.0, macOS x64).

### Key Highlights

### High-quality Image Adjustments

Add color refinements, create appealing duotone effects, and empower image handling with the latest version of the Python spreadsheet manipulation API.

### Chat Outlines and More

Create great-looking charts programmatically in your macOS x64-powered Python applications. Utilize star character outlines in charts and alignment of X-axis values to the right in chart images. 

### HTML and Data Conversions Accuracy

Expand your data export options with HTML and image conversions, such as converting Excel ranges to HTML format, and upgrade your existing Python Excel processing applications.

### Advanced Pivot Tables Functionality

Implement powerful data analysis feature with enhanced pivot table functionality to your macOS x64 applications using the newest Aspose.Cells for Python via .NET API release.

### Enhancements

- Exception handling for invalid template files (CELLSNET-55827, CELLSNET-55828)
- Pivot field "Show Values As" enhancements (CELLSNET-55698)
- Range to image conversion (CELLSNET-55580)
- Pivot table merging range improvements (CELLSNET-55680)

### Bug Fixes

Some of the bugs fixed in this release are:

- Asian multiline text break issues in shapes (CELLSNET-55715)
- Missing SmartArt hyperlinks in PDF output (CELLSNET-55830)
- Image recoloring rendering failures (CELLSNET-55862)
- Algorithm error in 'satMod' code (CELLSNET-55882)
