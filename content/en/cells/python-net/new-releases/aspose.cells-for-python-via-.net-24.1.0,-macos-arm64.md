### Compatibility with .NET 8

We are glad to announce .NET 8 support in Aspose.Cells for Python via .NET version 24.1.0. Python developers can seamlessly integrate the latest .NET 8 advancements into their applications.

### GridDesktop Autofill Enhancements

Experience increased data entry efficiency by leveraging the advanced autofill capabilities of the `GridDesktop` component on macOS.

### Conversion from CSV to Excel

When converting CSV files to Excel using the spreadsheet API for Python, you can retain header rows in each worksheet to preserve the data structure.

### Additional Highlights
- Improved display of active sheet backgrounds and added exception handling for corrupted file opening.
- Optimized autofill capabilities now extend to merged areas, enabling data filling with greater efficiency.
- Better management of large dynamic array formulas and manipulation of pivot fields.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Python via .NET 24.1.0 Release Notes](https://releases.aspose.com/cells/python-net/release-notes/2024/aspose-cells-for-python-net-24-1-release-notes/).
