Linux developers, rejoice! Aspose.Cells for Python via .NET 24.8.0 is here. It includes powerful Excel document processing capabilities, new features, enhancements, and fixes for your Linux-based Python applications. 

### New Features

- Add the ability to convert values to text with the `VALUETOTEXT` function to your apps with this Python Excel API release.
- Developers can use the functionality of pivot table sorting by pivot field item for finer control on Linux-based systems.
- The newest version of the Python library delivers empowered formula editing with blank space support for enhanced flexibility.
- Experience supercharged cross-platform development with `GridWeb` for .NET 6.0 on Linux using Aspose.Cells for Python via .NET v28.8.0.
- Enable visiting all cells containing "Place in Cell" pictures for accurate image management in your apps.

### Improvements

- Seamlessly prevent encryption of document properties while encrypting files for data security.

### Bug Fixes

Various bug fixes were added in the latest Aspose.Cells for Python via .NET release, including the following:

- Resolved character rendering issues in Excel to PDF conversion.
- Addressed inconsistent text content rendering in PDF output.
- Fixed `YEARFRAC` formula calculation errors.
- Corrected hyperlinks in PDF after formula calculation.
- Resolved `YEARFRAC` formula calculation problem with February 29th and January 1st.
- Addressed the text underline display issue during text file to PDF conversion.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Python via .NET 24.8.0 Release Notes](https://releases.aspose.com/cells/python-net/release-notes/2024/aspose-cells-for-python-net-24-8-release-notes/).
