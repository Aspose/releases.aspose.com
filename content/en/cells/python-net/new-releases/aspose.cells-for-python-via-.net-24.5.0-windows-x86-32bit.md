### XMATCH Function

Efficiently locate values based on specific criteria within a range using the `XMATCH` function using Aspose.Cells for Python via .NET 24.5.0.

### XOR Function

Easily perform logical exclusive `OR` operations for diverse calculations using the `XOR` function within the spreadsheet processing apps running on Windows 32-bit.

### Transpose Range Data

Use the latest Python Excel API release to effortlessly reorganize data layouts for improved analysis.

### Advanced Filtering with Enhanced Property Access

Utilize enhanced access and gain better control over filtering configurations in your Win x32 Python apps.

### Determine `ReferredArea`

We have included the ability to identify whether a `ReferredArea` represents an entire row or column in this API release.

### Behavior of The Floor Function

Handle `#NUM!` negative numbers consistently within the spreadsheet manipulation solutions.

### Export Cell Metadata

Aspose.Cells for Python via .NET now supports extracting cell data for inclusion in PDFs, images, and HTML formats on Windows x32 systems.

### RGBA Color Processing in Fonts

Develop feature-rich applications for manipulating Excel documents and handle font colors with style background colors effortlessly.

### Removing Pictures from Cells

The latest Python spreadsheet API release for 32-bit Windows lets you easily remove pictures when modifying cell values programmatically.

### Updated Format Conversion

We have added extended support for various `LoadFormat` conversions to extensions in this version, which enables flawlessly converting Excel documents.

- Corrected text box height calculation for Japanese text.
- Resolved SVG parsing errors when converting HTML to XLSX.

- Fixed `INDEX(FILTER(?);1)` behavior within LAMBDA functions.
- Ensured correct currency formatting for negative Canadian values.

- Improved formula evaluation accuracy.

- Maintained consistent text rendering when converting charts to images.
- Fixed Excel Combo chart control rendering issues.
- Addressed chart title rendering inconsistencies with `FarEastName` and `LatinName`.
- Enabled conditional formatting color scale display in `GridWeb`.
- Preserved image headers when saving Excel files as PDFs.
- Improved Excel page splitting behavior in PDF output.
- Enhanced skewed cell rendering during HTML conversion.
- Resolved HTML table element conversion issues when loading HTML.
