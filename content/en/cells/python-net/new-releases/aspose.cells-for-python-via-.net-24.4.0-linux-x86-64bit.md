### Streamlined Data Exchange on Linux

Developers can effortlessly export charts from Numbers files to XLSX format, ensuring seamless data exchange between different spreadsheet apps on Linux.

### Simplified Formula Management and Padding Space Control

- **Empty String Formulas:** We've introduced support for empty-string formulas in version 24.4.0. Cells with empty-string formulas will now be treated as blank when deleting rows or columns,  on your Linux machines.
- **Padding Space Control:** Take control of how padding spaces are handled for formulas within the `GridWeb` component on Linux machines.

### Enhanced User Experience on Linux

- **Interrupt Monitor:** A new, simplified interrupt monitor introduced in this version allows convenient interaction with spreadsheet processing tasks on Linux.
- **Custom Cell Color Settings:** Easily set custom red color values for cell text within the `GridDesktop` component, adding a touch of personalization to your spreadsheets.

### Embedding Enhancements

- **OLE Object Attachments:** Developers can now embed OLE objects as attachments during Excel to PDF conversion and preserve rich data for further analysis.
- **Image Embedding in XLSB Files:** Directly embed images into cells inside XLSB files to create visually appealing spreadsheets on your Linux systems.

### Optimized Numbers Exports on Linux

- **Hide Rows and Columns:** Choose to hide rows and columns when exporting Numbers files to XLSX format and maintain a clean and focused data presentation.
- **Merged Cell Preservation:** Users can now preserve merged cells in Numbers files during export to XLSX format.


### Image Handling Improvements

- **Export Pictures in Cell to HTML:** Efficiently export pictures within cells to HTML format and create dynamic reports or presentations.
- **Enhanced Image Management:** Enjoy a range of improvements to image handling, including copying embedded images, retrieving web image metadata, and more.

### Enhanced JSON Export 

Experience convenient export of spreadsheet ranges to JSON format as an array of arrays with version 24.4.0 of the spreadsheet processing Python API for a more structured data representation on Linux.

### Support for Dynamic Array Formula

Copy Dynamic Array Formulas with ease, leveraging this powerful feature in your Python apps running on Linux.

### Validation of External Links

With this update, users can effortlessly determine the validity of external links within spreadsheets using the new `Type` property.

### Bug Fixes

Numerous issues and bugs related to PDF export, text box alignment, formula calculation, chart rendering errors, HTML import/export, and more have been addressed in this version of the Python API.
