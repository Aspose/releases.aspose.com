#### Release Highlights

Aspose.Cells for C++ 24.12 (Linux) enables formula change tracking, BYROW/BYCOL support, and rich pivot filter enhancements. It also addresses Linux-specific crashes and improves HTML export and chart-to-PCL rendering.

#### New Feature

- **Track Formula Changes**: Monitor formulas when cells are inserted/deleted.
- **Lambda Evaluation**: Support for BYROW and BYCOL functions in formulas.

#### Enhancement

- **Fractional Input Parsing**: Convert strings like ‚Äú33 1/3‚Äù to numbers.
- **Expanded Chart Rendering**: Support PCL output with advanced brush types.
- **HTML Styling**: Apply inline styles inside HTML without relying on CSS.
- **New Pivot Filters**: Use specialized methods for date, value, and label filters.
- **Day-Based Date Filters**: Enhance filtering accuracy using full-day values.
- **Warning/Exception Unification**: Integrated enums for exception/warning categories.

#### Fix

- Resolved segmentation fault on Linux when using `libcurl` and `openssl`.
- Fixed formula and formatting errors when converting files to PDF.
- Corrected chart rendering bugs including bar chart label loss and arrow rendering.
- Addressed data corruption when copying worksheets with Python formulas.
- Fixed conditional formatting and query connection preservation bugs.
- Prevented crash when loading malformed files or accessing invalid DataMashup properties.
