#### Release Highlights

Aspose.Cells for C++ 25.4 (macOS) expands file format capabilities with editable DOCX chart export, improved `.numbers` compatibility, new SVG export control, and performance fixes for rendering and calculations.

#### New Feature

- **Editable Charts to Word**: Save charts as editable content within DOCX files.
- **HTML Sheet Selection**: Save specific worksheets when exporting to HTML.
- **Enhanced Numbers Export**: Supports fill settings and visibility toggles for rows/columns.
- **SVG Export Control**: Manage fonts, CSS prefix, and fit-to-viewport options with `SvgImageOptions`.
- **Sensitivity Label Support**: Access and set Microsoft sensitivity labels programmatically.

#### Enhancement

- **Low-Code Utility API**: Fast-track key operations via `Aspose::Cells::LowCode`.
- **Advanced Chart Data to DOCX**: Multi-level categories and date axis support in Word.
- **Optimized Shape Management**: Better handling of shape shifts during row deletion.

#### Fix

- Fixed missing or misplaced labels, legends, and arrows in complex charts.
- Addressed corrupted file generation in merging, sparkline creation, and worksheet move.
- Corrected formula outcomes in SORT, MATCH, TRANSPOSE, and table logic.
- Resolved layout bugs in PDF output including misaligned logos, column width shifts, and bullet issues.
- Eliminated exceptions when accessing PowerQuery, refreshing arrays, or applying HTML strings.
