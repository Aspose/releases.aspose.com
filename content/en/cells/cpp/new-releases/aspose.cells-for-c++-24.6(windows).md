Aspose.Cells for C++ 24.6 gives you the freedom to create, manipulate, and convert spreadsheets with even greater efficiency in your C++ applications running on the Windows platform. This release brings a wave of exciting new features and enhancements, alongside vital bug fixes.

### Key Highlights

### Effortless Image Manipulation:
- Recolor images seamlessly within your spreadsheets with stunning duotone effects (New Feature).
- The C++ Excel API lets you convert charts to high-quality images (New Feature).
- Ensure accurate WebP image handling in PDF and image conversions on Windows (Enhancement).

### Supercharge The Data Visualization and Export:
- Aspose.Cells for C++ 24.6 allows you to export chart data directly to HTML for convenient integration (New Feature).
- Developers can align chart axis values precisely for improved readability within their C++ apps (Enhancement).
- Leverage the new `Range.ToHtml` method for effortless HTML conversion on Windows-based systems (New Feature).

### Simplified PivotTable Management:
- Easily customize pivot table formatting with the `ShowValuesAs` method added in this C++ library version (New Feature).
- You can now easily access pivot table settings using `GetShowValuesSetting` (New Feature).
- Accurately convert data ranges to JSON for versatile data exchange (New Feature).

### Streamlined API Interaction

Aspose.Cells for C++ 24.6 enables a more intuitive API experience with additions like `PivotTable::GetFields` and deprecation of methods with alternative, clearer options.

### Bug Fixes

We have added numerous bug fixes in this release to ensure smoother operations and address issues related to charts, hyperlinks, data loss, and more.

### Public API and Backwards Incompatible Changes

### Added APIs

- Shape::IsAspectRatioLocked() method
- PivotTable::GetFields(Pivot::PivotFieldType) method
- JsonSaveOptions::ToExcelStruct method 
- Range::ToJson(JsonSaveOptions) method
- PivotField::ShowValuesAs(PivotFieldDataDisplayFormat displayFormat, int32_t baseField, PivotItemPositionType baseItemPositionType, int32_t baseItem) method
- PivotShowValuesSetting class and PivotField::GetShowValuesSetting() method
- Range::ToImage(const ImageOrPrintOptions& options) method
- Range::ToHtml(const HtmlSaveOptions& saveOptions) method
- PivotItemPositionType enum
- PivotItem::IsFormula()
- PivotItem::SetIsFormula(bool value) method

### Obsolete APIs

- Shape::IsLockAspectRatio() method
- GetDataDisplayFormat()
- SetDataDisplayFormat(PivotFieldDataDisplayFormat::Index)
- GetBaseFieldIndex()
- SetBaseFieldIndex(int32_t value)
- GetBaseItemPosition()
- SetBaseItemPosition(PivotItemPosition value)
- GetBaseItemIndex()
- SetBaseItemIndex(int32_t value)
