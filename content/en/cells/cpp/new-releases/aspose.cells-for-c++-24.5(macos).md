Mac developers, rejoice! Aspose.Cells for C++ 24.5 brings an array of enhancements to optimize your spreadsheet processing tasks. This update introduces an advanced set of new features and improvements to empower your C++ development on macOS.

### Work With XMATCH and XOR Functions

Supercharge your spreadsheet processing apps with `XMATCH` and `XOR` functions. You can now perform efficient data analysis seamlessly in C++ Excel solutions.

### Upgraded Data Handling

Aspose.Cells for C++ 24.5 supports effortless rearranging of data ranges and extracting advanced filtering settings in your Excel apps running on macOS.

### Refined Rendering & Export

With this release of the C++ Excel API, developers can exert finer control over cell formatting and metadata export for various file formats.

### Bug Fixes & Enhancements

Numerous reported bugs were fixed and enhancements added in this Excel API release allow users to experience a more robust and efficient library on Apple Macs.

### Public API and Backwards Incompatible Changes

Aspose.Cells for C++ 24.5 release delivers many changes to the public API as we have added, updated, and deprecated API members ensuring a great user experience.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for C++ 24.5 Release Notes](https://releases.aspose.com/cells/cpp/release-notes/2024/aspose-cells-for-cpp-24-5-release-notes/).
