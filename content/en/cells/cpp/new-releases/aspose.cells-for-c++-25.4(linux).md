#### Release Highlights

Aspose.Cells for C++ 25.4 (Linux) delivers support for exporting editable charts to Word, managing sensitivity labels, advanced SVG output customization, and improved rendering accuracy in PDF/HTML/image exports.

#### New Feature

- **Export Charts to DOCX**: Save editable charts into Word documents.
- **HTML Selective Sheet Export**: Choose specific sheets when saving as HTML.
- **Numbers Export Upgrade**: Hidden rows/columns and fill settings now supported.
- **SVG Options API**: Fine-tune SVG layout with `SvgImageOptions`.
- **Sensitivity Labels**: Apply and retrieve sensitivity labels for secure data management.

#### Enhancement

- **Low-Code API Module**: Fast access to common operations via `Aspose::Cells::LowCode`.
- **Chart Axis and Category Export**: Supports detailed axis and multi-category charts in Word output.
- **Shape Deletion Logic**: Shapes now shift correctly when deleting blank rows above them.

#### Fix

- Addressed geometry crashes, text layout distortion, and file corruption in PDF exports.
- Fixed rendering errors in pie/bar/line/radar charts across PDF/EMF/SVG.
- Improved formula calculation accuracy for TRANSPOSE, VLOOKUP, CHOOSEROWS, and data tables.
- Resolved file corruption in XLS/XLSX conversion, dynamic array refresh, and merged file creation.
- Eliminated exceptions in formula chains, HTML string addition, and PowerQuery access.
