#### Release Highlights

Aspose.Cells for C++ 25.3 (macOS) extends support for exporting to Apple Numbers format, retrieving chart layout ratios, accessing font binaries, and converting Excel to JSON. The release also includes crucial fixes for layout, formatting, and rendering.

#### New Feature

- **Export to `.numbers`**: Convert spreadsheets with formulas, styles, and comments to Apple Numbers format.
- **Workbook JSON Export**: Save workbook structures to structured JSON.
- **Font Data Access**: Retrieve binary data from fonts used in workbooks.
- **Display Style Access**: Get visual styles including adjacent border merges directly via `Cells`.
- **Chart Text Fitting**: Dynamically fit text using `Shape::FitToTextSize()`.

#### Enhancement

- **HTML Styling & Highlighting**: Apply inline formatting and reflect selected sheet in HTML output.
- **Chart Ratio Properties**: All chart areas, frames, titles use ratio-based layout settings.
- **Markdown Table Headers**: Customizable options for table export in Markdown.

#### Fix

- Fixed SVG/PDF rendering issues for charts with multiline labels and legends.
- Corrected XPS layout bugs: distorted 3D, wrong text orientation, missing font styles.
- Improved conditional format logic and MATCH, SUM, and complex name references.
- Resolved conversion issues between XLS/XLSX and malformed output files.
- Fixed alignment, layout, and bullet handling for LightCells and conditional formatting.
