#### Release Highlights

Aspose.Cells for C++ 24.10 (Windows) introduces key features like external pivot connections, compact pivot table creation, and shape-preserving cleanup logic. This release also delivers significant fixes for Excel-to-PDF conversion, formula evaluation accuracy, and chart layout integrity.

#### New Feature

- **Pivot Table Source Linking**: Retrieve pivot sources from external and DataModel connections.
- **Shape-Safe Clean-Up**: Preserves shapes when deleting blank rows/columns.
- **Compact Layout Tables**: Create compact pivot tables in addition to classic layouts.
- **Formula Parsing Options**: Specify parsing behavior with `SetFormula()` overloads.

#### Enhancement

- **Pivot Table Performance**: Better cache usage and area conversion.
- **ODS Support**: Enhanced pivot table handling and correct date formats on save.
- **Hyperlink Protocols**: Appends `file://` to shared file/directory links.
- **Stability Boost**: Prevents formula auto-calc hangs in large ranges.

#### Fix

- Resolved incorrect rendering of combo, histogram, and bubble charts in PDF.
- Fixed corrupted XLSX/XLSB when copying sheets or saving with pivots.
- Corrected pivot field position and layout issues in ODS files.
- Fixed protection loss on chart sheets and axis color rendering bugs.
- Resolved crashes during formula calculation and dynamic array evaluation.
