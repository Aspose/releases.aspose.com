#### Release Highlights

Aspose.Cells for C++ 24.11 (Linux) delivers advanced Excel automation with Excel-to-PCL conversion, enhanced pivot capabilities, and SmartArt rendering. This release also adds memory-friendly image placement and thread-safe formatting operations.

#### New Feature

- **Freeform Shape API**: Draw complex shapes using path commands like `MoveTo`, `LineTo`, and `ArcTo`.
- **Export to PCL**: Generate PCL format documents from Excel files.
- **Dynamic Image Embedding**: Handle images within formulas and in memory-optimized exports.
- **Conditional Image Handling**: Match and place images in multiple cells under rules.
- **Data Model & Pivot Links**: Get pivot data sources, relationships, and PowerQuery definitions.
- **Signature Support**: Insert digital signature lines like in Excel UI.

#### Enhancement

- **Rich Formatting Support**: Enhanced stability for multi-threaded formatting.
- **Image Resolution Management**: Customize default DPI for output images.
- **HTML to Excel Improvements**: Retains layout, borders, alignment from HTML.
- **Improved Formula Engine**: Refined error evaluation and dynamic array handling.
- **Chart Optimization**: Better rendering for legends, axes, and SmartArt in image/PDF output.
- **ODS & XLSB Stability**: Prevent file corruption and layout regression.

#### Fix

- Eliminated #NUM! errors caused by large formulas or incorrect logic.
- Fixed hanging during Excel-to-PDF conversion with SmartArt and charts.
- Resolved crash issues with digital signatures, pivot groups, and nested DIVs.
- Addressed memory corruption in multi-thread formula evaluations and sheet copy operations.
