#### Release Highlights

Aspose.Cells for C++ 25.2 (macOS) empowers developers with asynchronous API support, expanded file format compatibility (DIF/DBF), and customizable SVG and HTML export. Enhancements cover chart styling, formula accuracy, and PowerQuery reliability.

#### New Feature

- **Asynchronous API**: Non-blocking processing for heavy workbook operations.
- **Extended Format Support**: Added DIF and DBF compatibility.
- **LightCells RichText**: Preserve rich cell content in optimized save mode.
- **Multi-Sheet Name Detection**: Provide all worksheet names from formula refs.
- **INDIRECT Ref Evaluation**: Support defined names with `#` syntax.
- **WPS Format Embeds**: Ensure image behavior aligns with WPS expectations.

#### Enhancement

- **PowerQuery Descriptions**: Read and write metadata in queries.
- **PivotTable Source Retrieval**: Access original pivot data references.
- **Conditional Scope Application**: Apply pivot formatting by range/scope.
- **HTML Export Versions**: Choose HTML5 or XHTML output.
- **SVG Customization**: Set CSS prefixes for SVGs.
- **Axis Display Precision**: Use floating-point units for axis customization.

#### Fix

- Addressed incorrect shape/text rendering in image and SVG output.
- Resolved vertical alignment issues in HTML and callouts in charts.
- Eliminated workbook corruption and PDF rendering failures.
- Fixed PowerQuery load crashes and missing embedded query formulas.
- Resolved Excel style issues in XPS (double underline, superscript).
- Corrected formula miscalculations and invalid references in `CELL`, `RANK`, `XLOOKUP`.
