#### Release Highlights

The Linux version of Aspose.Cells for C++ 24.10 enhances data connectivity and Excel interoperability. It brings unified behavior for row/column deletion affecting shapes, adds compact pivot table creation, and resolves long-standing issues with chart rendering and pivot reliability.

#### New Feature

- **External Connection Support**: Access pivot source from external and DataModel links.
- **Drawing-Aware Clean-Up**: Prevents accidental deletion of rows with overlapping shapes.
- **New Pivot Layouts**: Create classic and compact pivot tables.
- **Formula Options API**: New overloads for formula input with parsing control.

#### Enhancement

- **OLE Object Parsing**: Improved error handling for invalid names.
- **Pivot Range Conversion**: Better accuracy and formatting.
- **Hyperlink Behavior**: Supports file path protocol for shared resources.
- **Auto-Calc Handling**: Prevents UI freeze during formula evaluation.

#### Fix

- Corrected rendering bugs in histogram, column, and bubble charts.
- Fixed pivot table errors in XLSB, ODS, and XLS exports.
- Resolved Excel repair prompts on file load due to external links.
- Fixed chart label overlap, color loss, and shape misplacement in PDF.
- Eliminated formula crashes and inconsistencies in calculated values.
