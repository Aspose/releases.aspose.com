#### Release Highlights

Aspose.Cells for C++ 25.2 (Windows) introduces asynchronous APIs, DIF/DBF file format support, rich text handling with LightCells, and broader formula engine improvements. It includes significant bug fixes in chart rendering, Excel-to-PDF, and PowerQuery handling.

#### New Feature

- **Async Operations**: Support for asynchronous loading, saving, and formula calculation.
- **New File Formats**: Read/write support for DIF and DBF (dBase) formats.
- **INDIRECT + Defined Name**: Calculate INDIRECT using defined names with `#`.
- **Multi-Sheet Reference Listing**: Retrieve all referenced sheet names in formulas.
- **Rich Text in LightCells**: Preserve styled cell content during LightCells saving.
- **WPS Image Compatibility**: Embed images compatible with WPS format.

#### Enhancement

- **Pivot Format Rules**: Improved application of conditional formatting scopes.
- **Web Pivot URL Fix**: Corrects protocol in external pivot data source.
- **Query Comments**: Preserve query formula comments across loads/saves.
- **SVG Styling Control**: Set prefix for SVG CSS when exporting.
- **HTML Version Control**: Save to HTML5 or XHTML via new `HtmlVersion` enum.
- **Chart Axis Precision**: Enhanced display unit handling via double precision.
- **Improved Pivot APIs**: Access original data sources and formatting scopes.

#### Fix

- Corrected `#VALUE!` errors from formulas like RANK, INDIRECT, and XLOOKUP.
- Fixed shape misalignment and incorrect text positions in charts/images/SVG.
- Resolved crashes in PDF conversion and loading corrupted XLSX files.
- Eliminated PowerQuery and Name-LAMBDA exceptions.
- Fixed bullet points and text breaks in LightCells HTML exports.
- Addressed Excel compatibility for strikethrough, underline, superscript, and pattern fills.
- Improved rendering and label alignment in multi-line axis labels and callouts.
