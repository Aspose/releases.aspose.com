C++ developers can take Excel document processing to the next level with Aspose.Cells for C++ 24.4 which packs a load of new features and improvements for your spreadsheet development workflow on macOS.Â 

### New Features

- **Optimized Image Handling:** Convert ChartX to images with ease, embed attachments when converting OLE objects to PDF, and effortlessly inject images within XLSB files, enriching the visual capabilities of your macOS apps.
- **Advanced Exporting for Numbers Files:** With this update, you can control your exports by hiding rows and columns, merging cells, and exporting charts during numbers to XLSX conversions.
- **Interrupt Monitoring:** Manage long-running operations effectively with the new `SystemTimeInterruptMonitor` class, allowing you to monitor and interrupt them as needed. 


### Enhancements
- Aspose.Cells for C++ 24.4 allows you to determine the validity of external links using the `Type` property.
- Incorporate advanced formula processing into your apps on macOS by treating empty formula results as blank cells when deleting rows/columns Dynamic Array Formulas copying support.
- Easily export cell ranges to JSON as an array of arrays.
- You can now control your web-based content on macOS by fetching the metadata of the web image files.

### Bug Fixes
This update includes many bug fixes related to formula calculation, PDF generation, Excel to various format conversions, and more.
