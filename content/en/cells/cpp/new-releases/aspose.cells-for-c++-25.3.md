#### Release Highlights

Aspose.Cells for C++ 25.3 (Linux) introduces Apple Numbers file export support, advanced display style access, workbook-to-JSON conversion, and performance/stability fixes for formulas, rendering, and layout management.

#### New Feature

- **Export to Numbers**: Save cell data, styles, formulas, and comments to `.numbers` files.
- **Workbook-to-JSON**: Convert entire Excel structure into structured JSON.
- **Font Binary Access**: Retrieve raw binary content of used fonts.
- **Track FormatCondition Changes**: Monitor formatting rule changes during cell shift.
- **Display Style via Cells**: Retrieve cell styling using row/column without full cell objects.

#### Enhancement

- **HTML Styling**: Inline formatting inside HTML elements for fidelity.
- **Highlighted Sheet Tab**: Show selected sheet tab in HTML exports.
- **Smart Border Merging**: Border rendering honors adjacent cell context.

#### Fix

- Corrected rendering of line charts, legends, treemaps, bar/radar charts in image/PDF/SVG.
- Fixed Excel to XPS issues: strikethrough, vertical text, layout shifts, and 3D distortions.
- Improved What-if and complex formula calculations including MATCH, SUM, defined names.
- Prevented invalid file generation or formula corruption during save/convert.
- Resolved duplicated line breaks, font style shifts, and conditional formatting errors.
- Fixed data label misalignment and dropped values in rendered chart images.
