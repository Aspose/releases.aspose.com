Aspose.Cells for C++ 24.8 delivers a set of dynamic features and improvements for you to effortlessly manage your Excel spreadsheet processing tasks on Windows.

### Key Highlights

- **Enhanced Pivot Table Functionality:** Create dynamic pivot tables, sort by specific pivot items, and improve overall pivot table performance with the latest C++ Excel API release.
- **Advanced Chart Rendering:** Experience superior chart visualization with accurate label positioning, consistent rendering, and support for various chart types on Windows OS.
- **Improved Formula Calculation:** Developers can benefit from reliable formula evaluation, including support for complex functions like `YEARFRAC` and `LET`.
- **Optimized Excel File Formats:** Aspose.Cells for C++ 24.8 efficiently handles different Excel file formats with enhanced conversion, import, and export capabilities.

### New Features

- We have now introduced support for visiting all cells containing "Place in Cell" pictures.
- The `VALUETOTEXT` function for flexible data manipulation was implemented in this C++ spreadsheet processing library version.
- The functionality of sorting pivot tables by pivot field items is now enabled for finer data analysis on Windows.
- Support for sorting by specific pivot items to refine data presentation has been added.
- The ability to edit formulas with blank spaces for improved formula creation is incorporated.

### Enhancements

- Utilize the functionality of encrypting documents for enhanced security control without encrypting document properties.
- Version 24.8 of the C++ API delivers refined character rendering in Excel to PDF conversion for accurate output.
- Text content rendering to PDF is optimized for improved visual consistency on Windows-powered machines.

### Bug Fixes

Here are some of the issues we fixed in Aspose.Cells for C++ 24.8:

- Resolved `YEARFRAC` formula calculation issue with February 29th and January 1st.
- Fixed hyperlinks in PDF not working after formula calculation.
- Addressed `LET` function evaluation error.
- Corrected spreadsheet conversion errors.
- Improved chart rendering by fixing missing data labels, incorrect arrow/bullet positioning, and label alignment issues.
- Addressed Excel file to PDF conversion rendering issues.
- Resolved text underline display errors in PDF conversion.
- Fixed HTML-rich text rendering issues.
- Corrected SpreadsheetML to XHTML conversion errors.
- Improved image positioning in XLSX to HTML conversion.
- Resolved pivot table area formatting loss after opening the resultant spreadsheet in Excel.
- Fixed worksheet corruption after saving with PivotTables.
- Addressed issue with sorting row field by values in a specific field.
- Corrected rendering of merged cells with blank rows removed.
- Retained math equations when opening Excel spreadsheets.
- Fixed result file crashes when resaving the sample file.
- Resolved error messages when opening XLSX files.
- Improved data accuracy and consistency when importing JSON into Excel.
- Corrected single node export as attributes when exporting to JSON.
- Retained outline borders after refreshing pivot tables.

### Public API Changes

- Added new methods for pivot table sorting and filtering.
- Introduced properties for customizing chart appearance.
- Updated calculation engine for improved accuracy.
