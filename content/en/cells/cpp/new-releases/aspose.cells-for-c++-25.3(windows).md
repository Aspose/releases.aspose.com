#### Release Highlights

Aspose.Cells for C++ 25.3 (Windows) brings new capabilities including `.numbers` export, access to chart object ratios, optimized display styling, and deep chart API refinements. This update also resolves key rendering and formatting bugs.

#### New Feature

- **Apple Numbers Export**: Output cell data, styles, formulas, and comments to `.numbers`.
- **Workbook-to-JSON Export**: Transform workbook to structured JSON.
- **Font Binary Retrieval**: Extract font binaries from workbook.
- **Track Conditional Format Formula Changes**: Detect formatting updates post cell shifts.
- **Cell Display Style Access**: Fetch display styles directly from `Cells` object.
- **Markdown Header Type Control**: Customize markdown table headers.

#### Enhancement

- **BorderType & StyleFlags Update**: Improved border logic with new enum values.
- **Refined Chart Positioning**: All `ChartArea`, `ChartFrame`, and `PlotArea` properties now support ratio-based X/Y/Width/Height.
- **Inline HTML Styling**: Export inline styles directly to HTML content.
- **Selected Sheet Tab Highlight**: Reflect UI-active tab in HTML output.

#### Fix

- Solved various layout issues in charts exported to images, SVG, and PDFs.
- Addressed chart area calculations and incorrect zero values in rendered output.
- Fixed multi-level category axis and bar chart misplacements.
- Resolved font rendering bugs (strikethrough, small caps, superscripts).
- Corrected complex formula evaluation for locales, What-if tables, and defined names with dots.
- Eliminated workbook crashes when saving XLSB or rendering with images.
