#### Release Highlights

Aspose.Cells for C++ 24.9 for Windows offers deep enhancements in formula extensibility, math equation rendering, and pivot table intelligence. PDF and HTML exports benefit from smarter chart rendering and checkbox support.

#### New Feature

- **Custom Formula Engine**: Extend Excel functions using `AbstractCalculationEngine`.
- **OfficeMath Support**:
  - Export grouped/limit equations to **MathML**, **LaTeX**, or **image**.
  - Control via `HtmlSaveOptions::SetOfficeMathOutputMode()`.
- **Pivot Table API Upgrades**:
  - Get formulas from calculated fields/items, slicer captions.
  - Display "Sum of" captions properly.
- **Export & UI Features**:
  - Add checkboxes for cell ranges.
  - Export cell address in HTML attributes.
- **Better Excel Compatibility**:
  - New XLS save options for WPS compatibility.
  - Equation array support via `ArrayEquationNode`.

#### Enhancement

- Optimized memory/image handling for PDF.
- Better layout of shapes, charts, and conditional formats.
- Improved OLE object detection.
- Improved grouping and formula evaluation speed.

#### Fix

- Fixed 40+ issues affecting:
  - PDF distortions, chart image scaling, text rendering, ActiveX controls.
  - Pivot data loss, corrupted files, incorrect slicer state, export crashes.
