#### Release Highlights

Aspose.Cells for C++ 24.12 (macOS) brings support for formula monitoring, lambda functions, detailed pivot filtering, and better chart export. It improves HTML styling and resolves significant stability and rendering issues.

#### New Feature

- **Monitor Formulas**: Observe formula changes when editing the sheet.
- **Lambda Expansion**: Full support for BYROW and BYCOL functions.

#### Enhancement

- **Smart Fraction Parsing**: Automatically convert fractions like ‚Äú33 1/3‚Äù in formulas.
- **Rich PCL Chart Support**: Gradient, hatch, and texture brushes now render in PCL output.
- **HTML Output Control**: Adds inline styles to exported HTML elements.
- **New Pivot API**: Replace deprecated filters with targeted filter methods.
- **Day-Precision Filtering**: Accurate full-day date-based pivot filters.
- **PowerQuery Enhancements**: Group name support and warning metadata access.

#### Fix

- Fixed rendering issues with XYScatter and bar charts in PDF/image outputs.
- Resolved bugs with text wrapping, subtitle rendering, and element clipping in exported files.
- Prevented Excel file corruption due to Python formulas or conditional formatting.
- Fixed workbook crashes on formula recalculation or loading broken XLSM/XLSB files.
- Addressed incorrect pivot views and table data issues in refreshed reports.
