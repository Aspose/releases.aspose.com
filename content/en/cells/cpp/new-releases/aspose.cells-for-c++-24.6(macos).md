C++ developers working on spreadsheet applications can use Aspose.Cells for C++ 24.6 to integrate image recoloring, refined pivot table functionalities, and the ability to accurately convert worksheet ranges to HTML and image formats into their apps on macOS systems.

### Key Highlights

### Powerful Data Visualization and Export Options:
- The C++ Excel API lets you export chart data directly to HTML for simplified integration (New Feature).
- Align chart axis values precisely in your C++ apps for better readability (Enhancement).
- Utilize the newly introduced `Range.ToHtml` method for accurate HTML conversion on macOS-based systems (New Feature).

### Refined Image Manipulation:
- Recolor images effortlessly within your spreadsheets with catchy duotone effects (New Feature).
- Aspose.Cells for C++ 24.6 enables converting charts to high-quality images (New Feature).
- Ascertain precise WebP image handling in PDF and image conversions on macOS (Enhancement).

### Supercharge Your PivotTable Management:
- Quickly customize pivot table formatting with the `ShowValuesAs` method introduced in the latest C++ library version (New Feature).
- Mac developers can now access pivot table settings using `GetShowValuesSetting` (New Feature).
- Convert data ranges accurately to JSON for optimized data exchange (New Feature).

### Upgraded API Interaction

Aspose.Cells for C++ 24.6 helps you with an enhanced API usage experience with additions like `PivotTable::GetFields` and the deprecation of methods with better alternatives.

### Bug Fixes

We have added numerous bug fixes in this release to ensure smoother operations and address issues related to charts, hyperlinks, data loss, and more.

### Public API and Backwards Incompatible Changes

### Added APIs

- Shape::IsAspectRatioLocked() method
- PivotTable::GetFields(Pivot::PivotFieldType) method
- JsonSaveOptions::ToExcelStruct method 
- Range::ToJson(JsonSaveOptions) method
- PivotField::ShowValuesAs(PivotFieldDataDisplayFormat displayFormat, int32_t baseField, PivotItemPositionType baseItemPositionType, int32_t baseItem) method
- PivotShowValuesSetting class and PivotField::GetShowValuesSetting() method
- Range::ToImage(const ImageOrPrintOptions& options) method
- Range::ToHtml(const HtmlSaveOptions& saveOptions) method
- PivotItemPositionType enum
- PivotItem::IsFormula()
- PivotItem::SetIsFormula(bool value) method

### Obsolete APIs

- Shape::IsLockAspectRatio() method
- GetDataDisplayFormat()
- SetDataDisplayFormat(PivotFieldDataDisplayFormat::Index)
- GetBaseFieldIndex()
- SetBaseFieldIndex(int32_t value)
- GetBaseItemPosition()
- SetBaseItemPosition(PivotItemPosition value)
- GetBaseItemIndex()
- SetBaseItemIndex(int32_t value)
