#### Release Highlights

Aspose.Cells for C++ 24.11 (macOS) enhances Excel features with support for SmartArt rendering, PCL export, and pivot data models. Improved HTML import, signature lines, and stability fixes deliver better performance and compatibility.

#### New Feature

- **SmartArt Rendering**: Maintain layout fidelity for shapes and diagrams in PDF/HTML.
- **Excel-to-PCL Export**: Generate print-friendly formats directly from workbooks.
- **Flexible Image Handling**: Embed images using formulas or memory-efficient workflows.
- **DataModel Extraction**: Access pivot sources, relationships, and PowerQuery parameters.
- **Dynamic Rich Value Handling**: Access rich/error cell values programmatically.

#### Enhancement

- **Multi-Thread Formatting Engine**: Stable performance for parallel formatting tasks.
- **HTML Rendering Fixes**: Correct layout issues in Safari and Mac-based HTML exports.
- **Custom Image DPI Settings**: Use `WorkbookSettings` to control image output resolution.
- **Advanced Shape Manipulation**: Draw, edit, and align custom shapes using `ShapePath`.

#### Fix

- Corrected missing images, font issues, and layout inconsistencies in exported charts.
- Fixed Smart Marker hyperlinks and signature rendering.
- Addressed formula errors in TEXTJOIN, LET, MAP with long or nested inputs.
- Prevented file corruption in saved XLSX/XLSB files with pivot or SmartArt.
- Resolved exceptions from formula recalculation and nested shape rendering.
