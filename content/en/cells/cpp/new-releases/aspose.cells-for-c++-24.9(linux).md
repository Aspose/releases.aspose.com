#### Release Highlights

Aspose.Cells for C++ 24.9 on Linux adds extensive formula handling, mathematical expression support, chart rendering fixes, and performance tuning for server-side spreadsheet processing and PDF/HTML conversion.

#### New Feature

- **Custom Function API**: Add user-defined functions via `AbstractCalculationEngine`.
- **Object Type Handling**: Pass and return Object-type values in formulas.
- **Math Equation Enhancements**:
  - Support limit/grouped equations, export to **LaTeX** and **MathML**.
  - Control export as image or MathML using `HtmlOfficeMathOutputType`.
- **HTML Output Expansion**:
  - Add cell address as `TD` attributes.
  - Better compatibility for OfficeMath objects.
- **Pivot Table API**:
  - Get formulas for calculated fields/items.
  - Translate "Sum of" labels and access slicer captions.
- **Checkbox Controls**: Add checkboxes for cell ranges programmatically.
- **Feature Bags Support**: Parse xlsx property bags via API.

#### Enhancement

- **PDF Rendering Optimization**: Improved memory and image data handling.
- **Pivot Calculation Performance**: Faster processing of grouped pivot fields.
- **OLE Format Detection**: Smarter identification of embedded OLE objects.
- **Equation to Image Export**: Improved math equation-to-image conversion.

#### Fix

- Resolved over **35+ issues** including:
  - Chart axis rotation, font mismatches in charts, shape rendering errors.
  - PDF/HTML distortions in checkboxes, images, borders, conditional formatting.
  - XLSX/XLSB corruption and exception crashes in file loading & formula evaluation.
