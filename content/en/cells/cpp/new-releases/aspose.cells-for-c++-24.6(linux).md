Supercharge your spreadsheet manipulation apps on Linux with Aspose.Cells for C++ 24.6 release. This release is packed with exciting new features, enhancements, and bug fixes to upgrade your spreadsheet processing capabilities. 

### Key Highlights

### Next-level Data Visualization and Export:
- With the latest C++ Excel API release, you can export chart data directly to HTML for convenient integration (New Feature).
- Easily align chart axis values accurately in your C++ apps for improved readability (Enhancement).
- The newly added `Range.ToHtml` method allows accurate HTML conversion on Linux-based systems (New Feature).

### Image Manipulation Improvements:
- Experience effortless recoloring of images in your spreadsheets with visually appealing duotone effects (New Feature).
- Aspose.Cells for C++ 24.6 offers the conversion of charts to high-quality images (New Feature).
- Enjoy accurate WebP image management in PDF and image conversions on Linux (Enhancement).

### Extensive PivotTable Management:
- Customize pivot table formatting seamlessly with the `ShowValuesAs` method added in the newest Spreadsheet library version (New Feature).
- Linux developers can now utilize pivot table settings with `GetShowValuesSetting` (New Feature).
- Transform data ranges precisely to JSON for enhanced data exchange (New Feature).

### Updated API Interaction

Aspose.Cells for C++ 24.6 enables an optimized API usage experience with additions like `PivotTable::GetFields` and the deprecation of methods with high-quality alternatives.

### Bug Fixes

We have added numerous bug fixes in this release to ensure smoother operations and address issues related to charts, hyperlinks, data loss, and more.

### Public API and Backwards Incompatible Changes

### Added APIs

- Shape::IsAspectRatioLocked() method
- PivotTable::GetFields(Pivot::PivotFieldType) method
- JsonSaveOptions::ToExcelStruct method 
- Range::ToJson(JsonSaveOptions) method
- PivotField::ShowValuesAs(PivotFieldDataDisplayFormat displayFormat, int32_t baseField, PivotItemPositionType baseItemPositionType, int32_t baseItem) method
- PivotShowValuesSetting class and PivotField::GetShowValuesSetting() method
- Range::ToImage(const ImageOrPrintOptions& options) method
- Range::ToHtml(const HtmlSaveOptions& saveOptions) method
- PivotItemPositionType enum
- PivotItem::IsFormula()
- PivotItem::SetIsFormula(bool value) method

### Obsolete APIs

- Shape::IsLockAspectRatio() method
- GetDataDisplayFormat()
- SetDataDisplayFormat(PivotFieldDataDisplayFormat::Index)
- GetBaseFieldIndex()
- SetBaseFieldIndex(int32_t value)
- GetBaseItemPosition()
- SetBaseItemPosition(PivotItemPosition value)
- GetBaseItemIndex()
- SetBaseItemIndex(int32_t value)


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for C++ 24.6 Release Notes](https://releases.aspose.com/cells/cpp/release-notes/2024/aspose-cells-for-cpp-24-6-release-notes/).
