#### Release Highlights

Aspose.Cells for C++ 25.2 (Linux) introduces DIF and DBF file format support, async APIs for long-running tasks, and SVG customization tools. Enhanced compatibility for charts, PowerQuery, and LightCells are also included in this release.

#### New Feature

- **Async APIs**: Non-blocking load/save and formula evaluation support.
- **File Format Expansion**: Added DIF and DBF (dBase) reading/writing.
- **INDIRECT with Name Ref**: Supports names with `#` for INDIRECT evaluation.
- **Multi-Sheet Formula References**: List all referenced worksheets.
- **RichText in LightCells**: Retain full text styling on export.
- **WPS-Compatible Embeds**: Ensure image elements align with WPS format specs.

#### Enhancement

- **PivotScope Controls**: Fine-tune conditional formatting using `ApplyTo()` API.
- **HTML Output Control**: Target XHTML/HTML5 via new enum and settings.
- **PowerQuery Enhancements**: Description fields and formula consistency improved.
- **Axis Unit Precision**: Now handles display units as doubles.
- **SVG Styling Control**: Prefix CSS classes for SVG export precision.
- **Query Comment Preservation**: Retain formula-level annotations.

#### Fix

- Resolved misaligned callouts, Y-axis labels, and plot areas in chart-to-image workflows.
- Fixed bullet wrapping and text spacing in LightCells and HTML exports.
- Eliminated crashes during PDF generation and PowerQuery parsing.
- Addressed `#VALUE!` and formula errors in XLOOKUP, RANK, CELL, and arrays.
- Fixed incorrect validation rules, Excel font corruption, and alignment shifts.
