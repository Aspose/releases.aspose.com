This page contains release notes for [Aspose.Cells for JasperReports 1.0.1](https://releases.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.0.1/)

We are happy to announceAspose.Cells for JasperReports 1.0.1!

What's new:

- Font sizes are adjusted appropriately to paste characters in Microsoft Excel format similar to characters in PDF format.
- Axis labels (of Charts) in date format are rendered correctly and accordingly.  
- Gradient Fill effects for the areas are supported including chart’s background (chart area, plot area etc.), legend’s background, bars and bubbles etc.

What's fixed:

- Fixed some bugs of filling wall and floor of 3D charts.
- Unable to render embedded object: File not found.

- Fixed a bug of pie chart labels.

- Fixed a bug regarding support of jasperserver 3.1.0.
