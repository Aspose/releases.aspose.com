### Optimized HTML Conversion

Aspose.Cells for Node.js via Java offers greater control over HTML output with options for setting default folders and handling empty cells during Excel to HTML conversion.

### Formula Calculations Improvements

You can now ensure accurate formula calculations with bug fixes addressing pivot table fields and multiple formatting conditions.

### Chart Rendering Accuracy

Easily generate visually accurate charts in PDF and PNG formats with fixes for chart elements and layout inconsistencies added in the latest update.

### Public API Changes

Several deprecated methods and properties have been removed in favor of more efficient alternatives in this update to the Node.js Excel API to help you streamline the code and development workflows.

### Bug Fixes

- Text characters shift when converting an Excel file to a PDF (CELLSJAVA-45876).
- Issues related to formatting (CELLSJAVA-45883).
- There are several other issues with formula calculations, cell formatting, and file conversion.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Node.js via Java 24.3 Release Notes](https://releases.aspose.com/cells/nodejs/release-notes/2024/aspose-cells-for-node-js-via-java-24-3-release-notes/).
