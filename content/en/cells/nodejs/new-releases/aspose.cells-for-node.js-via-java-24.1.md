### An Upgraded Mobile Rendering Experience

We have enhanced the user experience on mobile devices with the ability to generate HTML output with highlighted sheet tabs in Aspose.Cells for Node.js via Java 24.1.

### Use Smart Markers

This update to the Node.js Excel API allows developers to utilize JSON as a data source for smart markers and provides greater flexibility in data binding.

### Exceptional Excel to Image Conversion

While converting Excel worksheets to image files, you can ensure text within shapes is rendered flawlessly and deliver high-quality image outputs.

### Additional Improvements

This release brings various improvements to the existing API functionality. Some of these are:

- Enhanced CSV import functionality with user-defined rows and column limits.
- Import CSV data with specified header rows and columns for extended worksheets.
- Define additional CSS styles for the HTML formatter.
- Group number or date/time pivot fields using the PivotField.GroupBy method.
- Export Excel files to PowerPoint with options to view or print modes.

### Resolved Issues

Several issues have been resolved in the latest API update, such as:

- Accurate text color within shapes and textboxes when saving Excel files to HTML. (CELLSJAVA-45756)
- The formula calculation engine is not consistently delivering accurate results. (CELLSJAVA-45748, CELLSJAVA-45780)
- Improved rendering of various chart types during XLSX to HTML conversion. (CELLSJAVA-43795, CELLSJAVA-45784)
- Editing cell values in newly created sheets within GridWeb. (CELLSJAVA-45794)
- Resolved issues causing extra columns in exported HTML and incorrect smart marker layout. (CELLSJAVA-45771, CELLSJAVA-45708)
