### Accurate Formula Calculations

Aspose.Cells for Node.js via Java 24.7.2 delivers optimized functions like ASC, TEXT, and WIDECHAR to ensure precise results across various regions and character sets.

### Enhanced Text Rendering

The latest Node.js spreadsheet API release offers improved text box spacing that ensures accurate representation in converted PDF documents.

### Precise Font Measurements

Integrate the feature of measuring accurate width and height for the Aptos Narrow font, which guarantees consistent formatting in your Node.js Excel manipulation apps.

### Streamlined Image Handling

We have fine-tuned the printing functionality for specific print options to ascertain the image size does not exceed limitations.

### Bug Fixes and Refinements

By resolving various bugs related to PDF conversion, formula calculations, and chart data display, we offer a more robust and reliable solution to Node.js developers.

### Public API and Backwards Incompatible Changes

### Added API Members

- Added AbstractCalculationEngine.ForceRecalculate(string functionName) method.
- Added Picture.PlaceInCell() method.
- Added Shape.IsDecorative property.
- Added JsonLoadOptions.KeptSchema property.
- Added PivotTableCalculateOption class.
- Added PivotTable.CalculateData(Pivot.PivotTableCalculateOption) method.
- Added SlicerCacheItemSortType.Natural enum.
- Added SaveFormat.Azw3 enum.
- Added EbookSaveOptions(SaveFormat saveFormat) constructor.
- Added ChartDataTable.HasHorizontalBorder property.
- Added ChartDataTable.HasVerticalBorder property.
- Added ChartDataTable.HasOutlineBorder property.
- Added ChartPoint.DoughnutInnerRadius property.
- Added DataLabels.ApplyFont() method.
- Added Series.XValuesFormatCode property.
- Added SeriesCollection.SwapSeries(Int32,Int32) method.
- Added SparklineGroupCollection.Add(Charts.SparklineType) method.
- Added a SparklineGroup with a Sparkline to the collection.
- Added Cells.GetCellsWithPlaceInCellPicture() method.
- Added ConversionUtility class.
- Added GridJsWorkbook.JsonToStream(OutputStream stream, string filename) .
- Added GridJsWorkbook.JsonToStreamByUid(OutputStream stream, string uid, string filename).

### Deprecated API Members

- AxisBins.ResetOverflow() and AxisBins.ResetUnderflow() method.
- ChartDataTable: HasBorderHorizontal, HasBorderVertical and HasBorderOutline properties.
- ChartPoint.InnerRadiusPx property.
- SeriesCollection.ChangeSeriesOrder(Int32,Int32) method.
