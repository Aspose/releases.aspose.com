### Key Highlights

- **Accurate Font Measurements:** Aspose.Cells for Node.js via Java 24.7 brings improved width and height measurement for the Aptos Narrow font ensuring precise layout rendering during conversions.
- **Enhanced Macro Management:** You can seamlessly keep and remove custom JavaScript macros and provide greater control over spreadsheet behavior.
- **Formula Calculation Refinements:** Different refinements have been added for `ASC`, `WIDECHAR`, `TEXT` functions, and shared formula calculations to ensure reliable results.
- **Streamlined Conversions:** Experience enriched image handling and page break management during conversions to HTML and PDF formats for a seamless user experience using the latest Node.js spreadsheets API release.

### New Features for Developers
- `AbstractCalculationEngine.ForceRecalculate` method for custom formula recalculation control.
- `Picture.PlaceInCell` method for precise picture placement within cells.
- Shape.IsDecorative` property to identify decorative shapes in spreadsheets.
- Enhanced JSON loading options and pivot table calculation methods.

### Bug Fixes
- Incorrect spacing between text lines in text boxes during PDF conversions (CELLSJAVA-46022)
- Inaccurate `ASC` function values for specific characters (CELLSJAVA-46035)
- `WIDECHAR` function returning incorrect results for some Japanese characters (CELLSJAVA-46036)
- `TEXT` function calculation errors in certain regions (CELLSJAVA-46051)
- Incomplete chart data display when converting files to HTML (CELLSJAVA-46024)
- Image size exceeding the limit with `ImageOrPrintOptions.setOnePagePerSheet`(true) (CELLSJAVA-46029)
- Page break issues when using the Aptos Narrow font (CELLSJAVA-46039)
- `NullPointerException` during file conversion to HTML (CELLSJAVA-46010)
- `ArrayIndexOutOfBoundsException` with `Workbook.calculateFormula` method (CELLSJAVA-46031)
- Fixed "Invalid power query formula definition" error during file merging (CELLSJAVA-46030)
