### Learn to Embed Images in Cells

A brand new feature introduced in Aspose.Cells for Node.js via Java 24.4 allows you to directly insert pictures within cells and enhance the visual appeal of your spreadsheets.

### Accurate Column Width Retrieval

Gain more control over your spreadsheets by effortlessly retrieving the original width of hidden columns with this update to the Node.js Excel Processing API.

### Upgraded Excel File Conversion

Developers can now experience improved accuracy when converting Excel files to various formats, including PDF and HTML.

### Streamlined PivotTable Refresh

With the latest update, you can easily customize the refresh behavior of pivot tables within worksheets and workbooks for optimal performance.

### Bug Fixes

A multitude of bugs and issues related to image and shape copying, character conversion, chart rendering, and more have been fixed in this version of the API, ensuring a more robust development experience.
