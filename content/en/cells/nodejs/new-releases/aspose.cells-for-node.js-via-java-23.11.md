### New Features

- Effortlessly generate spreadsheets compatible with PDF versions 1.5 & 1.6 using Aspose.Cells for Node.js via Java 23.11.
- You can now choose whether to calculate formulas before saving HTML files using the `HtmlSaveOptions.CalculateFormula` property.
- Apply a static final color of your choice for the default font colors within your Excel processing solutions.
- Added the `HtmlSaveOptions.FormatDataIgnoreColumnWidth` property.
- The `CellsHelper.NeedQuoteInFormula()` method has been introduced.
- Developers can now use the `PivotGlobalizationSettings` methods for localized date/time text.
- Introduced the `PivotField.GroupBy()` methods for grouping pivot fields.
- `Range.AddHyperlink()` method has been added to the public API.
- `PivotFieldGroupType` enum is also added in this release.
- `CustomPivotFieldGroupItem` and `PivotDiscreteGroupSettings` classes are added to the API.
- The `PivotGroupByType.Numbers` enum added to this version supports grouping by numeric values.
- We have added the `PivotNumbericRangeGroupSettings` and `PivotDateTimeRangeGroupSettings` classes.

### Fixed Bugs

- `DAYS360` function issue with leap days.
- `LOOKUP` function error in Chinese environments.
- Sheet to image rendering errors.
- Slow PDF conversion times.
- Cell content display issues in HTML exports.
- Date format errors in HTML exports.
- Abnormal cell merging in HTML exports.
- Extra tags in HTML output for overflowing cells.
- Missing borders in HTML exports.
- Pivot table filter and refresh issues.
- Protected view issues when saving spreadsheets.
- Workbook licensing issues.
- Crashing with `treemap` charts.
- Exceptions during HTML conversion.

### Public API Changes

- We have updated the constants for `FormatConditionType`.
- Various obsolete methods have been replaced with new functionalities.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Node.js via Java 23.11 Release Notes](https://releases.aspose.com/cells/nodejs/release-notes/2023/aspose-cells-for-node-js-via-java-23-11-release-notes/).
