This page contains release notes for [Aspose.Cells for Java 7.7.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.7.2/)

{{% /alert %}} 

Aspose.Cells for Java has been updated to version 7.7.2 and we are pleased to announce this release brings the addition of about 20 new useful improvements. 
Using Aspose.Cells for Java you can work with XLS, SpreadsheetML, OOXML, XLSB, CSV, HTML, ODS, PDF, XPS and other formats in your applications. You can also generate, modify, convert, render and print workbooks all without using Microsoft Excel.
Visit the documentation to learn how to get started with Aspose.Cells for Java.
Note this download contains a fully working version of the product, however without a license set it will run in evaluation mode with some limitations. To test Aspose.Cells without these evaluation limitations you can request a free 30-day temporary license.
The following is a list of changes in this version of Aspose.Cells for Java.

Major Features
Support opening password protected MS Excel 2013 files

Other Improvements and Changes

New Features

(CELLSJAVA-40475) - Print comments on an Excel document
(CELLSJAVA-40732) - Support Justify Distributed Alignment Setting

Enhancements

(CELLSJAVA-40719) - Remove hardcode usage of com.sun.image.codec.jpeg.JPEGCodec
(CELLSJAVA-40708) - Unhide Row with previously assigned height

Bugs

(CELLSJAVA-40711) - Axis.setTickLabelSpacing() is not working for XLS version
(CELLSJAVA-40714) - Cell borders were rendered on opaque cells
(CELLSJAVA-40710) - Less than equals to calculation error
(CELLSJAVA-40715) - Problem with wildcard parsing in formula calculation
(CELLSJAVA-40707) - Some columns are narrow to contain text properly
(CELLSJAVA-40720) - IStreamProvider.setCustomPath not effect to file tabstrip.htm
(CELLSJAVA-40729) - Extracting formatted Chinese Date issue
(CELLSJAVA-40728) - getStringValue() returns #VALUE!
(CELLSJAVA-40722) - Validation not working after converting to XLS format
(CELLSJAVA-40690) - XLSX to PDF: Underline below the hyperlink is getting spread out to adjacent character positions

Exceptions

(CELLSJAVA-40629) - Exception when opening xlsx converted to xls
(CELLSJAVA-40630) - Exception when opening xlsx converted to xls
(CELLSJAVA-40717) - CellsException: Invalid cell name on Workbook constructor
(CELLSJAVA-40723) - Rendering cuased com.aspose.cells.CellsException: java.lang.String cannot be cast to java.lang.Double
(CELLSJAVA-40724) - com.aspose.cells.CellsException on SheetRender
(CELLSJAVA-40727) - Exception: java.lang.NullPointerException in rendering charts in a worksheet

Note
Since the code base of Aspose.Cells for Java matches the code of relevant .NET version, most of the changes, enhancements and fixes included in the Aspose.Cells for .NET v7.7.2 are also included in this Aspose.Cells for Java v7.7.2.
