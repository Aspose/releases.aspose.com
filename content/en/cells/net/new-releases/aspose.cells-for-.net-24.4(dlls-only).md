C# developers looking to integrate spreadsheet processing capabilities into their .NET applications can use the Aspose.Cells for .NET 24.4 (DLLs-only) package and experience chart conversion, embedding attachments, improved image handling, and more without needing a full installer.

### New Features

- **ChartX Conversion:** You can now convert ChartX charts to images for awesome visuals.
- **Easy Interrupt Monitoring:** Apply interrupt monitoring with version 24.4 of the C# Excel API seamlessly.
- **Customized Number Formatting:** Easily display cell text in red when using custom number formatting.
- **Add Attachments in PDF Conversion:** Embed attachments during OLE objects to PDF conversion.
- **Insert Images in XLSB Files:** With Aspose.Cells for .NET 24.4, users can add images directly into cells within XLSB files.
- **Freeze Pane Formatting:** Use freeze pane functionality while maintaining conditional formatting icons.
- **Hide Rows and Columns in XLSX Export:** Convert numbers files to XLSX with hidden rows and columns.
- **Cell Merging in Numbers to XLSX Export:** Merging cells when exporting from number files to XLSX format is now supported.
- **Export Charts from Numbers Files:** Users can now export charts from numbers files to optimize the data visualization workflows.


### Feature Enhancements

- **Padding in Formulas:** You can select keeping or ignoring padding space within `GridWeb` formulas.
- **`GridFontUnderlineType` Enhancements:** The `GridFontUnderlineType` property offers more font underlining control in Grid apps.
- **Convenient Formula Input:** This update supports automatic cursor positioning within parentheses after formula insertion in `GridJS`.
- **Export Pictures to HTML:** Export pictures embedded within cells to HTML format with ease. 
- **Enhanced JSON Export:** Developers can now export cells range to JSON format as an array of arrays.
- **Image Handling Updates:** Copying and managing embedded images within cells functionality is optimized.
- **Fetch Web Image Metadata:** Obtain detailed web image metadata embedded within cells with this update.
- **Support for Dynamic Array Formula:** Experience efficient spreadsheet manipulation with Dynamic Array Formulas.
- **Validate External Links:** Use the new `Type` property to determine the external links path validity.


### Bug Fixes

- Aspose.Cells for .NET 24.4 addresses numerous bugs to ensure optimal performance and stability. These fixes include:
- Missing SVG image in PDF files
- Textbox misalignment in comparison to Excel positioning
- Chart rendering errors in PDF conversions
- Postscript coordinate issues in Excel to PDF conversion
- Formula calculation inconsistencies
