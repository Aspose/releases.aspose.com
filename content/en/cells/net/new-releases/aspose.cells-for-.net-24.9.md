#### Release Highlights

Aspose.Cells for .NET 24.9 delivers extensive improvements in mathematical expression handling, interactive GridJs input, pivot table intelligence, and export flexibility across PDF, HTML, and LaTeX.

### ‚ú≥Ô∏è New Feature

- **Math Equation Support**: Added support for limit equations, grouped equations, and export to MathML and LaTeX.
- **GridJs Interactivity**: Introduced spell check, formula explanation, syntax validation, and time control for mobile in GridJs.
- **Pivot Table APIs**: Retrieve formulas from calculated items/fields and customize slicer item captions.
- **Equation Rendering**: Choose between image or MathML output for OfficeMath equations.
- **Checkbox Controls**: Added ability to insert checkboxes for cell ranges.
- **HTML Export**: Now supports exporting cell address as an attribute and equation output control.

- **PDF Conversion Optimization**: Better image memory handling and enhanced equation rendering.

- **Chart Display**: Optimized chart title positioning and font rendering consistency.
- **Mobile UX**: Improved multilingual date selector and validation error styling.
- **OLE Format Detection**: Enhanced format recognition and shape/text layout fidelity.
