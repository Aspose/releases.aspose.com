Aspose.Cells for .NET 24.4 (MSI installer) allows C# developers to streamline spreadsheet processing within their .NET applications with updated functionalities for converting charts, embedding attachments, and improving image handling. 

### New Features

- **Enhanced Chart Export:** Effortlessly convert ChartX charts to images for stunning presentations.
- **Interrupt Monitoring Made Easy:** Implement user-friendly interrupt monitoring with version 24.4 of the C# Excel API.
- **Custom Number Formatting:** Display cell text in red when using custom number formatting for better visual representation.
- **Embed Attachments in PDF Conversions:** You can now embed attachments when converting OLE Objects to PDF files.
- **Add Images to XLSB Files:** Insert images directly into cells within XLSB files and expand your C# spreadsheet capabilities.
- **Formatting in Freeze Pane:** Maintain conditional formatting icons even when using the freeze pane functionality.
- **Hiding Rows and Columns in XLSX Export:** Accurately preserve hidden rows and columns during the numbers files to XLSX format export.
- **Merge Cells in Numbers to XLSX Export:** You can now merge cells effectively when exporting from number files to XLSX format.
- **Chart Export from Numbers Files:** Export charts from number files to XLSX format to streamline your data visualization workflows.


### Feature Enhancements

- **Formula Handling and Padding:** Choose to keep or ignore padding spaces within formulas used in `GridWeb`.
- **Enhanced GridFontUnderlineType:** Use the new `GridFontUnderlineType` property for better control over font underlining in Grid apps.
- **Picture Export to HTML:** Developers can now export pictures embedded within cells to HTML format. 
- **Improved Image Handling:** Enhanced functionalities for copying and managing embedded images within cells are now available.
- **Retrieve Web Image Metadata:** Access detailed metadata associated with web images embedded within cells.
- **JSON Export Enhancements:** Export a range of cells to JSON format as an array of arrays with Aspose.Cells for .NET 24.4.
- **Dynamic Array Formula Support:** Work with Dynamic Array Formulas for efficient spreadsheet development experience.
- **External Link Validation:** The new `Type` property lets you determine the validity of paths associated with external links.
- **Formula Input Convenience:** Enjoy automatic cursor positioning within parentheses after inserting formulas in `GridJS`.

### Bug Fixes

- Aspose.Cells for .NET 24.4 addresses numerous bugs to ensure optimal performance and stability. These fixes include:
- Textbox misalignment in comparison to Excel positioning
- Missing SVG image in PDF files
- Formula calculation inconsistencies
- Postscript coordinate issues in Excel to PDF conversion
- Chart rendering errors in PDF conversions
