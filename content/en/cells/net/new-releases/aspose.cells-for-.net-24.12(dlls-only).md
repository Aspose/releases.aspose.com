#### Release Highlights

Aspose.Cells for .NET 24.12 (DLLs Only) introduces dynamic formula tracking, advanced pivot filtering, new lambda support, and enhanced GridJs controls.

### ‚ú≥Ô∏è New Feature

- **Track Formula Edits**: New API additions like `InsertOptions.FormulaChangeMonitor` enable tracking during range modifications.
- **Lambda BYROW/BYCOL**: Implement array operations using Excel-compatible lambda functions.
- **GridJs Update Monitor**: GridJs now includes an `UpdateMonitor` class to track UI data changes.

### üîß Enhancement

- **Formula Calculation**: Supports parsing mixed number fractions in expressions.
- **Advanced Pivot Filters**: Apply multiple filters, group PowerQuery items, and refine by full-day date settings.
- **HTML & PCL Rendering**: Adds precision rendering for brushes and enables inline CSS in HTML exports.
- **GridJs UI**: Smoother scroll behavior and center alignment for standard and rich text.

### üõ† Fix

- Addressed a variety of export issues including: incorrect pivot values, corrupted files from Python formula use, and multiple chart/image rendering flaws.
- Fixed formula evaluation errors and background color loss during save.
- Improved stability by resolving NullReferenceExceptions and unsupported model exceptions in formula evaluation and XLSM parsing.
