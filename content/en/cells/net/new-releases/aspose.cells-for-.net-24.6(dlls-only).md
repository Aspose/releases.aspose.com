Aspose.Cells for .NET 24.6 has been released, and it offers advanced features to developers for processing Excel spreadsheets. This version delivers exciting features like image recoloring and optimized export capabilities for WebP images.

### Key Highlights

### Streamlined Chart Rendering:
- Work with precise control over axis alignment for image output.
- Data label placement has been optimized for combined charts in this C# spreadsheet API release.

### Refined Image Processing and Recoloring:
- Change color saturation and temperature for images effortlessly.
- Developers can generate stunning duotone effects within images using this release.

### Save to Diverse Formats

- Transform workbook ranges to HTML and JSON formats accurately for extended data utilization with Aspose.Cells for .NET 24.5.
- Upgraded WebP image handling for PDF and image conversions is now available on Linux (NET 6.0, 7.0, and 8.0).

### Additional Improvements:
- Experience efficient handling of invalid data in template files within your C# Excel processing apps.
- We have resolved several bugs related to hyperlinks, pivot tables, and data conversion in this release.

### Public API and Backwards Incompatible Changes

### Added APIs

- Shape.IsAspectRatioLocked property
- PivotTable.GetFields(Pivot.PivotFieldType) method
- JsonSaveOptions.ToExcelStruct property
- Range.ToJson(JsonSaveOptions) method
- PivotField.ShowValuesAs() method
- PivotShowValuesSetting class
- PivotField.ShowValuesSetting property
- Range.ToImage() method
- Range.ToHtml() method
- PivotItemPositionType enum
- PivotItem.IsFormula property

### Removed APIs

- Shape.TextFrame obsoleted property
- Shape.ConnectionPoints obsoleted property
- Shape.ToImage(Stream stream, ImageFormat imageFormat) obsoleted method
- FormatCharacters(int startIndex, int length, Font font) obsoleted method
