Aspose.Cells for .NET 24.6 is here, offering advanced features to developers for working with Excel files. This release introduces exciting functionalities like image recoloring with duotone support and enhanced export capabilities for WebP images on Linux (.NET 6, 7, and 8). 

### Key Highlights

### Enhanced Chart Rendering:
- Experience precise control over axis alignment for image output.
- We have optimized data label placement for combined charts in the latest C# spreadsheet API version.

### Image Processing and Recoloring:
- Adjust color saturation and temperature for images effortlessly.
- This release helps .NET developers generate stunning duotone effects within images.

### Export to Diverse Formats

- Convert workbook ranges to HTML and JSON formats accurately for broader data utilization with Aspose.Cells for .NET 24.5.
- Improved WebP image handling for PDF and image conversions is now available on Linux (NET 6.0, 7.0, and 8.0).

### Additional Improvements:
- You can enjoy effective handling of invalid data in template files within your C# Excel manipulation apps.
- We have resolved several bugs related to hyperlinks, pivot tables, and data conversion in this release.

### Public API and Backwards Incompatible Changes

### Added APIs

- Shape.IsAspectRatioLocked property
- PivotTable.GetFields(Pivot.PivotFieldType) method
- JsonSaveOptions.ToExcelStruct property
- Range.ToJson(JsonSaveOptions) method
- PivotField.ShowValuesAs() method
- PivotShowValuesSetting class
- PivotField.ShowValuesSetting property
- Range.ToImage() method
- Range.ToHtml() method
- PivotItemPositionType enum
- PivotItem.IsFormula property

### Removed APIs

- Shape.TextFrame obsoleted property
- Shape.ConnectionPoints obsoleted property
- Shape.ToImage(Stream stream, ImageFormat imageFormat) obsoleted method
- FormatCharacters(int startIndex, int length, Font font) obsoleted method


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for .NET 24.6 Release Notes](https://releases.aspose.com/cells/net/release-notes/2024/aspose-cells-for-net-24-6-release-notes/).
