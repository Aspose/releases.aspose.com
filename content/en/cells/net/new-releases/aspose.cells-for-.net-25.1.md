#### Release Highlights

Aspose.Cells for .NET 25.1.1 (MSI) focuses on dynamic array spill preservation, external table formula parsing, HTML + GridJs interactivity, and refined pivot UX.

### ‚ú≥Ô∏è New Feature

- **Dynamic Array Integrity**: Preserve spilled ranges by skipping custom function recalculation.
- **Client-Side GridJs Events**: GridJs now supports insert/delete events, update hooks, and pre-check logic.
- **GridJs HTML Interaction**: New APIs to interact with and display HTML nodes within specific cells.
- **Locale Flexibility**: GridJs supports decimal separator per locale.

### üîß Enhancement

- **External Table References**: Custom engine now supports resolving and calculating from external workbook sources.
- **ODF Shape Export**: Save autoshapes and freeform shapes into `.ods` format.
- **HTML Output Optimization**: Adds CSS custom property support for optimized HTML export.
- **Blazor Compatibility**: Trimming improvements and upgraded SkiaSharp for better rendering in WebAssembly apps.
- **Pivot UX Boost**: Overhauled pivot conditional formatting API and improved table form settings.

### üõ† Fix

- Fixed incorrect behavior of `TEXTSPLIT` and `HYPERLINK` functions.
- Solved numerous rendering bugs for charts (e.g., strikethroughs, area backgrounds, text clipping).
- Addressed pivot table issues: invalid formats, incorrect filters, and corrupted results after refresh.
- Resolved data alignment, font styling, and workbook save corruption errors.
- GridJs: Fixed decimal validation, alignment issues, merge area JSON parsing, and formula retention bugs.
