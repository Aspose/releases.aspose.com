### Data Manipulation

Effortlessly copy and paste image and HTML content directly into your spreadsheets and leverage new options for exporting data to `DataTable` with empty cell detection using this release of the C# spreadsheet API.

### OXPS Support

Aspose.Cells for .NET now supports the OXPS (Open XML Paper Specification) format, expanding file compatibility options within your .NET apps.

### Data Grid Management

Manage and filter large datasets with enhanced performance, keyword search, clear filter options, and number filtering capabilities for `GridDesktop`.

### New Functions

This release lets you leverage new functions like TOROW/TOCOL and number detection for streamlined data handling.

### Improved Performance & Bug Fixes

Numerous enhancements and bug fixes are added, enabling developers to experience faster sorting for large datasets, formula removal during data import, chart direction changes, conditional formatting for accurate cell background color application, and more.


### Public API and Backward Incompatible Changes

### Added API Members

We have added several classes, properties, and methods in this release. Some of these are:
- FileFormatType.Oxps enum
- ChartCalculateOptions class
- Chart.Calculate(ChartCalculateOptions) method
- Cells.GetColumnWidthPixel(Int32, Boolean) method
- ExportTableOptions.AllowDBNull property


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for .NET 24.2 Release Notes](https://releases.aspose.com/cells/net/release-notes/2024/aspose-cells-for-net-24-2-release-notes/).
