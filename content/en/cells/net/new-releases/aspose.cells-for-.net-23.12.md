### PDF/A Compliance

Generate PDF documents (when exporting Excel to PDF) compliant with PDF/A-2b, PDF/A-2u, PDF/A-2a, PDF/A-3b, PDF/A-3u, and PDF/A-3a standards using this version of the C# Excel Apps API.

### Enhanced GridDesktop

Developers can now control formula warning messages in `GridDesktop`, export data to generic lists (GridWeb), and render pictures with rotation.

### Improved Pivot Table Support

Export to generic lists, keep missing data in `PivotTable` refresh and calculate discrete groupings by downloading Aspose.Cells for .NET 23.12 MSI installer.

### Bug Fixes

This release addresses issues related to file corruption, chart rendering, formula calculations, and more.

### Public API and Backward Incompatible Changes

### Added API Members

We have added several classes, properties, and methods in this release. Some of these are:
- PdfCompliance.Pdf2b, Pdf2u, Pdf2a, Pdf3b, Pdf3u, Pdf3a enum
- WorkbookDesigner.SetJsonDataSource() method
- Cells.ExportList() method
- Style.IsNumberFormatApplied property
- Style.IsProtectionApplied property
- GridExportTableOptions/GridImportTableOptions classes
- GridShapeDraw class
- GridDesktop.ShowFormulaWarning property

### Obsolete API Members

We have made several properties and methods obsolete in this release. Some of these are:
- Axis.AxisLabels property, added Axis.GetAxisTexts() method
- Cells.MergedCells property, added Cells.GetMergedAreas() method
- Comment.GetCharacters() method, added Comment.GetRichFormattings() method
- Shape.GetCharacters() method, added Shape.GetRichFormattings() method
- PivotField.GetPivotFilters() method, added PivotField.GetFilters() method


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for .NET 23.12 Release Notes](https://releases.aspose.com/cells/net/release-notes/2023/aspose-cells-for-net-23-12-release-notes/).
