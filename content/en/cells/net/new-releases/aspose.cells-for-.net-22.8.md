# Expand / Collapse Pivot Table Button

Added a button style to perform expand table and collapse table functionality for Pivot Table.

# Threaded Comments to HTML Conversion

Supports converting the threaded comments to `HTML`.

# Cell Text Alignment in Excel to PDF Conversion

Previously there was wrong vertical alignment of cell text while we converted any Excel&reg; file containing a pivot table to PDF format. This issue has now been resolved.
