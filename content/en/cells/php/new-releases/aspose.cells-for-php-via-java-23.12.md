### Work with MAKEARRAY function

PHP developers can leverage the `MAKEARRAY` function within their Excel apps using the latest updates in Aspose.Cells for PHP via Java 23.12.

### JSON Data Source

Utilizing JSON as a data source for smart markers is now supported, allowing you to enhance the feature set of your PHP apps.

### Enhanced Formatting Control

With this release of the PHP API, you can effortlessly retrieve "Style Includes" information for advanced formatting control.

### Updated API Structure

Various new APIs have been added and many were removed in this update to the Excel library, providing a smooth user experience.

### Bug Fixes

- Resolved issues in formula calculations (VLOOKUP, IRR, NumberValue)
- Addressed errors in file conversions (XLSX to PNG, XLSX to PDF, etc.)
- Fixed memory consumption issues
