### Improved Formula Calculations

In Aspose.Cells for PHP via Java 24.5, we have resolved issues with `WORKDAY` formula calculations and `TEXTBEFORE/TEXTAFTER` formulas.

### Enhanced Chart Rendering
 
The latest Excel API update brings fixed X-axis tickmark intervals and ensures an accurate display of trendline equations when saving to PDF.

### Streamlined File Format Handling

Developers can experience more precise extensions for tab-delimited text files (".tsv") and better font family extraction from XLSX files within their PHP spreadsheet processing apps.

### Advanced Filtering & HTML Export

Empower your Excel apps with the new `AdvancedFilter` class and properties for controlling table borders and generic fonts in HTML exports.

### Public API Changes

- `FileFormatUtil.LoadFormatToExtension` now returns the most specific extension (".tsv") for tab-delimited text files.
- New properties `ReferredArea.IsEntireRow/IsEntireColumn` indicates whether an area reference encompasses an entire row or column.
- `DefaultStyleSettings.BuiltInPreference` and `WorkbookSettings.DefaultStyleSettings` properties control formatting behavior when styles define both built-in numbers and custom patterns.
- `HtmlSaveOptions.AddGenericFont` allows adding a generic font to the CSS font family during HTML export.
- `HtmlSaveOptions.IsBorderCollapsed` manages collapsing table borders in HTML output.
- `Range.UnionRanges` method replaces the deprecated Range.UnionRang method for uniting ranges.
- `Range.Transpose` method transposes rows into columns and vice versa.
- `AdvancedFilter` class represents advanced filter settings.
- The `Worksheet.GetAdvancedFilter` method retrieves advanced filter settings.
- `CellsUnitType` enum defines unit types.
- `Cells.GetRowHeight` and `Cells.GetColumnWidth` methods offer more control over unit types when retrieving row and column dimensions, replacing several deprecated methods.
