### Key Highlights

- Enhanced rendering of 3D charts (Column & Pie) in PDF format.
- Optimized XML parsing for your Excel apps.
- Improved accuracy of chart labels and legends while exporting to various formats.
- New functionalities were introduced for custom function definitions and formula parsing with custom options.

### Bug Fixes

- Issues with rendering charts, PDFs, formulas, and cell data in various formats (HTML, PDF, SVG, EMF) are resolved.
- Fixed the bugs related to pivot tables, custom functions, and formula parsing.
- Addressed compatibility issues and memory leaks.

### Public API Changes

- Several new methods and classes have been added to the public API for improved functionality.
- Various obsolete methods have been deprecated in favor of the new ones.
