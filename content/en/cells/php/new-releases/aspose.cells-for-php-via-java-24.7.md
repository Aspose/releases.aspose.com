### Key Highlights

Aspose.Cells for PHP via Java 24.7 includes refined chart customization, efficient Excel data handling, accurate formula results, and expanded file format support for your PHP spreadsheet manipulation apps. 

### Improvements and Changes

### Enhanced Chart Capabilities:

- Developers can now customize chart data labels with precision using the new `DataLabels.ApplyFont()` method.
- Accurately represent doughnut chart dimensions through `ChartPoint.DoughnutInnerRadius` property.
- Streamline chart creation with the `SparklineGroupCollection.Add()` method in the latest PHP spreadsheet API version.

### Improved Formula Calculation:

- Aspose.Cells for PHP via Java 24.7 guarantees accurate calculations for shared formulas with the `AbstractCalculationEngine.ForceRecalculate()` method.
- The calculation errors that caused `NullPointerException` and `ArrayIndexOutOfBoundsException` are now addressed.

### Expanded File Format Support:

- Effortlessly convert Excel files to various formats using the new `ConversionUtility` class.
- Create Azw3 ebooks directly from Excel data using the `EbookSaveOptions` constructor in version 24.7 of the PHP library.
- Optimize JSON handling with `GridJsWorkbook.JsonToStream` and `GridJsWorkbook.JsonToStreamByUid` methods.

### Additional Enhancements:

- Work with optimized width and height measurement for Aptos Narrow font in this PHP Excel library release.
- Build and handle custom JavaScript macros seamlessly within your PHP solutions.
- You can precisely position pictures within cells using the `Picture.PlaceInCell()` method.
- Conveniently determine shape decoration with the `Shape.IsDecorative` property.
- Control JSON schema preservation during loading with the `JsonLoadOptions.KeptSchema` property.
- Fine-tune pivot table calculations with the `PivotTableCalculateOption` class and `PivotTable.CalculateData()` method.
- Manage slicer cache item sorting with the `SlicerCacheItemSortType.Natural` enum.

### Bug Fixes

- Corrected text line spacing in text boxes during PDF conversion.
- Fixed incorrect ASC function results for certain characters.
- Improved `WIDECHAR` function accuracy for Japanese characters.
- Addressed `TEXT` function calculation errors in specific regions.
- Corrected chart data display inconsistencies during HTML conversion.
- Resolved image size limitations when using `ImageOrPrintOptions.setOnePagePerSheet(true)`.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for PHP via Java 24.7 Release Notes](https://releases.aspose.com/cells/php/release-notes/2024/aspose-cells-for-php-via-java-24-7-release-notes/).
