### Improved Image Management

Aspose.Cells for PHP via Java 24.6 provides improved support for loading various image formats through third-party Imageio plugins. 

### Calculate Formulas Accurately

The PHP Excel API ensures accurate formula calculation in large workbooks, eliminating the need for manual recalculation in MS Excel within your PHP applications.

### Streamlined PDF & HTML Conversions

Take your PHP spreadsheet conversion applications to the next level with supercharged functionalities such as improved table formatting accuracy in HTML exports and more. 

### New Conversion Features

This version of Aspose.Cells for PHP via Java introduces functionalities to convert ranges to images (`Range.ToImage()`) and HTML (`Range.ToJson()`), expanding the output options of your PHP solutions.

### Handle Pivot Tables and Charts

We have added new features and enhancements in the latest PHP library release for pivot tables (`PivotTable.GetFields()`, `PivotField.ShowValuesAs()`) and deprecated properties with improved alternatives for better code maintainability.

### Bug Fixes

Several issues related to EMF image rendering (CELLSJAVA-45989), Wingdings font display (CELLSJAVA-45996), data loss in HTML conversion (CELLSJAVA-45997), and more have been fixed in the latest version.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for PHP via Java 24.6 Release Notes](https://releases.aspose.com/cells/php/release-notes/2024/aspose-cells-for-php-via-java-24-6-release-notes/).
