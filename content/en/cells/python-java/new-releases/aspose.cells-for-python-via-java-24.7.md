### Key Highlights

Great news for Python developers! Aspose.Cells for Python via Java 24.7 is released and it delivers a fresh wave of enhancements to excel your spreadsheet management capabilities across platforms. This release equips you to effortlessly create, edit, convert, and manipulate Excel files with precision and efficiency. 

### Text Handling:
- We have refined width and height measurements for Aptos Narrow font to ensure precise text rendering.
- Optimized `ASC` and `WIDECHAR` functions to ensure correct values are returned for specific characters.
- The spacing inconsistencies between text lines in text boxes when converting to PDF have been fixed in this Python spreadsheets API release.

### Charting:
- This version introduces options to customize chart data table appearance, including borders and background.
- Enjoy enhanced control over the doughnut chart's inner radius for precise visualization in this version of the Excel library.
- The chart data display feature during HTML conversion has been augmented.

### Performance:
- Fine-tuned formula calculation process to reduce processing time.
- Simplified image handling to prevent excessive image size during PDF conversion.


# Bug Fixes
- Corrected page break positioning when using Aptos Narrow font.
- Resolved NullPointerException occurring during HTML conversion.
- Addressed `ArrayIndexOutOfBoundsException` in `Workbook.calculateFormula()` method.
- Fixed "com.aspose.cells.CellsException: Invalid Power query formula definition" error during file merging.
- Other minor bug fixes to improve overall stability.

### Public API Changes

- Introduced `AbstractCalculationEngine.ForceRecalculate()` method for custom function recalculation control.
- Added `Picture.PlaceInCell()` method for precise image placement within cells.
- Expanded `Shape` properties with `IsDecorative` for shape identification.
- Enhanced JSON loading options with `JsonLoadOptions.KeptSchema`.
- Introduced `PivotTableCalculateOption` for flexible pivot table calculations.
- Extended `SlicerCacheItemSortType` with Natural enum for sorting control.
- Supported Azw3 file format with `SaveFormat.Azw3` and `EbookSaveOptions`.
- Deprecated obsolete methods and properties for API consistency.
- Added `Cells.GetCellsWithPlaceInCellPicture()` for efficient picture retrieval.
- Introduced `ConversionUtility` for file format conversions.
- Provided `GridJsWorkbook.JsonToStream()` and `GridJsWorkbook.JsonToStreamByUid()` for JSON output.
