### Key Highlights

Experience enhanced Excel data manipulation using Aspose.Cells for Python via Java 24.8 across different platforms with new functions for equation handling, pivot table management, and merged cell control. Optimize your workflows with faster file loading and improved data rendering.

### Major Updates

- **Equation Enhancements:** Precisely control equation elements with LowerLimit, UpperLimit, and Limit functions using the latest Python spreadsheet manipulation API release. Convert equations to LaTeX and MathML for seamless integration.
- **Comprehensive PDF conversion:** Enjoy accurate Excel to PDF conversions and include Excel pivot tables, row, and column labels while converting.
- **Pivot Table Refinements:** Aspose.Cells for Python via Java 24.8 allows effortlessly sorting pivot table fields and clear slicers for better data analysis. It offers insights into data sources with `PivotTable.SourceType`.
- **Merged Cell Mastery:** Customize merged cell behavior during row/column deletion with `MergedCellsShrinkType` within your cross-platform Excel processing apps.
- **Data Handling Improvements:** You can now prevent data loss with enhanced currency symbol preservation and accurate HTML rendering.
- **Performance Optimization:** Accelerate your projects with faster XLS file loading and optimized data display.

### Public API Changes

- Introduced `MergedCellsShrinkType` enum for merged cell shrinking strategies.
- Added `DeleteBlankOptions.MergedCellsShrinkType` property for merged cell processing.
- Included `EquationNodeType` enums for LowerLimit, UpperLimit, and Limit functions.
- Provided `EquationNode.ToLaTeX()` and `EquationNode.ToMathML()` methods for equation conversion.
- Added `SortOrder.Natural` enum to preserve original data order in `PivotTables`.
- Introduced `SaveOptions.EncryptDocumentProperties` property for encrypting document properties.
- Enhanced `PivotField` with `SortBy()` methods and `SortSetting` property for flexible sorting.
- Added `SlicerCollection.Clear()` method for clearing all slicers.
- Introduced `PivotTable.SourceType` property to indicate data source type.
- Deprecated `PivotItemCollection.ChangeitemsOrder()` and added `PivotItemCollection.SwapItem()` for item swapping.
- Added `HtmlSaveOptions.EncodeEntityAsCode` property for HTML character encoding.
- Introduced `JsonSaveOptions.Schemas` property for specifying JSON schema during Excel to JSON conversion.
- Added `Config.SkipInvisibleShapes` property for `GridJS` to skip invisible shapes.
