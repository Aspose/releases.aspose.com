### Improved Formula and Chart Rendering

Aspose.Cells for Python via Java now accurately calculates formulas and renders charts during Excel to PDF conversion.

### WORKDAY Formula Calculations
 
Using version 24.5 of the Excel API, developers can experience accurate calculations for the `WORKDAY` formula with holiday references.

### Enhanced Text Rendering

We have resolved the text rendering issues causing text offsets after the same line.

### Embedded Word Object Display

Boost Excel processing within your Python solutions as the partial data loss problem when saving files with embedded Word objects to PDF is now fixed.

### Gridline Thickness Control

Another important update in this Python API release is the gridlines rendering with the correct thickness when exporting to images.

### Font Extraction for HTML Export

With this version of Aspose.Cells for Python via Java, you can seamlessly extract font family information during HTML export from XLSX files.

### Public API Enhancements

Several improvements have been made to the public API, including new methods for range manipulation, unit handling, and advanced filtering.
