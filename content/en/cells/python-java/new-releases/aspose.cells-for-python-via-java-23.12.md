We bring some exciting new features and resolve several bugs for Python developers to enhance the spreadsheet processing experience in this release of Aspose.Cells for Python via Java.

### New Features

- Leverage the MAKEARRAY function for dynamic array creation. (CELLSJAVA-45737)
- Utilize JSON as a data source for smart markers. (CELLSJAVA-45705)
- Retrieve valuable "Style Includes" information. (CELLSJAVA-45750)

### Enhancements

- Improved performance for calculating shared formulas recursively. (CELLSJAVA-45759)

### Bug Fixes

- Addressed issues with `VLOOKUP` function calculation, IRR function results, and formula calculation loops. (CELLSJAVA-45707)
- Resolved errors in file data saving after formula calculation. (CELLSJAVA-45725)
- Enhanced rendering for charts, text, and images during various file conversions (XLSX to PNG, Excel to PDF, XLSX to HTML).
- Fixed problems with hidden cell visibility, image scaling, and font changes during conversions.

### Public API and Backwards Incompatible Changes

### Added API Members

We have added several methods and properties in this version. Some of these are:

- WorkbookDesigner.SetJsonDataSource() method
- Style.IsNumberFormatApplied property
- Style.IsAlignmentApplied property
- PptxSaveOptions.IgnoreHiddenRows property
- HtmlSaveOptions.IsJsBrowserCompatible property

### Obsolete API Members

We have made several methods and properties obsolete in this version. Some of these are:

- Axis.AxisLabels property is obsolete, use Axis.GetAxisTexts() method instead
- Cells.MergedCells property is obsolete, use Cells.GetMergedAreas() method instead
- Comment.GetCharacters() method is obsolete, use Comment.GetRichFormattings() method instead
- Shape.GetCharacters() method is obsolete, use Shape.GetRichFormattings() method instead
- PivotField.GetPivotFilters() method is obsolete, use PivotField.GetFilters() method instead


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Python via Java 23.12 Release Notes](https://releases.aspose.com/cells/python-java/release-notes/2023/aspose-cells-for-python-via-java-23-12-release-notes/).
