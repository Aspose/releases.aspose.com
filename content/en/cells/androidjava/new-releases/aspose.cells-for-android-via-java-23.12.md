### Key Highlights

### Expanded PDF Support

Aspose.Cells for Android via Java 23.12 lets you render spreadsheets to PDF format in versions 1.5 and 1.6 for wider compatibility.

### JSON Data Source Integration

Use the latest Excel processing library for Android to utilize JSON as a data source for smart markers and enable dynamic content populations within Excel documents.

### Optimized Formula Calculations

Developers can experience faster and more accurate calculations for shared formulas to improve overall spreadsheet performance.

### Color Palette Expansion

Access a static final Color for the default font color with Aspose.Cells for Android via Java v23.12 and simplify color management in your applications.

### Enhanced Style Information Retrieval

You can now effortlessly retrieve "Style Includes" information for in-depth style analysis.

### Formula Calculation Control

Determine whether to calculate formulas before saving to HTML using the enriched `HtmlSaveOptions.CalculateFormula` property.

### Top Bug Fixes

- Resolved issues with vertical text positioning and rendering.
- Corrected formula calculation errors and unexpected results.
- Improved chart rendering accuracy and appearance.
- Addressed cell data display, formatting, and merging problems in HTML output.
- Fixed pivot table refresh and calculation issues.
- Resolved various exceptions and errors during file operations.

### Public API and Backwards Incompatible Changes

- Introduced GlobalizationSettings.GetDefaultSheetName() to customize default sheet names.
- Added MsoFormatPicture properties for cropping pictures.
- Introduced Range.IsBlank() method to check for empty ranges.
- Added Range.AddHyperlink() method for creating hyperlinks within ranges.
- Added VbaModuleCollection.AddUserForm() method to add user forms to VBA projects.
- Modified FormatConditionType constant values.
- Added HtmlSaveOptions.CalculateFormula and HtmlSaveOptions.FormatDataIgnoreColumnWidth properties.
- Added CellsHelper.NeedQuoteInFormula() method for checking sheet name quotation requirements.
- Introduced new ErrorCheckType enums.
- Expanded PivotGlobalizationSettings methods for localized date and time text.
- Replaced PivotField.SxRng property with PivotField.GroupSettings.
- Modified PivotTable ungrouping and grouping methods.
- Introduced PivotFieldGroupType enum and related classes for pivot field grouping.
- Added ImageOrPrintOptions.EmfRenderSetting and XpsSaveOptions.EmfRenderSetting properties for EMF rendering settings.
- Expanded PdfCompliance enum to support PDF 1.4, 1.5, 1.6, and 1.7.
- Replaced Legend.LegendEntriesLabels with Legend.GetLegendLabels().
- Replaced Axis.AxisLabels with Axis.GetAxisTexts().
- Replaced Cells.MergedCells with Cells.GetMergedAreas().
- Replaced Comment.GetCharacters() with Comment.GetRichFormattings().
- Replaced Shape.GetCharacters() with Shape.GetRichFormattings().
- Replaced PivotField.GetPivotFilters() with PivotField.GetFilters().
- Replaced Range.Union() with Range.UnionRange().
- Added WorkbookDesigner.SetJsonDataSource() method for setting JSON data source for smart markers.
- Introduced Style properties to indicate applied formats.
- Added PptxSaveOptions.IgnoreHiddenRows and PptxSaveOptions.AdjustFontSizeForRowType properties.
- Added HtmlSaveOptions.IsJsBrowserCompatible and HtmlSaveOptions.IsMobileCompatible properties.
