### Key Highlights

### Enhanced Chart Precision

Aspose.Cells for Android via Java 23.9 allows you to experience more accurate and visually appealing chart rendering with improvements in label positioning, axis scaling, and overall chart appearance.

### Refined Formula Accuracy and Performance

Developers can benefit from refined formula calculations in the latest spreadsheet manipulation Android API release, including corrections for specific functions and optimizations for faster processing.

### Data Import and Export Refinements

Conveniently import XML data more effectively and export data to HTML with greater precision to ensure data integrity.

### New Features and Enhancements

- Users can retrieve button names and titles from ODS files for enhanced data extraction.
- Easily evaluate and enhance the rendering of HTML to Excel worksheets for improved data conversion accuracy.
- We have empowered the ability to parse XML with unescaped greater-than signs for broader XML compatibility.

### Top Bug Fixes

- Resolved formula calculation errors and inconsistencies.
- Corrected chart rendering issues, including label overlapping, axis inaccuracies, and legend positioning.
- Addressed data display and formatting errors in HTML output.
- Improved overall performance and stability.

### Public API and Backwards Incompatible Changes

- Modified Cells.StandardWidth behavior for consistent column width adjustments.
- Added FileFormatType.MicrosoftCabinet and FileFormatType.Rtf for file format detection.
- Introduced Shape.GetActualBox() method for retrieving shape dimensions.
- Expanded XmlLoadOptions for numeric and date conversion options.
- Added FormulaSettings.PreservePaddingSpaces for formula formatting preservation.
- Introduced Cells.DeleteRow() with update reference flag.
- Enhanced AbstractCalculationEngine and CalculationData for custom function parameter handling.
- Added Cell.InsertText() and Cell.Replace() methods for cell text manipulation.
- Introduced HtmlSaveOptions.IsIECompatible property for IE browser compatibility.
- Expanded CustomFunctionDefinition class for custom function definitions.
- Added Workbook.UpdateCustomFunctionDefinition() and FormulaParseOptions.CustomFunctionDefinition for custom function updates.
- Enhanced Worksheet.CalculateFormula() and Worksheet.CalculateArrayFormula() for custom formula calculations.
- Introduced Worksheet.ConvertFormulaReferenceStyle() for formula reference style conversion.
- Added CellValue class for representing cell values and types.
- Introduced ExportTableOptions.PreprocessExportedValue() for custom value preprocessing.
- Added Dispose() method for SheetRender and WorkbookRender for resource management.
- Added EbookLoadOptions and EbookSaveOptions for ebook file handling.
- Introduced GridWeb.DPI property for device DPI settings.
- Added FileFormatType.Epub, LoadFormat.Epub, and SaveFormat.Epub for EPUB file support.
- Introduced Chart.FilteredNSeries and Series.IsFiltered properties for chart filtering.
- Expanded CommentTitleType enum with Note and Reply options.
- Obsoleted CellsHelper.ConvertR1C1FormulaToA1() and ConvertA1FormulaToR1C1() methods.
- Added TableStyleCollection.DefaultTableStyleName and TableStyleCollection.DefaultPivotStyleName properties.
- Introduced WorksheetCollection.RefreshAll() method for refreshing all pivot tables and pivot charts.
- Added Validation.GetValue() method for retrieving validation results.
