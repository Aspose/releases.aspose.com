### Key Highlights

### Streamlined Data Source Integration

Leverage JSON as a data source for smart markers with Aspose.Cells for Android via Java 24.3, and enable dynamic content population within Excel documents.

### Enhanced Rendering Accuracy

Developers can experience improved rendering fidelity for shapes and text within images generated from Excel worksheets.

### Boosted Efficiency

Effortlessly reduce file size when exporting Excel worksheets to EMF images to optimize performance and storage utilization.

### Expanded Data Import

Aspose.Cells for Android via Java 24.3 allows importing data from CSV files with user-specified row and column limits, and duplicate headers for extended worksheets.

### Enhanced HTML Output

You can easily customize CSS styles for the HTML formatter and control the merging of empty cells using the latest Android library version.

### Other New Features and Enhancements

- Support for Jakarta Servlet in `GridWeb` Java is introduced.
- Highlighted sheet tabs in the output HTML for mobile devices.
- We have implemented the functionality of dialog box conversion to PDF documents.
- More timely response to terminate saving process when saving HTML is added.
- Support for setting default folders when saving file to HTML is now available.
- Show the first item of the date/time group pivot field if the field contains a blank item.
- Added the ability to reduce the file size of generated EMF image while exporting sheet to EMF image.
- Support for exporting Excel to PowerPoint as `View` or `Printing` is added.
- Easily calculate the custom position of plot area and axes in charts.
- Developers can get the original width of hidden columns in pixels.
- Null values in exported tables are now allowed.

### Top Bug Fixes

- Resolved issues with text inside shapes not displaying when converting Excel worksheets to images.
- Corrected formula calculation errors.
- Addressed issues with generated XLS file corruption due to linked source for chart's display unit label.
- Fixed inconsistent formula calculation results.
- Improved histogram chart rendering in XLSX to HTML conversion.
- Resolved data labels through conditions not being effective when converting to HTML.
- Fixed cell value editing in newly created sheets in GridWeb Java.
- Addressed scaling differences between Aspose.Cells and Excel-generated PDFs.
- Corrected extra columns in exported HTML output when Excel named range contains icons.
- Improved smart marker layout when setting line-by-line replacement to false.
- Prevented exception when a pivot field is grouped by quarters.
- Fixed invalid encoding issues when loading sample files.
- Corrected preview mode showing different values.
- Improved rendering of drawings and text for XLT file format in Excel to PDF.
- Resolved Excel conditional format duplicate value setting format errors after saving to HTML.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Cells for Android via Java 24.3 Release Notes](https://releases.aspose.com/cells/androidjava/release-notes/2024/aspose-cells-for-android-via-java-24-3-release-notes/).
