### Key Highlights

### Effortlessly Embed Images

Aspose.Cells for Android via Java 24.6 helps you directly insert pictures within cells, replicating the "Place in Cell" functionality from Microsoft Excel. 

### Enhanced Data Extraction

Easily retrieve the original column width for precise control over spreadsheet layout and empowered Excel manipulation on Android.

### Improved Image Handling

With the latest version of the Android spreadsheet processing library, you can seamlessly load various image formats supported by third-party image processing plugins.

### Top Bug Fixes

- Resolved issues with copying images and shapes between Excel files.
- Fixed errors in converting full-width characters to half-width characters during PDF conversion.
- Addressed dotted line rendering issues in charts saved as EMF images.
- Corrected row height calculations during HTML conversion.
- Improved rotated text rendering accuracy during HTML conversion.
- Fixed cell data positioning errors in HTML output.
- Resolved text overflow issues across columns in HTML conversion.
- Optimized performance of new Workbook() on HTML streams.
- Addressed pivot table filter selection and formula field removal problems.
- Corrected pivot table refresh rendering issues for charts.

### Public API and Backwards Incompatible Changes

- Introduced `SystemTimeInterruptMonitor` and `ThreadInterruptMonitor` classes for monitoring and interrupting time-consuming procedures.
- Added `DeleteBlankOptions` class to define blank cell checking behavior for deleting rows and columns.
- Removed obsolete `Worksheet.SparklineGroupCollection` property, replaced by `Worksheet.SparklineGroups`.
- Removed obsolete `Worksheet.SetBackground(byte[])` method, replaced by `Worksheet.BackgroundImage` property.
- Introduced `HtmlTableLoadOption.TableToListObject` and `HtmlTableLoadOptionCollection.TableToListObject` properties to indicate table-to-list object generation during import.
- Added `HtmlTableLoadOptionCollection.Add(â€¦)` method to add `HtmlTableLoadOption` to the list.
- Obsoleted `HtmlTableLoadOption.Name` property, replaced by `HtmlTableLoadOption.Id`.
- Introduced `HtmlSaveOptions.HideOverflowWrappedText` property to control overflow text hiding during HTML saving.
- Added `PdfSaveOptions.EmbedAttachments` property to control embedding of Ole objects as attachments during PDF conversion.
- Introduced `Cells.GetColumnOriginalWidthPoint()` method to retrieve the original width of a hidden column.
- Added `ExternalLink.PathType` property to get the relationship type of an external link.
- Introduced `FileFormatType.WebP` enum to support WebP image detection.
- Added `PivotTable.RefreshData(PivotTableRefreshOption)` method to customize pivot table refresh options.
- Introduced `Worksheet.RefreshData(PivotTableRefreshOption)` and `WorksheetCollection.RefreshData(PivotTableRefreshOption)` methods to customize pivot table refresh options at worksheet and workbook levels.
