# What's new in the Aspose.BarCode for Java 23.3

## Release Highlights

Aspose.BarCode for Java 23.3 delivers targeted reliability improvements. This update fixes a critical issue in DotCode generation and begins evaluation of a more robust Reed-Solomon error correction library for future enhancements.

## Enhancements

- **Reed-Solomon Library Evaluation**  
  Internal investigation initiated to assess replacement of the Reed-Solomon error correction mechanism, aiming for improved accuracy and performance in future releases.

## Bug Fixes

- **Unrecognized DotCode from BarcodeGenerator**  
  Fixed an issue where DotCode barcodes created via `BarcodeGenerator` could not be correctly recognized, restoring end-to-end support for this symbology.

## Public API and Backward Incompatible Changes

No changes to Public API or backward compatibility in this release.
