# What's new in the Aspose.BarCode for Java 23.12

## Release Highlights

Aspose.BarCode for Java 23.12 delivers significant performance improvements across core recognition routines. This release enhances support for Han Xin and QR symbologies and resolves a known issue with DotCode stability during automated testing.

## Enhancements

- **Han Xin Code Recognition Improved**  
  Upgraded decoding engine to better handle complex character patterns and regional encodings in Han Xin barcodes.

- **QR Code Accuracy Boosted**  
  Improved QR code detection algorithms result in higher recognition success rates under low contrast and rotated conditions.

- **Performance Optimizations**  
  Core recognition components now execute faster with reduced CPU usage, accelerating barcode processing in high-throughput environments.

## Bug Fixes

- **Unstable DotCode Test Resolved**  
  Investigated and fixed a rare test instability affecting DotCode recognition, improving test suite reliability.

## Public API and Backward Incompatible Changes

No changes to Public API or backward compatibility in this release.
