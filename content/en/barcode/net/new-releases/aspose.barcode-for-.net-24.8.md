#### Release Highlights

Aspose.BarCode for .NET 24.8 delivers precision control over postal barcodes and QR recognition. It adds support for PZN7 and PZN8 encoding/decoding, auto-calculates short bar heights for AustraliaPost and Postnet, and now captures QR version and error correction data from scanned codes.

### ‚ú≥Ô∏è New Feature

- **QR Version Detection**: Added `QRVersion`, `MicroQRVersion`, and `RectMicroQRVersion` properties in QRExtendedParameters for accurate decoding.
- **QR Error Level**: Access error correction level via `QRErrorLevel` in scanned results.
- **PZN7/PZN8 Encoding**: Supports automated checksum generation based on provided code text.

### üîß Enhancement

- **Postal Barcode Rendering**: Automatically calculates short bar height (0.26x for AustraliaPost, 0.5x for Postnet) if not manually defined.
- **Updated PZN Engine**: Improved encoder and decoder logic to align with pharmaceutical barcode specifications.

### üõ† Fix

- Fixed incorrect short bar rendering for AustraliaPost when no height is explicitly set.
- Resolved test failures under metered licensing scenarios.
- Addressed quality issues in QR, PZN, and Postal barcode regression coverage.
