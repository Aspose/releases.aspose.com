#### Release Highlights

Aspose.BarCode for C++ 25.2 for macOS enhances 1D barcode recognition speed and ensures uniform PDF output regardless of locale. Additional fixes improve barcode detection when handling rotated or flipped images.

#### New Features and Enhancements

- **1D Recognition Performance**:
  - Faster decoding and higher accuracy for key symbologies, especially **PatchCode** and **Pharmacode**.

- **Broad 1D Engine Upgrades**:
  - System-wide improvements in decoding logic and speed.

#### Fixes

- **PDF Encoder System Independence**:
  - Unified PDF barcode output across different macOS regional settings.

- **Barcode Region Accuracy**:
  - Improved region boundary detection when using rotated image sources.
