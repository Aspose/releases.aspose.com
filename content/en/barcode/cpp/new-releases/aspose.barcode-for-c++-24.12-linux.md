#### Release Highlights

Aspose.BarCode for C++ 24.12 for Linux enhances barcode generation and recognition through added QualitySettings, vector PDF support, and key bug fixes for symbology rendering and exception handling.

#### New Features and Enhancements

- **PDF Output Support**:
  - Generate vector barcodes as PDFs with experimental `BarCodeImageFormat::Pdf`.

- **Enhanced 1D Recognition**:
  - QualitySettings now supported by NBase and WBase processors.
  - Faster and more accurate reading of linear barcodes like GS1Code128, ITF14, and MSI.
