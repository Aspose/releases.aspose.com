#### Release Highlights

Aspose.BarCode for C++ 24.9 for Windows brings crucial stability and performance improvements. This release introduces thread-safe detection for QR code types and enhances AutoSizeMode behavior for better rendering control.

#### New Features and Enhancements

- **Thread-Safe Barcode Detection**:
  - QR, Micro QR, and Rectangular Micro QR barcode recognition is now thread-safe.
  - Improves stability and predictability on multicore Windows systems.

- **AutoSizeMode Fix**:
  - Corrected scaling behavior when rendering barcodes with automatic sizing.

#### Fixes

- **QRTargetDetector Bug Fix**:
  - Resolved exception thrown during QR code detection caused by a flawed sorting algorithm.
