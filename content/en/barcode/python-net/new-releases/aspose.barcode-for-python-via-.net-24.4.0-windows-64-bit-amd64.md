This release of Aspose.BarCode for Python via .NET 24.4.0 brings exciting enhancements for Windows 64-bit (x64) systems in the form of improved QR code reading and DataMatrix generation features.

### Unicode Character Detection in QR Codes

We have enhanced the QR code recognition functionality in the latest Python barcode API release. You can easily detect Unicode characters in the QR codes you read within your barcode processing applications on Windows 64-bit systems.

### Improved DataMatrix Barcode Recognition

Another important enhancement in this release of Aspose.BarCode for Python via .NET is the updated `DataMatrix` barcode generation capability with the new `QualitySettings` structure.

### Bug Fixes

Various bug fixes have also been added to this version of the Python API.
- Fixed Reed-Solomon error encoding for `DotCode` barcodes (BARCODENET-38962).
- `DotCode` is generated with an incorrect size (BARCODENET-38969).
- Fixed wide bars width of start-stop patterns for `Standard2of5` and `IATA2of5` encoders (BARCODENET-38961).
- QR code recognition test failed on .NET Core issue is fixed (BARCODENET-38963).
- Resolved the `BarCodeReader` detected unexpected `DecodeType` issue (BARCODENET-38943).


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.BarCode for Python via .NET 24.4.0 Release Notes](https://releases.aspose.com/barcode/python-net/release-notes/2024/aspose-barcode-for-python-via-dotnet-24-4-release-notes/).
