### Improved QR Code Recognition

In this release of Aspose.BarCode for Node.js via Java, we've updated recognizing QR codes featuring rounded targets and those with darkened or lightened right or bottom sides.

### Accurate PDF417 Recognition

We have fixed the issues causing errors with `PDF417` recognition in version 24.4 of the Node.js barcode API. 

### Enhanced Performance

Aspose.BarCode for Node.js via Java now operates at an optimized level, thanks to the performance improvements introduced in this release. Build feature-rich barcode processing applications with our Node.js library and seamlessly recognize and generate several types of barcodes.

### Greater Control over Aztec Barcodes

The new `QualitySettings` structure is now available for the Aztec barcode engine, which provides finer control over barcode generation.

### Simplified `MaxiCode` Usage

The `setPostalCode` method has been made public within `ComplexBarcode.MaxiCodeStructuredCodetext` to streamline `MaxiCode` integration.
