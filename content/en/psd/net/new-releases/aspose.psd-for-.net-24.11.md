#### Release Highlights

Aspose.PSD for .NET 24.11 introduces full Artboard layer support, enhanced control over FillSettings, and resolves key exceptions with vector bounds and AI file parsing.

### ‚ú≥Ô∏è New Feature

- **Artboard Layer API**: Added support for the `ArtboardLayer` type with access to dimensions, background color, and save functionality for each artboard separately.
- **Dynamic FillSettings Modification**: Introduced support for replacing and editing `FillSettings` (e.g., from gradient to solid fill) via public API.

- *(No enhancements in this release)*

### üõ† Fix

- **Unit Conversion Fix**: Resolved missing support for `UnitTypes.Millimeters` in vector origin bounds.
- **AI File Compatibility**: Added fallback handling for AI files with no OCG (layer) data.
- **FillLayer Resource Sync**: Fixed incorrect `FillSettings` refresh when modifying `SoCoResource`.
- **Crash-Free Load**: Ensured stability while reading PSD files containing malformed or partial gradient data.
