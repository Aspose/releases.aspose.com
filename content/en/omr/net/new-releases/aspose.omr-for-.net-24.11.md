#### Release Highlights

Aspose.OMR for .NET 24.11 brings powerful interactivity to forms with support for saving and recognizing machine-readable interactive PDFs, allowing users to fill forms electronically and developers to recognize them seamlessly.

### ‚ú≥Ô∏è New Feature

- **Interactive PDF Export**: Generate forms as fillable, machine-readable PDFs with checkboxes and input fields using `SaveAsInteractivePdf()`. Supports all standard OMR elements including grids, score groups, and barcodes.
- **Interactive PDF Recognition**: Aspose.OMR now recognizes both scanned and digitally filled interactive PDFs, unifying processing logic and improving flexibility for hybrid form collection workflows.

### üîß Enhancement

- **Bulk Form Interactivity**: Use `BatchGenerationResult.SaveAsInteractivePdf()` to create personalized interactive PDFs in batch processing mode.
- **Export Stream Support**: Export individual or personalized pages via stream using new `ExportPageInteractivePdf()` and `ExportFormInteractivePdf()` methods.

- No specific bug fixes in this release.
