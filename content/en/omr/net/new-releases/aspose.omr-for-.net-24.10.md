#### Release Highlights

Aspose.OMR for .NET 24.10 enhances form generation flexibility with support for hyphenating long words that exceed page width, while continuing the transition to a unified form generation API.

### ‚ú≥Ô∏è New Feature

- **Long Word Hyphenation**: Introduced `LongWordHandling.Hyphenation` in global page settings. This feature hyphenates lengthy words that extend beyond layout margins, ensuring better text wrapping and improved readability on generated forms.

### üîß Enhancement

- **Layout Flexibility**: Forms now better handle long unbroken text thanks to the new hyphenation setting, delivering cleaner output without manual layout fixes.
- **API Transition Support**: All legacy generation methods remain available but deprecated, easing the transition to the `OmrEngine.Generate()` universal method.

### üõ† Fix

- No specific bug fixes included in this release.

#### ‚ö†Ô∏è Deprecation Notice

The following APIs are deprecated and will be removed in version 25.1.0 (January 2025):
- `GenerateTemplate()`
- `GenerateJSONTemplate()`
- `GenerateJSONTemplateFromString()`

Continue to use them during the 2-month transition period.
