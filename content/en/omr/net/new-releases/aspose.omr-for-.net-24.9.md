#### Release Highlights

Aspose.OMR for .NET 24.9 introduces a streamlined universal form generation API, simplifying template rendering and future-proofing your projects by replacing multiple deprecated methods.

### ‚ú≥Ô∏è New Feature

- **Unified Form Generation API**: Added `OmrEngine.Generate()` to replace previous GenerateTemplate and GenerateJSONTemplate methods. It auto-detects the form's source format, making integration more intuitive and consistent.

### üîß Enhancement

- **Simplified Form Rendering**: The new API consolidates multiple legacy functions into a single, smart method, reducing complexity and improving developer productivity.

- No specific bug fixes included in this release.

#### ‚ö†Ô∏è Deprecation Notice

The following APIs are now deprecated and will be removed in version 25.1.0 (January 2025):

- `OmrEngine.GenerateJSONTemplate`

Your existing code will remain functional until the deprecation window ends in January 2025.
