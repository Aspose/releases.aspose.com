The latest release of Aspose.3D for Node.js via Java (version 24.6) brings significant performance upgrades and important bug fixes to the API. App developers can now experience faster Google Draco encoding and decoding and much more.

### Optimized Google Draco Encoding
Enhanced Google Draco encoding and decoding is a part of the newest Node.js 3D processing library. You can expect a noticeable performance boost while working with Google Draco compressed files and streamlining large-scale 3D data handling.

### Bug Fixes
- **GLB to FBX Conversion:** Resolved an issue where PBR materials lost their appearance during conversion to maintain visual consistency.
- **GLTF Redundant Materials:** Fixed the generation of redundant materials in GLTF output to reduce file size and improve model efficiency.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.3D for Node.js via Java 24.6 Release Notes](https://releases.aspose.com/3d/nodejs-java/release-notes/2024/aspose-3d-for-node-js-via-java-24-6-release-notes/).
