#### Release Highlights

Aspose.3D for .NET 24.9 introduces full support for the **3MF format**, including **material and texture export**, **thumbnail generation**, and **buildable part settings**. This release also includes **IFC file format import**, enhanced property metadata handling, and exposes internal axis system utilities to developers. Several legacy APIs have been deprecated for a more streamlined API design.

### ‚ú≥Ô∏è New Feature

- **3MF Export Support**: Export complete 3D scenes to Microsoft 3MF format using new APIs.
- **Texture & Material Export for 3MF**: Embed textures and materials during 3MF export.
- **IFC Import Support**: Load industry-standard Building Information Modeling (BIM) files.
- **Texture Import for 3MF**: Import 3MF models with textures seamlessly.

### üîß Enhancement

- **3MF Thumbnails**: Generate and embed thumbnail previews in exported 3MF files.
- **Expose Axis System Utilities**: Access internal axis system tools programmatically.
- **Improved Property Flags**: Added internal flags and extra data support for properties.

### üõ† Fix

- **Bug Fix (THREEDNET-1569)**: Implemented previously missing MetadataDecoder functionality.
