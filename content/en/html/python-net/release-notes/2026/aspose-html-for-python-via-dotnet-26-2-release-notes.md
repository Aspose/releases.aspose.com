This page contains release notes information for Aspose.HTML for Python via .NET 26.2.
{{% /alert %}}

As per the regular monthly update process of all APIs being offered by Aspose, we are honored to announce the February release of Aspose.HTML for Python via .NET.

This release includes several critical bug fixes and improvements. Key changes include fixing an infinite loop in table-in-table page break handling, resolving encoding detection issues , and addressing accessibility compliance with C28 rule fixes . Additionally,contains numerous rendering engine improvements across layout calculations for blocks, tables, flexbox, and grid layouts.
