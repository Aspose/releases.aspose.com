As per the regular monthly update process of all APIs being offered by Aspose, we are honored to announce the January release of Aspose.HTML for .NET.

### Release Notes

In this release, we have improved font processing when converting html to various formats. We have also improved the rendering of tables in HTML and their pagination. The public API for navigateAsync methods has also been changed, they will only be available starting with NETStandard 2.0.
