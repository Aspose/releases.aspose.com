## Release Summary

Aspose.HTML for .NET 23.8 is a significant release that includes a number of new features and bug fixes that make it a more powerful and reliable tool for C#, .NET App developers. The new `base64` encoding support, enhanced `HTML` and `CSS` parsing and rendering, and new Window object API can help developers to create more efficient, accurate, and reliable applications.

## Added API

- `IWindow`: A new interface that represents a window containing a DOM document. This interface includes the `Btoa()` and `Atob()` methods for converting content to and from the base64 encoding.

## Improved `base64` encoding support

Perform reliable and efficient base64 encoding and decoding via HTML API.

## Improved `HTML` and `CSS` parsing

More accurate and consistent parsing of HTML and CSS documents using HTML API.

## Improved `EPUB` to `PDF` Covnersion Support

- Fixed a bug that caused the `EPUB` to `PDF` conversion to produce an empty `PDF` file in some cases.
- Fixed a bug that caused the layout of the resultant `PDF` file to be incorrect in some cases.

## Improved `HTML` to `PDF` Covnersion Support

- Fixed a bug that caused the output `PDF` file to be incorrect in some cases.
- Fixed a bug that caused the `CSS` to not be retained upon conversion from `HTML` to `PDF` in some cases.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.HTML for .NET 23.6 Release Notes](https://releases.aspose.com/html/net/release-notes/2023/aspose-html-for-net-23-6-release-notes/).
