## Notable Performance Enhancements

Aspose.HTML for .NET 23.10 release of the HTML API, offers faster and more efficient functionality, thanks to the following improvements:

- Better attribute storage and utilization
- Increaseed HTML parsing speed
- Optimized data structures
- Reduced memory usage
- Fewer redundant object creations

## Accessibility Improvements

- The `AccessibilityRules` class has been updated with a new `GetRule(string ruleName)` method to make it easier to retrieve specific accessibility rules. 
- The `Validate` method of the `AccessibilityValidator` class has been fixed to improve the accuracy of accessibility validation.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.HTML for .NET 23.10 Release Notes](https://releases.aspose.com/html/net/release-notes/2023/aspose-html-for-net-23-10-release-notes/).
