This release of the HTML API offers the following new features to the C#, ASP.NET developers for incorporating into xtheir Apps:

## Improved work with Markdown

Easily convert Markdown documents to PDF files. This is useful for creating documentation, blog posts, and README files.

## Optimized memory usage and improved performance

Enables to convert large and complex HTML documents to PDF more quickly and efficiently.

## Javascript processing improved

Convert HTML documents that contain JavaScript code to PDF more accurately and reliably.

## Expanded the ability to convert HTML forms to PDF

Enahnced the ability to convert HTML forms to PDF.

## Bug fixes
- Fixed a number of bugs related to HTML to PDF conversion (ASCII text, Stack Overflow, Flatten HTML Form Fields), and HTML to Image conversion
- Improved the stability and performance of the API

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.HTML for .NET 23.1 Release Notes](https://releases.aspose.com/html/net/release-notes/2023/aspose-html-for-net-23-1-release-notes/).
