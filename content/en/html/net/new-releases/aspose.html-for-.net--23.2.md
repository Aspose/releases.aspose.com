## Improved `HTML` to `PNG` Layout Rendering

Improved layout rendering to ensure that HTML elements are positioned correctly.

## Improved `background-position` Rendering

Fixed an issue where the background-position CSS property was not having an effect.


## .NET Standard 2.0 Support
### Announcement

Starting from version 23.4, the Aspose HTML API will only be supporting .NET Standard 2.0.

### Impact to C#, .NET App Developers

Developers who are using the Aspose HTML API in .NET Framework applications will need to upgrade to .NET Standard 2.0 in order to continue using the API.
