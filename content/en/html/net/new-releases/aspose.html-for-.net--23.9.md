## `WCAG 2` Testing Support

The new `Aspose.Html.Accessibility` namespace provides a comprehensive API for testing `WCAG 2` (Web Content Accessibility Guidelines) requirements.

## Accurate `NOSCRIPT` Tag Treatment

Aspose.HTML now more accurately interprets the NOSCRIPT HTML tag, which can be used to provide alternative content for users who do not have JavaScript enabled.

## Bug Fixes:

- Fixed a bug that caused an infinite loop when converting HTML to PNG.
- Fixed a bug that caused HTML content to be cut off on a new page when converting from HTML to PDF.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.HTML for .NET 23.9 Release Notes](https://releases.aspose.com/html/net/release-notes/2023/aspose-html-for-net-23-9-release-notes/).
