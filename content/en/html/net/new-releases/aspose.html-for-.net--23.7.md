## Release Summary

The improvements in Aspose.HTML for .NET 23.7 make it easier for C#, .NET Apps developers to create and convert HTML documents. The improved `ECMAScript` 2020-2023 support can help developers to create more modern and efficient JavaScript code. Enhanced HTML conversion options to various formats. The bug fixes improve the overall stability and reliability of the Aspose.HTML API.

## Improved ECMAScript 2020-2023 support

This release of Aspose.HTML for .NET API offers an improved ECMAScript 2020-2023 support. It allows developers to use the latest JavaScript features in their web applications.

## Convert HTML to Popular Formats using API

An enhanced `Converter` class is provided that allows for converting HTML to other popular formats, such as PDF, XPS, image formats, etc.

The HTML to text conversion API can be useful for creating text-based versions of websites or for extracting the text content of HTML documents. It also provides various text saving options via `TextSaveOptions` class. It gives control to process the outside list markers of OL and UL elements.

## Improved HTML to PDF Conversion

Fixed a bug that caused the lower half of an HTML document to be cut off when converting it to PDF in some cases.
