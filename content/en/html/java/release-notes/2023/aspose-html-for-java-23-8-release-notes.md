We are honored to announce the Augusteber release of Aspose.HTML for.Java.

In this release, the functionality for working with MD format has been added.

## Public API and Backward Incompatible Changes

### Added  APIs

The functionality for working with MD format has located in the [**_Aspose.Html.Toolkit.Markdown_**](https://reference.aspose.com/html/java/com.aspose.html.toolkit.markdown.syntax/) namespace.
