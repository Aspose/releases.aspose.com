This page contains release notes for Aspose.HTML for Java 26.6

{{% /alert %}}

We are honored to announce the June release of Aspose.HTML for Java.

In this release, we focused on improving performance and stability by enhancing the library's core collection and string processing infrastructure.

Improved collection processing. Reworked the collection traversal logic by replacing recursive processing with an iterative approach.

Improved string processing. Corrected the implementation of character-based string processing.
