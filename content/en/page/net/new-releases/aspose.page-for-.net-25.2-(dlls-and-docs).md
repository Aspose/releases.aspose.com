#### Release Highlights

The DLLs-only release of Aspose.Page for .NET 25.2 provides enhanced control over font handling, introducing direct support for Type1 to TTF conversion and public API access to the font transformation logic.

### ‚ú≥Ô∏è New Feature

- **New Public API for Font Conversion**:
  - `PsDocument.ConvertType1FontToTTF(sourcePath, destinationPath)` lets developers convert embedded or standalone Type1 fonts into modern TTF format programmatically.
  
- **Automatic Font Conversion Option**:
  - New `SaveOptions.ConvertFontsToTTF` property allows enabling font conversion during document processing without manual intervention.

- **Extended Unicode Handling in XPS**:
  - Support for CMAP Table format 12 enhances XPS font mapping for large character sets and extended glyphs.
