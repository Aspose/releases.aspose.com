#### Release Highlights

Aspose.Page for .NET 25.2 brings major improvements in font handling and symbol rendering. Developers can now convert Type1 fonts to TTF during PS file processing, improving compatibility and text consistency in output files.

### ‚ú≥Ô∏è New Feature

- **Type1 to TTF Font Conversion**:
  - Conversion of Type1 fonts to TrueType (TTF) is now supported during PostScript processing for more consistent rendering across platforms.
  - A new public method `PsDocument.ConvertType1FontToTTF()` enables developers to convert fonts on demand.

- **SaveOptions API Extended**:
  - Added `SaveOptions.ConvertFontsToTTF` property to trigger automatic conversion of Type1 fonts to TTF in supported workflows.

- **XPS Support for CMAP Table Format 12**:
  - Improved Unicode compatibility in XPS rendering through support for CMAP Table format 12, enabling accurate character mapping for extended code points.
