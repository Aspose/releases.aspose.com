## **Improvements and Changes**

Aspose.Page for C++ 22.4 is is based on the latest [Aspose.Page for .NET 22.4](/page/net/release-notes/2022/aspose-page-for-net-22-4-release-notes/) and ready for use with Aspose C++ products ver 22.4.

- Support to set XMP metadata in EPS document was added.
- Support to merge multiple EPS/XPS documents to PDF was added.

[The developer guide on using merge.](https://docs.aspose.com/page/cpp/merge/)

You can consult Aspose.Page for C++ [API reference](https://reference.aspose.com/page/cpp/) for documentation.
