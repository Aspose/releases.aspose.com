Upgrade your XPS and PostScript manipulation solutions on macOS with Aspose.Page for C++ 24.1. This version includes utility methods to handle XPS (XML Paper Specification) documents and new font management functionality on Apple Macs.

### XPS Manipulation on macOS

This C++ API release optimizes XPS processing for Mac developers. It includes new utility methods offering finer control over XPS document structure, and optimizing document handling within your macOS applications.

### Advanced Font Management for Mac Apps

Take control of fonts with the introduction of the `BaseTrFont` class. You can easily access and manage font properties like type, encoding, character widths, and styles, ensuring exceptional text rendering in your macOS-based C++ applications.
