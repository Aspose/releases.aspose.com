Add improved document manipulation capabilities to your C++ apps with Aspose.Page for C++ 24.1. This release introduces utility methods for handling XPS (XML Paper Specification) and new font management features across Windows, macOS, and Linux platforms.

### XPS Manipulation Utilities

Easily streamline XPS handling with new utility methods introduced in the latest C++ XPS API release. These methods provide greater control over XPS document structure and help developers manipulate documents more efficiently.

### Enhanced Font Management

The `BaseTrFont` class added in this C++ library version offers effortless font management capabilities. Access and manage font properties like type, encoding, character widths, and styles for tailored text rendering within your C++ apps.
