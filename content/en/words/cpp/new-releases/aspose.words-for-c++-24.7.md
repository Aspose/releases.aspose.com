### Create Accessible PDFs

Aspose.Words for C++ 24.7 helps you generate documents in PDF/UA-2 format and offers an optimized user experience for individuals with disabilities.

### Seamless XLSX Export

Effortlessly export data with automatic date/time format detection in your XLSX files using the latest word-processing C++ API.

### ActiveX Control Mastery

Modify properties and interact with ActiveX objects like radio buttons and checkboxes within your Word document manipulation applications for dynamic functionality.

### Broader Markdown Control

This release of Aspose.Words for C++ lets you tailor hyperlink export behavior in Markdown format with the `LinkExportMode` property.

### Fine-tuned Chart Data Labels

Refine data label orientation and rotation inside your C++ applications for visually compelling presentations.

### VBA Project Protection Check

Conveniently verify VBA project protection status using the new `VbaProject.IsProtected` property and boost your document processing solutions.

### Custom List Level Numbering

Define custom number styling for list levels with Aspose.Words for C++ 24.7 using the `ListLevel.CustomNumberStyleFormat` setter.

### Limitations and API Differences

This release of the MS Word documents C++ library does not support:
- Metered license.
- LINQ and Reporting features.
- OpenGL 3D Shapes rendering.
- Loading PDF documents.

This release includes limited support for: 
- Database features - C++ doesn’t have a common API for DB like .NET `System.Data`.

This release includes support for: 
- Microsoft Visual C++ version 2017 or higher.
- Clang 3.9.1 or higher on Linux and only for the x86_x64 platform.
- MacOS Big Sur or later (11.5+) for 64-bit Intel Mac platform.
