This release offers a variety of improvements for C++ developers to streamline their Word document processing tasks. Empower your C++ apps and take them to the next level with Aspose.Words for C++ version 23.11. 

### Key Highlights

- **Revision Management:** Enhanced control over revision acceptance and rejection using the newly added `Accept` and `Reject` public methods to the [`RevisionCollection`](https://reference.aspose.com/words/cpp/aspose.words/revisioncollection/) class. Additionally, a new public interface `IRevisionCriteria` is made available.
- **Chart Legend Customization:** Experience optimized chart presentation consistency by applying default font formatting to all chart legend entries effortlessly with the newly introduced `ChartLegend.Font` property.
- **Flexible XLSX Section Handling:** Streamline document organization within spreadsheets by choosing between saving each document section in a separate worksheet or combining all sections into a single worksheet using the `XlsxSaveOptions.SectionMode` property. Another new addition is the enum `XlsxSectionMode`. 
- **Retrieve Stripped-down XML Content:**  With the new public property `WordOpenXMLMinimal` added to the `StructuredDocumentTagRangeStart` nodes, it is possible to retrieve minimal XML content in `FlatOpc` format.
- **Bug Fixes and Improvements:** Developers can leverage a variety of bug fixes and enhancements to ensure optimal performance and address reported issues.

Please download the Aspose.Words for C++ 23.11 version release today and upgrade your C++ solutions with the latest features.
