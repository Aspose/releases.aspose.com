#### Release Highlights

Aspose.Words for C++ 24.11 brings groundbreaking integration with AI summarization models (OpenAI and Google), advanced chart formatting control, PDF border rendering options, and enhanced document metadata access. This release includes 90+ improvements, reinforcing performance and cross-platform reliability across Windows, Linux, and macOS environments.

#### New Feature

- **AI-Powered Text Summarization**: Integrates OpenAI and Google generative models to provide intelligent document summarization features.
- **Chart Data Formatting**: Added ability to get and set **format codes** for chart data values.
- **PDF Form Field Borders**: New option to control **choice form field border visibility** in PDF output.
- **Extended Document Properties**: Exposes **extended document metadata** for structured analysis and reporting.
- **ActiveX Captioning**: Ability to **set custom captions** on ActiveX CommandButton controls.
- **Histogram Chart Improvements**: Now supports setting **bin count and label configuration** for histogram charts.
- **Logical Structure in PDF**: Table titles can now be exported as **logical structure element titles** in accessible PDFs.

#### Enhancement

- **PDF Export Quality**: Improved rendering of line charts, OfficeMath, rotated shapes, and shape alignment across document sections.
- **Metadata & TOC Handling**: More consistent field updates and better handling of bookmarks, captions, and document properties.
- **ActiveX & SDT Handling**: Enhanced control over ActiveX components and structured document tag (SDT) rendering.
- **Performance & Stability**: Reduced memory allocation and improved error handling for large files and malformed content.

#### Cross-Platform Notes (macOS and Linux)

- **macOS Support**: Continues support for **macOS Monterey 12.0+** on **64-bit Intel platforms**, ensuring full export, rendering, and metadata integration across formats.
- **Linux Support**: Compatible with **Clang 3.9.1 or newer**, built for **x86_x64 architecture**, with robust performance on server environments.
- Limitations: **LINQ Reporting**, **PDF document loading**, and **Metered License** functionality are still not available in the C++ variant.

#### Fix

- Resolved numerous issues affecting document rendering, field updates, TOC formatting, and chart data in PDF, JPEG, and PNG exports.
- Fixed misplaced content, missing shapes, incorrect text wrapping, and shape overlaps during layout rendering.
- Corrected encoding and font substitution problems for Chinese, Thai, French, and Big5 character sets.
- Eliminated application crashes (`IndexOutOfRangeException`, `FileCorruptedException`, `InvalidOperationException`) across document load, save, and page extraction workflows.
- Addressed missing bookmarks, inconsistent revision tracking, broken hyperlinks, and invisible elements in DOCX, RTF, and MHTML conversions.
