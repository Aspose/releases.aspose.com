#### Release Highlights

Aspose.Words for .NET 24.12 (MSI) introduces advanced AI document translation, customizable chart label positioning, and a powerful suite of new low-code APIs for comparison, mail merge, report building, and watermarking.

- **AI-Powered Document Translation**

- **Data Label Customization**  
  Precisely control label positioning for charts with new properties and modes, including doughnut chart label placement.

- **Low-Code API Suite**  
  New `LowCode.*` classes (Comparer, MailMerger, Replacer, ReportBuilder, Splitter, Watermarker) enable developers to perform complex document tasks with minimal code.

- **Mail Merge Refinement**  
  `RemoveEmptyTables` option enhances document cleanliness by automatically removing unused tables post-merge.
