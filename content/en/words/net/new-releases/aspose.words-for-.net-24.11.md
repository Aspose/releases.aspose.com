#### Release Highlights

Aspose.Words for .NET 24.11 (MSI) introduces OpenAI/Google-powered document summarization, new chart formatting options, extended document property access, and refined ActiveX and PDF export support.

### ‚ú≥Ô∏è New Features

- **AI-Powered Text Summarization**  
  Summarize documents using OpenAI or Google generative models with fine-grained length control.

- **Chart Data Formatting**  
  Apply number, date, and custom format codes to X, Y, and bubble chart values.

- **PDF Export Enhancements**  
  Control rendering of PDF choice form field borders using `RenderChoiceFormFieldBorder`.

- **Extended Document Properties**  
  Access `ScaleCrop`, `SharedDocument`, and `HyperlinksChanged` via `BuiltInDocumentProperties`.

- **Histogram Enhancements**  
  Customize the number of bins and labels for histogram charts.

### üõ† Public API Updates

- `ChartXValueCollection.FormatCode`, `ChartYValueCollection.FormatCode`, and `BubbleSizeCollection.FormatCode` for data formatting.
- `PdfSaveOptions.RenderChoiceFormFieldBorder` for form appearance control.
- New `AiModel`, `SummarizeOptions`, `SummaryLength` for generative summarization.
- `Forms2OleControl.Caption` now has a public setter.
- `RevisionOptions.InsertCellColor` and `DeleteCellColor` added with new `RevisionColor` presets.
