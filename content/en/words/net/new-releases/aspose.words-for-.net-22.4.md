# Read PhotoShop&reg; Metadata Resolution in JPEG

Support for reading the Photoshop&reg; metadata resolution in `JPEG` format images.

# Saving to PDF/A-4 Support

This version of the API has introduced the support of saving to PDF/A-4 which is based on ISO 32000-2 (PDF 2.0). This is great for long-term archiving of engineering specific documents.

# Handling of Chart Legend Entries

Feature has been added into API to manipulate and work with DrawingML chart legend entries.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.Words for .NET 22.4 Release Notes](https://releases.aspose.com/words/net/release-notes/2022/aspose-words-for-net-22-4-release-notes/).
