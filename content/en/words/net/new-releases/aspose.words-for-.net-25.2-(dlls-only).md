#### Release Highlights

The DLLs-only edition of Aspose.Words for .NET 25.2 introduces AI-generated summaries with Anthropic models, support for Microsoft Works documents, and enhanced typography with font number spacing.

### ‚ú≥Ô∏è New Features

- **Anthropic-Powered Summarization**  
  Generate summaries with Claude 3.x series models via the new `IAiModelText.Summarize()` API.

- **Support for .WPS / MS Works Format**  
  New parser introduced to detect and read MS Works documents using `LoadFormat.MsWorks`.

- **Single-Level List API**  
  `AddSingleLevelList()` simplifies list creation based on predefined styles.

- **Typographic Enhancements**  
  Customize numeral spacing with `Font.NumberSpacing`, enabling precise layout control for documents.

- Added: `AddSingleLevelList()` in `ListCollection`  
- New: `Font.NumberSpacing` for typography tuning  
- New: `IAiModelText.Summarize()` via Claude 3 models  
- New: `LoadFormat.MsWorks`, `ApplicationType.MsWorks`
