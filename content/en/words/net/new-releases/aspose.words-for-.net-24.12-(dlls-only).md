#### Release Highlights

Aspose.Words for .NET 24.12 (DLLs-only) delivers major enhancements with low-code API support for comparison, mail merge, watermarking, and reporting‚Äîalongside flexible chart data label positioning and AI-based translation capabilities.

- **Low-Code Document Automation**

- **Google AI Translation Support**  
  Easily translate documents via the `Translate()` method with support for multiple target languages.

- **Advanced Chart Label Placement**  
  Gain full control over label positions with new `Position`, `Top`, `Left` and layout mode properties.

  Remove empty tables automatically using `MailMergeCleanupOptions.RemoveEmptyTables`.
