#### Release Highlights

Aspose.Words for .NET 24.9 (MSI) brings advanced digital signing with XAdES-EPES, support for underline formatting in Markdown, grouped shape insertion, and a wide array of public API enhancements for developers building rich .NET document automation apps.

### ‚ú≥Ô∏è New Features

- **XAdES-EPES Digital Signatures**  
  Sign Word documents using XAdES-EPES level XML-DSig, enhancing compliance with advanced electronic signature standards.

- **Markdown Underline Formatting Support**  
  New option to interpret `++text++` as underline formatting during Markdown document import.

- **Group Shape Insertion**  
  Easily group multiple shapes via new `InsertGroupShape` methods in `DocumentBuilder`.

- **StructuredDocumentTag Insertion API**  
  Insert structured content controls like PlainText, Date, ComboBox using the `InsertStructuredDocumentTag()` method.

- **Footnote/Endnote Separator Access**  
  New classes to manipulate separator lines for footnotes and endnotes programmatically.

### üõ† Fixes & API Changes

- **ContextTableFormatting**  
  New option in `DocumentBuilderOptions` to isolate formatting changes applied within tables.

- **Removed Obsolete ChartAxis Properties**  
  Deprecated label positioning APIs removed in favor of `TickLabels` object properties.

- **Refined Public API Structure**  
  Streamlined formatting, shape handling, and digital signature classes to reflect modern usage scenarios and standards.
