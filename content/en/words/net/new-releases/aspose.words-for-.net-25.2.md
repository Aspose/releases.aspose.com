#### Release Highlights

Aspose.Words for .NET 25.2 (MSI) delivers robust list APIs, AI-driven text summarization using Anthropic models, support for Microsoft Works documents, and improved font formatting.

### ‚ú≥Ô∏è New Features

- **AI-Based Summarization**  
  Summarize documents using Anthropic Claude models with new `IAiModelText.Summarize` method support.

- **Microsoft Works Format Support**  
  Documents in Microsoft Works format can now be parsed and loaded using `LoadFormat.MsWorks`.

- **Simplified List Creation**  
  Added `AddSingleLevelList()` API to make single-level list generation easier and template-driven.

- **Enhanced Font Formatting**  
  Introduced `Font.NumberSpacing` for proportional or tabular number spacing in modern Word output.

### üõ† Public API Changes

- `ListCollection.AddSingleLevelList(ListTemplate)` for simplified list creation  
- `Font.NumberSpacing` for fine-grained typographic control  
- `IAiModelText.Summarize()` for AI-based content summarization  
- New enum support: `LoadFormat.MsWorks`, `ApplicationType.MsWorks`
