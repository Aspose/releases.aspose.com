# AZW3 (Kindle KF8) Format Loading Support

This version of the API introduces the loading support for the `AZW3` eBook format. It is also known as Kindle Format 8 or `KF8` format. So now the Word Processing API has been extended to offer the functionality to load the digital books.

# Fetch Custom Field Data via API

An API feature has been added to get custom field data that is associated with the field.

# Enable Saving Progress Notifications

The support of the `SaveOptions.ProgressCallback` has been extended to enable saving progress notifications for numerous HTML-based and XAML-based flow formats, such as; `HTML`, `MHTML`, `EPUB`, `XamlFlow`, and `XamlFlowPack`.

# Shapes to SVG Image Conversion

This API release offers an ability to convert Shapes to SVG images upon export to `HTML`, `MHTML` or `EPUB`.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.Words for .NET 22.2 Release Notes](https://releases.aspose.com/words/net/release-notes/2022/aspose-words-for-net-22-2-release-notes/).
