#### Release Highlights

Aspose.Words for .NET 24.10 (MSI) delivers powerful document enhancements including ActiveX CommandButton controls, shape visibility toggles, grouped shape insertion, Markdown HTML table export, and new Pie/Doughnut chart properties.

### ‚ú≥Ô∏è New Features

- **ActiveX Control Creation**  
  Insert `CommandButtonControl` using `DocumentBuilder.InsertForms2OleControl()` for macro actions.

- **Shape Visibility Property**  
  Use the new `Shape.Hidden` property to dynamically hide or show shapes.

- **Shape Grouping Enhancements**  
  Combine multiple shapes or group shapes and `GroupShape` elements with new overloads of `InsertGroupShape()`.

- **Markdown HTML Table Export**  
  Export tables as raw HTML in Markdown using `MarkdownSaveOptions.ExportAsHtml`.

- **Chart Customization**  
  Customize chart visuals with properties like `FirstSliceAngle`, `DoughnutHoleSize`, and `SecondSectionSize` in `ChartSeriesGroup`.

### üõ† API Changes and Enhancements

- New methods for inserting StructuredDocumentTags and grouped shapes.
- Added support for `XmlDsigLevel` in digital signature handling.
- Public access to footnote and endnote separators via `FootnoteSeparatorCollection`.
- Deprecated and replaced several legacy `ChartAxis` properties for modern compatibility.
