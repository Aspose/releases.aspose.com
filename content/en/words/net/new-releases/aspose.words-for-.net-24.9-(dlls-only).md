#### Release Highlights

Aspose.Words for .NET 24.9 (DLLs-only) delivers key feature upgrades for .NET developers, including grouped shape insertion, XAdES-EPES digital signatures, underline formatting in Markdown imports, and extensive public API improvements.

### ‚ú≥Ô∏è New Features

- **InsertGroupShape API**  
  Easily group shapes using new `InsertGroupShape()` overloads in `DocumentBuilder`.

- **XAdES-EPES Signature Level**  
  Introduced `XmlDsigLevel.XAdEsEpes` support in the digital signing workflow for advanced document compliance.

- **Markdown `++underline++` Recognition**  
  Toggle parsing of underline formatting in Markdown using `MarkdownLoadOptions.ImportUnderlineFormatting`.

- **Content Control (StructuredDocumentTag) Insertion**  
  Add content controls dynamically via the `InsertStructuredDocumentTag()` method.

- **Access to Footnote/Endnote Separators**  
  Programmatically manipulate separator styles and behaviors through new public classes.

### üõ† Fixes & API Changes

- **Chart Axis Obsolete Properties Removed**  
  Cleanup of outdated tick label properties in favor of `TickLabels`-based configuration.

- **New ContextTableFormatting Option**  
  Improved formatting behavior control in tables via `DocumentBuilderOptions.ContextTableFormatting`.

- **Expanded and Modernized Public API**  
  Enhanced consistency and clarity across shape, signature, and rendering APIs.
