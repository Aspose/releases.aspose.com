## **Improvements and Changes**

 - Increased smoothness for small paths processed by GraphicsPath.Flatten()(DRAWINGNET-1645)
 - Improved LockBits performance for True Color formats (DRAWINGNET-1642)
 - Corrected Bitmap.SetPixel()/GetPixel() implementation for True Color formats (DRAWINGNET-1642)
 - Implemented bitmap creation from a metafile (DRAWINGNET-1647)
 - Improved LockBits CMYK conversions correctness and performance (DRAWINGNET-1649)
