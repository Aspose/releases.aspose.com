## **Improvements and Changes**
- Implemented ImageAttributes.SetOutputChannelColorProfile() for RGB and CMYK color profiles (DRAWINGNET-985)
- Improved Chinese fonts support (DRAWINGNET-403)
- Implemented vector scaling for metafiles (DRAWINGNET-878)
- Corrected EMR_STRETCHDIBITS processing in EMF (DRAWINGNET-1092)
- Improved rectangle frame drawing correctness (DRAWINGNET-1088)
