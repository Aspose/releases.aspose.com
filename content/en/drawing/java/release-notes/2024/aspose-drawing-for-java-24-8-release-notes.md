## **Improvements and Changes**

 - Added support for RebeccaPurple color (DRAWINGNET-1427)
 - Enabled support for Macintosh font files (DRAWINGNET-1578)
 - Moved KnownColor and SystemColors from System.Drawing to System.Drawing.AsposeDrawing namespace for .NET Standard build (DRAWINGNET-1427)
 - Improved correctness of image file codecs (DRAWINGNET-1573)
 - Fixed stream ObjectDisposedException after saving TIFF (DRAWINGNET-1573)
