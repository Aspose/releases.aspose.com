## **Major Features**
Several improvements and features were added this month. The most notable are:

- Added support for LinearGradientBrush Clamp mode (DRAWINGNET-1604)
- Improved LockBits performance without format conversion for 1bpp and 8bpp indexed bitmaps (DRAWINGNET-1607)
- Improved correctness of image file codecs (DRAWINGNET-1596)
- Fixed container transformations (DRAWINGNET-1434)
- Fixed a frame selection hang on a TIFF file (DRAWINGNET-1596)
- Fixed OverflowException when drawing a large Bitmap with HighQualityBicubic interpolation mode (DRAWINGNET-1509)
