## **Major Features**
Several improvements and features were added this month. The most notable are:

- Added support for saving BMP in 24 bpp format (DRAWINGNET-1383)
- Improved Windows font fallback (DRAWINGNET-1397)
- Corrected transformation offset for text decorations (DRAWINGNET-1399)
- Corrected 1bpp bitmap palette loading (DRAWINGNET-1402)
- Corrected PathGradientBrush drawing in CompositingMode.SourceCopy mode (DRAWINGNET-1398)
- Corrected pixels encoding in UnlockBits for Format4bppIndexed (DRAWINGNET-1396)
