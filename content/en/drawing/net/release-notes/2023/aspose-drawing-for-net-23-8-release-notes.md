## **Major Features**
Several improvements and features were added this month. The most notable are:

- Implemented GraphicsPath.IsVisible(...) (DRAWINGNET-1367)
- Implemented experimental support for vector regions (DRAWINGNET-1354)
- Added support for loading JPEG resolution from the jfif segment (DRAWINGNET-1368)
- Improved PNG saving performance (DRAWINGNET-1363)
- Improved PathGradientBrush drawing performance (DRAWINGNET-1370)
- Fixed incorrect PathGradientBrush drawing for paths with zero bounds and paths represented by a line(DRAWINGNET-1369)
