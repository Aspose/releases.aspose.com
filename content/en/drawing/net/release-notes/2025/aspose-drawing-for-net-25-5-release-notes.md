This page contains release notes information for Aspose.Drawing for .NET 25.5.

{{% /alert %}} 
## **Major Features**
Several improvements and features were added this month. The most notable are:

- Fixed a color channel overflow in LinearGradientBrush drawing (DRAWINGNET-1620)
- Improved LockBits performance for Format32bppPArgb (DRAWINGNET-1631)

Support for .NET Framework v2.0 is deprecated and will be removed in 1 month. The minimum supported version will be .NET Framework v4.0.

## **Got a Query?**
Have any questions or queries about the API? Please feel free to reach us on [Aspose.Drawing support forum](https://forum.aspose.com/c/drawing) where our technical team will be more than happy to assist you further.
