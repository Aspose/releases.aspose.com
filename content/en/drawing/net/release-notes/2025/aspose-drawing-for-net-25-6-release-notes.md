## **Major Features**
Several improvements and features were added this month. The most notable are:

- Improved Region.Intersect performance (DRAWINGNET-1529, DRAWINGNET-1624)
- Corrected Region creation from RegionData (DRAWINGNET-1638)
- Added an alternative public library initialization method (DRAWINGNET-1636)
- Cleaned up XML documentation (DRAWINGNET-1637)
- Removed .NET Framework 2.0 support (DRAWINGNET-1635)
