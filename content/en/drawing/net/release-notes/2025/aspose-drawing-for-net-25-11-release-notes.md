## **Major Features**
Several improvements and features were added this month. The most notable are:

- Improved LockBits conversion performance (DRAWINGNET-1655)
- Improved image file codecs correctness and performance (DRAWINGNET-1656)
- Corrected geometry angle calculation (DRAWINGNET-1658)
