Aspose.CAD for .NET 24.7 (DLLs package) offers a flexible and efficient way to work with CAD files in your .NET projects across Windows, Linux, and macOS platforms. This version includes enhancements that improve performance, accuracy, and compatibility.

### STEP File Support

The latest release of the C# CAD processing API introduces enhanced support for STEP file format, a widely used format for exchanging 3D models.

### Scaling Improvements

Utilize more accurate scaling of line types with SHX symbols for impeccable representations in your cross-platform C# and VB.NET applications.

### CAD Conversion Enhancements

Easily convert between CAD formats with supercharged algorithms and finer handling of missing logos and line thicknesses.

### Smooth DXF and DWG Export

Aspose.CAD for .NET 24.7 delivers unified options for saving DXF and DWG files and streamlines your workflow.

### Other Highlights

### STEP File Optimizations:
- Improved loading and handling of STP files.
- Upgraded header properties for STP files.
- Fine-tuned reading and writing properties/handles for entities and objects.

### Advanced Exporting:
- Export PDFs with tables intact.
- Updated multiline text (SHX) rendering.
- Incorporated the capability to save colors into STP files.

### Accurate Conversions and Rendering:
- Enriched STP to PDF and STP to STP conversions.
- Better color parsing and export for STP files.
- Refined the conversion time of CadMText objects into APS objects.

### Elevated Compatibility and Performance:
- Introduced support for the DRC format.
- Enhanced DWG to DXF conversions that produced files that were difficult to open.
- Augmented support for unsupported color policies in CTB files.
