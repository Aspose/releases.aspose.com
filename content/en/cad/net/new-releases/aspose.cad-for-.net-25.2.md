#### Release Highlights

Aspose.CAD for .NET 25.2 (MSI) introduces a major upgrade in CAD processing capabilities including **simultaneous rendering of polylines and filled figures**, optimized **DWG to SVG color output**, and parsing of **composite and B-Spline surfaces**. New stereo vision APIs are now supported for point cloud generation, epipolar geometry, rectification, and 3D reconstruction. 

### ‚ú≥Ô∏è New Feature

- **Stereo Vision Toolkit**  
  Introduced a stereo image processing pipeline with the ability to compute essential/fundamental matrices, rectification, disparity maps, and 3D point clouds. [CADNET-9400, CADNET-9401, CADNET-9402, CADNET-9399, CADNET-9398]

- **Annotation API Support**  
  Allows adding and manipulating annotations in CAD formats. [CADNET-9083]

- **Support for Polyline + Fill Rendering**  
  Enables objects with both polyline outlines and fill regions to render simultaneously. [CADNET-10040]

- **AcDb:AcDsPrototype_1b Section Parser**  
  Parser for a new DWG internal section added. [CADNET-9851]

- **B-Spline Surface Reading**  
  Surfaces defined using B-splines are now fully readable and convertible. [CADNET-9921]

- **DWG to SVG Color Output Correction**  
  Ensures fidelity of colors post-export. [CADNET-10025]

- **DWG to DXF Export File Fix**  
  Eliminates structural issues in DXF output. [CADNET-9214]

- **Faster JPEG Export**  
  Significant performance boost for image generation. [CADNET-9580]

- **Improved PDF 3D Viewer Compatibility**  
  Fixes object misalignment across PDF viewers post-conversion. [CADNET-9552, CADNET-9377]

- **Improved DWG to 3DS Conversion Output**  
  Mesh visibility and integrity preserved. [CADNET-9559]

- **Updated IFC Component Reference**  
  Reflects updated documentation and object handling for IFC formats. [CADNET-9144]

- **Internal Parser Improvements**  
  Composite object parsing and modular project refactoring for U3D/3DS improves developer experience and internal logic. [CADNET-9961, CADNET-9962, CADNET-9963]
