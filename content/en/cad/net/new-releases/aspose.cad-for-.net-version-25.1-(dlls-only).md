#### Release Highlights

Aspose.CAD for .NET 25.1 (DLLs-only) enables direct support for **PostScript file import**, blind watermark embedding/removal, SVG layer extraction, and **optimized rendering of DWG, DWF, GLTF/GLB, STP, and OBJ**. Developers integrating directly via DLLs benefit from performance boosts and improved cross-format fidelity.

- **Embed/Extract Watermarks**  
  Blind watermark support now spans GLTF, GLB, PDF, DWG. [CADNET-9406, CADNET-9624, CADNET-9808]

- **Read Layers from SVG**  
  New support for extracting layer data from SVG drawings. [CADNET-9933]

- **Douglas-Peucker Simplification**  
  Vertex grouping added for smoother line approximation. [CADNET-9404]
