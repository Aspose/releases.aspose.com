Build high-performance CAD processing applications with Aspose.CAD for .NET 24.3 (MSI installer). It brings exciting enhancements for developers working with CAD drawings.

### Enhanced DWG/DXF Processing

Developers can experience improved DWG R13 reading, accurate handling of specific IFC files, and precise width and height calculations with the C# CAD API.

### Expanded Conversion Capabilities

Upgrade your CAD conversion solutions and convert DWG files to DXF with linearized text and ensure correct file opening after conversion.

### Improved Rendering & Export

With Aspose.CAD for .NET 24.3, you can benefit from optimized hatch export performance, accurate color preservation during DWF to PDF conversion, and efficient image loading for DWG files.

### Watermark Support

We have implemented blind watermark support for CGM and FBX formats for enhanced document security in this C# API update.

### Other Improvements

Several enhancements have been added for object sorting order, DICOM export, layout detection, and more.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.CAD for .NET 24.3 Release Notes](https://releases.aspose.com/cad/net/release-notes/2024/aspose-cad-for-net-24-3-release-notes/).
