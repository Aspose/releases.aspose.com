# Support for Big Font

Added the support for the Big Font in the API.

# Memory Efficient DWG to PDF Conversion

Previously when DWG was converted to the PDF format it used to eat a lot of memory. This issue has been resolved now. Now you may convert DWG to PDF with improved results.

# Improved DAE to OBJ Export

In some previous instances the DAE to OBJ export failed. This issue has now been resolved in the API.
