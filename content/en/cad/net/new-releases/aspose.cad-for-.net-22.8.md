# Preserve 3D Models (Wired)

Ability to preserve the 3D wired models.

# Render Double Side Planes

Ability to render double-side planes.

# Scale Up Watermarks

Added the support for scaling up of watermark.

# Improved DWG to PDF Conversion

It was observed that in some instances the `DWG` was not fully being rendered when converted to the `PDF` format. This release of the API has resolved the aforementioned issue.
