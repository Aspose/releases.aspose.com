#### Release Highlights

Aspose.CAD for .NET 25.4 (MSI) enhances the **STP to DWG pipeline**, introduces **DGN text extraction**, and adds parsing for **path construction operators** to support advanced 2D graphic workflows. Multiple stability and interoperability improvements are included for DWG to PDF, GLB to GLTF, and DXF transformations.

- **STP to DWG Conversion Fixes**  
  - Resolved lexer and surface type errors for improved STEP compatibility. [CADNET-10073]  
  - Fixed geometry parsing during complex STP-to-DWG transformation. [CADAPP-2267]

- **DWG to PDF Exception Resolved**  
  Handled System.NullReferenceException during export for more stable rendering. [CADNET-10078]

- **DXF-to-DXF Compatibility Improved**  
  Enhanced fidelity and correctness in round-trip DXF processing. [CADAPP-2146]

- **GLB to GLTF JSON Fix**  
  Fixed deserialization bug when converting GLB to GLTF format. [CADNET-10081]

- **DGN Text Extraction Support**  
  Added ability to programmatically extract text elements from DGN files. [CADNET-10079]

- **DWG Polylines with Text Preserved**  
  Ensures polyline-attached text is retained during format conversion. [CADNET-10067]
