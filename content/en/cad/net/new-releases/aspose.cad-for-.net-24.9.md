#### Release Highlights

Aspose.CAD for .NET 24.9 (MSI) introduces major export enhancements, including improved DWG-to-PDF quality, DWF/DXF interoperability, CTB color table rendering, and support for DWG block insertion workflows.

### ‚ú≥Ô∏è Enhancement

- **DWG-to-PDF Quality**:  
  Improved DWG export engine resolves distortion issues in rendered PDF outputs. [CADNET-9863]

- **CTB Color Table Accuracy**:  
  Corrected export logic for DWG drawings using CTB plot styles, ensuring correct color mappings in target formats. [CADNET-9890]

- **SVG to DXF Stability**:  
  Fixed internal server error during SVG-to-DXF conversion to ensure consistent vector format translation. [CADNET-9900]

- **Embed DWG into DWG as Block**:  
  Now supports inserting one DWG drawing into another as a block entity, expanding CAD composition workflows. [CADNET-9887]

- **DWF to DXF Conversion Fix**:  
  Improved handling of DWF layers and geometry for reliable DXF export. [CADNET-9885]

- **DXF Export Dimension Handling**:  
  Addressed damaged dimension elements when exporting DXF to DXF, ensuring layout precision. [CADNET-9882]

- **Redistributable Licensing Fix**:  
  Resolved false watermark rendering with valid redistributable licenses. [CADNET-9868]
