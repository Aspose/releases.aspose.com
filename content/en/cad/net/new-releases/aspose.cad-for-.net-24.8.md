Aspose.CAD for .NET 24.8 (MSI) brings important updates to enhance CAD file handling. This release provides a range of new features and enhancements to streamline the management of DWG, DXF, and other CAD formats.


### New Support for SHX Files

We have added the capability to load SHX files from streams in the latest .NET CAD files API version to improve flexibility in CAD operations.

### Enhanced DWG Export

Export performance for `CadViewport` and `CadInsert` objects has been optimized for faster processing within your C# and VB.NET apps.

### Advanced Polylines Handling

Better support for polylines with linestyles containing SHX is introduced in version 24.8 of Aspose.CAD for .NET, which enhances visual accuracy.

### 3D Mesh Lines

Support for storing lines in 3D mesh is now available to expand the utility of CAD models in CAD file management solutions.

### Autodesk Forge SVF Model Support

This release also includes support for Autodesk Forge SVF Model format. You can take your C# applications to the next level with this feature.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.CAD for .NET 24.8 Release Notes](https://releases.aspose.com/cad/net/release-notes/2024/aspose-cad-for-net-24-8-release-notes/).
