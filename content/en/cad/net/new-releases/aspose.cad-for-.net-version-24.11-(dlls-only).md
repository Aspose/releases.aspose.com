#### Release Highlights

Aspose.CAD for .NET 24.11 (DLLs-only) expands CAD export options with **DWF and DWFx** support and introduces meaningful optimizations for DXF and DWG workflows. The update includes parsing improvements, new DWG coordinate control, enhanced line style rendering, and fixes for font and Collada-related issues.

- **Export to DWF & DWFx Formats**  
  Official support for outputting CAD documents to DWF/DWFx, enhancing interoperability. [CADNET-9542, CADNET-9674]

- **Improved DWG-to-DXF Speed**  
  Faster and more memory-efficient conversion, especially with larger DWG files. [CADNET-9916]

- **Entity Position Control for DWG Export**  
  Users can now set base coordinates for DWG entity positioning. [CADNET-9899]

- **Simplified Hatch Parsing**  
  Removed unnecessary Hatch attributes to improve compatibility and rendering performance. [CADNET-9901]

- **Support for Custom Line Styles**  
  Adds precision rendering support for customized line patterns. [CADNET-9871]

- **Resilient DXF Generation**  
  Now supports robust DXF creation from scratch with proper shape handling. [CADNET-8952, CADNET-9042]

- **Fix for SHX Font PDF Export**  
  Resolved export issues caused by simplex.shx font inconsistencies. [CADNET-9951]

- **Improved Collada (.dae) Import**  
  Fixes load errors and enhances model parsing. [CADNET-9857]

- **Parsing of Simple CAD Objects**  
  Increased parser coverage to accommodate lightweight CAD objects. [CADNET-9947]

- **Improved STPImage API Docs**  
  Made the STPImage API easier to understand with enhanced documentation. [CADNET-9891]
