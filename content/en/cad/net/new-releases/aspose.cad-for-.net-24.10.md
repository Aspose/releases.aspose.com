#### Release Highlights

Aspose.CAD for .NET 24.10 (MSI) brings robust support for IGS/IGES file handling, hatch gradient rendering, improved DWG-to-OBJ exports, and better text-to-3D object translation. This version strengthens the CAD-to-PDF conversion pipeline and introduces fixes for DWF-to-OBJ workflows.

### ‚ú≥Ô∏è Enhancement

- **IGS/IGES File Format Support**:  
  Added robust import and conversion support from IGS to DWG, DXF, and PDF formats. [CADNET-9157, CADNET-9192, CADNET-9180]

- **Hatch with Linear Gradient Rendering**:  
  Implemented gradient fills in hatches, improving DXF/DWG rendering fidelity. [CADNET-9462, CADNET-9888]

- **DWG to OBJ Enhancements**:  
  Corrected mesh fidelity issues and optimized export speed. [CADNET-9870]

- **Text to 3D Object Export**:  
  Improved transformation of text entities into accurate 3D object representations. [CADNET-9360]

- **PDF Export with Attribute Precision**:  
  Fixed layout shift when attribute values are changed before PDF rendering. [CADNET-9831]

- **DWF to OBJ Workflow Improvements**:  
  Enhanced geometry extraction and conversion accuracy from DWF to OBJ. [CADNET-9197]
