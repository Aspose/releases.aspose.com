#### Release Highlights

Aspose.CAD for .NET 25.3 (DLLs-only) introduces key enhancements for advanced .NET developers working with CAD. This release adds **support for Navisworks NWF files**, enhances **DXF and DWG reliability**, and upgrades **3D transformation capabilities** for FBX, OBJ, and DXF formats.

- **Navisworks (.NWF) File Format Support**  
  Developers can now read and interpret Navisworks project sets for BIM-related workflows. [CADNET-1176]

### üîß Enhancement

- **Hatch + LWPolyline Writing to DXF 2018**  
  Enhanced fidelity for drawings with filled and polyline-based elements. [CADNET-10065]

- **Contour-to-Bold Conversion (Paper-to-CAD)**  
  Detect and render contour lines as solid vectors. [CADNET-10033]

- **Merged MText Handling**  
  Streamlines MText processing and layout rendering. [CADNET-9980]

- **Triangulation for B-Spline Surfaces**  
  Provides accurate rendering of NURBS-like geometry. [CADNET-9929]

- **STP Rendering & Conversion Fixes**  
  Addresses rendering issues and improves STP to DWG, DWG to STP workflows. [CADNET-9908, CADNET-9853, CADNET-9839]

- **Support for 3D Scaling in FBX, DXF, OBJ**  
  Enables geometric scaling of 3D content across formats. [CADNET-9292, CADNET-9297]

- **Improved Thread Safety**  
  Fixed multithreading bugs that caused data collisions. [CADNET-9495]

- **On-the-Fly Rendering Memory Optimization**  
  Reduces memory consumption during complex rendering tasks. [CADNET-8542]

- **Arnold Algorithm for Watermark Scrambling**  
  Allows encrypted image watermarking via Arnold transformation. [CADNET-9405]

- **Support for PostScript and Other Enhancements**  
  Includes smaller refactors and support structure upgrades across formats. [Multiple]
