Aspose.CAD for Python via .NET 24.7.0 offers a range of new features and improvements to empower Python developers working on Linux. Effortlessly integrate these enhancements into your Python CAD AND BIM drawings processing pipeline natively on any Python framework applications.

### Generic conversion and export API

Aspose.CAD for Python via .NET 24.7.0 delivers an extended conversion API with new features for greater customization. You can now control the format of various CAD elements, including metadata and visuals. This Python code example demonstrates conversion of DXF file into PDF: 

```Python
import aspose.cad as cad

image = cad.Image.load("file.dxf")
rasterizationOptions = cad.imageoptions.CadRasterizationOptions()
rasterizationOptions.layouts = ["Model"]

pdfOptions = cad.imageoptions.PdfOptions()
pdfOptions.vector_rasterization_options = rasterizationOptions

image.save("result.pdf", pdfOptions)

```
