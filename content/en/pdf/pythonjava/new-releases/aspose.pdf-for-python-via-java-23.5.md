Python developers, get ready! We are happy to announce the new release of Aspose.PDF for Python via Java (version 23.5). This update delivers a new feature for you to upgrade your Python PDF apps. 


### Acro Forms Support

This release sees a significant addition to the reptoire of Aspose.PDF for Python via Java API. You can now work with Acro forms within your Python apps. This enables creating, editing, and manipulating interactive PDF forms with ease.

Please download and explore version 23.5 of the Python PDF processing API to leverage the exciting new enhancements.
