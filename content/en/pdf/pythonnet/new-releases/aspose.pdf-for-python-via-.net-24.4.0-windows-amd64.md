Aspose.PDF for Python via .NET 24.4.0 includes different features and enhancements for Python developers to create, manipulate, and analyze PDF documents seamlessly within their Python apps running on Windows 64-bit systems.

### Key Highlights

- **Enhanced Text Processing:** This release brings improved subscript and superscript text detection and conversion to ensure accurate text representation in various output formats.
- **Streamlined PDF/A Conversion:** A file size optimization option for PDF/A conversion and better compatibility with different PDF/A variants has now been added.
- **Advanced Printing Control:** With version 24.4.0, you can gain more control over paper selection during printing with the ability to choose the paper source based on PDF page size directly through the API.
- **Improved Image Handling:** Enjoy support for applying clipping masks to images for precise image manipulation within PDFs on Windows x64 systems.
- **Asynchronous Saving:** Aspose.PDF for Python via .NET 24.4.0 enables saving PDF documents to DOCX format asynchronously for faster processing.
- **Richer Annotations:** Seamlessly create annotations with a wider variety of dashed line styles for enhanced document aesthetics.

### Bug Fixes

We have fixed numerous bugs in this Python API release, addressing issues like document flattening, XFA to standard PDF conversion, PDF resizing, and more.

### Public API and Backward Incompatible Changes

### Added APIs

We have added several new methods, properties, and types to the public API in this release. Some of these are:
- aspose.pdf.annotations.Dash.#ctor(list[int])
- aspose.pdf.devices.TiffDevice.binarize_bradley(stream,stream,bool)
- aspose.pdf.Document.pick_tray_by_pdf_size bool
- aspose.pdf.facades.ViewerPreference.PICK_TRAY_BY_PDF_SIZE int
- aspose.pdf.facades.Form.export_json(stream,bool)
- aspose.pdf.facades.Form.import_json(stream)
- aspose.pdf.forms.Field.export_value_to_json(stream, bool)
- aspose.pdf.forms.Field.import_value_from_json(stream) bool
- aspose.pdf.FileHyperlink.new_window aspose.pdf.ExtendedBoolean
- aspose.pdf.plugins.SignOptions
- aspose.pdf.plugins.Signature
- aspose.pdf.XImage.add_stencil_mask(stream)

### Removed APIs
We have removed the following public API members from this release:
- aspose.pdf.devices.DocumentDevice.binarize_bradley(stream,stream,float)
