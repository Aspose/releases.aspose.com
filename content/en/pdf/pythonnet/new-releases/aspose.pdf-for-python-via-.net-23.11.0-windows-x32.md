Take your Python development to the next level with the newest version of Aspose.PDF for Python via .NET 23.11.0 on Windows x32. This latest release is designed to make your PDF processing tasks easier than ever, with improved merging features and exciting additions to the API.

### Key Highlights
- **Effortless PDF Merging:** Increased flexibility for merging PDF documents with minimum coding in your Python applications using various overload methods. 
- **Expanded Color Depth Options:** Developers can gain better control over the color depth of exported PDFs using the new `FORMAT_24BPP` enumeration.
- **Adjustments to Text Replacement:** Text replacement functionality gains new properties for right and left adjustments.
- **Manage Dictionaries:** The introduction of the `DictionaryEditor` class simplifies manipulating PDF document dictionaries and further enhances the profile of your PDF apps in Python.
- **Additional API Enhancements:** You can explore a range of improvements in this API release for advanced PDF document manipulation.

### Public API and Backward Incompatible Changes

### Added APIs

We have added several new classes, properties, and methods, mainly focusing on mergers, dictionaries, text adjustments, and multithreading. Some of these are:
- aspose.pdf.devices.ColorDepth.FORMAT_24BPP
- aspose.pdf.Document.merge(aspose.pdf.Document[])
- aspose.pdf.Document.merge(aspose.pdf.Document.MergeOptions,str[])
- aspose.pdf.Document.MergeOptions
- aspose.pdf.multithreading.IInterruptMonitor
- aspose.pdf.multithreading.IInterruptMonitor.interrupt
- aspose.pdf.text.TextReplaceOptions.right_adjustment float
- aspose.pdf.utils.DictionaryEditor
- aspose.pdf.utils.DictionaryEditor.add(str,aspose.pdf.utils.publicdata.ICosPdfPrimitive)


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.PDF for Python via .NET 23.11.0 Release Notes](https://releases.aspose.com/pdf/pythonnet/release-notes/2023/aspose-pdf-for-python-net-23-11-release-notes/).
