Aspose.PDF for Python via .NET 24.3.0 is here for Windows 64-bit systems! This release empowers Python developers with many new features and enhancements for streamlined PDF manipulation.

### Key Highlights
- **Inline Image Size Retrieval:** Get the length (L key) of inline images for precise control with this update of the Python PDF API.
- **Effortless PDF/A Conversion:** Seamlessly convert your PDFs to the PDF/A archive format within your PDF conversion apps on Windows 64-bit systems.
- **Advanced Text Search:** Aspose.PDF for Python via .NET 24.3.0 lets you locate specific phrases within `TextFragmentAbsorber` for targeted content manipulation.
- **Table Conversions to Markdown:** Effortlessly convert PDF tables into well-formatted Markdown with a new feature added to this release.
- **Persistent Empty Signature Fields:** Maintain PDF/A compliance with empty signature fields on every PDF document page

### Other Enhancements
- **Improved Word Wrapping in TextStamp:** Ensure proper text handling with automatic word breaks.
- **Asynchronous PDF Saving:** Save PDF documents asynchronously for a more responsive user experience.
- **Enhanced JPEG Rasterization:** We have enhanced JPEG rasterization allowing you to experience optimized JPEG rendering quality.
- **Long Word Splitting:** Break lengthy words within `TextStamp` when `Wordwrap` is enabled.
- **Faster Image Extraction:** Extract images from PDFs with improved performance and better accuracy.
- **Precise Text Replacement:** You can now replace multiple phrases accurately without overlapping or clipped text.
- **Bitmap Image Size Property:** A new enhancement lets you determine the size of a bitmap image efficiently.

### Bug Fixes
- We have resolved numerous bugs related to XPS conversion, JavaScript execution, PDF/A compliance, text formatting, and more in this release of the Python API.

### Public API and Backward Incompatible Changes

### Added APIs

We have added several new classes, methods, and properties to the public API in this release. Some of these are:
- aspose.pdf.forms.Field.execute_field_java_script(aspose.pdf.annotations.JavascriptAction)
- aspose.pdf.Image.bitmap_size aspose.pdf.Rectangle
- aspose.pdf.plugins.PdfAConvertOptions
- aspose.pdf.plugins.PdfAConvertOptions.outputs
- aspose.pdf.plugins.PdfAConvertOptions.add_output(aspose.pdf.plugins.IDataSource)
- aspose.pdf.plugins.PdfAConverter
- aspose.pdf.plugins.PdfAOptionsBase
- aspose.pdf.plugins.PdfAValidateOptions
- aspose.pdf.plugins.PdfDoc.process(aspose.pdf.plugins.IPluginOptions)
- aspose.pdf.plugins.PdfToDocOptions.#ctor(aspose.pdf.plugins.SaveFormat,aspose.pdf.plugins.ConversionMode)

### Removed APIs

We have removed several classes, methods, and properties from the public API in this release. Some of these are:
- aspose.pdf.plugins.PdfWord
- aspose.pdf.plugins.PdfWord.process(aspose.pdf.plugins.IPluginOptions)
- aspose.pdf.plugins.PdfExcel
- aspose.pdf.plugins.PdfToWordOptions
- aspose.pdf.plugins.PdfToWordOptions.#ctor
- aspose.pdf.plugins.PdfToExcelOptions.ExcelFormat
- aspose.pdf.plugins.PdfToExcelOptions.ExcelFormat.XLSX
