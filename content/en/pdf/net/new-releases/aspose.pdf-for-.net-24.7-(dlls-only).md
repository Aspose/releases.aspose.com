Great news for .NET developers! Aspose.PDF for .NET 24.7 (DLLs-only) is now available, and it brings a host of new features and improvements. The DLLs package ascertains easy integration with your existing projects and smooth working of your solutions across Windows, Linux, and macOS platforms.

### Form Data Import and Export in JSON Format

Confidently import and export data from forms in JSON format using the newest C# PDF API version and simplify your .NET data extraction workflows.

### Introducing Aspose.PDF AI Copilot

Aspose.PDF delivers a groundbreaking AI-powered copilot, aimed to help you with different PDF processing tasks through natural language instructions.

### Support for .NET 7 on Docker

Utilize Aspose.PDF feature set smoothly on Docker containers running .NET 7 and boost your development environment on Windows, Linux, or macOS.

### Apply Tags Automatically

Developers can now incorporate innovative auto-tagging functionality to easily identify and categorize key elements in their PDF documents.

### Document Processing with Multithreading

Enjoy a noticeable uplift in processing speed by enabling multithreaded conversion of PDFs to DOC format within your .NET PDF processing applications.

### Simplified Path Flattening

Develop solutions to streamline your path manipulation tasks with the capability to flatten `Path` objects for straightforward processing and editing.

### Smart Buttons for Full-Screen Mode

Aspose.PDF for .NET 24.7 enables the creation of interactive buttons that help viewers enter and leave full-screen mode within the HTML document created from the PDF.

### Next Level PDF Content Comparison

Enrich your cross-platform PDF manipulation applications by integrating the capability to carry out detailed PDF comparisons, such as annotations and side-by-side output for seamless identification of changes. 

### Alignment and `TextStamp` Options

Users can now enjoy better control over `TextStamp` objects by setting alignment and other visual properties.

### Fine-tuned XPS to PDF Conversion

Experience high-quality performance and accuracy while converting XPS documents to PDF format within your C# and VB.NET document conversion applications.

### No Image Redaction Dependency

The C# PDF apps API lets you redact specific images across multiple pages of PDFs without affecting other content.

### High-performance PDF to Image Conversion

Obtain high-quality images from PDFs without having to worry about whether the original fonts are available on the system.

### Improved Page Label Processing

The latest C# PDF processing API version ascertains precise page label processing during document merging operations using `Facades.PdfFileEditor`.

### Exclude Area and Tables in PDF Comparison

Apply exclusion of specific areas and tables while performing PDF comparisons for focused analysis on Windows, Linux, or MacOS.

### Improvements and Bug Fixes

- Resolved a critical issue that caused an exception during HTML to PDF conversion when encountering BASE64 strings. (PDFNET-53536)
- Numerous other bug fixes and improvements throughout the Aspose.PDF for .NET library.
- Fixed an error that led to an exception when opening a PS document. (PDFNET-57515)
- Addressed a memory leak that occurred during document opening. (PDFNET-57493)


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.PDF for .NET 24.7 Release Notes](https://releases.aspose.com/pdf/net/release-notes/2024/aspose-pdf-for-net-24-7-release-notes/).
