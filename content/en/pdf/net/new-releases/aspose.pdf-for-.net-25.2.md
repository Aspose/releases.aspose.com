#### Release Highlights

Aspose.PDF for .NET 25.2 adds full support for PDF/X-4 conversion, fine-grained digital signature management using `SignatureName`, and advanced control over text box form fields.

### ‚ú´Ô∏è New Feature

- **PDF/X-4 Compliance**: Convert PDFs to PDF/X-4 standard with external ICC profiles and output intent for accurate print-ready output.
- **Signature Delegate Control**: Avoid dual invocation of `CustomSignHash` using `AvoidEstimatingSignatureLength`.
- **Advanced Signature Name Handling**: Use `SignatureName` type to handle and verify uniquely named digital signatures.
- **Multi-Widget TextBoxField**: Add TextBoxField with multiple widget annotations and customized appearances.

### üîß Enhancement

- **Image Optimization**: Improved compression for embedded images without quality loss.
- **Security**: SHA-3 based signature verification enhancements.
- **PDF Repair**: Expanded repair support for `Annotation.Rect` and resource dictionaries.
- **Vulnerability Fix**: Updated `System.Text.Json` dependency to mitigate vulnerabilities.

### üõ† Fix

- Fixed duplicate link annotations and missing content during PS to PDF conversion.
- Resolved `NullReferenceException`, `IndexOutOfRangeException`, and memory leaks across signature and text extraction workflows.
- Corrected visual formatting for TextStamps, Bookmarks, and PDF/A conversions.
- Improved handling of rotated text, SVG rendering, and Thai language in HTML export.
