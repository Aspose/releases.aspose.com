#### Release Highlights

Aspose.PDF for .NET 24.12 introduces smarter PDF generation with custom ICC-based PDF/X-1a output, intelligent font substitution for conversions, and dynamic stamp sizing. Includes over 30 precision fixes for performance, compatibility, and rendering.

### ‚ú≥Ô∏è New Feature

- **PDF/X-1a Output Intent**: Set output intent with custom ICC profiles during PDF/X-1a conversion.
- **Font Analyzer**: Identify the most suitable fonts for generation, conversion, and text replacement.
- **Auto-Fit Text Stamp**: Automatically adjust font size to fit the bounding rectangle of text stamps.

### üîß Enhancement

- **Form CheckBox Styling**: Support setting no-fill background for CheckBox form fields.
- **Font Substitution Flexibility**: Use custom fonts for characters not found in originals.
- **Raw Image Extraction**: Access raw embedded image data for integrity validation.
- **Document Privileges**: Retrieve document-level permissions information programmatically.

### üõ† Fix

- Fixed font changes and increased file size during PDF/A conversion.
- Corrected misaligned redaction overlays and improper text wrapping with Unicode characters.
- Resolved null reference, index out of range, and stack overflow exceptions across APIs.
- Improved EPUB, SVG, and TIFF conversion stability on Linux.
- Addressed regressions in underline rendering, Optimize method performance, and page concatenation.
