#### Release Highlights

Aspose.PDF for .NET 25.1 introduces PDF to HTML conversion with image removal, certificate validation via CA servers, and SHA-3 signature verification support.

### ‚ú≥Ô∏è New Feature

- **Image-Free HTML Export**: Convert PDFs to HTML without saving raster images using `RasterImagesSavingModes.DontSave`.
- **Certificate Authority Validation**: Support for validating digital signatures via OCSP and CRL with strict or auto modes.
- **SHA-3 Signature Support**: Cross-platform support for PDF signatures using SHA-3-256, SHA-3-384, and SHA-3-512 hashing algorithms.

### üîß Enhancement

- Optimized paragraph positioning with the new `ParagraphPositioningMode`.
- Enhanced platform-independent support for font rendering in OFD on Linux.

### üõ† Fix

- Resolved PDF file bloat after save operations.
- Fixed printing issues for XPS generated files.
- Corrected Chinese character rendering and PDF stamping artifacts.
- Stabilized annotation flattening and SVG rendering accuracy.
- Addressed verification bugs, form field formatting, and font-related exceptions across platforms.
