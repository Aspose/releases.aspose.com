#### Release Highlights

Aspose.PDF for C++ 25.2 for Linux improves OpenJpeg decoder speed, introduces new PDF editing structures, and enables support for PDF/X-4 conformance. It also offers programmatic access to signatures, form fields, and font control, ideal for automated PDF workflows.

#### New Feature

- **OpenJpeg Decoder Optimization**: Improved performance for OpenJpeg raster decoding.
- **CosPdfDictionary Editing**: Modify internal PDF structure with a key/value interface on Linux systems.
- **PDF/X-4 Conformance**: New `PdfFormat::PDF_X_4` compliance added for professional publishing.
- **Signature & Form APIs**:
  - Access signature metadata via `SignatureName`.
  - Detect validation mismatches with `SignatureLengthMismatchException`.
- **Stamping Enhancements**: Get `XIndent` and `YIndent` from `ImageStamp` for precise placement.
- **HTML Export Control**: Improved `SaveFullFont` accessors in `HtmlSaveOptions`.

#### Enhancement

- `Rectangle::Empty` replaced by `get_Empty()` for better runtime compatibility in Linux builds.

#### Fix

- No removals. All changes maintain stability and backward compatibility for Linux PDF workflows.
