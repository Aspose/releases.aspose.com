The latest release of Aspose.PDF for C++ (version 24.6) for macOS delivers valuable enhancements for Mac developers. In this release, we have optimized the process of saving PDFs to multipage TIFF files, offering a 2x speed increase and the ability to create TIFFs larger than 2GB.

### PDF to Multipage TIFF Conversion Boost on Apple Mac

Mac developers can enjoy a huge performance boost (2x faster) while exporting PDFs as multipage TIFF images with this C++ PDF library version. This release enables the creation of TIFF files exceeding 2 GB in size.

### The New Text PDF Comparer Class

Aspose.PDF for C++ 24.6 introduces a newÂ `TextPdfComparer` class, facilitating you to perform extensive text comparisons between PDF documents. Carry out page-by-page differences, identify insertions and deletions, generate in-depth comparison statistics, and do much more on the Linux platform.

### Optimized Text Extraction from Repaired PDFs

The optimized `Document.Repair` method offers better control over the repair process in the latest C++ PDF library version. Utilize the newly added `RepairOptions` class to select whether to restore indirect object generations during repairs.
