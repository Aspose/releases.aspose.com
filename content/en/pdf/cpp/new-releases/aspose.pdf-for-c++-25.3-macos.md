#### Release Highlights

Aspose.PDF for C++ 25.3 on macOS introduces Markdown conversion, comprehensive Bates numbering insertion, and structure-aware alignment for tagged PDF content. Ideal for legal, archival, and documentation workflows on macOS environments.

#### New Feature

- **Markdown Conversion**: Export PDF to Markdown format using newly introduced conversion logic.
- **Bates Numbering Tools**:
  - Use `BatesNArtifact` with start number, prefix, suffix, and digit length settings.
  - Automate numbering across pages using `AddBatesNumbering()`.
- **PDF Tagging Controls**:
  - `PositionSettings` provides control over alignment, margin, and paragraph behavior.
  - Layout elements (`LinkElement`, `TableElement`, `IllustrationElement`) support `AdjustPosition()` for structured output.
- **Pagination Enhancements**:
  - Subset control (`All`, `Even`, `Odd`) with `PaginationArtifact`.
  - Custom artifacts using `PageCollectionExtensions`.

#### Enhancement

- macOS-native tagging and layout systems now benefit from advanced artifact APIs for accessibility and automation.

#### Fix

- Removed deprecated MCRElement content access methods and old pagination logic for streamlined macOS performance.
