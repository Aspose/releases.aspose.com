#### Release Highlights

Aspose.PDF for C++ 24.10 on Linux enables import of CorelDRAW images, introduces shape layout validation with `CheckBounds()`, and enhances structured drawing control using `BoundsCheckableList`. Optimized for Clang-based Linux servers.

#### New Feature

- **CorelDRAW Support**: Load `.CDR` files into PDF using `CdrLoadOptions` under Linux environments.
- **Shape Boundary Validation**: Add geometric safety to drawing operations with `CheckBounds()` on all major shape types.
- **BoundsCheckableList API**: Use container-aware drawing lists for high-integrity rendering.
- **Crash Diagnostics**: Use `CrashReportOptions` to log structured crash data with customizable messages.
- **Rectangle Containment Improvements**: Use enhanced `Contains()` with strict logic toggle.

#### Enhancement

- **CLI Rendering Resilience**: New `AddImage()` overloads handle streams and HOCR mappings in Linux-native pipelines.
- **Matrix & Transformation**: Use `GetFlipMatrix()` for mirrored drawing output in Linux containers.

#### Fix

- Deprecated APIs removed to support safer image placement and shape coordination during PDF generation on Linux.
