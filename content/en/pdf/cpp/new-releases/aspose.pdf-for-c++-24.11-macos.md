#### Release Highlights

Aspose.PDF for C++ 24.11 for macOS introduces DJVU file support, header/footer layout APIs for pagination control, and new cryptographic signature details. Optimized for Intel-based macOS platforms with enhanced PDF security and structured layout rendering.

#### New Feature

- **DJVU to PDF on macOS**: Convert `.djvu` images to PDF with `DjvuLoadOptions` under macOS Monterey or later.
- **Custom Headers & Footers**: Use `HeaderFooterSettings`, `PageRange`, and `HorizontalAlignment` classes to define structured headers and footers per page range.
- **Signature Metadata API**: Gain insights into digital signatures using `SignatureAlgorithmType`, `KeyedSignatureAlgorithmInfo`, and supported standards like `Rfc3161`.

#### Enhancement

- **macOS PDF Layout Control**: Header/footer features allow consistent cross-platform rendering using alignment and pagination APIs.
- **Digital Signing Support**: Better validation and reporting of signature parameters within PDF workflows on Apple hardware.

#### Fix

- Obsolete `CurveTo::get_Points()` method removed to streamline PDF rendering compatibility with macOS graphics subsystems.
