#### Release Highlights

Aspose.PDF for C++ 25.4 for macOS includes vital fixes for Markdown rendering and introduces AutoTagging tools for accessibility compliance. The release also brings powerful encryption handling and advanced style control in `TextState`.

#### New Feature

- **Markdown Stability**: Resolved rendering crash during Markdown export.
- **Advanced Tagging**:
  - Added `AutoTaggingSettings`, `HeadingLevels`, and new heading strategies.
  - Improved structural tagging for PDF/UA outputs.
- **Custom Encryption APIs**:
  - Implement `ICustomSecurityHandler` for full encryption algorithm control.
  - Use `EncryptionParameters` to inspect existing security attributes.
- **TextStyle Control**: Numerous new `TextState` flags for color, spacing, font sizing, and visibility.
- **License Reset Support**: Clear license using `License::ClearLicense()` on macOS environments.
- **HeadingStyle Control**: Customize Markdown heading output via `MarkdownSaveOptions`.

#### Enhancement

- Additional tagging lifecycle methods: `ITaggedContent::PreSave()` and `Save()` improve document readiness before export.

#### Fix

- Deprecated `PdfToMarkdown::HeadingLevels` removed to reduce conflict with new AutoTagging system.
