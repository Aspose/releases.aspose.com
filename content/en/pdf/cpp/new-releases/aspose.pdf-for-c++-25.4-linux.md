#### Release Highlights

Aspose.PDF for C++ 25.4 on Linux delivers Markdown stability fixes, new layout tagging tools, and programmable encryption configuration. The update boosts server-side rendering, accessibility, and security compliance for Linux-based applications.

#### New Feature

- **Markdown Crash Fix**: Fixed `NullReferenceException` in Markdown conversions.
- **AutoTagging API**:
  - `AutoTaggingSettings`, `HeadingLevels`, and `HeadingRecognitionStrategy` now allow fine-tuned structure tagging.
- **Encryption & Security**:
  - Define `EncryptionParameters` such as filters, sub-filters, owner/user keys, and key length.
  - Implement `ICustomSecurityHandler` for bespoke encryption flows.
- **TextState Flags Added**:
  - More than 15 new properties added to `TextState` for tracking font, alignment, and rendering states.
- **MarkdownSaveOptions HeadingStyle**: New property controls heading level formatting during markdown export.

#### Enhancement

- Added `License::ClearLicense()` to support dynamic licensing resets in Linux environments.
- PDF tagging APIs enhanced with `PreSave()` and `Save()` for ITaggedContent.

#### Fix

- Removed legacy `HeadingLevels` class and migrated to the new `AutoTaggingSettings` structure.
