#### Release Highlights

Aspose.PDF for C++ 25.2 for macOS resolves a segmentation fault in SVG rendering on x86 platforms, introduces professional PDF/X-4 conformance, and adds low-level access to internal PDF dictionaries. Font and image stamp controls are also enhanced.

#### New Feature

- **Crash Fix**: Fixed segmentation fault on macOS x86 when processing SVG content.
- **PDF/X-4 Format Support**: Use `PdfFormat::PDF_X_4` for high-fidelity print compatibility.
- **CosPdfDictionary Editor**: Full dictionary manipulation via key/value pair support on macOS.
- **Stamp & Form Enhancements**:
  - Get `XIndent`, `YIndent` for `ImageStamp` alignment.
  - Create `TextBoxField` using rectangle arrays for flexible layout.
- **Signature API Enhancements**: Access names and exceptions via `SignatureName` and `SignatureLengthMismatchException`.
- **Font Management**: `SaveFullFont` now uses clean getter/setter for better property control.

#### Enhancement

- Namespaces updated for runtime consistency: `Rectangle::Empty` is now `Rectangle::get_Empty()` on macOS.

#### Fix

- SVG rendering now fully stable on macOS x86 builds.
