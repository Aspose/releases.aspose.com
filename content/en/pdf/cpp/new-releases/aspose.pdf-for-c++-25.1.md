#### Release Highlights

Aspose.PDF for C++ 25.1 for Windows delivers powerful digital signature validation APIs, the ability to flatten redaction and form fields, and introduces precise layout control with `FloatingBox`. This release also includes positioning enhancements and advanced HTML rendering options.

#### New Feature

- **Digital Signature Validation**: Verify PDF signatures using `ValidationOptions`, `ValidationResult`, and associated enums (`ValidationMode`, `ValidationMethod`, `ValidationStatus`).
- **Flatten Redactions & Fields**:
  - `RedactionAnnotation::Flatten()` to permanently apply redactions.
  - `Forms::Field::Flatten()` to lock field values.
- **FloatingBox Layout Enhancements**: Use `ParagraphPositioningMode::Absolute` for precise object placement.
- **Improved HTML Rendering**: `HtmlSaveOptions::RasterImagesSavingModes::DontSave` added for optimized image handling during export.
- **VerifySignature API**: Validate signatures with detailed status and timeout configurations using `PdfFileSignature::VerifySignature` and `Forms::Signature::Verify()`.

#### Enhancement

- Better layout control in dynamic document design through absolute positioning with `FloatingBox`.

#### Fix

- Resolved rare `FontNotFoundException` when rendering HTML on Linux without `ttf-mscorefonts` (Windows users benefit from consistent fallback behavior).
