#### Release Highlights

Aspose.PDF for C++ 24.12 on Windows introduces low-level font analysis, advanced TextStamp customization, and streamlined namespace structures. This release also improves page-level rendering, link optimization, and memory-level access to embedded image data.

#### New Feature

- **Font Analysis Support**: New `RenderingOptions::get_AnalyzeFonts()` for inspecting and analyzing embedded font structures.
- **Advanced TextStamp Features**: New API for `TextStamp` includes `FontSize`, `AutoAdjustFontSizeToFitStampRectangle`, and font precision options.
- **Embedded Image Access**: Extract raw image data from `XImage` using `GetRawImageData()`.
- **OutputIntent Enhancements**: Configure color profile output using the `OutputIntent(String)` constructor and associated accessors.
- **Link Optimization Control**: New property `LinkDuplicateStreamsScanLevel` to optimize redundant PDF streams.
- **Geometry Enhancements**: Use `Rectangle::MoveBy(x, y)` for precise object movement on PDF pages.
- **Initialization Checks**: Added `PdfAction::get_IsInitialized()` to verify readiness of interactive actions.

#### Enhancement

- **Major Namespace Simplification**: Over 50 types from deeply nested namespaces are now available via flat structure (e.g., `Aspose.Pdf.Center`, `Aspose.Pdf.PageRange`, etc.) for simpler access and cleaner API usage.
- **TextFragment Editing**: New `get_TextEditOptions()` supports rich control over in-place text editing.

#### Fix

- No removed APIs in this release. Refactored APIs remain backward compatible with improved discoverability.
