#### Release Highlights

Aspose.PDF for C++ 24.12 on Linux enhances PDF document processing with better font analysis, raw image data extraction, and streamlined namespace usage. Ideal for document automation in Linux-based CI/CD environments.

#### New Feature

- **Analyze Fonts on Linux**: Use `RenderingOptions::get_AnalyzeFonts()` to inspect embedded fonts for validation and optimization.
- **Enhanced TextStamp API**: Control font sizing and layout precision using new `TextStamp` attributes.
- **Image Data Extraction**: Access raw embedded images with `XImage::GetRawImageData()`.
- **Object Relocation**: Move PDF content with pixel-precision via `Rectangle::MoveBy(double, double)`.
- **Link Optimization**: Set `LinkDuplicateStreamsScanLevel` to reduce stream duplication in exported files.
- **OutputIntent Support**: Enable ICC profile-based output setup with the new `OutputIntent(String)` constructor.
- **PdfAction Status Check**: Ensure interactive elements are initialized using `get_IsInitialized()`.

#### Enhancement

- **Linux Namespace Simplification**: Flattened class access (e.g., `Aspose.Pdf.Footer`, `PageNumber`, etc.) makes development faster and cleaner in Linux IDEs and CLI tooling.

#### Fix

- No API removals. All changes maintain backward compatibility and improve API discoverability for Linux-based systems.
