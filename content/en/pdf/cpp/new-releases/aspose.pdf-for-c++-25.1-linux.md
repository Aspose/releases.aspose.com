#### Release Highlights

Aspose.PDF for C++ 25.1 on Linux adds robust PDF signature validation, redaction flattening capabilities, and a fix for `FontNotFoundException` during HTML conversion. This version enhances layout control and form processing within Linux-based document pipelines.

#### New Feature

- **Fix: HTML Conversion on Linux**: Resolved `FontNotFoundException` when `ttf-mscorefonts` is missing.
- **Redaction & Field Flattening**:
  - Use `RedactionAnnotation::Flatten()` to seal redacted content.
  - Use `Forms::Field::Flatten()` to finalize field values.
- **Digital Signature APIs**:
  - `ValidationOptions`, `ValidationResult`, and `ValidationStatus` enums introduced.
  - Supports OCSP, CRL, and timeout-controlled verification.
- **FloatingBox Positioning**: Introduces `ParagraphPositioningMode::Absolute` for layout consistency in complex Linux PDF generation.
- **Raster Image Export Mode**: Skip raster image saving during HTML export with new enum value `DontSave`.

#### Enhancement

- Enhanced rendering control with pixel-precise FloatingBox positioning on Linux using `PositioningMode`.

#### Fix

- Eliminated dependency crash in HTML export without TTF fonts â€” now handles gracefully in Linux servers.
