#### Release Highlights

Aspose.PDF for C++ 24.11 delivers full DJVU file support, structured pagination layout using new header/footer APIs, and extended cryptographic metadata for PDF signing. Based on Aspose.PDF for .NET 24.11, this release is optimized for Windows-based document workflows.

#### New Feature

- **DJVU Import Support**: Convert `.djvu` image documents to PDF using `DjvuLoadOptions`.
- **Header/Footer Pagination API**: Introduced classes like `HeaderFooterSettings`, `PageRange`, `Header`, `Footer`, and `HorizontalAlignment` for page-level content placement.
- **Digital Signature Metadata**:
  - `SignatureAlgorithmInfo` for signature description
  - `SignatureAlgorithmType` enum (`Rsa`, `Ecdsa`, etc.)
  - `CryptographicStandard` enum (`Pkcs1`, `Pkcs7`, `Rfc3161`)
  - `KeyedSignatureAlgorithmInfo` with key size details

#### Enhancement

- **Horizontal Alignment Options**: Use `Left`, `Center`, and `Right` containers inside `HeaderFooterData` to align page numbers and dates.
- **Improved Signing Flexibility**: Define cryptographic preferences for signing workflows on Windows systems.

#### Fix

- Removed deprecated `CurveTo::get_Points()` API to streamline curve path management and avoid legacy layout issues.
