## Public API and Backward Incompatible Changes

### Added APIs

Organize PDF functions:
* **AsposePdfDeleteTextHeaders** - Delete text headers from a PDF-file
* **AsposePdfDeleteTextFooters** - Delete text footers from a PDF-file

### Removed APIs

Complete details of API can be referenced from [Aspose.PDF for JavaScript via C++ API Reference Guide](https://reference.aspose.com/pdf/javascript-cpp/).
