We have expanded the conversion capabilities of Aspose.PDF for Node.js via C++ in version 24.1. Developers can also incorporate PDF attachment manipulation functionality into their Node.js apps with this release.

### TXT to PDF Conversion in Node.js

This version allows users to easily convert plain text files (TXT) to rich and formatted PDF documents using `AsposePdfFromTxt`. It is easier to streamline the process of creating PDFs from textual content with this new addition.

### Manipulate Attachments in PDF Documents

Aspose.PDF for Node.js via C++ 24.1 includes two new additions: `AsposePdfAddAttachment` and `AsposePdfGetAttachment`, that support adding and extracting the attachments to and from your PDF documents within the Node.js projects. You can generate information PDF files with the help of these additions.

Upgrade to version 24.1 of the Aspose.PDF for Node.js via C++ API and leverage these powerful new features and enhancements. Get started today!
