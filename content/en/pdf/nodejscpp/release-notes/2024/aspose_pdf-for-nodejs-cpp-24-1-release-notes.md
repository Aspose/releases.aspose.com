## Public API and Backward Incompatible Changes

### Added APIs

* **AsposePdfFromTxt** - Convert a TXT-file to PDF
* **AsposePdfAddAttachment** - Add attachment to a PDF-file
* **AsposePdfGetAttachment** - Get attachment from a PDF-file

### Removed APIs

Complete details of API can be referenced from [Aspose.PDF for Node.js via C++ API Reference Guide](https://reference.aspose.com/pdf/nodejs-cpp/).
