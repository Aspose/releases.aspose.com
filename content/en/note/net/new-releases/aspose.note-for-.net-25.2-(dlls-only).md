#### Release Highlights

Aspose.Note for .NET 25.2 (DLLs-only) brings expanded capabilities for developers working with Microsoft Loop components in OneNote files. You can now detect, access, and extract loop-related data directly from the document structure using the new `Loop` class and enhanced `DocumentVisitor`.

### ‚ú≥Ô∏è New Feature

- **Programmatic Access to Loop Structures**  
  Introduced `Aspose.Note.Loop` class with properties:
  - `LastModifiedTime`
  - `Link`
  - Loop visitor support via `VisitLoopStart()` and `VisitLoopEnd()`

### üîß Public API Changes

- **Class Added**: `Loop`
- **Enum Extended**: `NodeType.Loop`
- **Visitor Methods**:
  - `VisitLoopStart(Loop)`
  - `VisitLoopEnd(Loop)`
- **Methods**:
  - `Loop.Accept(DocumentVisitor)`
  - `Loop.#ctor(NodeType)`
