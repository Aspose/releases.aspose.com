Aspose.Note for .NET 24.2 (MSI installer) offers improved OneNote processing capabilities to .NET developers. Work with new properties for RichText elements, and other features in this release.

### Rich Text Enhancements

Developers can extract valuable information like titles, dates, and times from `RichText` elements using the new `IsTitleText`, `IsTitleDate`, and `IsTitleTime` properties introduced in this API release.

### Effortless Image Replacement

The latest C# OneNote API update supports replacing entire images in a OneNote file with a single call using the enhanced `Image.Replace` method.
