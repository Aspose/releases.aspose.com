#### Release Highlights

Aspose.Note for .NET 25.4 (MSI) introduces vital improvements in PDF conversion accuracy. This release resolves OneNote export issues involving incorrect page scaling, malformed table columns, and runtime exceptions that hindered reliable rendering.

### ðŸ›  Fix

- **Text Layout Compression & Artifacts**  
  Fixed a bug where text appeared squeezed into narrow columns and artifacts emerged after converting OneNote to PDF. Now rendering closely matches the original layout. *(NOTENET-5784)*

- **Scaling & Table Artifacts in PDF Export**  
  Resolved layout distortions that affected tables and page scaling in exported PDF documents. *(NOTENET-5856)*

- **PDF Export Exception Handling**  
  Addressed `System.InvalidOperationException` triggered by empty sequences when exporting specific OneNote files to PDF. *(NOTENET-5871)*
