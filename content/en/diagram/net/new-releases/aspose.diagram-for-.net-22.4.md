# Image Conversion Issue Resolution

Image conversion was not working when converting Microsoft Visio&reg; VSDX to PDF. The current release has fixed this issue.

# Visio&reg; Shape Background Color Conversion

Previously in some instances the background color of the shapes was being converted incorrectly during the convert VSDX to PDF process. This issue has now been resolved.

# Text Position Handling

Better handling of text position during Visio to SVG format conversion.

# Better Table Rendering in Visio to PNG Conversion

In this release of the API the formatting problems in rendering of table and radio buttons during VSDX to PNG conversion process have been resolved.
