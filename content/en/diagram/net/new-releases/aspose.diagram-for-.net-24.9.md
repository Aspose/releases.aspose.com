#### Release Highlights

Aspose.Diagram for .NET 24.9 introduces native VSD file saving support, enhanced SVG attribute control, and crucial fixes for connector positioning, background text rendering in PDF exports, and grouped shape elements.

- **VSD File Export**:

- **SVG Export Quote Handling**:  
  Modified SVG output behavior to use double quotes (`"`) instead of single quotes (`'`) in attributes for improved compatibility.

### üõ† Fix

- **Grouped Shape Export**:  
  Resolved missing connections and texts when exporting grouped shapes to SVG.

- **Connector Arrow Placement**:  
  Corrected arrowhead misplacement during SVG page export.

- **Connector Jump Value**:  
  Fixed `SetConnectorJumpValue` method to apply correctly in routing logic.

- **Custom Property Bug**:  
  Enabled setting of link ID as a custom property in Visio documents.

- **PDF Text Issue**:  
  Fixed issue where background text was lost when saving to PDF format.
