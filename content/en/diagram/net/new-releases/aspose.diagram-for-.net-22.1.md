# Improved External Resource Embedding

Previously in some instances, while saving the Microsoft Visio&reg; diagrams to HTML stream did not embed the external resources. However, this issue has now been resolved.

# Better Handling of Shape Shading

While converting a Visio&reg; VSDX to PDF had an issue where shading of shapes was missing. This release resolves that issue and offers a better handling of shape shades.

# Save HTML to a Single Stream

Added the support for saving HTML to a single stream.

# Save Diagram to HTML with Embedded Resources

Ability to save Visio&reg; diagrams to HTML with or without the embedded resources.
