# Data Graphics in Visio&reg; to PDF Conversion

In previous releases of the API within some instances while converting Visio&reg; to the PDF format, the data graphics encountered problems. This issue has now been resolved within the current release of the API.

# Export Visio&reg; to HTML

The exception that was being thrown when a Visio&reg; diagram was exported to the HTML format, has now been resolved.
