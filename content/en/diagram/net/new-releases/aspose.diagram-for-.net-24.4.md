This release of Aspose.Diagram for .NET (MSI installer) introduces .NET 8.0 framework support and focuses on improving the accuracy of VSDX conversions and resolving various issues encountered during Visio file processing.

### .NET 8.0 Support

With .NET 8.0 framework support, developers can seamlessly integrate Aspose.Diagram features with their existing .NET development environment.

### Enhanced VSDX to PNG Conversion

The issues related to shade colors in shapes rendering incorrectly during VSDX to PNG conversion have now been resolved.

### Improved Text Rendering

Process Visio files with consistent text rendering using the C# diagram API 24.4.

### Image Handling Improvements

Display your images accurately after conversion within your .NET diagram processing apps, as we have fixed the issues causing this problem.

### Visio Import/Export Bug Fixes

Aspose.Diagram for .NET 24.4 resolves several bugs related to missing shapes, incorrect page title encoding, connector visibility, auto-layout functionality, and character encoding during VSD to VSDX conversion.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Diagram for .NET 24.4 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2024/aspose-diagram-for-net-24-4-release-notes/).
