We are pleased to announce the release of Aspose.Diagram for .NET 24.1 for developers. This version delivers exciting new enhancements and bug fixes to refine your diagramming experience within .NET 6 apps.

### Key Highlights

- **Enhanced .NET 6 Support:** You can now seamlessly integrate Aspose.Diagram capabilities with .NET 6 applications, and utilize its powerful diagramming features in your latest projects.

### Bug Fixes

We have fixed several issues in version 24.1 of Aspose.Diagram for .NET:

- **Corrected Shape Flipping (DIAGRAMNET-53239):** An issue causing shapes to be flipped incorrectly during VSD to VSDX conversion has been resolved in this C# Visio API update.
- **Hidden Master Visibility (DIAGRAMNET-53240):** Masters hidden in VSS files are no longer erroneously displayed after converting to VSSX format.
- **Improved LocPinY Handling (DIAGRAMNET-53242):** Manage the `LocPinY` property seamlessly and ensure proper positioning of elements within diagrams.
- **Shape Formula Retention (DIAGRAMNET-53243):** Grouping shapes no longer leads to formula loss, maintaining data integrity within your diagrams.
- **Accurate Shape Height Calculation (DIAGRAMNET-53244):** Shape heights are now calculated precisely, resulting in visually accurate representations.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Diagram for .NET 24.1 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2024/aspose-diagram-for-net-24-1-release-notes/).
