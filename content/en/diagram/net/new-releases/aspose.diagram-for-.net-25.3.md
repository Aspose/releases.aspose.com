#### Release Highlights

Aspose.Diagram for .NET 25.3 MSI brings full support for saving to VSS/VST formats, compatibility with .NET 9.0, and enhances shape action control. The release also resolves critical Visio rendering, PDF layout, and HTML export issues for improved reliability.

### ‚ú≥Ô∏è Enhancement

- **Save to VSS and VST**:  
  Added support for exporting diagrams as VSS (Stencil) and VST (Template) binary formats.

- **.NET 9 Support**:  
  The library now supports targeting .NET 9.0 runtime environments.

- **Shape Actions API**:  
  Introduced `InheritActs` property to manage inherited shape actions.

- **Security Update**:  
  Resolved NuGet dependency issue by mitigating `System.Drawing.Common.4.7.0` vulnerability (CVE-2021-24112).

- **Visio Formatting & Layout**:

- **HTML Export**:  
  Resolved truncation issues where portions of diagrams were cut off in exported HTML.

- **PDF Export**:  
  Fixed text layout bugs and conversion issues with `SaveForegroundPagesOnly`.
