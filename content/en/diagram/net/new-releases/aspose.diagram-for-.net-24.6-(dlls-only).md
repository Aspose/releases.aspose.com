.NET developers, rejoice! Aspose.Diagram for .NET 24.6 (DLLs-only package) has been released, and it brings several enhancements to improve your C# app development experience across Windows, macOS, and Linux platforms.

### Dynamic Shape Insertions

Add shapes to the page, extend it automatically during shape addition, and fit the shape on the page with the latest C# diagram API release.

### Supercharged Visio Conversion

The Visio conversion feature related to missing connections and cell values has been optimized, and it has enhanced converting VSD diagrams to VSDX format.

### Page Handling Improvements

We have fixed the problems with the `PageSize`Â option not working correctly in Aspose.Diagram for .NET 24.6, ensuring refined page size management.

### New addition: Shape Theme Color Property

The newly added `HasThemeColor` property in this .NET library release simplifies checking if a shape uses a theme color, streamlining shape customization workflows.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Diagram for .NET 24.6 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2024/aspose-diagram-for-net-24-6-release-notes/).
