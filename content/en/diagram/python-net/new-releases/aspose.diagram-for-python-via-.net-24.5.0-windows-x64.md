This release of Aspose.Diagram for Python via .NET 24.5.0 brings several enhancements and fixes to improve your Visio diagram processing experience on Windows (x64) machines.

### Support for .NET 8

We have added .NET 8 support in this version of the Python Visio API, allowing developers to utilize .NET integration for developing feature-rich diagram manipulation apps. 

### Improved VSDX Conversion

Aspose.Diagram for Python via .NET 24.5.0 offers improved accuracy when converting VSDX files to PNG, including correct shade colors within shapes on the Win64 platform. 

### Text Rendering Updates

Python developers can integrate consistent text rendering across various scenarios and ensure accurate representation in their Visio diagrams.

### Connector Visibility Fix

With the fixes to connector visibility, version 24.5.0 of the diagram API ensures connectors remain visible after moving shapes. 

### Character Encoding Fix

Develop Python apps targeting the Windows x64 platform using the latest Aspose.Diagram for Python via .NET release, offering optimized character encoding during VSD to VSDX conversion. 

### Image Handling Enhancements

The issues with image display and file corruption during conversion have now been resolved. 

### Grayscale Printing Fix

Another enhancement in Aspose.Diagram for Python via .NET 24.5 is the upgraded grayscale printing for Visio files on 64-bit Windows machines. 


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Diagram for Python via .NET 24.5.0 Release Notes](https://releases.aspose.com/diagram/python-net/release-notes/2024/aspose-diagram-for-python-via-net-24-5-release-notes/).
