We are pleased to announce the release of the Linux version of Aspose.Diagram for Python via .NET (v24.6.0). This release provides elevated page handling and theme color management to ascertain a great user experience on Linux.

### Handle Pages Seamlessly

Incorporate impeccable page size modification and prevent content overflow with improved page handling capability in this release of the Python Visio API.

### Personalize Shapes in Python Apps

Aspose.Diagram for Python via .NET v24.6.0 lets you determine if a shape has a theme color and provides more meticulous styling and control over your diagrams.

### Bug Fixes

- Lost connection for saving VSD to VSDX when moving shape.
- Missing cell values after converting the VSD diagram.
- Missing connections after converting VSD to VSDX.
- Visio hyperlinks are not clickable after saving to HTML.
- The page size option is not working properly.
- `AutoFitPageToDrawingContent`Â spilling over.
- Visio to HTML cropped image.

### Public API and Backward Incompatible Changes

We have added `HasThemeColor` property to the `Shape class`, to identify if the shape has a theme color.
