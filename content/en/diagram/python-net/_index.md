### Visio Python via .NET On-premise API Features

- Create visio files via API.
- High-quality file format conversion & rendering.
- Print Microsoft Visio&reg; files to physical or virtual printers.
- Combine, modify, protect, or parse Visio&reg; pages.
- Convert shapes to images or PDF.
- Configure and apply page setup for the pages.
- Convert Visio&reg; files to various other formats.

Please visit the [official documentation](https://docs.aspose.com/diagram/python-net/) for a more detailed list of features.

### Read Visio File Formats

**Microsoft Visio**: VSD, VDW,VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM

### Save Visio Files AS

**Microsoft Visio**:  VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM
**Fixed Layout**: PDF, XPS
**Images**: JPEG, PNG, BMP, SVG, TIFF, GIF, EMF
**Web**: HTML
### System Requirements

Your machine does not need to have Microsoft Visio&reg; software installed.

### Supported Operating Systems

**Microsoft Windows&reg;:** Windows Desktop & Server (`x64`, `x86`)\
**Linux:** Ubuntu, OpenSUSE, CentOS, and others\
**Other:** Any operating system (OS) that can install Mono(.NET 4.0 Framework support) or use .NET core.
