# What's new in Aspose.Diagram for Java 25.3

## Release Highlights

Aspose.Diagram for Java 25.3 delivers expanded format support with the ability to save diagrams in Microsoft Visio VSS and VST binary formats. This release also introduces enhancements for shape inheritance and addresses a critical bug in VSD to PDF conversion.

## New Features

- **Save to VSS Format:**  
  Users can now export diagrams as Microsoft Visio VSS binary stencils, broadening compatibility with Visio template workflows.

- **Save to VST Format:**  
  Introduced support for saving diagrams in Microsoft Visio VST binary template format, enabling streamlined template-based Visio automation.

- **Shape Inherited Actions Access:**  
  Added API support to retrieve actions inherited from master shapes using the `getInheritActs()` method, enhancing programmability for complex diagram structures.

## Bug Fixes

- **VSD to PDF Conversion Exception Resolved:**  
  Fixed an issue that caused exceptions during the conversion of VSD files to PDF, improving the reliability of document exports.

## Public API and Backward Incompatible Changes

### Added APIs:

- `SaveFileFormat.VSS`  
  Enables saving diagrams in Microsoft Visio VSS binary format.

- `SaveFileFormat.VST`  
  Enables saving diagrams in Microsoft Visio VST binary format.

- `Shape.getInheritActs()`  
  Retrieves inherited actions from the master shape associated with the current shape.
