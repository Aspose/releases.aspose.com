.NET developers, rejoice! Aspose.Email for .NET 24.7 (DLLs only) is here with advancements in `iCalendar` processing and .NET 8 compatibility to elevate your email development experience across Windows, Linux, and macOS platforms.

### Supercharged .NET 8 Support

This release of the .NET email API provides effortless integration with .NET 8 projects and equips development flexibility within your cross-platform C# email processing solutions.

### Fine-tuned `iCalendar` Loading

Enriched `iCalendar` (.ics) loading in the latest version of Aspose.Email for .NET ensures accurate processing of email events and appointments.

### Resolved Issues

- Fixed: Refined management of attachment names containing specific Japanese characters (EMAILNET-41364).
- Fixed: Resolved problems with synchronous `MboxrdStorageReader.SplitInto` methods in .NET 4.5 (EMAILNET-41381).
- Fixed: Exception thrown when using `Aspose.Email.Printing` with a .NET 8 project (EMAILNET-41329).
- Fixed: Improved loading of `iCalendar` files with specific email addresses (EMAILNET-41374).
- Fixed: Addressed tapered text rendering in HTML output when converting MSG files (EMAILNET-41394).
- Fixed: Enhanced handling of valid `RRULE` recurrence patterns in iCalendar files (EMAILNET-41392).
- Fixed: Resolved the problems encountered with `VCardContact` (EMAILNET-41370).
- Fixed: Correctly saved message body changes (EMAILNET-41387).
- Fixed: Guaranteed successful return from `FromVCard` call (EMAILNET-41379).


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Email for .NET 24.7 Release Notes](https://releases.aspose.com/email/net/release-notes/2024/aspose-email-for-net-24-7-release-notes/).
