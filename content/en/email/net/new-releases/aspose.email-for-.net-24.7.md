This release of Aspose.Email for .NET 24.7 (MSI installer) delivers several important enhancements and bug fixes to streamline your email processing tasks within C# and VB.NET applications.

### Streamlined .NET 8 Support

The latest .NET email API release offers seamless integration with your .NET 8 projects and empowers development flexibility within your C# email processing applications.

### Enhanced `iCalendar` Loading

We have implemented refined `iCalendar` (.ics) loading in this version of Aspose.Email for .NET to ascertain accurate processing of email events and appointments.

### Resolved Issues

- Fixed: Exception thrown when using `Aspose.Email.Printing` with a .NET 8 project (EMAILNET-41329).
- Fixed: Improved loading of iCalendar files with specific email addresses (EMAILNET-41374).
- Fixed: Resolved an exception ("Object reference not set to an instance of an object") encountered during PST file splitting (EMAILNET-41380).
- Fixed: Addressed tapered text rendering in HTML output when converting MSG files (EMAILNET-41394).
- Fixed: Improved handling of valid RRULE recurrence patterns in iCalendar files (EMAILNET-41392).
- Fixed: Resolved issues encountered with `VCardContact` (EMAILNET-41370).
- Fixed: Ensured message body changes are saved correctly (EMAILNET-41387).
- Fixed: Guaranteed successful return from `FromVCard` call (EMAILNET-41379).
- Fixed: Improved handling of attachment names containing specific Japanese characters (EMAILNET-41364).
- Fixed: Resolved issues with synchronous `MboxrdStorageReader.SplitInto` methods in .NET 4.5 (EMAILNET-41381).


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Email for .NET 24.7 Release Notes](https://releases.aspose.com/email/net/release-notes/2024/aspose-email-for-net-24-7-release-notes/).
