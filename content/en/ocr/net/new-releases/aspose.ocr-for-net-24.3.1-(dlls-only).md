Aspose.OCR for .NET 24.3.1 (DLLs-only package) streamlines the process of integrating optical character recognition (OCR) functionality into your .NET applications on your favorite platform.

### Enhanced TIFF Image Support

The C# OCR API now supports 16-bit per-pixel depth TIFF images, expanding your image processing capabilities with the ability to extract text from TIFF images easily.

### Language Code Deprecation Timeline

A clear 9-month notice is provided for the removal of legacy language codes in version 24.3.1 of Aspose.OCR for .NET to ensure a smooth upgrade path for your applications.
