Aspose.OCR for C++ 25.2.0 (Windows) introduces OCR engine optimizations that improve image-to-text conversion speed and reliability. No changes to public APIs ensure seamless upgrades for existing projects.

- **Faster Text Recognition**: OCR engine tuned for better throughput on supported Windows OS.
- **Improved Stability**: Resolved inconsistencies in complex OCR jobs and batch scans.

- Minor internal fixes to improve recognition consistency in long-running OCR processes.

- **No API Changes**: All existing methods and classes remain unchanged.
