Manage and convert Microsoft Publisher documents with the latest Aspose.PUB for C++ 24.8 release. This release lets you convert multiple Publisher files in one go, saves time, and improves productivity for developers working on C++ projects across platforms.

### Multi-document Conversion

Refine publisher document conversion functionality with the latest release of the C++ publisher documents API and easily convert multiple Publisher files in one process, streamlining document management.

### New API Enhancements

We have expanded the API functionality with newly added public API members in this version to power your C++ publisher file solutions on Windows, Linux, and macOS. 

### Public API and Backward Incompatible Changes

### Added APIs

- Interface Aspose::Pub::IPubPackageConverter
- Class Aspose::Pub::PackageDocumentCollection
- Class Aspose::Pub::PackageDocumentItem
- Class Aspose::Pub::PackageFileItem
- Class Aspose::Pub::PackageStreamItem
- Enumeration Aspose::Pub::PubDocumentType
- Method Aspose::Pub::PubFactory::CreatePubPackageConverter
