Maximize C# app development efficiency with the latest Aspose.PUB for .NET 24.8 (DLLs only) release. It delivers enriched Publisher document conversion capabilities in the form of batch conversion functionality, allowing you to process multiple Publisher files at once.

### Integrate Batch Conversion in C# Apps

Upgrade your cross-platform .NET applications with the capability to convert multiple Publisher files using the latest .NET MS Publisher API release across Windows, Linux, and macOS platforms.

### Support for .NET 7.0 Target Framework

.NET developers can leverage the latest .NET improvements with compatibility for the .NET 7.0 target framework in this .NET API version.

### Public API and Backward Incompatible Changes

### Added APIs

- Interface Aspose.Pub.IPubPackageConverter
- Class Aspose.Pub.PackageDocumentCollection
- Class Aspose.Pub.PackageDocumentItem
- Class Aspose.Pub.PackageFileItem
- Class Aspose.Pub.PackageStreamItem
- Enumeration Aspose.Pub.PubDocumentType
- Method Aspose.Pub.PubFactory.CreatePubPackageConverter
