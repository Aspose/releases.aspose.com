Aspose.PUB for .NET 24.3 (DLLs-only package) equips developers with a refined set of features for PUB file conversion within their platform-independent C# and VB.NET applications. This update introduces broader export format options, improved text placement during conversion, and advanced font handling mechanisms.

### Broader Export Formats Support

The latest C# PUB files library release lets you convert PUB files to 12+ formats such as PDF, JPG, PNG, TIFF, DOCX, XLSX, CSV, PPTX, HTML, MHTML, SVG, and more with impeccable accuracy. Boost your document conversion solutions effortlessly on Windows, Linux, or macOS platforms.

### Enhanced Text Placement

Enjoy a considerable boost in text placement accuracy when converting files with Aspose.PUB for .NET 24.3 and supercharge your document processing workflows.

### Seamless Font Handling

You can now load fonts from external sources easily and get rid of any font substitution issues within your cross-platform MS Publisher file processing applications. 

### Support for Table Borders

The C# PUB API captures table borders accurately for a visually consistent representation in the output document when converting your PUB files to the PDF format on your favorite platform.

### .NET 6.0 Compatibility

Aspose.PUB for .NET now supports the latest .NET 6.0 target framework alongside .NET 5.0, .NET Standard 2.0/2.1, and .NET Framework 4.0. This update boosts the compatibility of your C# and VB.NET solutions.
