#### Release Highlights

Aspose.PUB for .NET 24.10 introduces an important fix for background image rendering issues in Publisher-to-PDF conversions. Additionally, this version confirms support for the .NET 7.0 framework, ensuring future-ready application development.

### ✳️ New Feature

- **.NET 7.0 Framework Support**  
  Developers can now build Publisher conversion tools targeting the latest .NET 7.0 framework, in addition to .NET 6.0, .NET 5.0, .NET Standard 2.0/2.1, and .NET Framework 4.0.

### 🛠 Fix

- **Cropped Background Image in PDF Resolved**  
  Fixed an issue where background images were improperly cropped during Microsoft Publisher to PDF conversion. The layout now renders accurately with full image integrity.

### 🔧 Public API Changes

> ❌ No changes were made to the public API in this release.

### 🚨 Backwards Incompatible Changes

> ✅ This version does not introduce any breaking changes.
