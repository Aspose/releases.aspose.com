#### Release Highlights

This DLLs-only distribution of Aspose.PUB for .NET 24.10 includes a critical fix for Publisher file conversion accuracy and now supports .NET 7.0 development. Ideal for integrating Microsoft Publisher document rendering into lightweight .NET apps.

### ✳️ New Feature

- **Supports .NET 7.0 in DLL Form**  
  Enables application development using the latest .NET standards without the need for an installer package.

### 🛠 Fix

- **Corrected PDF Export Background Cropping**  
  Addressed a bug causing background images to appear cropped in PDF exports. Visual consistency has been restored.

### 🔧 Public API Changes

> 🚫 No public APIs were changed in this DLLs-only release.

### 🚨 Backwards Incompatible Changes

> ✅ No backward-incompatible changes reported.
