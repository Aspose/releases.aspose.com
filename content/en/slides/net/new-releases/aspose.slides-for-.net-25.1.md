#### Release Highlights

Aspose.Slides for .NET 25.1 introduces powerful features like EMF export for slides and SVGs, embedded font import from PDFs, detailed document statistics, and control over thumbnail generation. The release also improves performance on .NET 9 and fixes key issues in rendering, SVG parsing, and chart accuracy.

### ‚ú≥Ô∏è New Feature

- **EMF Export for Slides**: Save individual slides as EMF files with `ISlide.WriteAsEmf()`.
- **SVG to EMF Conversion**: Convert SVG images to EMF using `ISvgImage.WriteAsEmf()`.
- **PDF Font Import**: Automatically import embedded fonts from PDFs for enhanced fidelity.
- **Extended Document Metadata**: Access slides, words, paragraphs, multimedia count, and more via `IDocumentProperties`.
- **Thumbnail Control**: Disable thumbnail generation using `PptxOptions.RefreshThumbnail`.

### üîß Enhancement

- **.NET 9 Compatibility**: Ensures smooth execution with Aspose.Slides.NET6.CrossPlatform.
- **Performance Boost**: Optimized memory usage when exporting large PPTX files to PDF with notes.

### üõ† Fix

- Fixed font substitution, chart link handling, and color loss during export.
- Resolved SVG regional dependency and Chinese Windows image extraction bug.
- Corrected PPT to PDF rendering including footers, 3D model positioning, and gradient fidelity.

### üîç Public API and Backward Incompatible Changes

- **New APIs**:
  - `ISlide.WriteAsEmf(Stream)` ‚Äì Export slides as EMF.
  - `ISvgImage.WriteAsEmf(Stream)` ‚Äì Export SVG images as EMF.
  - `IPictureFillFormat.CompressImage(deleteCroppedAreas, resolution)` ‚Äì Compress pictures based on shape size and DPI.

- **New Properties in `IDocumentProperties`**:
  - Metadata: `Slides`, `Notes`, `HiddenSlides`, `Paragraphs`, `Words`, `MultimediaClips`.
  - Flags: `ScaleCrop`, `LinksUpToDate`, `HyperlinksChanged`.
  - Grouping: `HeadingPairs`, `TitlesOfParts`.

- **New Option**:
  - `RefreshThumbnail` in `PptxOptions` to preserve existing thumbnail on save.
