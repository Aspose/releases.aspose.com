#### Release Highlights

Aspose.Slides for .NET 25.1 (DLLs only) offers flexible APIs for EMF export, embedded PDF font import, detailed metadata access, and thumbnail generation control. Ideal for advanced integrations without MSI overhead.

### ‚ú≥Ô∏è New Feature

- **EMF Export via API**: Use `ISlide.WriteAsEmf()` and `ISvgImage.WriteAsEmf()` to export visuals as metafiles.
- **PDF Font Import**: Seamlessly load embedded fonts when importing from PDF.
- **Rich Metadata**: Access presentation stats like slide count, hidden slides, multimedia clips, etc.
- **Thumbnail Control**: Opt-out of PPTX thumbnail regeneration via `RefreshThumbnail`.

### üîß Enhancement

- Increased compatibility with .NET 9 cross-platform builds.
- Reduced memory usage in PDF export with speaker notes.

### üõ† Fix

- Corrected font mismatches, chart rendering bugs, and image degradation.
- Improved SVG rendering regardless of locale/regional settings.
- Fixed 3D and chart layout issues during conversion/export.
