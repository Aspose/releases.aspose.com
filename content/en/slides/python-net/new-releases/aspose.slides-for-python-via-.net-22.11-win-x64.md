# Export PDF 1.6 & PDF 1.7 Support

Support has been implemented for PDF to be exported to the PDF 1.6 and PDF 1.7 formats.

# Export PDF with A2a, A2b Compliance Levels

This Python API release introduces the support of PDF export feature with numerous compliance levels, i.e., A2a, A2b, A2u, A3a, and A3b.

# Manage Hyperlink Sound in Presentations

A new property ([`Hyperlink.sound`](https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/)) has been added that represents the played sound of a hyperlink.

# Animation Effect Stops the Previous Sound

A new property ([`Effect.stop_previous_sound`](https://reference.aspose.com/slides/python-net/aspose.slides.animation/effect/)) has been introduced that specifies if the animation effect stops the previous sound.
