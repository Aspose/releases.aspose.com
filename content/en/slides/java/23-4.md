# What's new in the Aspose.Slides for Java 23.4

## Release Highlights

Aspose.Slides for Java 23.4 delivers several key stability fixes including improved support for media in ODP to PPTX conversions and better chart rendering during PDF export. This release also addresses issues with MathML imports and application hangs when generating slide thumbnails.

## Bug Fixes

- **Text Change in Table Causes Save Failure:**  
  Fixed an issue where saving a PPTX after modifying table text did not complete successfully.

- **Incorrect Chart Rendering in PDF:**  
  Resolved a visual inconsistency when exporting certain charts from PPTX to PDF format.

- **Missing Media in ODP ↔ PPTX Conversion:**  
  Addressed audio and video export issues during conversion between ODP and PPTX formats.

- **Slide Thumbnail Generation Hang:**  
  Fixed a problem that caused the application to hang when retrieving slide thumbnails.

- **Incorrect MathML Rendering from HTML:**  
  Corrected rendering issues for MathML equations during HTML-to-PPTX import.

- **Cone Chart Display Errors in PDF:**  
  Resolved incorrect rendering of 3D cone charts when converting to PDF.

## Public API and Backward Incompatible Changes

No changes to Public API or backward compatibility in this release.
