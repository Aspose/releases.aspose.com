#### Release Highlights

Aspose.Slides for C++ 24.12 for macOS provides developers with enhanced spreadsheet capabilities within presentations. The update introduces full support for popular spreadsheet formulas in chart data and modernizes export settings by deprecating older layout properties.

#### New Features and Enhancements

- **Chart Worksheet Formulas**:
  - Use common spreadsheet functions directly in embedded chart worksheets.
  - Enhances dynamic data visualization in macOS-based presentation apps.

#### API Refinements

- **Deprecated Layout Properties Removed**:
  - `get_NotesCommentsLayouting()` deprecated across multiple export classes.
  - Use `SlidesLayoutOptions` methods instead for a streamlined export configuration.
