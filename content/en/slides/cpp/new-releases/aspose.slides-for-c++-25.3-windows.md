#### Release Highlights

Aspose.Slides for C++ 25.3 for Windows delivers major multimedia and export enhancements. Developers can now manage video captions, control HTML/HTML5/PNG exports, and customize JPEG output quality. It also improves text layout management with support for column-wise text splitting.

#### New Features and Enhancements

- **Video Caption Tracks**:
  - Add, extract, and remove WebVTT captions from video frames.

- **Export Enhancements**:
  - Skip JavaScript links during export (HTML, HTML5, PDF).
  - Toggle ligatures via `set_DisableFontLigatures()` for HTML5.
  - Set JPEG quality using the new `IImage::Save()` overloads.

- **Text Frame Enhancements**:
  - `SplitTextByColumns()` method allows retrieving text organized by columns.

- **Threading & Performance**:
  - Enhanced concurrency handling in `GetThumbnail()` when converting slides to images.

- **New InkActions Class**:
  - Support for ink annotations (serialization/rendering included).
