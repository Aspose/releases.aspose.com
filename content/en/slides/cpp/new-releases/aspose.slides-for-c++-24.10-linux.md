#### Release Highlights

Aspose.Slides for C++ 24.10 for Linux brings advanced animation features, including the ability to retrieve vanishing point options for Zoom animations and new effect subtypes for FadedZoom transitions, enhancing the visual storytelling in presentations.

#### New Features and Enhancements

- **Zoom Animation Vanishing Point**: Access the vanishing point settings of Zoom animations to create more engaging slide transitions.
- **Additional EffectSubtype Enum Members**: Introduced `SlideCenter` and `ObjectCenter` to the `EffectSubtype` enum, providing more options for FadedZoom effects.
