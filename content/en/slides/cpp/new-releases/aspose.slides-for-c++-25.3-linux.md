#### Release Highlights

Aspose.Slides for C++ 25.3 for Linux introduces support for closed captions in video frames and new export customization options. Gain more control over presentation rendering and file optimization with updated APIs for image quality, font rendering, and hyperlink filtering.

#### New Features and Enhancements

- **VideoFrame Caption Support**:
  - Work with WebVTT subtitles in PowerPoint presentations.

- **Smarter Export Handling**:
  - Avoid rendering JavaScript-based links.
  - Disable ligatures in HTML5 exports.
  - Configure JPEG quality with fine-grained control.

- **Advanced Text Access**:
  - Easily split text into columns via `SplitTextByColumns()`.

- **Image & Ink Handling**:
  - Use `IImage::Save()` for high-quality JPEG output.
  - New `InkActions` class supports ink annotation elements.
