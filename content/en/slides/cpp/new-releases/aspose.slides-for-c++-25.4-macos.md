#### Release Highlights

Aspose.Slides for C++ 25.4 for macOS empowers developers with enhanced image rendering and shape analysis capabilities. This update includes the ability to identify geometric adjustment types and extract multi-column text content with precision.

#### New Features and Enhancements

- **Shape-to-Image Quality Boost**:
  - Better rendering quality when converting vector shapes to images.

- **Shape Adjustment Metadata**:
  - Retrieve fine-grained adjustment type info using `IAdjustValue::get_Type()`.

- **Column-Based Text Extraction**:
  - Supports content extraction based on `TextFrame` column settings.

- **PDF Export Fixes**:
  - Resolves text wrapping issues from EMF sources in PDF outputs.
