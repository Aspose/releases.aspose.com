#### Release Highlights

Aspose.Slides for C++ 25.3 for macOS enhances media handling and export configuration for developers building on Apple Silicon or Intel platforms. With video caption support and new export filters, this release offers greater flexibility and presentation quality control.

#### New Features and Enhancements

- **Captions for Videos**:
  - Embed and extract closed captions in WebVTT format for accessibility.

- **Export Controls**:
  - Filter out JavaScript-based links in all export formats.
  - Disable ligatures and adjust JPEG quality per output need.

- **Enhanced TextFrame API**:
  - Retrieve text per column using `SplitTextByColumns()`.

- **Media & Annotation Support**:
  - Save images in JPEG format at custom quality levels.
  - Use `InkActions` to manage freehand drawings and annotations.
