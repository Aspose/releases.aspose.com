#### Release Highlights

Aspose.GIS for .NET 25.1 introduces full edit support for OpenStreetMap (OSM XML) layers, smarter in-memory file stream handling with MultiStreamPath, and critical bug fixes for OSM resaving workflows.

### ‚ú≥Ô∏è New Feature

- **Edit OpenStreetMap (OsmXML) Layers**:
  - You can now create and modify `.osm` files directly.
  - Duplicate features are automatically merged for consistency.
  - Ideal for automated OSM editing pipelines.

- **Stream-Based Layer Access via MultiStreamPath**:
  - New `MultiStreamPath` enables opening and editing of multi-file formats (like Shapefiles) directly from in-memory streams.
  - Supports runtime GIS data generation and virtualized storage environments.

### üîß Enhancement

- **Improved `AbstractPath` Capabilities**:
  - Now supports bundling multiple file streams under a single logical path using `MultiStreamPath`.
  - Simplifies access to complex GIS formats in memory without file system dependency.

### üõ† Fix

- **OSM XML Resaving Bug Fixed**:
  - Resaved `.osm` files no longer become unreadable.
  - Ensures accurate preservation of features and attributes during I/O operations.
