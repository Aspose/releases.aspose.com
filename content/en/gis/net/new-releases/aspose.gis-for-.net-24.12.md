#### Release Highlights

Aspose.GIS for .NET 24.12 enhances data export and database access by introducing connection-based data source building (no SQL), automatic map sizing, and reliable OSM tag handling.

### ‚ú≥Ô∏è New Feature

- **Access DB Layers Without SQL Queries**:
  - New API `PostGisDriver.FromDefinition()` enables database layer definitions via schema structure, without writing any SQL.
  - Supports `IdentityAttribute`, `GeometryField`, and attribute definitions with native `DbType` binding.
  - Build editable layers and submit changes with full async support.

- **Auto-Calculated Map Size**:
  - You can now render maps without explicitly defining dimensions.
  - The `Map()` constructor automatically calculates bounding boxes and size based on layer content.
