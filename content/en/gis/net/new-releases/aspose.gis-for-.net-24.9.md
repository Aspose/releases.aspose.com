#### Release Highlights

Aspose.GIS for .NET 24.9 significantly enhances data transformation workflows with new APIs for LINQ filtering, geometry aggregation, and in-memory layer creation from filtered feature sequences.

### ‚ú≥Ô∏è New Feature

- **LINQ-Based Feature Filtering**:
  - Introduced `WhereLinq()` method on `FeaturesSequence` to allow expressive feature queries using C# LINQ syntax.
  - Enables easy slicing of large GIS datasets based on complex property conditions.

- **Geometry Collection Extraction**:
  - New method `ExtractGeometryCollection()` collects all geometries in a layer into a single `GEOMETRYCOLLECTION`, including support for empty and null-safe scenarios.

- **Create InMemory Layers from Feature Sequences**:
  - Use `InMemoryDriver.CreateLayer(FeaturesSequence)` to generate a layer from filtered data, making format conversion and export seamless.
