MacOS developers can upgrade their project data manipulation solutions with empowered project import, calendar reading, and more features using Aspose.Tasks for Python via .NET 24.7.

### XER Project Import Boost

The latest Python project management API release equips you with XER project import functionality on macOS x64. You can also import multiple projects from XER files and ensure a seamless representation of your project data.

### Finer Calendar Handling on macOS

Get details such as minutes per day, week, and month easily by reading and interpreting project calendars from XER format with Aspose.Tasks for Python via .NET.

### Effective Resource Management

You can manage resource data efficiently, including resource rates and assignment work, on macOS x64 to boost project planning and execution.

### Bug Fixes

- Resolved issues with MPP9 file saving and resource-leveling.
- Corrected time-phased data generation errors.
