This update to Aspose.Tasks for Python via .NET 24.7 introduces improvements in project import, calendar reading, and resource management to make it easier to integrate project management functionality into your Linux applications.

### Seamless XER Project Import

Developers can now import projects conveniently from XER files, including the ones having multiple projects, with the newest version of the Python project management API and ascertain a smooth representation of project data.

### Handle Calendars Effortlessly on Linux

Reading and interpreting project calendars from XER format and fetching details like minutes per day, week, and month becomes super convenient with Aspose.Tasks for Python via .NET.

### Manage Resources Efficiently

Effectively manage resource data, such as resource rates and assignment work on Linux, and boost project planning and execution.

### Bug Fixes

- Resolved issues with MPP9 file saving and resource-leveling.
- Corrected time-phased data generation errors.
