Python developers on Linux can upgrade to Aspose.Tasks for Python via .NET 24.3 and utilize a range of improvements for managing MPP files.

### Key Highlights

- **Visual Fidelity:** Open and save your MPP files with confidence, knowing that all visual elements like charts and graphs will be preserved.
- **Streamlined External Task Handling:** This update tackles issues related to reading and writing external task data and its unique identifiers, ensuring accurate data exchange on Linux.
- **Clean and Efficient Files:** No more unexpected resources cluttering your project files! This update of Aspose.Tasks for Python via .NET eliminates the bug that caused an additional resource to appear when opening MPP9 files on Linux.
- **Reliable Cost Calculations:** We have fixed the issues with inaccurate cost variance calculations in version 24.1 of the Python project management API, providing you with dependable cost data.


### Public API and Backwards Incompatible Changes

### Added API Members

We have added different new methods and properties in this release which are shared below:
- Aspose.Tasks.FilterCriteria.IsFieldValue
- Aspose.Tasks.FilterCriteria.SetValueField(Aspose.Tasks.Field)
