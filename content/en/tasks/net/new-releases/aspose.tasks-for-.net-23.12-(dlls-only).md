Aspose.Tasks for .NET 23.12 (DLLs-only) grants developers flexibility in integrating project management and data validation functionalities into their .NET applications. This update delivers valuable enhancements for a smoother development experience.

### Key Highlights

- **Faster XML Saving:** Save your project schedules to XML format in a flash! Aspose.Tasks for .NET 24.3 delivers a significant performance boost for this operation, along with smooth data handling.
- **Apply Unique Calendar Names:** Keep your project data organized and free from confusion. Developers can now leverage the C# project API to ensure that calendar names within your projects are unique.
- **Write Powerful MPP Filters:** This update expands your project management capabilities. The .NET API now offers enhanced support for writing task and resource filters in MPP14+ formats, giving you more control over how you view and analyze project data.
- **Seamless Filter Compatibility:** No need to worry about losing your filters! This version ensures that filters you add using previous versions are displayed accurately when projects are opened in Microsoft Project.
- **Accurate Cost & Duration Tracking:** Get precise cost and duration calculations you can rely on. This update ensures accurate results when reading project data from MPP and XML formats.
- **Lightweight and Efficient:** As a .NET developer, you can leverage the DLLs-based installation for a streamlined development environment.


### Public API and Backwards Incompatible Changes

### Added API Members

We have added the following methods and properties in this release: 
- Aspose.Tasks.Saving.MPPSaveOptions.WriteFilters

### Removed API Members

We have removed the following methods and properties in this release: 
- Aspose.Tasks.TaskBaseline.DurationFormat
