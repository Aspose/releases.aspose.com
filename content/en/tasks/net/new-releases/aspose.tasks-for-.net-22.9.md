# Improved XML to MPP Conversion

Fixed the exception that occurred when saving a specific XML file to MPP format. The following C# code snippet demonstrates how a Primavera P6 XML file can be converted to a Microsoft Project&reg; file using API:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Improved Read Feature of Primavera XER Resource Rates

Fixed the issue of resource rates reading from the Primavera XER files.

# Improved Conversions from Various Formats to MPP

Fixed the errors which triggered while saving specific files to MPP format.
