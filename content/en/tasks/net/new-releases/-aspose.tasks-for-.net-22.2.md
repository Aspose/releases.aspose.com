# Better Memory Allocation to Read MPP Data

This release offers reduced number of memory allocations while reading View data from the `MPP` file.

# Project Timescale Start & Finish for MS Project&reg;

Previously in some instances the project timescale Start / Finish was not being shown properly by Microsoft Project&reg;. In the current version of the API this issue has been fixed.
