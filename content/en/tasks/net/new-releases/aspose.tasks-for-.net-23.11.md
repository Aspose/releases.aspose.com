This release of Aspose.Tasks for .NET 23.11 (MSI) equips .NET developers with a powerful toolkit for seamless project task and resource management. Leverage new features like time-phased work calculations and task status access to streamline workflows and gain deeper project insights on Windows.
### Key Highlights

- **Effortless Timephased Work Calculations:** Calculate assignment work for specific date ranges, streamlining project resource allocation and cost estimation within your C# task management solutions. 
- **Task Status at Your Fingertips:** You can get a clear view of task completion status, enabling better project tracking and progress monitoring. 
- **Enhanced Project File Management:** Aspose.Tasks for .NET 23.11 allows developers to write project calendar GUIDs to MPP files for seamless data exchange. 
- **Additional Improvements:** Feel free to explore the other enhancements introduced in this update for a more robust project management experience.


### Public API and Backwards Incompatible Changes

### Added API Members

We have added the following methods and properties in this release: 
- Aspose.Tasks.ResourceAssignment.GetTimephasedWork(DateTime, DateTime, TimephasedDataType)
- Aspose.Tasks.Task.Status
