Aspose.Tasks for .NET 24.6 (MSI) delivers a powerful resource leveling feature for your C# and VB.NET project management applications and enhances the overall API stability and performance.

### Resource Leveling (Experimental)

The .NET project management API now empowers you to level resources within your project schedules. This experimental feature helps you identify and resolve resource overallocation, leading to more efficient project planning and execution within your C# and VB.NET applications.

### FIPS-Compliant Algorithms

Experience augmented security with the removal of non-FIPS-compliant algorithms in Aspose.Tasks for .NET 24.6 for upgraded project data protection. 

### Bug Fixes

- Precise time-phased data management (TASKSNET-11208)
- Compatibility resolution (TASKSNET-11198, TASKSNET-11193)
- Accurate calendar reading (TASKSNET-11161)
- MPT template handling improvement (TASKSNET-11160)
