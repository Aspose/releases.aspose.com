# Import Multi Project XER Files

Extended the Primavera P6 `XER` import and it now supports the import of `XER` files with more than one (1) projects.

# Improved Import of Task Constraints for XER

Added the ability to specify the behavior of task constraints import for the `XER` format.

# Read User Defined Fields for XER Format

Added the support of reading User Defined Fields for the Primavera `XER` format.
