Aspose.Tasks for .NET 24.6 (DLLs-only package) provides developers with a platform to update their C# and VB.NET project management applications with the all-new resource leveling feature. This version also improves the API's stability and performance in general.

### New Feature: Powerful Resource Leveling (Experimental)

In the latest .NET project management API release, we have introduced the ability to level resources within your project schedules. With this experimental feature, you can manage resource overallocation easily, leading to more efficient project planning and execution within your C# applications.

### Enhanced FIPS-Compliant Algorithms

.NET developers can experience a boost in security with the removal of non-FIPS-compliant algorithms in Aspose.Tasks for .NET 24.6 for enhanced project data protection. 

### Bug Fixes

- Precise time-phased data management (TASKSNET-11208)
- Compatibility resolution (TASKSNET-11198, TASKSNET-11193)
- Accurate calendar reading (TASKSNET-11161)
- MPT template handling improvement (TASKSNET-11160)
