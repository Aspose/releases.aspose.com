#### Release Highlights

Aspose.Tasks for .NET 24.10 introduces full support for graphical indicators in MPP files, extended attribute customization, and validation for resource availability periods.

### ‚ú≥Ô∏è New Feature

- **Graphical Indicators API**: Added ability to **read and modify graphical indicators** for extended attributes and save them to MPP files.
- **Custom Indicator Logic**: Define conditions and assign images using `GraphicalIndicatorCriteria` and `GraphicalIndicatorCriteriaValue`.
- **Extended Attribute Control**: New `GraphicalIndicatorsInfo` class allows managing visibility rules, tooltips, and row-type inheritance.

### üîß Enhancement

- **Resource Availability Validation**: Introduced validation logic to ensure accurate modeling of availability periods.
- **Improved API Model**: Added new exception `ResourceValidationException` for resource property validation feedback.

### üõ† Fix

- Corrected serialization of `AvailabilityPeriod` when writing to MPP.
- Fixed `MinutesPerDay`, `MinutesPerWeek`, and `HoursPerMonth` parsing from XER files (v6.0.0).
- Addressed issue where `TaskLink.LagFormat` did not receive the correct `Duration.TimeUnit`.
