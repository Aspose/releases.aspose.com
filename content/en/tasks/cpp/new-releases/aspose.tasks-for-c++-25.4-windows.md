#### Release Highlights

Aspose.Tasks for C++ 25.4 for Windows delivers precise control over Gantt chart legend bars, improved calendar-based duration calculations, and customizable timescale fitting. Also resolves critical issues in Primavera P6 XML export and TaskUsage rendering for Microsoft Project compatibility.

#### New Features

- **Custom Gantt Legend Bars**: Use `LegendItems` to define specific bar types in PDF or image exports.
- **ICalendar Interface**: Perform date/time calculations based on merged resource-task calendars.
- **Timescale Fit Behavior**: Align timescale rendering with new `TimescaleFitBehavior` enum.
- **Legend API**: New `PageLegendItem` type for enhanced legend bar labeling.

#### Enhancement

- Support for end-of-page timescale control using `TimescaleFitBehavior`.
- Calendar intersection for precise working hours via `Calendar.GetIntersectionCalendar`.
- Cancellation tokens now supported in `LoadOptions` and `LevelingOptions`.

#### Fix

- Primavera P6 XML exports now import correctly.
- Summary task assignment export fixed for Primavera formats.
- Resolved rendering errors for TaskUsage views and final timescale page areas.
- Addressed manual/auto task scheduling inconsistency.

#### Public API Additions

- **Types**: `ICalendar`, `PageLegendItem`
- **Methods**: Calendar intersection, working time calculations, Gantt legend controls
- **Properties**: `SaveOptions.LegendItems`, `SaveOptions.TimescaleFitBehavior`
- **Enums**: `TimescaleFitBehavior` additions for end alignment control
