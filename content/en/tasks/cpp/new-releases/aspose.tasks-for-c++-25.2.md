#### Release Highlights

Aspose.Tasks for C++ 25.2 (Bundled) improves timephased and baseline data accuracy across all supported platforms. This update ensures consistent visual and calculation behavior between exported MPP files and Microsoft Project.

#### Fixes

- Baseline work now reflects correct assignment values after baseline setting.
- Fixed inaccuracies in timephased data start/finish markers.
- MPP files now display zero-value cost data correctly in MS Project.
- Task start dates without time values default to `Prj.DefaultStartTime`.
- Timephased logic corrected for pre-working-hour assignment starts.
- Cost distribution for timephased and baseline data now behaves predictably.
