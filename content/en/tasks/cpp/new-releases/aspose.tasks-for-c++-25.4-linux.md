#### Release Highlights

Aspose.Tasks for C++ 25.4 for Linux expands developer control with Gantt chart bar legend customization, cross-calendar scheduling via `ICalendar`, and streamlined end-of-page timescale rendering. This release includes major bug fixes for exporting Primavera P6 XML and enhancing TaskUsage output.

#### New Features

- Define legend bar types using `LegendItems` in Linux-exported PDFs.
- Calculate durations using task/resource calendar intersections.
- New `TimescaleFitBehavior` enables precise timescale scaling.
- Use `PageLegendItem` to define custom legend labels.

#### Enhancement

- Render Gantt timescale aligned to page end.
- New `ICalendar` functions for duration, date, and working hour computations.
- Cancellation control through tokens in load/leveling operations.

#### Fix

- Exported Primavera files now validate and import correctly.
- Corrected TaskUsage rendering issues in Linux builds.
- Eliminated overlapping timescale and calendar rendering bugs.

#### Public API Additions

- **Classes**: `ICalendar`, `PageLegendItem`
- **Methods**: Calendar logic, legend bar control
- **Enums**: `TimescaleFitBehavior` with enhanced options
- **Options**: Legend item customization, summary assignment skipping
