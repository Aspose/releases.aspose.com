#### Release Highlights

Aspose.Tasks for C++ 24.12 (macOS) focuses on precision improvements for Gantt chart rendering and file exports. With this update, macOS developers can better manage visual consistency across formats like PDF and XLSX.

#### Enhancements

- **Gantt Bar Label Styling**:  
  Apply custom fonts to task bar text labels in charts.

- **Multi-page Label Logic**:  
  Labels in long timescales now repeat correctly with `RenderLabelOnEachPage`.

- **Image Export Boost**:  
  Enhanced TIFF support and layout accuracy for macOS rendering engines.

#### Fixes

- Addressed incorrect label alignment and missing texts at tier boundaries.
- Prevented addition of unnecessary columns in XLSX outputs.
- Improved gridline visibility and chart accuracy.
- Corrected Baseline2 field handling in formulas.

#### Public API Changes

- **Added**:  
  - `TimescaleTier::RenderLabelOnEachPage`

- **Removed**:  
  - `SaveOptions::LegendOnEachPage`
