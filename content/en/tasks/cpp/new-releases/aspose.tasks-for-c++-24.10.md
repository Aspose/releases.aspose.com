#### Release Highlights

Aspose.Tasks for C++ 24.10 (Bundled) enables graphical indicator configuration and resource availability validation across Windows, Linux, and macOS. Developers can now embed visual logic into extended attributes and ensure schedule correctness using new public types and APIs.

#### New Features

- **Graphical Indicators for Extended Attributes**:  
  Model project visuals using criteria types, value conditions, and icon assignments across all platforms.

- **Resource Scheduling Validation**:  
  Detect errors in availability periods with a purpose-built exception handler.

#### Fixes and Improvements

- Compatibility fixes for MPP and XER properties.
- Accurate lag formatting for task links.
- Initialization improvements in imported project schedules.

#### Public API Additions

- **New Classes**:
  - `GraphicalIndicatorCriteria`
  - `GraphicalIndicatorCriteriaValue`
  - `GraphicalIndicatorsInfo`
  - `ResourceValidationException`

- **New Enumerations**:
  - `GraphicalIndicatorCriteriaType`
  - `FilterComparisonType::IsAnyValue`

- **ExtendedAttributeDefinition API**:
  - `get_GraphicalIndicator()` / `set_GraphicalIndicator()`
