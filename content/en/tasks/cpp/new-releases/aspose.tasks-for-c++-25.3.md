#### Release Highlights

Aspose.Tasks for C++ 25.3 (Bundled) integrates vital Primavera P6 import enhancements, new negative slack capabilities, and broad-spectrum fixes for task scheduling, baseline accuracy, and calendar-driven workflows.

#### Enhancements

- Full support for >10 Primavera activity codes in XER/XML.
- Start/Finish/Total slacks now support negative values.
- Implicit assignment actual work data now correctly exported to MPP.

#### Fixes

- Fixed material resource baseline work TD contributions.
- Correct Task.Resume and Task.Stop values.
- Adjusted remaining work handling after calendar edits.
- Removed duplicate key errors from task linking.
- Safeguarded actual TD when calendar changes.
- No crash when outdenting tasks.
