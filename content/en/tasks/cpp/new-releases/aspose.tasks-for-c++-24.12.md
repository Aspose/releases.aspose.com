#### Release Highlights

Aspose.Tasks for C++ 24.12 delivers consistent visual fidelity for project charts across Windows, Linux, and macOS. Whether exporting to PDF, XLSX, or TIFF, your Gantt chart layout now stays true to settings and styles.

#### Key Improvements

- **Custom Fonts for Gantt Task Labels**:  
  Style task bar texts independently for precise visual presentation.

- **Page-spanning Timescale Labels**:  
  Ensure timescale date labels appear on each page during multi-page rendering.

- **TIFF Format Rendering on *nix**:  
  Enhanced support for exporting to TIFF in non-Windows environments.

#### Bug Fixes

- Timescale alignment and label placement resolved at page borders.
- Removed phantom column from XLSX export at column index 26.
- Gantt bars now render accurately with gridline color and positioning.
- Formula evaluation improved for baseline reference fields.

#### Public API Changes

- **Added Property**:  
  - `TimescaleTier::RenderLabelOnEachPage(bool)`

- **Removed Property**:  
  - `SaveOptions::LegendOnEachPage`
