#### Release Highlights

Aspose.Tasks for C++ 24.10 (Linux) expands project scheduling capabilities by adding graphical indicator APIs and improving MPP and XER file compatibility. It enables Linux developers to apply logic-based visuals and validate availability periods effectively.

#### New Features

- **Extended Graphical Indicators API**:  
  Define indicator rows and logic-based visuals using `GraphicalIndicatorsInfo`, `GraphicalIndicatorCriteria`, and related value types.

- **Resource Availability Validation**:  
  Catch improper scheduling setups via the new `ResourceValidationException`.

#### Fixes and Improvements

- Resource availability period logic corrected for Linux-based workloads.
- XER import now initializes `MinutesPerDay`, `HoursPerMonth`, and related units reliably.
- Task link duration lag format now respects `Duration.TimeUnit`.

#### Public API Additions

- **Classes**:  
  `GraphicalIndicatorCriteria`, `GraphicalIndicatorCriteriaValue`, `GraphicalIndicatorsInfo`, `ResourceValidationException`

- **Enumerations**:  
  `GraphicalIndicatorCriteriaType` (with row types), `FilterComparisonType::IsAnyValue`

- **Method Enhancements**:
  - Multiple constructors for flexible value setup
  - Flags for indicator inheritance in summary rows and project summaries
