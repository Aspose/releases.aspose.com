#### Release Highlights

Aspose.Tasks for C++ 25.1 (macOS) introduces the creation of resource-specific tables and resolves several issues affecting baseline rendering and task duration accuracy after MPP save and reload in Microsoft Project.

#### Enhancement

- **Define Resource Tables**:  
  Customize how resource data is structured and presented within MPP files.

#### Fixes

- Adjusted Stop/Resume behavior for timephased data with trailing zero-work segments.
- Corrected read-in of ActualStart/Finish fields from MPP files.
- Fixed recalculated finish dates turning NA in MS Project.
- Repaired baseline timephased data visuals in exported Gantt charts.
