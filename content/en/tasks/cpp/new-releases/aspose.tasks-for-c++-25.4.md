#### Release Highlights

The ZIP bundle for Aspose.Tasks for C++ 25.4 unifies platform-wide enhancements including Gantt chart legend bar filters, intersection calendar APIs, and streamlined timescale layout rendering. Suitable for deployment across Windows, Linux, and macOS environments.

#### New Features

- Gantt legend customization using `SaveOptions.LegendItems`.
- Perform advanced scheduling calculations via `ICalendar` abstraction.
- Enhanced timescale rendering with `TimescaleFitBehavior` options.
- Summary task assignment control in Primavera exports.

#### Enhancement

- Platform-independent support for timescale alignment and calendar calculation.
- Flexible working time and duration methods for cross-calendar logic.
- API support for legend item filtering and visual alignment.

#### Fix

- Unified fixes for Primavera XML/XER exports across platforms.
- Consistent timescale rendering and legend layout on all systems.
- Improved baseline and assignment rendering.

#### Public API Additions

- **New Types**: `ICalendar`, `PageLegendItem`, `TimescaleFitBehavior`
- **New Methods**: Work duration, timescale behavior, legend rendering control
- **Deprecated**: `FitTimescaleToEndOfPage` (replaced by `TimescaleFitBehavior`)
