Aspose.Tasks for C++ 24.1 delivers various enhancements, focusing on improved data integrity with MPP files and providing a more robust development experience for Linux users.

### Key Highlights

- **Enhanced MPP Management on Linux:** Aspose.Tasks for C++ 24.1 improves how you work with MPP files on Linux, ensuring accurate handling of visual indicators and external task IDs.
- **Bug Fixes and Improvements:** This release equips you with brand-new API methods and properties specifically designed to elevate your project management capabilities on Linux. Additionally, we've added several bug fixes to ensure a smoother and more efficient development experience.

### Added API Members

- Aspose.Tasks.FilterCriteria.IsFieldValue
