#### Release Highlights

Aspose.Tasks for C++ 25.3 (Windows) adds Primavera activity code reading beyond 10 entries, introduces support for negative slack calculation, and improves timephased and baseline behavior across scheduling scenarios.

#### Enhancements

- **Primavera P6 Integration**: Read >10 activity codes from XER/XML formats.
- **Negative Slack Calculations**: Support for Start, Finish, and Total Slack < 0.
- **Actual Work TD Exceptions**: Implicit actual work timephased exceptions now saved to MPP files.

#### Fixes

- Prevented material resources from affecting baseline work TD.
- Corrected Task.Resume and Task.Stop inconsistencies.
- Accurate generation of remaining work after resource calendar changes.
- Resolved key duplication exception when adding task links.
- Preserved actual timephased data when modifying base calendars.
- Fixed exceptions on task outdenting actions.
