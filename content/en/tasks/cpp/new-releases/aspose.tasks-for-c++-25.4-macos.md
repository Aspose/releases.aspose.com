#### Release Highlights

Aspose.Tasks for C++ 25.4 for macOS improves project rendering fidelity with configurable Gantt chart legend bar types, advanced `ICalendar` APIs for scheduling, and refined timescale layout to match visual expectations. This release also resolves Primavera export and view rendering issues.

#### New Features

- Configure which Gantt bar types show in the legend using `LegendItems`.
- Calculate start/end dates across intersecting calendars with `Calendar.GetIntersectionCalendar`.
- Align project timelines to page boundaries with new `TimescaleFitBehavior`.
- Create custom bar labels using `PageLegendItem`.

#### Enhancement

- Support for calendar-based project calculations across task and resource schedules.
- Cancellation tokens available for loading and leveling operations.
- Refined rendering options for page-end layout and TaskUsage clarity.

#### Fix

- Proper import/export of Primavera P6 formats.
- TaskUsage rendering improved on macOS platforms.
- Bug fixes for bar alignment and scheduling edge cases.

#### Public API Additions

- **New**: `ICalendar`, `PageLegendItem`, `TimescaleFitBehavior`
- **Added**: LegendItems API, calendar intersection utilities
- **Fixed**: Timescale gaps, Primavera export corruption, baseline logic
