#### Release Highlights

Aspose.Tasks for C++ 24.12 (Linux) empowers project developers with refined control over visual scheduling exports. Label alignment and Gantt layout are now more accurate in multi-page PDF output and image formats.

#### Enhancements

- **Custom Task Label Fonts**:  
  Added font customization support for task bars in Gantt charts.

- **Cross-page Label Rendering**:  
  Timescale labels now appear on every page using `RenderLabelOnEachPage`.

- **TIFF Generation Enabled on Linux**:  
  Full TIFF support for .NET 6+ environments on *nix platforms.

#### Fixes

- Adjusted multi-page label display for timescale tiers.
- Eliminated unintentional column insertions in exported spreadsheets.
- Restored gridline color and task bar layout precision in Gantt views.
- Formula references to baseline fields are now calculated correctly.

#### Public API Changes

- **Added**:  
  - `TimescaleTier::RenderLabelOnEachPage` (get/set)

- **Removed**:  
  - `SaveOptions::LegendOnEachPage` (get/set)
