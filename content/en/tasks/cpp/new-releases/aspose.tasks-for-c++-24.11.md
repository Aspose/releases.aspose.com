#### Release Highlights

Aspose.Tasks for C++ 24.11 (Bundled) introduces cross-platform features for Windows, Linux, and macOS: remove VBA macros, export task/resource groups, and address key MPP formatting issues.

#### New Features

- **VBA Macro Removal for MPP Files**:  
  Secure project files by removing macros at save-time using `ClearVba` in `MPPSaveOptions`.

- **Group Export for MPP**:  
  Visual and logical grouping support via `GroupAssignments` and `WriteGroups` simplifies large project file rendering.

#### Fixes and Enhancements

- Task formatting, group intervals, and baseline `TimephasedData` fixed across platforms.
- Improved handling of fonts and duplicate field entries in complex MPPs.
