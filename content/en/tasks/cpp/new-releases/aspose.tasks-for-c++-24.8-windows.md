Enhance project scheduling and data accuracy with the latest release of Aspose.Tasks for C++ (version 24.8, Windows). This release is ideal for developers working on Windows-based project management solutions. 

### Task Scheduling Logic

Work with improved accuracy of uncompleted work rescheduling in this release of the project data manipulation C++ API and deliver more reliable project plans for your end users.

### MS Project Compatibility

Any discrepancies between Aspose.Tasks and MS Project scheduling data have been take care of to provide you with a great API usage experience on Windows.

### Enhanced Task Property Calculations

Aspose.Tasks for C++ 24.8 includes corrected Stop/Resume properties and overall task duration values for a more accurate project overview.

### Bug Fixes
- Resolved date discrepancies between Aspose.Tasks and MS Project. (TASKSNET-11173)
- Fixed task Stop/Resume property calculations. (TASKSNET-11249)
- Addressed issues in calculating `ActualDuration`, `Duration`, and `Finish` for summary tasks. (TASKSNET-11252)


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Tasks for C++ 24.8 Release Notes](https://releases.aspose.com/tasks/cpp/release-notes/2024/aspose-tasks-for-cpp-24-8-release-notes/).
