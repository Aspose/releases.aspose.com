This release of Aspose.Tasks for C++ (Windows-only) brings different improvements focusing on enhanced MPP processing, bug fixes, and API advancements designed to streamline development on the Windows platform.

### Key Highlights

- **Improved MPP Processing for Windows Developers:** Aspose.Tasks for C++ 24.1 offers better handling of MPP files on Windows, including preserving visual indicators and accurately managing external task IDs during file operations.
- **API Enhancements and Bug Fixes:** You can leverage new API methods and properties with this release of the C++ API to experience matchless project management on Windows. We have also addressed several bugs that enhance your API usage experience. 

### Public API and Backwards Incompatible Changes

### Added API Members

We have added different new methods and properties in this release which are shared below:
- Aspose.Tasks.FilterCriteria.IsFieldValue
- Aspose.Tasks.FilterCriteria.SetValueField(Aspose.Tasks.Field)
