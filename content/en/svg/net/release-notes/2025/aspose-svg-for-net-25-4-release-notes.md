## Major Features

We are pleased to announce the April release of **Aspose.SVG for .NET 24.5.0**. In this version, we have introduced the new ability to load GZIP-compressed `.svgz` documents, expanding the range of supported formats. This addition improves flexibility and efficiency when working with large or compressed SVG content in .NET applications.

## Enhancements and Fixes

- **New `.svgz` Document Loading**  
  Added support for loading GZIP-compressed `.svgz` files, allowing for reduced file size and faster loading times.
