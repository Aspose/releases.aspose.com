#### Release Highlights

Aspose.SVG for .NET 25.1 (DLLs-only) unlocks SVGZ export, rendering option refinements, and streamlined extension methods ‚Äî ideal for developers embedding compact and optimized SVG processing in custom applications.

### ‚ú≥Ô∏è New Feature

- **Compressed SVGZ Export Support**  
  Developers can now generate `.svgz` files using new `SVGSaveFormat.SVGZ` and extended `SVGDocument.Save()` overloads.

- **Enhanced Rendering Customization**  
  New rendering flags allow toggling antialiasing and font hinting for greater control over image and text quality.

- **More Predictable Event & Animation Extension Methods**  
  API changes to animation (`Accumulate`, `Additive`) and event (`TransformationMatrixChanged`) mechanisms offer cleaner integration paths.

### üõ† Fix

- **Legacy API Cleanup**  
  Outdated rendering properties (`SmoothingMode`, `TextRenderingHint`), font methods, and the `IBitmap` interface have been removed for clarity and maintainability.

### üîß Public API Changes

- **Additions**: `UseHinting`, `UseAntialiasing`, `SVGSaveFormat.SVGZ`, new `SVGDocument.Save()` overloads  
- **Modifications**: Replaced `FontStyle` with `WebFontStyle`; events now use updated delegates  
- **Removals**: `IBitmap`, `ITrueTypeFont.FontData`, `TextRenderingHint`, etc.

### üö® Backwards Incompatible Changes

> ‚ùó Several properties and types were removed in this DLLs-only package. Review usage of font rendering and bitmap APIs to ensure compatibility.
