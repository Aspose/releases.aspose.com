We are excited to announce the release of Aspose.SVG for .NET 24.5.0 (MSI installer)! This update introduces valuable enhancements for developers working with SVG graphics in C# and VB.NET applications.

### Text Overflow Ellipsis Support 

The C# SVG processing API now renders text exceeding container boundaries with ellipsis according to W3C specifications and ensures cleaner SVG to PNG conversions.

### `SkiaSharp` Module Update

The `Aspose.SVG.Drawing.SkiaSharp` module utilizes `SkiaSharp` version 2.88.8 to provide improved stability, performance, and compatibility with the latest rendering advancements.

### `SkiaSharp` Rendering Enhancements

We have improved text rendering accuracy for italic and bold styles within SVGs, guaranteeing a closer match between rendered elements and their specified styles.

### Bug Fixes

Aspose.SVG for .NET 24.5.0 includes bug fixes for `SkiaModule` where page size settings caused inconsistencies when rendering directly to `SKCanvas`. The module now correctly adjusts to the standard 96 dpi resolution when page sizes are set in pixels.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.SVG for .NET 24.5.0 Release Notes](https://releases.aspose.com/svg/net/release-notes/2024/aspose-svg-for-net-24-5-release-notes/).
