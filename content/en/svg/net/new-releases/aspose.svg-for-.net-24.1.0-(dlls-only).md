Aspose.SVG for .NET 24.1.0 (DLLs-only package) includes improved image vectorization, CSS style parsing, and more enhancements for your .NET SVG processing applications. You can use the DLLs package across any platform seamlessly.

### Upgraded `ImageVectorizer` with `SkiaSharp`

The latest SVG API for .NET version delivers an updated `ImageVectorizer` which boasts advanced `SkiaSharp` support and provides more control over image vectorization. This functionality is now fully released within [`Aspose.SVG.Drawing.SkiaSharp`](https://www.nuget.org/packages/Aspose.SVG.Drawing.SkiaSharp/) NuGet package.

### CSS Style Parsing Boost

Experience refined CSS style parsing for a more refined and standards-compliant SVG rendering. With this feature, you can expect visually appealing SVG outputs.

### Improved Bezier Path Building

Aspose.SVG for .NET 24.1.0 offers optimized `Build` methods within `BezierPathBuilder`, `IPathBuilder`, and `SplinePathBuilder` to create efficient SVG paths with minimal segments while preserving the original trace accuracy.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.SVG for .NET 24.1.0 Release Notes](https://releases.aspose.com/svg/net/release-notes/2024/aspose-svg-for-net-24-1-release-notes/).
