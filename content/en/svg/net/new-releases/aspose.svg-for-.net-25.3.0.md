#### Release Highlights

Aspose.SVG for .NET 25.3.0 brings powerful rendering enhancements and introduces support for the CSS `transform-origin` property. These improvements provide developers with better control over SVG transformations, leading to visually accurate outputs across platforms.

- **Support for `transform-origin` in SVG Transformations**  
  Developers can now control the pivot point for SVG element transformations, allowing accurate scaling, rotation, and translation with the newly implemented `transform-origin` property.

- **Improved SVG Rendering Engine**  
  This release enhances color fidelity and rendering precision, ensuring SVG documents are visualized consistently across rendering targets.

- **Refined Transformation Logic**  
  The internal transformation engine has been streamlined for better predictability, benefiting developers working with complex transformation matrices or animations.
