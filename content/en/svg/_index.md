## Why download Aspose.SVG for .NET?

This .NET on-premise API helps you [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) functionality into your C#, VB.NET, ASP.NET & other .NET based Apps.

## SVG File Processing Features

- [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/) format files.
- Convert SVG to other [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- DOM Tree manipulation as per official SVG specs.
- Support for content navigation via [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), CSS Selectors, Element, and Document Traversal features.
- Support for quality rendering.
