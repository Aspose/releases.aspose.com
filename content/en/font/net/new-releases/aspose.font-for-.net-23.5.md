## .NET Standard 2.0 Support

This API release offers the .NET Standard 2.0 (.NET Core 2.0/2.1) support.

It enables this Font manipulation API to help C#, ASP.NET developers to build more flexible and powerful apps to work with the various fonts such as; True Type Font (TTF), True Type Collections (TTC), Scalable Computer Fonts (OpenType), Web Open Font Format (WOFF), Web Open Font Format version 2.0 (WOFF2), Embedded OpenType format (EOT), Scalable Vector Graphics format (SVG), Compact Font Format (CFF), PostScript Fonts (Type1).

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.Font for .NET 23.5 Release Notes](https://releases.aspose.com/font/net/release-notes/2023/aspose-font-for-net-23-5-release-notes/).
