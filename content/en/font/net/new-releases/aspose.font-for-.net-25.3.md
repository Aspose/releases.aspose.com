#### Release Highlights

Aspose.Font for .NET 25.3 delivers deeper control over CFF font programs through an expanded `TopDictDataProvider` API. This release empowers developers building custom font editors, converters, or metadata validators for CFF-based fonts.

### ‚ú≥Ô∏è New Feature

- **Advanced Access to TopDict Metadata**  
  The updated `TopDictDataProvider` now supports additional properties and methods to interact with core CFF font fields like `FontName`, `UnderlineThickness`, `FontBBox`, and `XUID`. These enhancements offer developers precise control over how embedded font metadata is read and manipulated.

### üõ† Fix

- No fixes reported for this release.

### üîπ Public API Changes

Added:
- `Aspose.Font.CffDataProviders.TopDictDataProvider` class  
- New public properties:
  - `UnderlineThickness`
  - `FontMatrix`
  - `CIDFontOperatorPresent`
  - `Xuid`
- New methods:
  - `SetFontNameSid(int sid)`
  - `SetUnderlineThickness(double thickness)`
  - `SetRos(int registrySid, int orderingSid, int supplement)`
  - `GetRos(out int registrySid, out int orderingSid, out int supplement)`

These updates give low-level programmatic access to CFF font internals.

### üö´ Backwards Incompatible Changes

*No breaking changes introduced in this release.*
