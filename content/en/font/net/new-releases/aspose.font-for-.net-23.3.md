## Change Truetype Font Chracter Set Encoding

This release of API has added the feature of converting TrueType fonts using a specific character set.

It means that using this Font API you can change the encoding of a TrueType font to another character set.

This can be useful for a number of reasons, such as:

- To make a TrueType font compatible with a specific language or operating system.
- To reduce the size of a TrueType font for improving the response time of your Web App.
- To improve the performance of a TrueType font.

> For a complete list of features, enhancements, and bug fixes in this release please visit, [Aspose.Font for .NET 23.3 Release Notes](https://releases.aspose.com/font/net/release-notes/2023/aspose-font-for-net-23-3-release-notes/).
