With the latest release of Aspose.Font for .NET (v24.7, DLLs only), you can experience a flexible approach to font manipulation within your C# and VB.NET projects across Windows, Linux, and macOS. This release incorporates enhancements that will refine your font handling capabilities.

### Empower Font Conversion in .NET

The .NET font processing library now includes support for the 'vhea' table in `TrueType` font conversion and guarantees highly accurate results across multiple platforms.

### Broader Functionality

Aspose.Font for .NET 24.7 introduces new properties for the `TtfHheaTable` class to provide greater control over font metrics in your platform-independent .NET solutions.

### Public API and Backwards Incompatible Changes

### Added API Members

- Aspose.Font.TtfTables.TtfHheaTable.Version
- Aspose.Font.TtfTables.TtfHheaTable.MetricDataFormat
- Aspose.Font.TtfTables.TtfHheaTable.CaretOffset
