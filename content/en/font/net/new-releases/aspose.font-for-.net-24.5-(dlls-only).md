The latest Aspose.Font for .NET (DLLs-only package) release is here! Version 24.5 brings exciting new features and improvements to empower your C# and VB.NET development projects.

### Support for .NET 7 Framework

.NET developers can now experience flawless integration with the .NET 7 framework and expand their application's compatibility for modern .NET development. This C# font API release allows you to expand the capabilities of your font processing solutions.

### Optimized WOFF2 to SVG Conversion

We have resolved the issues causing `ArgumentOutOfRange` exception while converting WOFF2 to SVG to ensure smoother font format conversions in .NET.


> You can view the list of all new features, enhancements, and bug fixes introduced in this release by visiting [Aspose.Font for .NET 24.5 Release Notes](https://releases.aspose.com/font/net/release-notes/2024/aspose-font-for-net-24-5-release-notes/).
