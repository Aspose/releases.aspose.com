Seamless access and manipulate font properties within your platform-independent C++ apps using the latest Aspose.Font for C++ release (v24.8). This release helps you simplify font manipulation tasks and improve overall performance. 

### Enhanced Font Metrics Access

Retrieve detailed information about font metrics, including version, metric data format, and caret offset, using the new `TtfHheaTable` properties available in the latest C++ font processing API version.

### Fine-tuned Font Manipulation

You can now experience smoother and more efficient font manipulation operations, thanks to underlying performance optimizations.

### Cross-Platform Compatibility

Aspose.Font for C++ 24.8 is fully compatible with Windows, macOS, and Linux to ensure a consistent development experience across different platforms.

### Public API and Backwards Incompatible Changes

### Added API Members

- Property Aspose::Font::TtfTables::TtfHheaTable::get_Version()
- Property Aspose::Font::TtfTables::TtfHheaTable::get_MetricDataFormat()
- Property Aspose::Font::TtfTables::TtfHheaTable::get_CaretOffset()
