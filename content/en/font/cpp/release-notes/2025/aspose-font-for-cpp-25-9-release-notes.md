## Major Features

Aspose.Font for C++ 25.9 is based on latest [Aspose.Font for .NET 25.9](/font/net/release-notes/2025/aspose-font-for-net-25-9-release-notes/) and ready for use with Aspose C++ products ver 25.9.


Fixed a **CFF** font merging issue where glyphs from a secondary font had the wrong width.
