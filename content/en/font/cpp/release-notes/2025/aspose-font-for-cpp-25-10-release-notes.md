## Major Features

Aspose.Font for C++ 25.10 is based on latest [Aspose.Font for .NET 25.10](/font/net/release-notes/2025/aspose-font-for-net-25-10-release-notes/) and ready for use with Aspose C++ products ver 25.10.


Support for saving **CFF** *CID-keyed* fonts has been added.
