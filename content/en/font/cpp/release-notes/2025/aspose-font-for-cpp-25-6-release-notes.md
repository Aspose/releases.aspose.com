## Major Features

Added support for converting fonts from Compact Font Format (**CFF**) to OpenType font format (**OTF**).

You can use method [Aspose.Font.Font.Convert](https://reference.aspose.com/font/cpp/aspose.font/font/convert/) or 
method [Aspose.Font.Font.SaveToFormat](https://reference.aspose.com/font/cpp/aspose.font/font/savetoformat/) to convert font from **CFF** to **OTF** format.
