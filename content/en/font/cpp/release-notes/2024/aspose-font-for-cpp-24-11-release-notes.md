## Major Features

Aspose.Font for C++ 24.11 is based on the latest [Aspose.Font for .NET 24.11](/font/net/release-notes/2024/aspose-font-for-net-24-11-release-notes/) and ready for use with Aspose c++ products ver 24.11.

Support for **Grid-fitting and Scan-conversion Procedure (gasp)** table in **TrueType** font converter has been added.
