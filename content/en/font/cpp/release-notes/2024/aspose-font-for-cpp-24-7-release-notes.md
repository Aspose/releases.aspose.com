## Major Features

Support for '*vhea*' table in **TrueType** font converter has been added.

For more details, see [Aspose.Font for .NET 24.7](/font/net/release-notes/2024/aspose-font-for-net-24-7-release-notes/).

## Public API and Backwards Incompatible Changes

### Added APIs
* Property Aspose::Font::TtfTables::TtfHheaTable::get_Version()
* Property Aspose::Font::TtfTables::TtfHheaTable::get_MetricDataFormat()
* Property Aspose::Font::TtfTables::TtfHheaTable::get_CaretOffset()
