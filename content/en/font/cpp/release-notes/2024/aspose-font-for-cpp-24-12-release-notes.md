## Major Features

Aspose.Font for C++ 24.12 is based on the latest [Aspose.Font for .NET 24.12](/font/net/release-notes/2024/aspose-font-for-net-24-12-release-notes/) and ready for use with Aspose c++ products ver 24.12.


Support for updating the name and style of a Compact Font Format (**CFF**) font has been added.
