## Major Features

Aspose.Font for C++ 23.10 is based on the latest [Aspose.Font for .NET 23.10](/font/net/release-notes/2023/aspose-font-for-net-23-10-release-notes/) and ready for use with Aspose c++ products ver 23.10.

New functonality for the **Morse code** encoding has been added.
