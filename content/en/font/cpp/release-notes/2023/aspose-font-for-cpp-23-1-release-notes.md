## Major Features

Aspose.Font for C++ 23.1  is based on the [Aspose.Font for .NET 23.1](/font/net/release-notes/2023/aspose-font-for-net-23-1-release-notes/) and ready for use with Aspose c++ products ver 23.1.

Support for converting fonts to `SVG` format has been added.

## Public API and Backwards Incompatible Changes

### Added APIs
* Enumeration member **Aspose.Font.FontSavingFormats.SVG**
