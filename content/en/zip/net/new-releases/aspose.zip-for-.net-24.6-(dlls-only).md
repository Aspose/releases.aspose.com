The standalone DLLs release of Aspose.ZIP for .NET 24.6 is here! This update enables developers to leverage new features and enhancements for their cross-platform .NET file compression and archiving apps and provides a lightweight integration approach.

### Introducing ISO Archive Support

Manage ISO archives with the latest C# archiving library release effortlessly. Our library helps you create, modify, and extract ISO images, broadening your archive management options on your favorite platform.

### Extract ZIP Archives with Implode Method

Implement the ability to extract ZIP archives compressed with the `Implode` method. With this addition, we have expanded the decompression abilities of Aspose.ZIP for .NET library.

### Manage Zstandard Archives Easily

The introduction of `Save` methods by path optimizes the functionality of managing Zstandard archives. This addition lets you develop feature-rich archive processing apps boasting upgraded file handling.

### Bug Fixes

This release addresses different bugs, including incorrect commentaries for XAR archives and issues encountered during ZIP extraction using the Implode compression method.
