#### Release Highlights

Aspose.ZIP for .NET 25.1 expands archive format support by adding ARJ extraction and refines the handling of LHA metadata while improving .NET compatibility.

### ‚ú≥Ô∏è New Feature

- **ARJ Archive Extraction**:
  - Introduced full support for `.arj` files with the new `ArjArchive` and `ArjEntryPlain` classes.
  - Key APIs:
    - `ArjArchive.Entries`, `Name`, and `Commentary`
    - `ArjArchive.ExtractToDirectory(string)`
    - `ArjEntryPlain.Extract()`, `Name`, `CompressedSize`, and `UncompressedSize`

This enables extraction of legacy ARJ-compressed content, widely used in older DOS/Windows environments.

### üîß Enhancement

- **Improved LHA Metadata**:
  - `LhaArchiveEntry` now includes `IsDirectory` and `LastModified` for better file system integration.
  - Removed obsolete `LhaDirectoryEntry` to simplify API.

- **.NET Framework Cleanup**:
  - Removed superfluous dependencies for .NET Framework 4.6.1+, reducing application size and improving compatibility.

### üõ† Fix

- **Stream Length Guard for LHA**:
  - Prevents errors by avoiding extraction from invalid or too-short LHA archive streams.
