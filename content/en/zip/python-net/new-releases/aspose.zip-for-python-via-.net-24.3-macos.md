This release of Aspose.ZIP for Python via .NET (v24.3) empowers MacOS developers with Zstandard archive format support and broadened CPIO archive features.

### Zstandard Compression on MacOS

Integrate the Zstandard compression functionality into your Python applications on macOS and achieve exceptional compression efficiency. Zstandard format is ideal for large datasets and allows you to optimize your compression workflows.

### Enhanced CPIO Management

Easily manage CPIO archives using different compression methods, such as LZIP, LZMA, and Zstandard, with this release of the Python ZIP archive library. CPIO archives support greater control over archive size and performance in your macOS-based applications.
