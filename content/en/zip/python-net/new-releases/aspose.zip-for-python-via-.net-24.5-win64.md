We are thrilled to announce the release of Aspose.ZIP for Python via .NET 24.5 (Windows x64)! This update focuses on augmenting performance and optimizing resource usage for specific archive formats.

### Improved LZMA and XZ Compression and Decompression

Experience faster processing times for TAR.LZMA and CPIO.LZMA archives with the introduction of pipeline streams in the latest version of the Python file compression API. This update enables you to zip files faster on Windows 64-bit systems.

### Evaluation Version Limits Adjusted

We have adjusted the evaluation version limitations to ensure optimal usage patterns for our users. You can test and review the Python API with a streamlined trial option.
