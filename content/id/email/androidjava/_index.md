### Email Fitur Android

- Buka atau simpan email di Microsoft Outlook & format lainnya.
- Konversi file email ke berbagai format.
- Parsing, baca & simpan email MS Outlook, file PST & OST.
- Ekstrak objek dari berbagai format penyimpanan email.

### Membaca & Menulis Format Email

**Microsoft Outlook速:** MSG, PST, OST, OFT\
**Email:** EML, EMLX, MBOX\
**Lainnya:** ICS, VCF, HTML, MHTML

### Baca Format Email

**Mac Outlook速:** OLM

### Lingkungan yang Didukung

- Setiap platform tempat lingkungan Android Runtime diinstal.
- OS Android 2.0 atau lebih tinggi.

> Aspose.Email untuk Android melalui Java tidak perlu menginstal Microsoft Outlook速.
