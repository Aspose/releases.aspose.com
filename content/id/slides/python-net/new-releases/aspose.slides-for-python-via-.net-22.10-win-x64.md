# Set bingkai rendering & perilaku rotasi

Kemampuan untuk menentukan dan mengontrol perilaku rotasi bingkai dan ukuran bingkai sambil memberikan bentuk dari `pptx` dan format presentasi yang didukung lainnya.

# Kompres font tertanam untuk mengurangi ukuran presentasi

Jika presentasi Anda berisi font tertanam maka rilis presentasi kami pemrosesan API kami menawarkan kemampuan untuk melakukan kompresi dari semua font tertanam tersebut dan pada gilirannya mengurangi ukuran file presentasi seperti `pptx`.

# Set Presentation Leader Lines Color

Rilis API ini memperkenalkan kemampuan untuk mengatur warna garis pemimpin dalam slide presentasi Anda.

> Untuk daftar lengkap fitur, peningkatan, dan perbaikan bug di rilis ini silakan kunjungi, [Aspose.slides untuk catatan rilis Python 22.10] (https://docs.aspose.com/slides/python-net/aspose-slides- for-python-net-22-10-release-notes/).
