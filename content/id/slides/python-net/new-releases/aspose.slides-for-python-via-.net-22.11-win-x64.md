# Ekspor PDF 1.6 & PDF 1.7 Dukungan

Dukungan telah diimplementasikan untuk PDF untuk diekspor ke format PDF 1.6 dan PDF 1.7.

# Ekspor PDF dengan A2A, Tingkat Kepatuhan A2B

Rilis API Python ini memperkenalkan dukungan fitur ekspor PDF dengan berbagai tingkat kepatuhan, yaitu, A2A, A2B, A2U, A3A, dan A3B.

# Kelola suara hyperlink dalam presentasi

Properti baru ([`hyperlink.sound`] (https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/)) telah ditambahkan yang mewakili suara hyperlink yang diputar.

# Efek animasi menghentikan suara sebelumnya

Properti baru ([`Effect.stop_previous_sound`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/effect/)) telah diperkenalkan yang menentukan jika efek animasi menghentikan yang sebelumnya menghentikan yang menghentikan hal suara.
