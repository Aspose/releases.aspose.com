# Dukungan yang ditingkatkan untuk Ubuntu Focal

Diamati bahwa saat mengubah Microsoft PowerPoint & Reg; `Pptx` ke` pdf` format Beberapa masalah muncul di Ubuntu Focal. Masalah -masalah itu sekarang telah diselesaikan.

Dukungan bagan kolom # 3D

Rilis API ini memperkenalkan dukungan dari grafik kolom 3D. Sekarang Anda dapat menambahkan grafik dengan data default dan kemudian mengatur properti `rotation3d` (seperti sumbu sudut kanan, persen kedalaman, dll.).

# Peningkatan dukungan efek teks bahasa Ibrani

Sebelumnya diamati bahwa dalam beberapa kasus bayangan teks Ibrani tidak benar ketika presentasi `pptx` dikonversi menjadi format` pdf`. Masalah ini sekarang telah diselesaikan.
