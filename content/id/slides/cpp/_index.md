## Mengapa mengunduh Aspose.Slides untuk C++?

Pustaka kelas C++ mandiri ini membantu Anda membuat, membaca, menulis, mengedit & mengonversi presentasi Microsoft PowerPoint® tanpa memerlukan PowerPoint® atau Office Automation.

## Fitur Pemrosesan Presentasi

- Buat, buka, dan kerjakan presentasi tanpa Microsoft PowerPoint®.
- Konversi presentasi ke salah satu format file yang didukung.
- Menambahkan, memformat, dan memanipulasi bagan, bentuk, slide, SmartArt, tabel, dan teks dalam presentasi.

### Membaca & Menulis File PowerPoint®

**Microsoft PowerPoint®:** PPT, POT, PPS, PPTX, POTX, PPSX, PPTM, PPSM, POTM\
**OpenDocument:** ODP, OTP\
**Metafile:** EMF\
**Gambar:** TIFF, XML

### Simpan Presentasi Sebagai

**Tata Letak Tetap:** PDF, XPS\
**Gambar:** JPEG, PNG, GIF, BMP, SVG\
**Web:** HTML
