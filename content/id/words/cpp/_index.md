Pustaka kelas C++ mandiri untuk memuat, menyimpan, memformat & mengonversi dokumen Microsoft Word® tanpa Office Automation.

## Mengapa mengunduh Aspose.Words untuk C++?

Aspose.Words untuk C++ adalah API on-premise yang kuat yang dapat digunakan untuk berbagai tugas pemrosesan dokumen seperti; bekerja dengan dokumen, bagian, tabel, bookmark, bidang, bidang formulir, DocumentBuilder, rentang, komentar, kontrol konten SDT, gambar, gaya, bagan, tanda air, daftar, bentuk, gabungan surat, rendering, pencetakan, dan banyak lagi.

## Fitur Pemrosesan File Kata

- Panggil konstruktor Dokumen tanpa parameter untuk membuat dokumen kosong baru secara terprogram.
- Deteksi format file dan periksa kompatibilitas format file.
- Konversi Dokumen Word ke format lain yang didukung.
- Serialisasi objek dokumen untuk mengambil array byte dan sebaliknya.
- Mendukung kepatuhan konversi dengan berbagai standar PDF.
- Ekspor font ke HTML dalam penyandian `Base64`.
- Simpan dokumen kata terenkripsi kata sandi apa pun.
- Tentukan spesifikasi OOXML.
- Terapkan kata sandi ke dokumen terenkripsi dan abaikan data `RoutingSlip` saat menyimpan.
- Kompres semua metafile, baik kecil atau besar.
- Aktifkan tanda dua arah untuk menambahkan dukungan untuk bahasa kanan ke kiri.
- Akses Proyek VBA untuk memperluas fungsionalitas.
- Membaca, menulis, atau memodifikasi Makro VBA.
- Proyek Klon VBA atau Modul VBA.

### Membaca & Menulis Format Dokumen

**Microsoft Word®:** DOC, DOT, DOCX, DOTX, DOTM, FlatOpc, FlatOpcMacroEnabled, FlatOpcTemplate, FlatOpcTemplateMacroEnabled, RTF, WordML\
**OpenDocument:** ODT, OTT\
**Web:** HTML, MHTML\
**Penurunan harga:** MD\
**Tata Letak Tetap:** PDF\
**Teks:** TXT

### Baca Format Dokumen

**Microsoft Word®:** DocPreWord60\
**eBuku:** MOBI, CHM

### Simpan Dokumen Sebagai

**Tata Letak Tetap:** XPS, OpenXps\
**PostScript:** PS\
**Printer:** PCL\
**eBuku:** EPUB\
**Markup:** XamlFixed, HtmlFixed, XamlFlow (Beta), XamlFlowPack (Beta)\
**Gambar:** SVG, TIFF, PNG, BMP, JPEG, GIF\
**Metafile:** EMF
