# Manajemen Memori Lebih Baik

Kebocoran memori diamati saat mengonversi AutoCAD® DXF ke format TIFF (grafik raster). Jadi rilis API ini menawarkan manajemen memori yang lebih baik untuk Konversi DXF ke TIFF.

# Peningkatan Ekspor DWG ke WMF

Meningkatkan ekspor AutoCAD® `DWG` ke `WMF` (Windows Metafile).

# Ekspor Gambar Tersemat

Kemampuan untuk mengekspor gambar dari data gambar yang disematkan.

# Rendering `DWG` yang ditingkatkan

Diamati bahwa `DWG` dirender tanpa warna. Masalah ini sekarang telah diselesaikan dalam rilis API ini.
