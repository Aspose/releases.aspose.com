## Mengapa mengunduh Aspose.Font untuk C++?

Aspose.Font for C++ adalah C++ API yang menawarkan perpustakaan kode C++ yang tersedia untuk pemrosesan dan manipulasi font. API kami memungkinkan aplikasi C++ Anda memuat, mengedit, dan menyimpan data font serta merender Glyph atau teks yang diinginkan.

## Fitur Pemrosesan Font

- Muat file font dari disk serta streaming.
- Baca informasi font dan simpan file font yang diperbarui ke disk.
- Dukungan untuk *TrueType* dan *OpenType* Font.
- Bekerja dengan font CFF & Type1 di Aplikasi C++ Anda.
- Baca informasi Glyphs dan Metrik dari file Font.
- Mendeteksi Simbol Latin di Font.
- Ekstrak informasi lisensi yang disematkan dari file font.
- Render teks menggunakan font Glyphs.

### Membaca & Menulis Format Font

TTF

### Baca Format Font

TTC, Tipe Terbuka, CFF, Tipe1, WOFF/WOFF2

### Platform yang Didukung

Aspose.Font untuk C++ adalah pustaka asli. Ini mendukung sistem operasi `32-bit` serta `64-bit`. Selain itu, Anda dapat menggunakan Aspose.Font untuk C++ untuk mengembangkan aplikasi di lingkungan pengembangan apa pun yang mendukung C++, tetapi lingkungan yang terdaftar secara eksplisit didukung.

**Microsoft Windows®:** Microsoft Windows desktop (XP, Vista, 7, 8, 10) dan sistem operasi server (2003, 2008, 2012), dll.
**Lingkungan Pengembangan:** Microsoft Visual C++ versi 2017 atau lebih tinggi.
