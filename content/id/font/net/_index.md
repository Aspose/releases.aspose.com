## Mengapa mengunduh Aspose.Font untuk .NET?

Aspose.Font untuk .NET adalah pustaka yang memungkinkan aplikasi .NET Anda memuat, mengedit, dan menyimpan data font. Aspose.Font untuk .NET juga memungkinkan aplikasi .NET Anda menggambar teks dengan font yang ditentukan.

## Fitur Pemrosesan Font

- Muat file font dari disk serta streaming.
- Baca informasi font dan simpan file font yang diperbarui ke disk.
- Baca informasi Glyphs dan Metrik dari file Font.
- Mendeteksi Simbol Latin di Font.
- Ekstrak informasi lisensi yang disematkan dari file font.
- Render teks menggunakan mesin terbang font.
- Render teks menggunakan antarmuka khusus.

### Membaca & Menulis Format Font

TTF

### Baca Format Font

TTC, Tipe Terbuka, CFF, Tipe1, WOFF/WOFF2
