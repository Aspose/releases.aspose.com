> Ini adalah perpustakaan kelas mandiri yang memungkinkan interaksi dengan dokumen Microsoft OneNote® untuk pemrosesan dan konversi.

## Mengapa mengunduh Aspose.Note untuk .NET?

Aspose.Note untuk .NET dapat digunakan untuk mencetak SATU dokumen serta manipulasi halaman, gambar, teks, tabel, lampiran, tag, tugas, gaya teks, dan hyperlink, tanpa memerlukan Microsoft OneNote®.

## Fitur Pemrosesan File Microsoft OneNote®

- Muat, edit, dan simpan dokumen Microsoft OneNote® melalui API.
- Menavigasi melalui OneNote® Document Object Model (DOM).
- Sisipkan gambar ke dalam file OneNote®.
- Parsing dan ekspor berbagai format daftar bernomor.
- Ekstrak teks dari bagian mana pun dari dokumen OneNote®.
- Ekspor dokumen OneNote® sebagai format populer lainnya.

### Baca & Tulis Format OneNote®

**Microsoft OneNote®:** SATU

### Simpan File OneNote® Sebagai

**Tata Letak Tetap:** PDF\
**Gambar:** PNG, GIF, JPEG, BMP, TIFF

### Baca Format

**Microsoft OneNote®:** ONETOC2
