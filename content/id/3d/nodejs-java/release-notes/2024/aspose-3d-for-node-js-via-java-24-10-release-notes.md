## **Peningkatan dan Perubahan**

|**Key**|**Ringkasan**|**Kategori**|
| :- | :- | :- |
| THREEDNET-1601 | Data bGP tidak didukung dan array tidak teratasi dalam berkas blender. | Tugas |
| THREEDNET-1594 | Dukungan profil garis tengah | Fitur Baru |
| THREEDNET-1603 | Tambahkan dukungan Boolean dalam IFC | Peningkatan |
| THREEDNET-1605 | Transformasi adegan IFC yang diimpor tidak benar | Perbaikan Bug |
| THREEDNET-1606 | Berkas IFC dengan IfcRevolvedAreaSolid tidak dapat menghasilkan jaring yang benar | Perbaikan Bug |
| THREEDNET-1614 | Tingkatkan kompatibilitas IFC | Tugas |

## Perubahan API ##

### Ditambahkan kelas **com.aspose.threed.CenterLineProfile**

Profil ini memungkinkan Anda memberikan ketebalan pada kurva, yang dapat digunakan dengan fasilitas pemodelan prosedural seperti LinearExtrusion.
