# Dukungan Operator `XFORM` Kustom

Menambahkan dukungan operator `xform` khusus dalam pengimpor USD / USDZ.

# Konversi Model 3D FBX ke USDZ yang ditingkatkan

Sebelumnya Model yang dikonversi dari FBX ke USDZ berubah menjadi serba hitam. Masalah ini telah diselesaikan dalam rilis API saat ini.

# Ekspor Atribut Material yang Ditingkatkan

Atribut material tidak diekspor saat tidak ada tekstur yang dilampirkan. Masalah ini sekarang telah diselesaikan.
