# Peningkatan pemuatan format 3D

API 3D melempar pengecualian saat memuat file `GLTF`,` X`, dan `VRML`. Format 3D ini sekarang sepenuhnya kompatibel dan dimuat tanpa masalah.

# Tangent/Binormal Recognition di GLB

Sebelumnya dalam beberapa kasus tangen/binormal yang didefinisikan dalam `GLB` dengan draco diaktifkan tidak diakui. Masalah ini telah diselesaikan dalam rilis API ini.

# Peningkatan konversi format 3D

Perbaikan telah dilakukan untuk konversi format 3D berikut:

- ** ASE ke 3DS ** Autodesk & Reg; Ascii Scene Export (ASE) Format file ke konversi format studio 3D.

- ** ase to obj ** Autodesk & Reg; Format file Ekspor Adegan ASCII (ASE) untuk OBJ Format Konversi.

- ** USD ke DRC ** Format file deskripsi universal Pixar (USD) untuk format Google Draco (DRC).

> Untuk daftar lengkap fitur, peningkatan, dan perbaikan bug di rilis ini, silakan kunjungi, [Aspose3d untuk catatan rilis Net 231](https://releases.aspose.com/3d/net/release-notes/2023/aspose-3d-for-net-23-1-release-notes/).
