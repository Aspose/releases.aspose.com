# Encode / decode lampu

Versi API 3D ini memungkinkan pengkodean dan penguraian cahaya arah serta cahaya spot.

# Peningkatan penanganan silinder 3D

Masalah berikut telah diselesaikan dalam rilis ini:

- ** ThreedNet-1294 ** Cylinder Defined UV Tapi Tidak Ada Data

- ** ThreedNet-1306 ** Silinder dengan Radius Top = 0 Akan Menghasilkan Model Salah

# Konversi dari `dxf` ke` gltf`/`glB`

Kemampuan untuk mengubah file `dxf` menjadi format` GLTF` atau `GLB`. GLTF (Format Transmisi GL) digunakan sebagai format file standar untuk bekerja dengan adegan dan model 3D.

Harap dicatat bahwa file `GLTF` dapat memiliki ekstensi file` GLTF` atau `GLB`.
