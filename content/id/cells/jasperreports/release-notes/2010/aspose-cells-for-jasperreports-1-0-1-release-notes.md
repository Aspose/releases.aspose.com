 Halaman ini berisi catatan rilis untuk[Aspose.Cells for JasperReports 1.0.1](https://releases.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.0.1/)

{{% /alert %}}

Kami dengan senang hati mengumumkanAspose.Cells for JasperReports 1.0.1!

Apa yang baru:

- Ukuran font disesuaikan dengan tepat untuk menempelkan karakter dalam format Excel Microsoft serupa dengan karakter dalam format PDF.
-  Label sumbu (Bagan) dalam format tanggal ditampilkan dengan benar dan sesuai.
- Efek Isian Gradien untuk area didukung termasuk latar belakang bagan (area bagan, area plot, dll.), latar belakang legenda, batang dan gelembung, dll.

Apa yang diperbaiki:

- Memperbaiki beberapa bug pengisian dinding dan lantai grafik 3D.
- Tidak dapat merender objek yang disematkan: File tidak ditemukan.

- Memperbaiki bug pada label diagram lingkaran.

- Memperbaiki bug terkait dukungan jasperserver 3.1.0.
