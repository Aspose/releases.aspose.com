 Pelanggan yang terhormat,

 Kami telah merilis Aspose.Cells for Java V1.7.3! Penyempurnaan dan perbaikan berikut disertakan:

- Mendukung untuk menyisipkan/menghapus rentang.
 Mendukung untuk membuka dan menyimpan file Excel terenkripsi.
 Meningkatkan metode Cell.getStringValue() untuk mendukung lebih banyak format tanggal/angka.
 Memperbarui demo web.
 Mendukung untuk menghitung rumus dengan JDK1.4.
 Memperbaiki bug metode License.setLicense().
 Memperbaiki bug pengaturan format angka khusus.
 Memperbaiki bug saat menghapus lembar kerja dari buku kerja.
