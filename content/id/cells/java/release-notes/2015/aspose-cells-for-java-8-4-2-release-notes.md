 Catatan

 Karena basis kode Aspose.Cells for Java cocok dengan kode versi .NET yang relevan, sebagian besar perubahan, penyempurnaan, dan perbaikan yang disertakan dalam Aspose.Cells for .NET v8.4.2 juga disertakan dalam Aspose.Cells for Java v8.4.2 ini.
