Kita
 dengan senang hati mengumumkan Aspose.Cells for Java v7.1.2!

 Fitur baru

 – Mendukung metode Cells.importCustomObjects() untuk mengimpor POJO

 Peningkatan

- Penyempurnaan dilakukan untuk metode autoFitColumns()
- Performa untuk fitur Sheet-to-Image dengan garis grid aktif

 Pengecualian

- NullPointerException terjadi ketika memanggil metode Cell.getDisplayStyle()

Bug

- Nilai maksimum untuk bagan Sebar tidak dapat disetel dengan benar
- Teks dalam bentuk hilang atau dipindahkan ke luar rentang bentuk dalam file PDF yang dihasilkan
- Nama seri tidak ditampilkan untuk Label Tanggal di bagan
