 Halaman ini berisi catatan rilis untuk[Aspose.Cells for Java 7.1.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.1/)

{{% /alert %}} 

Kita
 dengan senang hati mengumumkan Aspose.Cells for Java v7.1.1!

 Fitur baru

- Periksa apakah kolom kosong

 Peningkatan

- Menyimpan ke XLSX dengan LightCells API dengan merender langsung ke OutputStream

 Pengecualian

- Metode Cell.setHtmlString() menampilkan NullPointerException
- Membaca file templat XLSX melemparkan NullPointerException

Bug

- Sumbu kategori sekunder bagan SCATTER menjadi sumbu teks
- Konten di Kotak Teks belum dirender dengan benar
- Fungsi ROUND tidak dapat membulatkan nilai yang lebih besar dari 922337.20
- Bagan tidak terbaca dengan benar dari file ODStemplate
- Format Nomor Khusus:"dd.MM.yyyy" tidak disimpan dengan benar untuk file ODS
- LineShape menjadi lebih pendek dalam dokumen PDF yang dihasilkan
