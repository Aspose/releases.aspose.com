Aspose.Cells untuk .NET adalah API Pemrograman Spreadsheet Excel® untuk mempercepat pengelolaan dan pemrosesan spreadsheet. Dukungan Excel® .NET API untuk membangun aplikasi lintas platform yang memiliki kemampuan untuk menghasilkan, memodifikasi, mengonversi, merender, dan mencetak spreadsheet. Ini memungkinkan pengembang untuk mengelola lembar kerja, baris, kolom & sel, membuat konten dan gaya spreadsheet dari awal, mengimpor data ke lembar kerja dari sumber data yang berbeda, menambahkan rumus matematika, keuangan, dan teks yang umum dan kompleks, membuat & memanipulasi tabel pivot, bagan , hyperlink, komentar, menggambar objek, dan banyak lagi.

Aspose.Cells untuk .NET menyediakan solusi Grid dengan dua kontrol .NET berbasis GUI. Aspose.Cells.GridDesktop mendukung aplikasi desktop dan Aspose.Cells.GridWeb dirancang khusus untuk aplikasi web berbasis .NET. Kedua kontrol Gird memungkinkan Anda untuk mengimpor/mengekspor file Excel®, memanipulasi data & pemformatan, menyesuaikan desain dan tata letak kisi, mengelola beberapa lembar kerja, membuat dan menghitung rumus Excel®, dan berbagai operasi serupa Excel lainnya.

## Mengapa mengunduh Aspose.Cells untuk .NET?

Aspose.Cells untuk .NET memungkinkan aplikasi .NET, C#, ASP.NET, dan VB.NET Anda untuk membuat, memformat, merender, mencetak, dan mengonversi spreadsheet Microsoft Excel® dan OpenDocument (XLS, XLSX, ODS, dll.) ke berbagai format yang didukung, tanpa perlu menginstal Microsoft Excel® atau perangkat lunak pihak ketiga apa pun. Mesin kalkulasi rumusnya yang kuat mendukung berbagai rumus/fungsi MS Excel® standar dan lanjutan.

> Aspose.Cells untuk .NET ditulis dalam C# terkelola dan dapat digunakan dalam segala jenis Aplikasi Desktop Windows berbasis .NET atau Aplikasi Web ASP.NET. Pustaka kami juga dapat digunakan dengan ASP klasik, dan PHP.

Model objek kayanya terdiri dari 100+ kelas publik yang menawarkan pengembang .NET untuk mengintegrasikan fitur manipulasi spreadsheet yang menarik ke dalam aplikasi mereka, mulai dari membuat bagan pivot, menggabungkan spreadsheet, mengurai data, hingga mengenkripsi dan mengekspor file spreadsheet. Render halaman spreadsheet ke gambar raster (TIFF, TIFF multihalaman, PNG, JPEG, BMP) atau vektor (EMF). Aspose.Cells untuk .NET juga mendukung bekerja dengan XLSB, XLT, XLTX, XLTM, XLSM, XML, HTML, CSV, TSV, HTML/MHTML, dll. Ia mampu mengonversi spreadsheet ke format PDF dan XPS sambil mempertahankan visual tertinggi kesetiaan. Anda dapat menambahkan bagan standar dan yang disesuaikan serta mengatur data secara dinamis untuk menyegarkan bagan. Grafik juga dapat dikonversi ke format gambar PDF dan raster berkualitas tinggi.

## Fitur Pemrosesan File Spreadsheet

- Pembuatan & manipulasi spreadsheet melalui API.
- Konversi & rendering format file berkualitas tinggi.
- Cetak file Microsoft Excel® ke printer fisik atau virtual.
- Menggabungkan, memodifikasi, melindungi, atau mengurai lembar Excel®.
- Terapkan format lembar kerja.
- Konfigurasi dan terapkan pengaturan halaman untuk lembar kerja.
- Buat & sesuaikan bagan Excel®, Tabel Pivot, aturan pemformatan bersyarat, pemotong, tabel & garis percikan.
- Konversi grafik Excel® ke gambar & PDF.
- Konversi file Excel® ke berbagai format lain.
- Mesin kalkulasi rumus yang mendukung semua fungsi Excel® dasar dan lanjutan.

### Format Baca & Tulis yang Didukung

**Microsoft Excel®:** XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM, CSV, TSV, TabDelimited, SpreadsheetML\
**OpenOffice®:** ODS, SXC, FODS\
**Teks:** TXT\
**Web:** HTML, MHTML\
**iWork®:** Angka\
**Lainnya:** SXC, FODS

### Simpan File Excel® Sebagai

**Microsoft Word®:** DOCX\
**Microsoft PowerPoint®:** PPTX\
**Microsoft Excel®:** XLAM\
**Tata Letak Tetap:** PDF, XPS\
**Pertukaran Data:** DIF\
**Grafik Vektor:** SVG\
**Gambar:** TIFF,PNG, BMP, JPEG, GIF\
**Metafile:** EMF\
**Penurunan harga:** MD
