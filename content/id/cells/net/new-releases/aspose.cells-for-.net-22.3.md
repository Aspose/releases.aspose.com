#  Dukungan untuk Format Templat Spreadsheet

Rilis API ini telah memperkenalkan dukungan untuk format templat spreadsheet untuk Excel&reg; (XLT) & OpenOffice Calc&reg; (OTS) program. Ini didukung sebagai format output melalui `SaveFormat.Ots` & `SaveFormat.Xlt`.

#  Tetapkan Batas Dalam

Kemampuan untuk mengatur batas dalam.

#  Urutkan PivotField per Baris atau Kolom yang Dipilih

Dukungan pengurutan untuk nilai Pivot Field telah ditambahkan. Penyortiran Pivot Field dapat dilakukan berdasarkan baris atau kolom yang diinginkan.
