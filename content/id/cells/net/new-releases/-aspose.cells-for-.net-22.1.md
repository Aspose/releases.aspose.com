#  Tetapkan Nama Pekerjaan Pencetak

Menambahkan dukungan untuk menyediakan dan mengatur nama pekerjaan pencetakan dengan `PrinterSettings` saat merender spreadsheet ke printer.

#  Deteksi otomatis Format File `CSV` dan `TSV`

Rilis API ini memperkenalkan kemampuan untuk mendeteksi secara otomatis file teks sebagai format file `CSV` yang dipisahkan koma dan `TSV` yang dipisahkan tab menggunakan API.

#  Kecualikan Baris/Kolom Tersembunyi dari Rentang

Properti range mencakup semua sel dalam koleksi sel. Dari rilis ini fungsi ini telah ditingkatkan untuk mengabaikan/mengecualikan baris dan kolom tersembunyi saat mendapatkan `Cells.MaxDisplayRange`.

 > Untuk daftar lengkap fitur, penyempurnaan, dan perbaikan bug pada rilis ini, silakan kunjungi,[Aspose.Cells for .NET 22.1 Catatan Rilis](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-1-release-notes/).
