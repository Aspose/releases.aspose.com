#  `.NUMBERS` Dukungan Penyimpanan Data 5.0

Menambahkan dukungan penyimpanan data 5.0 sebesar `.numbers`.

#  Peningkatan Rendering XLSX menjadi PDF

Terlihat bahwa saat mengonversi XLSX menjadi PDF, diagram batang tidak ditampilkan dengan benar. Masalah ini kini telah teratasi.

#  Peningkatan Excel&reg; Penanganan Label Bagan

Rilis API ini menawarkan beberapa perbaikan mengenai Excel&reg; label bagan. Itu adalah sebagai berikut:

- Label bagan tidak pas dengan kotaknya.
- Label bagan disejajarkan secara vertikal, bukan horizontal.

#  Simpan Rumus Tabel ke XLSB

Dukungan telah ditambahkan untuk menyimpan rumus tabel ke `XLSB`.

#  Peningkatan Dukungan di Linux

Terlihat bahwa karakter atau keterangan yang ditulis pada objek seperti bentuk dan kotak teks tampak kacau di Linux. Masalah ini kini telah teratasi.
