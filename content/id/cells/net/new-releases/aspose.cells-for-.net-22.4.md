# Peningkatan Rendering Bentuk WordArt

Diamati bahwa Microsoft Excel&reg; Efek dan bentuk WordArt tidak ditampilkan dengan benar selama konversi Excel ke PDF. Masalah ini kini telah teratasi.

#  Peningkatan Konversi XLS menjadi PDF

Sebelumnya dalam beberapa kasus, ketika kita mengonversi `XLS` menjadi `PDF`, ukuran baris teks kosong di kotak teks tidak ditampilkan dengan benar. Masalah ini kini telah teratasi.

#  Hapus Trailing Blank Cells dari CSV

Rilis API ini memperkenalkan kemampuan untuk menghapus sel kosong tailing sambil menyimpan file Comma Separated Values (CSV).
