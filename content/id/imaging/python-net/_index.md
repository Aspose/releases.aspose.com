Fitur API pencitraan ###

Berikut ini adalah Aspose. Fitur Inti Pencimaging:

- Buat gambar
- Baca gambar
- Tulis gambar
- Gambar gambar
- Proses gambar (termasuk modifikasi per-piksel)
- Konversi di antara format
- CROP, ROTATE & UKURAN IMAGE VIA API.
- DE-SKEW & mengubah gambar.
- Atur properti gambar.

Silakan kunjungi [dokumentasi resmi](https://docs.aspose.com/imaging/python-net/) untuk daftar fitur yang lebih rinci.

### Baca & Tulis Format Gambar

** Format Raster: ** JPEG2000, JPEG, BMP, TIFF, BIGTIFF, GIF, PNG, APNG \
** Pencitraan Medis: ** DICOM \
** Metafiles: ** EMF, WMF, EMZ, WMZ \
** Truevision: ** TGA \
** Grafik Vektor: ** SVG, SVGZ

### Simpan gambar sebagai

** diperbaiki: ** pdf \
** Photoshop: ** psd \
** Web: ** HTML5 Canvas

### Baca format gambar

** ebook: ** djvu \
** kamera digital mentah: ** dng \
** OpenOffice: ** ODG, OTG \
** Bitmap: ** DIB \
** Gambar Web: ** Webp \
** coreldraw: ** cdr (x6, x7), cmx (v2.0 32-bit) \
** PostScript: ** EPS

Silakan kunjungi [format file yang didukung](https://docs.aspose.com/imaging/python-net/supported-file-formats/) untuk rincian lebih lanjut.

### Persyaratan sistem

Aspose. Imaging untuk Python via .NET mendukung Windows 64-bit, sistem operasi Linux 64-bit di mana Python 3.5 atau lebih baru diinstal.

### Sistem operasi yang didukung

** Microsoft Windows: ** Windows Desktop & Server (`x64`) \
** Linux: ** Ubuntu, OpenSuse, Centos, dan lainnya.
