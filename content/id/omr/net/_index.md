Ini adalah API on-premise mandiri yang membantu mengenali data yang ditandai manusia dari gambar dokumen yang dipindai, survei, kuesioner, kuis, dan kertas ujian.

## Mengapa mengunduh Aspose.OMR untuk .NET?

Aspose.OMR untuk .NET tidak hanya bertindak sebagai mesin Pengenalan Tanda Optik. Ini juga menyediakan kontrol grafis praktis yang dapat digunakan untuk penyetelan ambang batas dan markup manual untuk melihat perubahan secara real-time.

## Fitur API OMR

- Pengenalan gambar dan foto yang dipindai.
- Kemampuan untuk memproses gambar yang diputar dan perspektif (tampak samping).
- Mengenali data dari tes, ujian, kuesioner, survei, dll.
- Tingkat akurasi tinggi & kemampuan untuk mengekspor hasil dalam format CSV.
- Buat template OMR.

### Baca Gambar untuk OMR

JPEG, PNG, GIF, TIFF, BMP

### Simpan Hasil OMR Sebagai

CSV, JSON, XML
