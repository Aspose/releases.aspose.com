# Penyematan Sumber Daya Eksternal yang Ditingkatkan

Sebelumnya dalam beberapa kasus, saat menyimpan diagram Microsoft Visio® ke aliran HTML tidak menyematkan sumber daya eksternal. Namun, masalah ini sekarang telah diselesaikan.

# Penanganan Shading Bentuk yang Lebih Baik

Saat mengonversi Visio® VSDX ke PDF memiliki masalah ketika bayangan bentuk tidak ada. Rilis ini menyelesaikan masalah itu dan menawarkan penanganan corak bentuk yang lebih baik.

# Simpan HTML ke Aliran Tunggal

Menambahkan dukungan untuk menyimpan HTML ke satu aliran.

# Simpan Diagram ke HTML dengan Sumber Daya Tertanam

Kemampuan untuk menyimpan diagram Visio® ke HTML dengan atau tanpa sumber daya yang disematkan.
