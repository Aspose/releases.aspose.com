# Grafik Data dalam Konversi Visio® ke PDF

Dalam rilis API sebelumnya dalam beberapa kasus saat mengonversi Visio® ke format PDF, grafik data mengalami masalah. Masalah ini sekarang telah diselesaikan dalam rilis API saat ini.

# Ekspor Visio® ke HTML

Pengecualian yang dilemparkan saat diagram Visio® diekspor ke format HTML, kini telah diselesaikan.
