# Resolusi Masalah Konversi Gambar

Konversi gambar tidak berfungsi saat mengonversi Microsoft Visio® VSDX ke PDF. Rilis saat ini telah memperbaiki masalah ini.

# Konversi Warna Latar Belakang Bentuk Visio®

Sebelumnya dalam beberapa kasus, warna latar belakang bentuk diubah secara tidak benar selama proses konversi VSDX ke PDF. Masalah ini sekarang telah diselesaikan.

# Penanganan Posisi Teks

Penanganan posisi teks yang lebih baik selama konversi format Visio ke SVG.

# Rendering Tabel yang Lebih Baik dalam Konversi Visio ke PNG

Dalam rilis API ini, masalah pemformatan dalam rendering tabel dan tombol radio selama proses konversi VSDX ke PNG telah diselesaikan.
