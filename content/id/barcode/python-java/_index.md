### Pemrosesan Barcode Python melalui Fitur API Java On-premise

- Menawarkan dukungan untuk sebagian besar spesifikasi dan standar barcode populer.
- Anda dapat membaca dan mengekspor kode batang dalam berbagai format gambar, misalnya JPEG, PNG, GIF & BMP.
- Kemampuan untuk bekerja dengan label barcode.
- Dukungan untuk dimensi X serta dimensi Y untuk barcode 2D.
