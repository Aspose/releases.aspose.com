## Mengapa mengunduh Aspose.Finance untuk .NET?

Aspose.Finance untuk .NET adalah API on-premise mandiri yang terdiri dari kelas C# yang memungkinkan Anda untuk memproses dan memanipulasi format terkait keuangan, seperti, XBRL dan iXBRL dari dalam aplikasi .NET Anda.

## Fitur Pemrosesan Keuangan

- Buat instance XBRL dari awal.
- Baca format XBRL & iXBRL.
- Mendukung validasi XBRL & iXBRL.
- Impor dan ekspor file format XBRL & iXBRL.
- Menangani elemen abstrak dalam taksonomi XBRL.
