### Обработка штрих-кода Python с помощью функций Java On-premise API

- Он предлагает поддержку большинства популярных спецификаций и стандартов штрих-кода.
- Вы можете считывать и экспортировать штрих-коды в различных форматах изображений, например, JPEG, PNG, GIF и BMP.
- Возможность работы с этикетками со штрих-кодом.
- Поддержка X-размера, а также Y-размера для 2D штрих-кодов.
