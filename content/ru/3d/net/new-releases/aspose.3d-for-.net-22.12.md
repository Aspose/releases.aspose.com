# Кодировать / декодировать свет

Эта версия 3D API позволяет кодировать и декодировать направленное свет, а также точечный свет.

# Улучшения в обработке 3D цилиндров

Следующие проблемы были решены в этом выпуске:

- ** ThreedNet-1294 ** цилиндр определял ультрафиолетовое ультрафиолетовое излучение, но нет данных

- ** Threednet-1306 ** цилиндр с Radius top = 0 будет генерировать неправильную модель

# Преобразовать из `dxf` в` gltf`/`glb`

Возможность преобразовать файл `dxf` в форматы` gltf` или `glb`. GLTF (формат передачи GL) используется в качестве стандартного формата файла для работы с 3D -сценами и моделями.

Обратите внимание, что файл `gltf` может иметь расширение файла gltf` или` glb.

> Для полного списка функций, улучшений и исправлений ошибок в этом выпуске, пожалуйста, посетите, [Aspose3d для Net 2212.](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-12-release-notes/).
