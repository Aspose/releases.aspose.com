# Поддержка временных файлов для FBX Importer

Добавлена поддержка внутренней временной файловой системы для импортера FBX.

# Улучшенный экспорт GLTF

Экспорт `GLTF` был не так хорош в предыдущих версиях. Однако в текущей версии 3D API функция экспорта файлов формата GLTF была улучшена.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.3D for .NET 22.9 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-9-release-notes/).
