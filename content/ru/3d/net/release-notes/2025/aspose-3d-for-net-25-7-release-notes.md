## Изменения API ##

### Добавлен класс **Aspose.ThreeD.Group**

Этот класс определяет логические группировки отношений, отличные от класса `Node`, который представляет иерархические отношения для рендеринга. В настоящее время эта функциональность доступна только при работе с форматом IFC.
