# Экспорт PDF 1.6 & PDF 1.7 Поддержка

Поддержка была реализована для экспорта PDF в форматы PDF 1.6 и PDF 1.7.

# Экспорт PDF с уровнями соответствия A2A, A2B

Этот выпуск Python API вводит поддержку функции экспорта PDF с многочисленными уровнями соответствия, то есть A2A, A2B, A2U, A3A и A3B.

# Управление гиперссылкой в ​​презентациях

Было добавлено новое свойство ([`Hyperlink.sound`] (https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/)), который представляет воспроизводимый звук гиперссылки.

# Эффект анимации останавливает предыдущий звук

Новое свойство ([`Effect.stop_previous_sound`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/effect/)) было введено, что указывает, останавливается ли эффект анимации предыдущий звук.
