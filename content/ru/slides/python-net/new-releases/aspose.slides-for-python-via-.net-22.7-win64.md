# Улучшенная поддержка Ubuntu Focal

Было отмечено, что при преобразовании Microsoft PowerPoint & Reg; `Pptx` to` pdf` форматирует несколько проблем на Ubuntu Focal. Эти проблемы теперь решены.

# 3D -диаграммные диаграммы

Этот выпуск API представляет поддержку 3D -диаграмм. Теперь вы можете добавить диаграмму с данными по умолчанию, а затем установить свойства `rowation3d` (например, ось правого угла, процент глубины и т. Д.).

# Улучшенная поддержка эффекта текста на иврите

Ранее было отмечено, что в некоторых случаях тень на иврите текста была неверной, когда презентации «pptx» преобразуются в формат `pdf`. Эта проблема теперь была решена.
