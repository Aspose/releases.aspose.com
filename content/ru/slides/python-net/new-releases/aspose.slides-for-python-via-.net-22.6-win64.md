# Обновленное поведение блокировки источника презентации

Устаревший презентация lockingbehavior `LOAD_AND_RELEASE_LEGACY_TEMPORARY_DEFAULT` enum enum был удален. `LOAD_AND_RELEASE` - это поведение блокировки презентации по умолчанию.

Эта настройка по умолчанию блокирует источник только для времени выполнения конструктора «ipresentation».
