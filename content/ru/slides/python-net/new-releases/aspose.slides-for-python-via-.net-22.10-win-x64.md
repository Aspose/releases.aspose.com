# Установить кадр и поведение ротации

Способность указывать и контролировать поведение вращения кадры и размера кадра, одновременно отображая формы из `PPTX` и других поддерживаемых форматов презентаций.

# Сжатие встроенных шрифтов, чтобы уменьшить размер презентации

Если ваша презентация содержит встроенные шрифты, то этот выпуск нашей презентации обработки Python API дает возможность выполнять сжатие всех таких встроенных шрифтов и, в свою очередь, уменьшает размер файлов презентации, таких как `PPTX`.

# Установите линии презентации линии линий линий

Этот релиз API вводит способность устанавливать цвет лидеров лидеров в ваших слайдах презентации.
