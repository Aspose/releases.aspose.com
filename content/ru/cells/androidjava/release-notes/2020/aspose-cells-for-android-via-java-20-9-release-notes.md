## **Public API и обратно несовместимые изменения**

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Android via Java. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом. на форуме поддержки Aspose.Cells.

### **Помечает интерфейс ICustomFunction как устаревший.**

Этот интерфейс иногда вызывает двусмысленность и непонимание у пользователей. Вместо этого пользователю следует использовать AbstractCalculationEngine, который предоставляет более удобные и гибкие API-интерфейсы для управления пользовательскими функциями.

### **Помечает свойство CalculationOptions.CustomFunction как устаревшее.**

 Пожалуйста, используйте**АннотацияРасчетДвигатель** вместо**ICustomFunction** к**CalculationOptions.CustomEngine** имущество.

### **Помечает метод Workbook.CalculateFormula(bool, ICustomFunction) как устаревший.**

 Пожалуйста, используйте**Workbook.CalculateFormula(CalculationOptions)** метод вместо этого.

### **Помечает метод Worksheet.CalculateFormula(bool, bool, ICustomFunction) как устаревший.**

 Пожалуйста, используйте**Worksheet.CalculateFormula(CalculationOptions, bool)** метод вместо этого.

### **Помечает метод Cell.Calculate(bool, ICustomFunction) как устаревший.**

 Пожалуйста, используйте**Cell.Рассчитать(Параметры расчета)** метод вместо этого.

### **Добавляет класс DocxSaveOptions и перечисление SaveFormat.Docx.**

Представляет параметры и перечисление для сохранения книги в виде файлов .docx.

### **Добавляет класс PptxSaveOptions и перечисление SaveFormat.Pptx.**

Представляет параметры и перечисление для сохранения книги в виде файлов .pptx.

### **Добавляет класс PowerQueryFormulaFunction**

Представляет функцию формулы мощного запроса.

### **Добавляет SaveOptions.UpdateSmartArt и удаляет свойство OoxmlSaveOptions.UpdateSmartArt.**

Указывает, обновляется ли текст интеллектуальных фигур при сохранении файлов.

### **Добавляет метод SlicerCollection.Add(ListObject table, int index, string destCellName)**

Добавьте новый слайсер, используя ListObjet в качестве источника данных.

### **Добавляет метод SlicerCollection.Add(ListObject table, ListColumn listColumn, string destCellName)**

Добавьте новый слайсер, используя ListObjet в качестве источника данных.

### **Добавляет метод SlicerCollection.Add(ListObject table, ListColumn listColumn, int row, int column)**

Добавьте новый слайсер, используя ListObjet в качестве источника данных.

### **Добавляет метод Cells.RemoveDuplicates().**

 Перегруженный метод**Cells. Удалить дубликаты (...)** для удобства пользователя удалить повторяющиеся строки во всем листе.

### **Добавляет свойство TickLabels.DisplayNumberFormat.**

Получает и задает числовой формат отображения меток деления.

### **Добавляет методы Cells.GetViewRowHeight() и Cells.GetViewRowHeightInch().**

Получает высоту строки представления.

### **Добавляет перечисление SheetType.InternationalMacro.**

Добавляет новый тип листа: международный макрос.

### **Добавляет метод PivotFieldCollection.iterator().**

Получает перечислитель по элементам в этой коллекции в правильной последовательности.

### **Добавляет метод PivotItemCollection.iterator().**

Получает перечислитель по элементам в этой коллекции в правильной последовательности.

### **Добавляет свойство Workbook.IsWorkbookProtectedWithPassword.**

Указывает, защищены ли структура и окно паролем.

### **Добавьте классы PowerQueryFormulaParameters и PowerQueryFormulaParameter.**

Представляет параметры формулы мощного запроса.

### **Добавляет свойство AbstractCalculationEngine.ProcessBuiltInFunctions**

Из соображений производительности и удобства пользователя мы добавляем это свойство и делаем его значение по умолчанию равным false, чтобы пользователь мог сосредоточиться на тех функциях, которые не поддерживаются встроенным движком. Если существующая реализация пользователя**АннотацияРасчетДвигатель** изменен расчет некоторых встроенных функций, пользователь должен переопределить это свойство, чтобы сделать его как**истинный** от**20.9**.

### **Добавляет свойство TxtLoadOptions.HasTextQualifier.**

Указывает, есть ли текстовый квалификатор для значений ячеек в файле шаблона.

### **Добавляет свойство TxtLoadOptions.TextQualifier.**

Задает квалификатор текста для значений ячеек в файле шаблона.

### **Добавляет свойство HtmlSaveOptions.ImageScalable.**

 Указывает, следует ли использовать масштабируемую единицу для описания ширины изображения. Значение свойства по умолчанию равно**истинный**.

### **Добавляет свойство Slicer.AlternativeText**

Получает или задает описательную (альтернативную) текстовую строку объекта Slicer.

### **Добавляет свойство Slicer.ColumnWidthPixel**

Получает или задает ширину в пикселях для каждого столбца среза.

### **Добавляет свойство Slicer.HeightPixel**

Получает или задает высоту указанного среза в пикселях.

### **Добавляет свойство Slicer.IsLocked**

Указывает, заблокирована ли форма среза.

### **Добавляет свойство Slicer.IsPrintable**

Указывает, доступен ли для печати объект среза.

### **Добавляет свойство Slicer.LeftPixel**

Получает или задает смещение формы среза по горизонтали относительно левого столбца в пикселях.

### **Добавляет свойство Slicer.LockedAspectRatio.**

Указывает, блокируется ли соотношение сторон.

### **Добавляет свойство Slicer.Placement**

Представляет способ прикрепления объекта рисования к ячейкам под ним. Свойство управляет размещением объекта на рабочем листе.

### **Добавляет свойство Slicer.RowHeightPixel**

Возвращает или задает высоту в пикселях каждой строки в указанном срезе.

### **Добавляет свойство Slicer.Title**

Указывает заголовок текущего объекта Slicer.

### **Добавляет свойство Slicer.TopPixel**

Получает или задает вертикальное смещение формы среза от ее верхней строки в пикселях.

### **Добавляет свойство Slicer.WidthPixel**

Получает или задает ширину указанного среза в пикселях.

### **Добавляет свойство Worksheet.PaneState и перечисление PaneStateType.**

Представляет состояние панели на листе.

### **Добавляет свойство OdsLoadOptions.RefreshPivotTables.**

Указывает, обновляется ли сводная таблица при загрузке файлов .ods.

### **Добавляет свойство FilterColumn.IsDropdownVisible.**

Указывает, видна ли кнопка автофильтра для этого столбца.

### **Устарело свойство Filter.Visibledropdown.**

 Использовать**FilterColumn.IsDropdownVisible** вместо.

### **Добавляет свойство CopyOptions.KeepMacros.**

Указывает, сохраняются ли макросы в целевой книге. Это действует только в том случае, если исходная книга не содержит макросов.

### **Добавляет перегруженный метод Workbook.Copy(Workbook,CopyOptions).**

Копирует книгу с параметрами.

### **Добавляет перечисление WarningType.InvalidAutoFilterRange.**

Представляет тип предупреждения о том, что диапазон не может быть отфильтрован автоматически.

### **Добавляет свойство Chart.DisplayNaAsBlank.**

Указывает, отображается ли #N/A как пустое значение.

### **Добавляет перечисление CrossType.Minimum.**

Представляет пересечение осей при минимальном значении.

### **Добавляет свойство XlsbSaveOptions.ExportAllColumnIndexes.**

Указывает, следует ли экспортировать индексы столбцов для всех ячеек при сохранении файлов XLSB.
