### **Изменения для сохранения рабочей книги с LightCells**

Чтобы сделать функции, связанные с формулами, доступными для LightCells, в старых версиях мы сохраняем все данные формулы в модели ячеек в памяти при сохранении книги с помощью LightCells. Это вызвало недопонимание и неправильное использование LightCells у некоторых пользователей. Пользователь думал, что данные формулы ячейки были выпущены за пределы StartCell(Cell), но на самом деле это не так. Для большинства пользователей, использующих LightCells, основной проблемой является производительность (затраты на память). Мало кто будет использовать функции, связанные с формулами, кроме установки простой формулы в ячейку в процессе сохранения книги. Итак, с этой версии мы добавляем некоторые ограничения на изменение объекта ячейки в области действия метода StartCell(Cell). Теперь нельзя задавать общие формулы, формулы массива для данного объекта ячейки в методе StartCell(Cell). Если такие формулы необходимы, пользователь должен установить их перед сохранением книги. Благодаря этому изменению мы улучшили производительность для большинства пользователей, которым необходимо сохранить простую формулу для ячеек в результирующем файле электронной таблицы с помощью LightCells.
