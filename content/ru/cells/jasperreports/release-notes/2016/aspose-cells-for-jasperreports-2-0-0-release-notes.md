### **Public API и обратно несовместимые изменения**
Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for JasperReports. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.
#### **Поддерживает JasperServer V6.3.0**
Мы удалили те версии, которые поддерживают старые версии JasperReports, и оставили в архиве только ту, которая поддерживает JasperServer v6.3.0. Если пользователям необходимо работать со старыми версиями JasperReports, загрузите старые версии Aspose.Cells for JasperReports.
