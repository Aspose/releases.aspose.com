#  `.NUMBERS` Поддержка хранилища данных 5.0

Добавлена поддержка хранилища данных 5.0 `.numbers`.

#  Улучшен рендеринг с XLSX по PDF.

Было замечено, что при преобразовании XLSX в PDF гистограммы отображались неправильно. Теперь эта проблема решена.

#  Улучшенный Excel&reg; Обработка меток диаграммы

Этот выпуск API содержит несколько исправлений, касающихся Excel&reg; метки диаграммы. Они заключаются в следующем:

- Метки диаграмм не вписывались в коробку.
- Метки диаграммы выравнивались по вертикали, а не по горизонтали.

#  Сохраните формулы таблицы на номер XLSB.

Добавлена поддержка сохранения формул таблиц по номеру `XLSB`.

#  Улучшенная поддержка в Linux

Было замечено, что символы или подписи, написанные на таких объектах, как фигуры и текстовые поля, в Linux выглядели искаженными. Теперь эта проблема решена.

 > Полный список функций, улучшений и исправлений ошибок в этом выпуске см. на странице[Aspose.Cells for .NET 22.6 Примечания к выпуску](https://releases.aspose.com/cells/net/release-notes/2022/aspose-cells-for-net-22-6-release-notes/).
