# Улучшенный рендеринг фигур WordArt.

Было замечено, что Microsoft Excel&reg; Эффекты и фигуры WordArt не отображались должным образом во время преобразования Excel в PDF. Теперь эта проблема решена.

#  Улучшено преобразование XLS в PDF.

Раньше в некоторых случаях, когда мы преобразовывали `XLS` в `PDF`, размер пустой текстовой строки в текстовом поле отображался неправильно. Теперь эта проблема решена.

#  Удалить конечный пробел Cells из CSV

В этом выпуске API появилась возможность удалять оставшиеся пустые ячейки при сохранении файла значений, разделенных запятыми (CSV).

 > Полный список функций, улучшений и исправлений ошибок в этом выпуске см. на странице[Aspose.Cells for .NET 22.4 Примечания к выпуску](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-4-release-notes/).
