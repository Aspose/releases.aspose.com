#  Поддержка форматов шаблонов электронных таблиц

В этом выпуске API появилась поддержка форматов шаблонов электронных таблиц как для Excel&reg; (XLT) & OpenOffice Calc&reg; (ОТС) программы. Они поддерживаются в качестве выходных форматов через `SaveFormat.Ots` и `SaveFormat.Xlt`.

#  Установить внутренние границы

Возможность установки внутренних границ.

#  Сортировка PivotField по выбранной строке или столбцу

Добавлена поддержка сортировки значений сводных полей. Сортировку сводного поля можно выполнить на основе нужной строки или столбца.

 > Полный список функций, улучшений и исправлений ошибок в этом выпуске см. на странице[Aspose.Cells for .NET 22.3 Примечания к выпуску](https://releases.aspose.com/cells/net/release-notes/2022/aspose-cells-for-net-22-3-release-notes/).
