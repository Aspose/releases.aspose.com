## **Public API и обратно несовместимые изменения**

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for .NET. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.

### **Добавляет метод Cell.SetDynamicArrayFormula (string arrayFormula, параметры FormulaParseOptions, логическое вычисление).**

Поддерживает установку формулы динамического массива и распространение в соседние ячейки, если это возможно.

### **Добавляет метод Workbook.RefreshDynamicArrayFormulas (вычисление логического значения).**

Поддерживает обновление формул динамического массива и перенос в соседние ячейки в соответствии с текущими данными.

### **Добавляет свойство Cell.Comment.**

Получает комментарий к ячейке.

### **Добавляет свойство HtmlSaveOptions.ExportExtraHeadings.**

Указывает, экспортируются ли дополнительные заголовки, когда длина текста превышает максимальную отображаемую колонку.
Значение по умолчанию неверно. Если вы хотите импортировать html-файл в Excel, оставьте значение по умолчанию.

### **Добавляет свойство HtmlSaveOptions.ExportFormula.**

Указывает, будет ли экспортироваться формула при сохранении файла в html. Значение по умолчанию верно.
Если вы хотите импортировать выходной HTML-код в Excel, оставьте значение по умолчанию.


### **Добавляет свойство HtmlSaveOptions.MergeEmptyTdForcely.**

Указывает, будет ли принудительно объединяться пустой элемент TD при экспорте файла в html.
Размер html-файла будет значительно уменьшен после установки значения true. Значение по умолчанию неверно.
Если вы хотите импортировать html-файл в Excel или экспортировать идеальные линии сетки при сохранении файла в html,
пожалуйста, оставьте значение по умолчанию.

### **Добавляет свойство LoadOptions.AutoFilter.**

Указывает, выполняется ли автоматическая фильтрация данных при загрузке файлов.
Иногда, хотя автофильтр установлен, соответствующие строки не скрываются в файле. Теперь работает только для файла SpreadSheetML.

### **Добавляет свойство WorkbookSettings.Author**

Получает и задает автора книги.

### **Добавляет метод MultipleFilterCollection.Add().**

Добавляет строку фильтра автофильтра.
