{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 4.9.0](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-4.9.0/)

{{% /alert %}} 

 Мы рады сообщить Aspose.Cells for .NET v!

 Что изменилось в Aspose.Cells:

- Вычисляет формулы, которые превышают ограничение на количество строк/столбцов Excel2003.
- Поддерживает спарклайны MS Excel 2010.
- Реализует 3D-эффекты на графиках
- Удаляет повторяющиеся значения
- Поддерживает разрешение пользователям редактировать диапазоны
- Усовершенствован механизм расчета формул.
- Обнаруживает и удаляет подключения к внешним данным
- Устанавливает связанное изображение в файле xlsx
- Включает ICustomTypeDesciptor в качестве источника данных интеллектуальных маркеров.
- Внешние связанные исходные данные в функции Chart to Image поддерживаются
- Поддерживаются параметры настройки внешней страницы MS Excel 2007 в файле xls.
- Указывает содержимое HTML в качестве значения ячейки или примечания к комментарию.
- 112 исправлений и других улучшений.

 (Примечание: класс Aspose.Cells.Charts переименован в класс Aspose.Cells.ChartCollection.)

 Что изменилось в Aspose.Cells.GridWeb:

- Добавлены новые форматы даты: «М/д/гг», «ММ/дд/гггг» и «дд.мм.гггг».
- Устанавливает сообщение об ошибке в заголовке ячейки, когда проверка не удалась на стороне клиента
- Получает имя столбца для ячейки на стороне клиента
- Сохраняет файлы Excel в поток или загружает файлы Excel из потока
- Поддерживает новые функции: ИНДЕКС, ПРОСМОТР, ПОИСКПОЗ
- 8 исправлений и других улучшений.

 Что изменилось в Aspose.Cells.GridDesktop:

- Устанавливает форматы даты «ММ/ДД/ГГГГ» и «М/Д/ГГГГ» в диалоговом окне «Формат Cells».
- Импортирует формат файла CSV
- Поддерживает формулы: ПРОСМОТР, ПОИСКПОЗ
- 3 исправления и другие улучшения.

Проблемы, решенные в Aspose.Cells for .NET v4.9.0
