## **Public API и обратно несовместимые изменения**


 Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for .NET. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.



 Добавляет свойство TxtLoadOptions.HasFormula.

 Указывает, содержит ли CSV-файл формулу.



 Добавляет свойство ColorScale.Is3ColorScale.

 Указывает, является ли условное форматирование трехцветной шкалой.



 Удаляет устаревшее свойство Workbook.SaveOptions.

 Вместо этого используйте метод Workbook.Save(Stream,SaveOptions) или Workbook.Save(string,SaveOptions).



 Добавляет метод Protection.VerifyPassword.

 Проверяет пароль защиты рабочего листа.



Добавляет свойство Proptition.IsProtectedWithPassword.

 Указывает, защищен ли рабочий лист паролем.



 Добавляет метод VbaProject.Sign.

 Подпишите проект VBA с помощью DigitalSignature.



 Добавляет свойство VbaProject.IsValidSigned.

 Указывает, действительна ли подпись проекта VBA.



 Добавляет свойство VbaProject.CertRawData.

 Получает необработанные данные сертификата, если проект VBA подписан.



 Добавляет свойство PdfSaveOptions.OptimizationType.

 Получает и устанавливает тип оптимизации PDF.
