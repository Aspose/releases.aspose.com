 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 8.6.3](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-8.6.3/)

Ниже приведен список улучшений и изменений в этом выпуске Aspose.Cells.

## 1) Aspose.Cells

### **Другие улучшения и изменения**

### **Новые особенности**

(CELLSNET-44131) - Некоторые неправильные формулы появляются вокруг в зависимости от количества вставленных строк.

(CELLSNET-44128) - Сохранить в XLSB разбивает формулы, такие как =СУММ(Таблица[Столбец])

(CELLSNET-44076) — ListObject.ListColumns[i].Name неверно, когда рабочая книга открывает файл XLS.

(CELLSNET-44028) — сводная таблица не обновляется при нажатии кнопки «Данные»> «Обновить все».

(CELLSNET-43084) — Выходной файл поврежден при копировании листа.

(CELLSNET-43083) - Ссылка на именованную ячейку недействительна "#Name?" при копировании листа

### **Public API и обратно несовместимые изменения**

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for .NET. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.

Добавляет свойство ImportTableOptions.IsHtmlString.

Указывает, содержит ли строковое значение в таблице данных теги html.

Добавляет метод Workbook.CreateBuiltinStyle (тип BuiltinStyleType).

Создает встроенный стиль по заданному типу.

Устарело Cells.Конечное свойство.

Вместо этого используйте свойство Cells.LastCell.

Добавляет метод Cells.ImportGridView (GridView gridView, int firstRow, int firstColumn, параметры ImportTableOptions).

Импортирует вид сетки в эти ячейки с параметрами

Добавляет свойство ImportTableOptions.ConvertGridStyle.

Указывает, применяется ли стиль представления сетки к ячейкам.

 Устарело Cells. Метод ImportGridView (GridView gridView, int firstRow, int firstColumn, bool insertRows, bool convertStringToNumber, bool convertStyle).

Используйте Cells.ImportGridView (GridView gridView, int firstRow, int firstColumn, параметры ImportTableOptions).

Добавляет свойство LoadDataOption.OnlyVisibleWorksheet.

Загружает только данные видимого рабочего листа.

Устарел метод Worksheet.CopyConditionalFormatting.

Вместо этого используйте метод Cells.CopyRows() или Range.Copy().
