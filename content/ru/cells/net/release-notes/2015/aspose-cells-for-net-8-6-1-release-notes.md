\1) Aspose.Cells 


## **Другие улучшения и изменения**

## **Новые особенности**


 (CELLSNET-43905) — Поддержка изменения целевого атрибута гиперссылки HTML на «_blank».

 (CELLSNET-43885) — Возможность получения ConnectionString для ExternalConnection типа WebQuery.

 (CELLSNET-43935) — игнорирует скрытый столбец, если для параметра ExportTableOptions.PlotVisibleColumns установлено значение true.

 (CELLSNET-43925) — Добавление ссылки на макросы VBA в книгу.


## **Улучшения**


 (CELLSNET-43933) - Cell. Формула может принимать недопустимую формулу, и API пытается ее исправить.

(CELLSNET-43476) - API необходимо проверить, загружена ли лицензия или нет

 (CELLSNET-43310) — Переименование повторяющихся имен рабочих листов при объединении рабочих книг.

 (CELLSNET-42518) — Возможность доступа к подобъектам с помощью интеллектуальных маркеров.

 (CELLSNET-43984) - Пароль для изменения преобразуется в Пароль для открытия при повторном сохранении XLSM.

 (CELLSNET-43983) - Пароль для изменения преобразуется в Пароль для открытия при повторном сохранении XLSX.

 (CELLSNET-43982) - Пароль для изменения преобразуется в Пароль для открытия при повторном сохранении XLTM.

(CELLSNET-43981) - Пароль для изменения теряется при повторном сохранении XLTM.

 (CELLSNET-43980) - Пароль для изменения преобразуется в Пароль для открытия при повторном сохранении XLTX.

 (CELLSNET-43979) — шрифт SetStyle не применяется для некоторых шрифтов.

 (CELLSNET-43977) - Пароль для изменения теряется, когда XLTX повторно сохраняется с Aspose.Cells.

 (CELLSNET-43976) - Несколько попыток открыть защищенный паролем XLSX

 (CELLSNET-43973) - Пароль для изменения утерян после повторного сохранения XLSM

 (CELLSNET-43968) - приложение Excel рекомендует открыть результирующий XLSB только для чтения

 (CELLSNET-43967) - Защищенный паролем XLT становится поврежденным после повторного сохранения

 (CELLSNET-43966) — формула ISNONTEXT возвращает неверное значение после CalculateFormula.

 (CELLSNET-43965) — DetectFileFormat (FileStream) потребляет много памяти для zip-файла.

 (CELLSNET-43951) - Пароль для изменения потерян в файлах OOXML.

 (CELLSNET-43950) - Проблемы с идентификацией защиты в Aspose.Cells

(CELLSNET-43944) - Размер картинки не правильный и меняется после сброса

 (CELLSNET-43943) — невозможно прочитать верхний индекс из символа ячейки.

 (CELLSNET-43940) - При открытии отчета выбрана неправильная ячейка

 (CELLSNET-43931) — Удаление строк из именованного диапазона приводит к повреждению результирующего XLSX.

 (CELLSNET-43928) — значение автора DocumentProperty читается неполным

 (CELLSNET-43927) - #REF в формуле, ссылающейся на объект списка на другом листе

 (CELLSNET-43891) - Проблема операций Workbook.VBAProject.Modules

 (CELLSNET-43737) — FileFormatInfo.IsEncrypted имеет неправильное значение для устаревших форматов.

 (CELLSNET-42120) — неправильное значение DisplayStringValue в некоторых сценариях.

 (CELLSNET-42110) — граница, установленная для диапазонов в XLSX, не отображается в PDF.
