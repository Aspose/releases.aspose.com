 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 7.2.2](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.2.2/)

Мы
 рад сообщить Aspose.Cells for .NET v7.2.2!

\1) Aspose.Cells 

 Улучшения

- Несколько промежуточных итогов добавить дополнительную строку
- Последующие формулы промежуточных итогов не используют правильный диапазон
- Доступ к данным фигур в MS Excel
- Свойства карты не сохраняются при сохранении копии файла Excel
- Отключить проверку совместимости в MS Excel
- Неправильная реализация/интерпретация для открытия файла неправильного формата

 Исключения

- NullReferenceException при открытии/сохранении файла XLSX
- Исключение: на листе слишком много форматов.

 -Арифметическая операция привела к переполнению

 -NullReferenceException при сохранении для этого простого примера книги

 Ошибки

 -Срочно: функциональность изменена в «FindStringContains».

 -Выполнение промежуточного итога в столбце «Дата» отображается как «Дата».

 -#Рекурсивная ссылка! Ошибка

- Копирование файла с использованием Aspose Cells требует ремонтных работ
- Думая, что файл Excel 97 XLS является документом Word

 -Экспорт в диаграмму PDF отличается от версии Excel

- Экспорт в диаграмму PDF отличается от версии Excel -2

 Дополнительные линии встроены в графики

- Текст в кавычках в файле CSV
- Пользовательские свойства повреждены из-за разрыва строки
- Расчеты MS Excel не завершаются до сохранения документа
- Формула не работает нормально
- WorkSheetCollection.AddCopy не обновляет ссылки
- Проблема при удалении источника данных в сводной таблице
- Проблема с копированием сводной таблицы (форматирование)
- Сохранение в формате Excel97-2003 повреждает файл
- Проблемы при очистке частичного содержимого таблицы
- Не удалось открыть файл, созданный OXML SDK.
