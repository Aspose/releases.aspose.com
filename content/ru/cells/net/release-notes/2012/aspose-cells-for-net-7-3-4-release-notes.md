 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 7.3.4](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.3.4/)

 Рады сообщить Aspose.Cells for .NET v7.3.4!



\1) Aspose.Cells 



 Новые особенности

- Поддержка 3D-диаграмм Open Office
- Вычислить средневзвешенное значение в строке промежуточных итогов между двумя столбцами (SmartMarkers)
- Обнаружение вертикального или горизонтального источника данных диаграммы



 Улучшения

- Найти и заменить внутренние тексты



 Спектакль

- Метод CalculateFormula рабочей книги занимает более 30 секунд
- Снижение производительности Office 2007 по сравнению с 2003 г.

 -CalculateFormula занимает около 3 минут на 8-ядерном компьютере.

- Aspose Cell замена оболочки Excel
- Сохранение документа Excel занимает более одной минуты



 Исключения

- Исключение «Недопустимая формула» при открытии файла XLSX
- Aspose.Cells вызывает исключение «ArgumentNullException» при открытии файла шаблона
- Сохранение файла MHtml, чтение Aspose.Cells - проблема



 Ошибки

- Формула не рассчитана должным образом
- Элементы ActiveX повреждают книгу
- 4 Электронные таблицы не удалось переписать
- Диаграммы Excel блокируются после сохранения
- Ошибка при копировании листов

 - Заполненное изображение радарного графика, отображаемое со скрытыми отметками осей с помощью метода Chart.ToImage.

 - Проблема с форматированием меток данных

- Проблема с расчетом диаграммы Excel
- Проблема с гистограммой, имеющей обе оси
- Несколько рассчитанных сводных полей приводят к нечитаемому файлу.
- Проблема с пользовательскими частями XML
- Этот файл поврежден после сохранения

 -Преобразование XLS в XLSX и обратно создает неверный файл XLS.

 -Преобразование XLS в XLSX создает неверный документ

- Рендеринг файла MS Excel в документ PDF имеет проблему с содержимым
