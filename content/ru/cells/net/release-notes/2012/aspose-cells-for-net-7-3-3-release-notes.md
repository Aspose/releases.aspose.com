 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 7.3.3](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.3.3/)

Мы
 рад сообщить Aspose.Cells for .NET v7.3.3!

\1) Aspose.Cells 

 Исключения

- Поврежденные файлы с новой версией Aspose.Cells
- Получение ошибки при сохранении файла Excel

 Ошибки

- Ошибка применения метода SetArrayFormula
- Обнаружена ошибка при открытии сгенерированного файла с диаграммой
- Cells.DeleteRange с

 не обновляет ссылки

- Сопоставьте столбцы и строки
- Некоторые электронные таблицы не будут работать при загрузке/сохранении.
- Проблемы с методом Workbook.CalculateFormula()
- Chart.CategoryAxis.TickLabels.RotationAngle не работает после копирования листов
- Проблема с удалением названия диаграммы/оси
- Форматирование легенды диаграммы теряется при сохранении книги
- Проблема с визуализацией диаграмм в PDF
- Установить проблемы с цветом градиента
- Сохранение файла MHT, чтение в Aspose.Cells является проблемой
- Некоторое условное форматирование отображается неправильно
- Плохое позиционирование текста в PDF со шрифтами разного размера
- Экспорт в PDF из файла Excel изменяет формулы
- Рабочий лист и проблемы рендеринга PDF
- Проблема с автоподбором столбцов
- Проблемы с темой сводных таблиц
- Не удается обновить сводную таблицу
- Проблема при обновлении файла XLS с макросом до формата XLSM
