 Эта страница содержит примечания к выпуску для[Aspose.Cells for .NET 7.7.1](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.7.1/)

{{% /alert %}} 

 Aspose.Cells for .NET был обновлен до версии 7.7.1, и мы рады сообщить, что в этом выпуске добавлено более 80 новых полезных улучшений.
Используя Aspose.Cells for .NET, вы можете работать с XLS, SpreadsheetML,OOXML,XLSB, CSV, HTML, ODS, PDF, XPS и другими форматами в ваших приложениях. Вы также можете просматривать, генерировать, изменять, преобразовывать, визуализировать и печатать рабочие книги, не используя Microsoft Excel.
Посетите документацию, чтобы узнать, как начать работу с Aspose.Cells for .NET.
Обратите внимание, что этот загружаемый файл содержит полностью рабочую версию продукта, однако без набора лицензий он будет работать в ознакомительном режиме с некоторыми ограничениями. Чтобы протестировать Aspose.Cells без этих ознакомительных ограничений, вы можете запросить бесплатную 30-дневную временную лицензию.

 Прекращена поддержка for .NET Framework 1.1

Начиная с этого выпуска Aspose.Cells больше не распространяется с DLL для использования в .NET Framework 1.1. Поддержка этой версии фреймворка была прекращена, чтобы упростить функции и исправления, поскольку сложно добавить некоторые функции в эту старую версию фреймворка.



 Добавляет новое свойство ImageOrPrintOptions.Transparent.

 Он указывает, является ли фон сгенерированных изображений (преобразование рабочего листа в изображение) прозрачным.



 Добавляет новое свойство Shape.ControlData.

 Он используется для получения двоичных данных управления.



 Удаляет устаревший метод Worksheet.Move



 Добавляет свойство FindOptions.Style

 Поддерживает поиск ячеек по форматированию.



 Добавляет свойство FindOptions.ConvertNumericData.

 Поддерживает ли преобразование искомого строкового значения в числовые данные.



 Добавляет AutoFitterOptions.MaxRowHeight

 Устанавливает максимальную высоту строки (в точках) при автоподгонке строк.



 Добавляет метод Clear() в PivotFieldCollection.

Он используется для удаления всех сводных полей в списке.



 Устаревшие OleFileType и OleObject.FileType

 Он заменен на FileFormatType



 Обрабатывает файлы шаблонов в режиме LightCells

 Для поддержки обработки файла шаблона в режиме LightCells
