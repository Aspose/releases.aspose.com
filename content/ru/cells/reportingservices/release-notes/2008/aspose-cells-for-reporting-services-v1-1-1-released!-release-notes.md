 Эта страница содержит примечания к выпуску для[Aspose.Cells for Reporting Services Версия 1.1.1 выпущена!](https://releases.aspose.com/cells/reportingservices/new-releases/aspose.cells-for-reporting-services-v1.1.1-released/)

{{% /alert %}} 

 Что улучшено?

-  Переписан клиентский инструмент, улучшена его стабильность
- Поддерживается дизайн подотчета
- Поддерживается диапазон отчетов ручного ввода
- Поддерживаемые композитные маркеры

 Что исправлено?

- Исправлена ошибка установки, когда несколько экземпляров SQL Server Reporting Serverices были установлены на одном компьютере.
- Исправлена ошибка вставки формулы, когда не был выбран набор данных.
