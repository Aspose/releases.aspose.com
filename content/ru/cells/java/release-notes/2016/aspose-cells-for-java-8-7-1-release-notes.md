Найти объект только из исходных значений без формата.
## **Запись**
Поскольку кодовая база Aspose.Cells for Java соответствует коду соответствующей версии .NET, большинство изменений, улучшений и исправлений, включенных в Aspose.Cells for .NET v8.7.1, также включены в этот Aspose.Cells for Java v8.7.1.
