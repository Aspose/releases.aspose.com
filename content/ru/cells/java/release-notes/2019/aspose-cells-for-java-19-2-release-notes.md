Public API и обратно несовместимые изменения

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Java. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.
#### **Добавляет свойство Cells.CountLarge**
Функционально то же самое, что и свойство Count, за исключением того, что свойство Count может генерировать ошибку переполнения, когда создано слишком много объектов Cell.
#### **Добавляет метод Hyperlink.Delete()**
Удаляет эту гиперссылку.
#### **Добавляет свойство Range.Hyperlinks**
Получает все гиперссылки в диапазоне.
#### **Добавляет enum CopyFormatType**
Представляет тип формата копирования при вставке строк.
#### **Добавляет класс InsertOptions и метод Cells.InsertRows(int, int, InsertOptions)**
Вставка строк с некоторыми параметрами.
#### **Добавляет методы FileFormatUtil.DetectFileFormat(Stream,String) и FileFormatUtil.DetectFileFormat(String,String).**
Определяет формат зашифрованного файла OOXML.
#### **Добавляет свойства ListObject.AlternativeDescription и ListObject.AlternativeText.**
Получает и задает альтернативный текст и описание таблицы.
#### **Добавляет свойство Line.ThemeColor**
Получает и задает цвет темы линии.
#### **Добавляет классы Mode3d и MsoModel3dFormat.**
Инкапсулирует объект, представляющий одну 3D-модель в электронной таблице.
#### **Добавляет перечисление ImageType.Gltf**
Представляет тип 3D-модели.
