## **Public API и обратно несовместимые изменения**
Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Java. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.
### **Добавляет перечисление LoadDataFilterOptions.DefinedNames**
Указывает, загружаются ли определенные объекты Name при загрузке файла шаблона.
### **Изменяет поведение перечисления LoadDataFilterOptions.Formula.**
В более старых версиях мы всегда загружаем определенные объекты Name при загрузке формул. Теперь мы явно предоставляем отдельное перечисление для определенных объектов Name, поэтому перечисление Formula будет обозначать только то, что формулы должны быть загружены сейчас, независимо от того, будут загружены определенные объекты Name или нет. Однако следует отметить одну вещь: обычно определенные объекты Name используются формулами, если пользователь загружает только формулы и не загружает определенные объекты Name, формула ячейки, которая ссылается на эти имена, будет повреждена и может вызвать исключение. Таким образом, обычно, если пользователь хочет загрузить формулы, определенные объекты Name также должны быть загружены. Но пользователь может загружать только определенные объекты Name без загрузки формул.
### **Добавить перечисление SheetType.Dialog**
Представляет диалоговый лист.
### **Добавляет свойство WorkbookSettings.MaxRowsOfSharedFormula.**
Получает и задает максимальное количество строк общей формулы. Общая формула будет разделена на несколько общих формул, если общее количество строк общей формулы больше ее.
### **Добавляет свойство WorkbookSettings.StreamProvider.**
Получает и задает поставщика потоков для внешнего ресурса.
### **Добавляет свойство ShapeTextAlignment.IsAutoMargin.**
Указывает, является ли поле текстового фрейма автоматическим.
### **Добавляет свойство ImportTableOptions.IsFormulas**
Представляет, какой столбец в таблице данных следует импортировать в виде формул.
