{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.0.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.0.1/)

{{% /alert %}} 

 Мы рады сообщить Aspose.Cells for Java v7.0.1!

 Улучшения

 30556 Улучшить функцию преобразования рабочего листа в изображение

 30557 Улучшение преобразования Excel в PDF

30458 Добавить новые API: Worksheet.calculateFormula(), Row.getFirstCell(), Row.getLastCell()

 30487 Поддержка лицензии Aspose.Total

 29838 Визуализация изображения в пользовательскую графику

 30558 Включить метод FormatCondition.getPriority()

 30505 Включить метод Cells.checkColumn(int)

 30504 Включить геттеры/сеттеры для атрибутов Cells.StandardHeightPixels/StandardWidthPixels

 30459 Включить методы WorkbookSettings.setLocale(Locale) и Protection.getPasswordHash()

 30469 Включить геттеры/сеттеры для WorkbookSettings.WindowHeightInch/WindowWidthInch

 Исключения

 29285 Получить исключение при сохранении PDF с французским языковым стандартом

 30459 Получить исключение при чтении файла XLS с языковым стандартом по умолчанию, который не имеет кода страны

 30489 Worksheet.autoFitColumn() и Chart.toImage() могут вызвать «java.awt.HeadlessException»

 30587 Получить исключение при чтении файла шаблона XLS

 Ошибки

30103 Шрифты диаграммы изменены после повторного сохранения файла XLSX

 28858 Сортировка имен приводит к изменению связанной ячейки элемента управления

 30486 Вертикальное выравнивание изменено в пересохраненном файле XLSX
