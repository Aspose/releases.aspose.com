 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 2.5.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.5.1/)

 Мы рады сообщить Aspose.Cells for Java 2.5.1!

 Что изменилось:

· Поддерживает прерывание процесса открытия/сохранения при работе с большими файлами.

 · Обеспечивает поддержку для установки и расчета большего количества формул.

 · Поддерживает получение/установку абсолютной позиции формы.

 · Исправлены ошибки чтения/сохранения файлов Excel 2007.

 · Исправлены ошибки чтения/сохранения ODS файлов.

 · Исправлены ошибки чтения/сохранения файлов Excel 97-2003.

 · Усовершенствования сделаны для функций Chart to Image, Sheet to Image и Excel to PDF.

 · Внесены улучшения для разбора формул.

 · Улучшено вычисление формул.

 · 22 исправления и улучшения.
