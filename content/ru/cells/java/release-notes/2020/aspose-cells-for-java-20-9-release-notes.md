## **Public API и обратно несовместимые изменения**

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Java. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.

### **Добавляет свойство AbstractCalculationEngine.ProcessBuiltInFunctions**

 Из соображений производительности и удобства пользователя мы добавляем это свойство и делаем его значение по умолчанию равным false, чтобы пользователь мог сосредоточиться на тех функциях, которые не поддерживаются встроенным движком. Если существующая пользовательская реализация AbstractCalculationEngine изменила вычисление некоторых встроенных функций, пользователь должен переопределить это свойство, чтобы сделать его как**истинный** от**20.9**.

### **Добавляет свойство TxtLoadOptions.HasTextQualifier.**

Указывает, есть ли текстовый квалификатор для значений ячеек в файле шаблона.

### **Добавляет свойство TxtLoadOptions.TextQualifier.**

Задает квалификатор текста для значений ячеек в файле шаблона.

### **Добавляет свойство HtmlSaveOptions.ImageScalable.**

 Указывает, следует ли использовать масштабируемую единицу для описания ширины изображения. Значение свойства по умолчанию равно**истинный**.

### **Добавляет свойство Slicer.AlternativeText**

Получает или задает описательную (альтернативную) текстовую строку объекта Slicer.

### **Добавляет свойство Slicer.ColumnWidthPixel**

Получает или задает ширину в пикселях для каждого столбца среза.

### **Добавляет свойство Slicer.HeightPixel**

Получает или задает высоту указанного среза в пикселях.

### **Добавляет свойство Slicer.IsLocked**

Указывает, заблокирована ли форма среза.

### **Добавляет свойство Slicer.IsPrintable**

Указывает, доступен ли для печати объект среза.

### **Добавляет свойство Slicer.LeftPixel**

Получает или задает смещение формы среза по горизонтали относительно левого столбца в пикселях.

### **Добавляет свойство Slicer.LockedAspectRatio.**

Указывает, блокируется ли соотношение сторон.

### **Добавляет свойство Slicer.Placement**

Представляет способ прикрепления объекта рисования к ячейкам под ним. Свойство управляет размещением объекта на рабочем листе.

### **Добавляет свойство Slicer.RowHeightPixel**

Возвращает или задает высоту в пикселях каждой строки в указанном срезе.

### **Добавляет свойство Slicer.Title**

Указывает заголовок текущего объекта Slicer.

### **Добавляет свойство Slicer.TopPixel**

Получает или задает вертикальное смещение формы среза от ее верхней строки в пикселях.

### **Добавляет свойство Slicer.WidthPixel**

Получает или задает ширину указанного среза в пикселях.

### **Добавляет свойство Worksheet.PaneState и перечисление PaneStateType.**

Представляет состояние панели на листе.

### **Добавляет свойство OdsLoadOptions.RefreshPivotTables.**

Указывает, обновляется ли сводная таблица при загрузке файлов .ods.

### **Добавляет свойство FilterColumn.IsDropdownVisible.**

Указывает, видна ли кнопка автофильтра для этого столбца.

### **Устарело свойство Filter.Visibledropdown.**

Вместо этого используйте FilterColumn.IsDropdownVisible.

### **Добавляет свойство CopyOptions.KeepMacros.**

Указывает, сохраняются ли макросы в целевой книге. Это действует только в том случае, если исходная книга не содержит макросов.

### **Добавляет перегруженный метод Workbook.Copy(Workbook,CopyOptions).**

Копирует книгу с параметрами.

### **Добавляет перечисление WarningType.InvalidAutoFilterRange.**

Представляет тип предупреждения о том, что диапазон не может быть отфильтрован автоматически.

### **Добавляет свойство Chart.DisplayNaAsBlank.**

Указывает, отображается ли #N/A как пустое значение.

### **Добавляет перечисление CrossType.Minimum.**

Представляет пересечение осей при минимальном значении.

### **Добавляет свойство XlsbSaveOptions.ExportAllColumnIndexes.**

Указывает, следует ли экспортировать индексы столбцов для всех ячеек при сохранении файлов XLSB.
