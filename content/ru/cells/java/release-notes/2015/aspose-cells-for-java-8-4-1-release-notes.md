 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 8.4.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-8.4.1/)

Ниже приведен список улучшений и изменений в этом выпуске Aspose.Cells.

## Aspose.Cells

### **Основные характеристики**

Основная кодовая база перенесена на Java 6 (также поддерживаются Java 7 и 8). Прекращена поддержка for Java 5 и 1.4.

### **Другие улучшения и изменения**

### **Новые особенности**

(CELLSJAVA-41235) — Поддержка RenderToSize API для изображения рабочего листа

(CELLSJAVA-41234) — Поддержка маркеров при использовании SmartMarkers или метода Cell.setHtmlString.

### **Ошибки**

(CELLSJAVA-41229) - Aspose.Cells не создает отдельные файлы HTM и CSS для листов в Excel для преобразования HTML

(CELLSJAVA-41170) — SheetRender.toImage визуализирует изображение с метками «(пусто)» на оси X диаграммы.

(CELLSJAVA-41270) - Проблема с Cells.insertRange(), так как объединенная область не смещается нормально

(CELLSJAVA-41240) — Текст шрифтом Arial обрезается сверху при отображении электронной таблицы в PDF.

(CELLSJAVA-41238) - БУМАГА_А_2 не работает должным образом при сохранении как PDF

(CELLSJAVA-41217) - Если данные категории ряда содержат запятую, легенды круговой диаграммы отображаются неправильно.

(CELLSJAVA-41194) - Перекрытие записей легенды при преобразовании диаграммы в изображение

(CELLSJAVA-41002) - Пунктирная линия отсутствует на диаграмме 1

(CELLSJAVA-40993) - На диаграмме роста отсутствуют горизонтальные линии сетки.

(CELLSJAVA-41259) — Установка Name.setRefersTo и пересчет формул приводят к неправильному значению при преобразовании электронной таблицы в HTML.

(CELLSJAVA-41258) - Загрузка и сохранение XLSX с Aspose.Cells приводит к повреждению результирующей электронной таблицы.

(CELLSJAVA-41255) — пользовательская кнопка становится изображением, а заголовок исчезает в выводе XLSX

(CELLSJAVA-41254) - Microsoft Excel аварийно завершает работу при открытии выходного файла XLSX

(CELLSJAVA-41253) — Выпадающий список исчезает в выходном файле XLSX.

### **Исключения**

(CELLSJAVA-41266) — ошибка java.lang.NumberFormatException при открытии файла шаблона XLSX.

(CELLSJAVA-41248) — исключение нулевого указателя при открытии исходного файла XLSX.

(CELLSJAVA-41265) — Исключение: «java.lang.NullPointerException» при открытии файла SpreadsheetML.

(CELLSJAVA-41264) — Исключение при использовании Cell.getStringValueWithoutFormat

(CELLSJAVA-41246) - Исключение: недопустимая динамическая формула... с использованием индексной функции при использовании динамических формул интеллектуальных маркеров

## **Public API и обратно несовместимые изменения**

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Java. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.

Добавляет свойство Workbook.IsLicensed.

Указывает, была ли установлена лицензия.

Устаревший метод Workbook.ValidateFormula.

Вместо этого используйте свойство Cell.Formula.

Добавляет свойство ImageOrPrintOptions.SVGFitToViewPort.

Указывает, подходит ли сгенерированное изображение SVG для порта просмотра.

Добавляет метод ImageOrPrintOptions.SetDesiredSize.

Устанавливает желаемую ширину и высоту изображения.

Добавляет метод Aspose.Cells.GridDesktop.WorksheetCollection.MoveTo.

Перемещает лист с указанным индексом на другой индекс.

**Запись**

Поскольку кодовая база Aspose.Cells for Java соответствует коду соответствующей версии .NET, большинство изменений, улучшений и исправлений, включенных в Aspose.Cells for .NET v8.4.1, также включены в этот Aspose.Cells for Java v8.4.1.
