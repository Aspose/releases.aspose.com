## **Ошибки**


 (CELLSJAVA-41323) - Водяные знаки отображаются неправильно.

 (CELLSJAVA-41319) — слайсер удаляется в Excel 2010/2013 после сохранения.

 (CELLSJAVA-41317) — символы верхнего регистра, созданные с помощью функции «Все прописные» в Excel, не сохраняются в PDF.

 (CELLSJAVA-41315) - Неправильная визуализация фигур на странице 5.

 (CELLSJAVA-41308) — неверное расположение формы при отображении электронной таблицы в формате PDF.

 (CELLSJAVA-41282) - увеличены наконечники стрелок при преобразовании электронной таблицы с чертежами в PDF.

 (CELLSJAVA-41249) — WordArt неправильно отображается в выходном файле PDF.

 (CELLSJAVA-41243) — Горизонтальный текст в фигурах отображается вертикально при преобразовании электронной таблицы в PDF.

(CELLSJAVA-41242) — рисунок искажается при рендеринге электронной таблицы на PDF.

 (CELLSJAVA-41241) — круглые формы преобразуются в овалы при рендеринге электронной таблицы в PDF.

 (CELLSJAVA-41233) — объекты рисования и фигуры в шаблоне Excel неправильно отображаются в выходных данных PDF с использованием Aspose.Cells.

 (CELLSJAVA-41302) — В последней версии Aspose не удается сохранить файл, сохраненный в более ранней версии.

 (CELLSJAVA-41299) — файл повреждается при сохранении xls в формате xlsx.

 (CELLSJAVA-41284) - Workbook.copy() повреждает выходной файл Excel.

 (CELLSJAVA-41283) - Ошибка вычисления с функцией ИЛИ

 (CELLSJAVA-41281) — Операция автоподбора столбцов не работает с несколькими ячейками с включенным параметром ShrinkToFit.

 (CELLSJAVA-41313) — Сплошные линии отображаются как пунктирные.

 (CELLSJAVA-41306) - Проблема преобразования Excel в PDF - обрезана правая сторона

 (CELLSJAVA-41316) — Некоторые метки данных отсутствуют в обработанном файле HTML при копировании книги.

(CELLSJAVA-41314) — метки данных на диаграмме не отображаются на визуализируемом изображении EMF.

 (CELLSJAVA-41311) - Отсутствуют метки оси диаграммы при преобразовании в EMF

 (CELLSJAVA-41301) — еврейские символы в «умном искусстве», представленные в Excel, зеркально отражены в представлении PDF.

 (CELLSJAVA-41300) - Неверные данные круговой диаграммы при отображении через Aspose

 (CELLSJAVA-41285) — Высота области графика диаграммы увеличивается после объединения рабочих книг.

 (CELLSJAVA-41277) - Отображение только круговой диаграммы приводит к пустому значению PDF

 (CELLSJAVA-41276) — Chart.toImage создает пустое изображение для круговой диаграммы.

 (CELLSJAVA-41275) - Две метки оси отсутствуют в результирующем EMF при преобразовании диаграммы в изображение.

 (CELLSJAVA-40606) - Визуализированное изображение диаграммы неверно (от диаграммы к изображению).

 (CELLSJAVA-40368) - Chart.calculate() неправильно размещает метки осей X и Y.
