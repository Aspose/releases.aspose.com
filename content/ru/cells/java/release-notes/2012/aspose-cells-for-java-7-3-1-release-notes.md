 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.3.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.3.1/)

{{% /alert %}} 

Мы
 рад сообщить Aspose.Cells for Java v7.3.1 для пользователей!

 Новые особенности

 -Скрипты анализа в верхних/нижних колонтитулах страницы настройки

- Условное форматирование — включить отсутствующие поля типа DataBar
- Условное форматирование — включить отсутствующие значения типа IconSet.
- Условное форматирование - Поддержка
- Чтение правил условного форматирования с межстраничными формулами
- Поддержка свойств Cells.MinDataColumn и Cells.MinDataRow.
- Поддержка цветов фона Cell с условным форматированием (MS Excel 2010)
- Поддерживаются фильтры данных сводной таблицы.
- Поддерживается расширенная проверка данных MS Excel 2010.

 (Запись:

Все вышеуказанные билеты
изначально добавляются от имени пользователей .NET. Начиная с нашей версии Java
продукт в равной степени соответствует (в отношении функций и улучшений) с .NET
сейчас, поэтому мы включили эти новые функции/улучшения в версию Java.
 также продукта.
) 

 Улучшения

 -Временные файлы шрифтов создаются более одного раза при сохранении в PDF

 - Дата в верхнем/нижнем колонтитуле не была отформатирована в соответствии с локальными настройками рабочей книги.

- Опция Supportnew: Aspose.Cells.Disable=SunFontManager вместо java.awt.headlessдля сбоя JVM Open JDK

 -Экспорт автоформ для файла HTML

 Исключения

- Исключение: «Ошибка преобразования формы в изображение» при сохранении в PDF

 -Shape to Image проблема при сохранении в PDF

- «NullPointerException» для метода Chart.calculate()

 -Сохранение XLS как PDF вызвало исключение

 -Сохранение XLS как PDF вызвало Исключение II

 Ошибки

 - Перекрывающийся текст и пропущенные линии сетки для сохранения PDF

 При пересохранении показывались лишние границы

 файл шаблона как файл XLS

 -Чтение имени со ссылкой "!$A$1" вызвало исключение

 -PDF генерация не удалась с определенными данными диаграммы

 -Формулы неверны после вставки диапазона

 - Сгенерированный PDF из книги Excel имел большее количество страниц.

 - Метки диаграммы смещались и перекрывались при копировании листа.
