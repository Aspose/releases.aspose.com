 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.1.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.2/)

{{% /alert %}} 

Мы
 рад сообщить Aspose.Cells for Java v7.1.2!

 Новые особенности

 – Поддержка метода Cells.importCustomObjects() для импорта POJO.

 Улучшения

- Внесены улучшения в методы autoFitColumns().
- Производительность функции «Лист-в-изображение» с включенными линиями сетки

 Исключения

- NullPointerException возникло при вызове метода Cell.getDisplayStyle()

 Ошибки

- Максимальное значение для точечной диаграммы не может быть правильно установлено
- Текст в фигурах был потерян или перемещен за пределы диапазона фигур в сгенерированном файле PDF.
- Название серии не отображалось для меток даты на диаграмме
