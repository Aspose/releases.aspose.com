{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.3.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.3.2/)

{{% /alert %}} 

Мы
 рад сообщить Aspose.Cells for Java v7.3.2!

 Новые особенности

- Shape.getRight()/getBottom() для получения смещения фигуры от правого нижнего угла до ячейки
- Установить цвет вкладки рабочего листа на цвет по умолчанию

 Улучшения

- Закрыть внутренние потоки для освобождения ресурсов памяти в преобразовании PDF
- Копировать спарклайны при копировании листа

 Исключения

- Открытие файла XLS вызвало StackOverflowError
- Сохранение PDF вызвало исключение
- Worksheet.getFreezedPanes() вызвало исключение NullPointerException
- Открытие пустого файла XML вызвало исключение
- Сохранение в PDF вызвало исключение -I
- Сохранение в PDF вызвало исключение -II
- Сохранение в PDF вызвало исключение -III
- Сохранение в PDF вызвало исключение -IV
- Исключение при открытии файла шаблона HTM
- IllegalArgumentException при открытии файла XLS
- NullPointerException при открытии XLS-файла
- Исключение ArrayIndexOutOfBounds при открытии файла XLS
- Сохранение сводной таблицы вызвало ClassCastException после настройки автосортировки
- Ошибка преобразования формы в изображение при сохранении PDF с помощьюAspose Cells
- CellsException при печати линий сетки в PDF

 Ошибки

- Некоторые регионы, сохраненные в шаблонах XLS, не могли быть распознаны при чтении
- Chart-to-Image не удается отобразить шкалу для линейчатых диаграмм
- Потеря данных после преобразования с XLSX на PDF
- Круговая и точечная диаграммы не работают, если Cellcollection очищен
