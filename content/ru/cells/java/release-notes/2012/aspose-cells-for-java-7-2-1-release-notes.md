{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.2.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.1/)

{{% /alert %}} 

Мы
 рад сообщить Aspose.Cells for Java v7.2.1!

 Новые особенности

- Формат сводной таблицы с пользовательскими стилями для разных категорий (изменение экспресс-стиля сводной таблицы)

 Улучшения

- Cells.findString()/find() поддерживает поиск RegExand в определенном диапазоне
- Поддержка Picture.setTitle()/getTitle()
- Сохранить диаграммы MS Excel в файле ODS
- Сделать файл Aspose.Cells созданным XLS совместимым с POI

 Исключения

- Чтение файла XLSX дает: «java.lang.ClassCastException: org.dom4j.Namespace».

 Ошибки

- Сохраненный файл XLSX выдает ошибку: "Datamay был потерян"
- Неправильный формат номера в сгенерированном PDF (были потеряны тысячи групповых символов)
- Гистограмма не отображалась в сгенерированном PDF для версии JDK6.
- Ссылки не обновляются при расширении диапазона
