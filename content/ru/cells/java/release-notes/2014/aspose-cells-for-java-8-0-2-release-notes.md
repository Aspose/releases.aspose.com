{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 8.0.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-8.0.2/)

{{% /alert %}} 

 Aspose.Cells for Java был обновлен до версии 8.0.2, и мы рады сообщить, что в этом выпуске добавлено более 10 новых полезных улучшений.
Используя Aspose.Cells for Java, вы можете работать с XLS, SpreadsheetML, OOXML, XLSB, CSV, HTML, ODS, PDF, XPS и другими форматами в ваших приложениях. Вы также можете создавать, изменять, преобразовывать, визуализировать и печатать рабочие книги, не используя Microsoft Excel.
Посетите документацию, чтобы узнать, как начать работу с Aspose.Cells for Java.
Обратите внимание, что этот загружаемый файл содержит полностью рабочую версию продукта, однако без набора лицензий он будет работать в ознакомительном режиме с некоторыми ограничениями. Чтобы протестировать Aspose.Cells без этих ознакомительных ограничений, вы можете запросить бесплатную 30-дневную временную лицензию.
Ниже приведен список изменений в этой версии Aspose.Cells for Java.


Другие улучшения и изменения

Улучшения

(CELLSJAVA-40788) - Поддержка пользовательской темы для свойств формы.
(CELLSJAVA-40803) - Установите подсказки по визуализации для изображений при экспорте электронных таблиц на HTML

Ошибки

(CELLSJAVA-40793) - Диапазон не относится к правильной области
(CELLSJAVA-40768) - Метод WorkbookRender.toPrinter() не печатает изображение
(CELLSJAVA-40669) — Большая проблема сводного столбца при рендеринге с XLTX по PDF.
(CELLSJAVA-40801) — Cell проблемы с выравниванием в отрендеренном файле PDF
(CELLSJAVA-40406) — преобразование файла Excel с большим количеством столбцов в файл PDF.
(CELLSJAVA-40794) — AutoFitColumns не работает при использовании с другими настройками шрифта.
(CELLSJAVA-40816) - Курсор все еще перемещается к последнему столбцу после использования Cells.DeleteColumn() для его удаления.
(CELLSJAVA-40786) - Сгенерированная форма ЭДС отличается от исходной.
(CELLSJAVA-40806) — закладки Excel не создаются при преобразовании в PDF


Исключения

(CELLSJAVA-40797) - Cell.getDependents() генерирует исключение NullPointerException
(CELLSJAVA-40800) — Исключение CellsException при преобразовании электронной таблицы в PDF в ОС MAC.

Public API и обратно несовместимые изменения

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for .NET. Если у вас есть сомнения по поводу каких-либо перечисленных изменений, сообщите об этом на форум поддержки Aspose.Cells.

Добавляет свойство Shape.TextDirection
Получает/задает направление потока текста для фигуры.

Добавляет свойство HTMLLoadOptions.ConvertFormulasData.
Указывает, следует ли преобразовывать строку в формулу, если строковое значение, начинающееся с символа '=', является строкой формулы, значение по умолчанию — false.

Добавляет свойство HtmlSaveOptions.ImageOptions.
Получает/устанавливает параметры рендеринга при сохранении html-файлов.

Устарело свойство HtmlSaveOptions.ExportChartImageFormat.
Вместо этого использует HtmlSaveOptions.ImageOptions для параметров формата изображения при сохранении html-файлов.


Запись
Поскольку кодовая база Aspose.Cells for Java соответствует коду соответствующей версии .NET, большинство изменений, улучшений и исправлений, включенных в Aspose.Cells for .NET v8.0.2, также включены в этот Aspose.Cells for Java v8.0.2.
