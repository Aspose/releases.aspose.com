{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 8.1.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-8.1.2/)

{{% /alert %}} 

 Aspose.Cells for Java был обновлен до версии 8.1.2, и мы рады сообщить, что в этом выпуске добавлено более 20 новых полезных улучшений.
Используя Aspose.Cells for Java, вы можете работать с XLS, SpreadsheetML, OOXML, XLSB, CSV, HTML, ODS, PDF, XPS и другими форматами в ваших приложениях. Вы также можете создавать, изменять, преобразовывать, визуализировать и печатать рабочие книги, не используя Microsoft Excel.
Посетите документацию, чтобы узнать, как начать работу с Aspose.Cells for Java.
Обратите внимание, что этот загружаемый файл содержит полностью рабочую версию продукта, однако без набора лицензий он будет работать в ознакомительном режиме с некоторыми ограничениями. Чтобы протестировать Aspose.Cells без этих ознакомительных ограничений, вы можете запросить бесплатную 30-дневную временную лицензию.
 Ниже приведен список изменений в этой версии Aspose.Cells for Java.

\1) Aspose.Cells

Другие улучшения и изменения

Новые особенности

(CELLSJAVA-40875) — получать предупреждения о замене шрифта при рендеринге электронных таблиц.

Улучшения

(CELLSJAVA-40900) - Обфускация общедоступных методов API
(CELLSJAVA-40891) — Процесс зависает при загрузке полностью поврежденной электронной таблицы.
(CELLSJAVA-40883) - Проблема с форматом даты при импорте CSV
(CELLSJAVA-40872) - worksheet.getCells().importResultSet, время в столбце даты всегда равно 00:00.

Ошибки

(CELLSJAVA-40866) — Преобразование в HTML не учитывает ImageFormat в SaveOptions.
(CELLSJAVA-40854) — HtmlHiddenRowDisplayType.HIDDEN приводит к смещению ячеек в результате HTML (проблема охвата)
(CELLSJAVA-40835) — проблема экспорта скрытых столбцов в файле HTML с визуализацией.
(CELLSJAVA-40879) - Проблема при создании изображения диапазона данных (от листа к изображению).
(CELLSJAVA-40878) - Установка разрешения по вертикали и горизонтали при сохранении электронной таблицы в изображение Jpeg не действует.
(CELLSJAVA-40877) - Excel для PDF - Плохое качество изображения, полученное с помощью Aspose Cells 8.xx
(CELLSJAVA-40910) — изображения теряются при рендеринге PDF с использованием PdfSaveOptions.setImageType(ImageFormat.getPng()).
(CELLSJAVA-40907) - Маркеры точек данных отсутствуют на диаграмме, когда файл шаблона Excel сохраняется как HTML.
(CELLSJAVA-40904) - Некоторые диаграммы не отображаются нормально в формате файла HTML.
(CELLSJAVA-40899) - Проблема с усечением данных на листе 18
(CELLSJAVA-40898) - Проблема с усечением данных на листе 17
(CELLSJAVA-40886) — метки серии нарушаются при повторном сохранении файла Excel.
(CELLSJAVA-40885) - Форма отсутствующей точки данных экспорта диаграммы в формате выходного изображения
(CELLSJAVA-40869) — В уравнениях отсутствуют глифы, а определенный форматированный текст усекается в визуализируемом файле PDF.
(CELLSJAVA-40865) - Изображение в выходном PDF-файле нечеткое.
(CELLSJAVA-40860) - Изменены свойства пузырьков на диаграмме при повторном сохранении файла шаблона XLSX.
(CELLSJAVA-40859) - Изменены свойства пузырьков на диаграмме при повторном сохранении файла шаблона XLSX.
(CELLSJAVA-40858) — свойство Column100PercentStacked или Bar label было потеряно
(CELLSJAVA-40817) - Изображение в выходном PDF-файле становится нечетким.
(CELLSJAVA-40880) — тип DateTime не определяется при добавлении значения DateTime во время выполнения через Aspose.Cells.
(CELLSJAVA-40851) - Изменена ширина фигур и потеряно другое форматирование в скопированной книге.
