{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 8.2.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-8.2.1/)

{{% /alert %}} 

 Aspose.Cells for Java был обновлен до версии 8.2.1, и мы рады сообщить, что в этом выпуске добавлено более 30 новых полезных улучшений.
Используя Aspose.Cells for Java, вы можете работать с XLS, SpreadsheetML, OOXML, XLSB, CSV, HTML, ODS, PDF, XPS и другими форматами в ваших приложениях. Вы также можете создавать, изменять, преобразовывать, визуализировать и печатать рабочие книги, не используя Microsoft Excel.
Посетите документацию, чтобы узнать, как начать работу с Aspose.Cells for Java.
Обратите внимание, что этот загружаемый файл содержит полностью рабочую версию продукта, однако без набора лицензий он будет работать в ознакомительном режиме с некоторыми ограничениями. Чтобы протестировать Aspose.Cells без этих ознакомительных ограничений, вы можете запросить бесплатную 30-дневную временную лицензию.
 Ниже приведен список изменений в этой версии Aspose.Cells for Java.

\1) Aspose.Cells

Другие улучшения и изменения

Новые особенности

(CELLSJAVA-40955) - Абсолютное позиционирование формы
(CELLSJAVA-40943) — Добавлен параметр PasteOptions для вставки только видимых ячеек из диапазона.

(CELLSJAVA-40953) — некоторые границы ячеек были немного расширены при преобразовании excel в html.
(CELLSJAVA-40980) — значение связанной ячейки не обновляется из внешней книги.

(CELLSJAVA-40979) — Некоторые квадраты прикреплены к меткам данных на круговой диаграмме в представленном PDF.
(CELLSJAVA-40975) - Преобразование XLSX в Jpeg - Производительность

(CELLSJAVA-40833) - Низкое качество изображения диаграммы - Преобразование диаграммы в изображение

(CELLSJAVA-40946) - Метки данных находятся в неправильном положении внутри изображения диаграммы.
(CELLSJAVA-40821) - Текстовое поле отсутствует, когда диаграмма сохраняется как EMF с помощью ToImage
(CELLSJAVA-40819) - Неправильные значения оси, когда диаграмма сохраняется как EMF с использованием ToImage
(CELLSJAVA-40818) - Отсутствует заголовок оси, когда диаграмма сохраняется как EMF с использованием ToImage

Public API и обратно несовместимые изменения

Ниже приведен список любых изменений, внесенных в общедоступный номер API, таких как добавленные, переименованные, удаленные или устаревшие члены, а также любые несовместимые с предыдущими изменениями, внесенные в номер Aspose.Cells for Java.  Добавляет метод Cell.GetValidation()
Получает проверку, которая применяется к ячейке.

 Добавляет метод Cell.GetValidationValue()
Указывает, допустимо ли значение ячейки.

 Добавляет метод WorkbookRender.ToPrinter(PrinterSettings PrinterSettings)
Рендеринг книги на принтер через PrinterSettings.


Запись
Поскольку кодовая база Aspose.Cells for Java соответствует коду соответствующей версии .NET, большинство изменений, улучшений и исправлений, включенных в Aspose.Cells for .NET v8.2.1, также включены в этот Aspose.Cells for Java v8.2.1.
