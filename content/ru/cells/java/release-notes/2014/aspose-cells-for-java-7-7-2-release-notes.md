{{% alert color="primary" %}} 

 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.7.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.7.2/)

{{% /alert %}} 

 Aspose.Cells for Java был обновлен до версии 7.7.2, и мы рады сообщить, что в этом выпуске добавлено около 20 новых полезных улучшений.
Используя Aspose.Cells for Java, вы можете работать с XLS, SpreadsheetML, OOXML, XLSB, CSV, HTML, ODS, PDF, XPS и другими форматами в ваших приложениях. Вы также можете создавать, изменять, преобразовывать, визуализировать и печатать рабочие книги, не используя Microsoft Excel.
Посетите документацию, чтобы узнать, как начать работу с Aspose.Cells for Java.
Обратите внимание, что этот загружаемый файл содержит полностью рабочую версию продукта, однако без набора лицензий он будет работать в ознакомительном режиме с некоторыми ограничениями. Чтобы протестировать Aspose.Cells без этих ознакомительных ограничений, вы можете запросить бесплатную 30-дневную временную лицензию.
Ниже приведен список изменений в этой версии Aspose.Cells for Java.

Основные характеристики
Поддержка открытия защищенных паролем файлов MS Excel 2013

Другие улучшения и изменения

Новые особенности

(CELLSJAVA-40475) - Печать комментариев к документу Excel
(CELLSJAVA-40732) - Поддержка настройки распределенного выравнивания по выравниванию

Улучшения

(CELLSJAVA-40719) — Удалено использование жесткого кода com.sun.image.codec.jpeg.JPEGCodec.
(CELLSJAVA-40708) - Показать строку с ранее назначенной высотой

Ошибки

(CELLSJAVA-40711) — Axis.setTickLabelSpacing() не работает для версии XLS.
(CELLSJAVA-40714) - Cell границы были отрисованы на непрозрачных ячейках.
(CELLSJAVA-40710) - Меньше равно ошибке вычисления
(CELLSJAVA-40715) — Проблема с анализом подстановочных знаков при расчете формулы.
(CELLSJAVA-40707) - Некоторые столбцы слишком узкие, чтобы правильно вмещать текст.
(CELLSJAVA-40720) — IStreamProvider.setCustomPath не влияет на файл tabstrip.htm
(CELLSJAVA-40729) - Извлечение форматированной китайской даты
(CELLSJAVA-40728) - getStringValue() возвращает #ЗНАЧ!
(CELLSJAVA-40722) - Проверка не работает после преобразования в формат XLS
(CELLSJAVA-40690) - с XLSX по PDF: подчеркивание под гиперссылкой распространяется на соседние позиции символов

Исключения

(CELLSJAVA-40629) — Исключение при открытии xlsx, преобразованного в xls
(CELLSJAVA-40630) - Исключение при открытии xlsx, преобразованного в xls
(CELLSJAVA-40717) — CellsException: недопустимое имя ячейки в конструкторе рабочей книги.
(CELLSJAVA-40723) — Рендеринг cuased com.aspose.cells.CellsException: java.lang.String нельзя преобразовать в java.lang.Double
(CELLSJAVA-40724) — com.aspose.cells.CellsException на SheetRender
(CELLSJAVA-40727) — Исключение: java.lang.NullPointerException при отображении диаграмм на листе.

Запись
Поскольку кодовая база Aspose.Cells for Java соответствует коду соответствующей версии .NET, большинство изменений, улучшений и исправлений, включенных в Aspose.Cells for .NET v7.7.2, также включены в этот Aspose.Cells for Java v7.7.2.
