 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 2.4.3](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.3/)

{{% /alert %}} 

 Мы рады сообщить Aspose.Cells for Java 2.4.3!

 Что изменилось:

- Поддерживает получение цветов темы Excel 2007.
- Экспортирует диаграммы/фигуры в виде изображений для преобразования файлов HTML.
- Поддерживает чтение/запись разных верхних/нижних колонтитулов для первых/четных/нечетных страниц.
- Исправлены некоторые внутренние ошибки чтения/сохранения файлов Excel 2007.
- Исправлены некоторые внутренние ошибки чтения/сохранения файлов ODS.
- Исправлены некоторые внутренние ошибки чтения/сохранения файлов Excel 97-2003.
- Улучшение сделано для функции Chart to Image.
- Улучшение сделано для функции Direct Excel до PDF.
- 14 исправлений и улучшений.
