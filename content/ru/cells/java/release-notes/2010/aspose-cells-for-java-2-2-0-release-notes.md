 Мы рады сообщить Aspose.Cells for Java 2.2.0!

 Что изменилось:

- Устанавливает формулы со строкой/столбцом/параметрами, превышающими лимит MS Excel 2003
 Поддерживает сохранение исходных данных, считанных из файла шаблона MS Excel 2010.
 Управление спарклайнами MS Excel 2010
 Предоставляет расширенные стили, сохраненные MS Excel 2007 для файлов XLS.
 Поддерживает автоматическое определение типа формата файла при открытии файла шаблона без указания формата для файлов Html и SpreadSheeML.
 Удаляет диаграмму из коллекции Charts
 Позволяет удалять пустые строки/столбцы на листе
Поддерживает сохранение цвета до ближайшего соответствующего цвета в палитре, когда указанный пользователем цвет отсутствует в стандартной палитре.
 Экспорт атрибута поворота текста для функции Excel в Pdf
 Экспорт диаграмм в виде изображений для функции Excel в PDF
 Удаляет существующую область печати
 Включает улучшения для сохранения объединенных областей: проверяйте и удаляйте или объединяйте те дублированные/перекрывающиеся области, которые могут привести к тому, что сгенерированный файл будет отображать предупреждающее сообщение при открытии в MS Excel.
 Включает улучшения для добавления разрывов страниц: проверяйте и удаляйте повторяющиеся разрывы страниц перед сохранением.
 Включает улучшение функции Chart to Image
 65 исправлений и других улучшений.

 Проблемы, решенные в Aspose.Cells для Jav

 Заметные изменения для пользователей:



 В старых версиях методы Workbook.save(String) и Workbook.save(InputStream) всегда будут сохранять результирующий файл в формате файла Excel97TO2003.

Начиная с этой версии, если был указан тип формата рабочей книги, методы Workbook.save(String) и Workbook.save(InputStream) будут сохранять результирующий файл в формате, указанном рабочей книгой. Тип формата книги можно задать методом Workbook.setFileFormatType(int). Или его можно установить в качестве формата входного файла шаблона автоматически при открытии существующего файла шаблона.

 Кроме того, ограничение количества строк/столбцов формул и ограничение количества параметров формул также зависят от типа формата рабочей книги. Прежде чем вы превысите лимит строк/столбцов/параметров формул для MS Excel 2003, вы должны явно установить формат рабочей книги для некоторых других типов, таких как EXCEL2007.
