 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 2.2.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.2.1/)

 Мы рады сообщить Aspose.Cells for Java 2.2.1!

 Что изменилось:

-  Включает поддержку расширенного условного форматирования, предоставляемого MS Excel 2007.
-  Поддерживает расширенные записи условного форматирования, сохраненные MS Excel 2007 для файла XLS.
- Сохраняет неподдерживаемые формулы при чтении файлов MS Excel 2007
-  Читает изображения из верхнего/нижнего колонтитула в файле шаблона MS Excel 2007
-  Внесены улучшения для чтения файлов шаблонов
-  17 исправлений и других улучшений.
