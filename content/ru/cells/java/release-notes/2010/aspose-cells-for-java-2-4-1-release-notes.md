 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 2.4.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.1/)

{{% /alert %}} 

Мы рады сообщить Aspose.Cells for Java v2.4.1!

 Что изменилось:

- Поддерживает более гибкую модель для расчета пользовательских функций.
- Включает параметры проверки ошибок для ячеек.
- Добавлен математический расчет параметров Range в формулах.
- Предоставляет поддержку для установки связанных ячеек для рамок диаграммы, таких как заголовок, метки данных.
- Поддерживает управление свойствами положения и размера окна для книги.
- Улучшено чтение/сохранение файлов Excel 2007.
- Функция Chart to Image улучшена.
- Функция Direct Excel to PDF улучшена.
- Улучшено форматирование значений Cell.
- Улучшена операция чтения файлов Excel 97-2003.
- Доработан модуль сохранения графиков.
- Усовершенствован механизм расчета формул.
- 21 исправление и улучшение.
