 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 1.9.0.0](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.0.0/)

 Дорогие клиенты,

 Мы выпустили Aspose.Cells for Java V1.9.0!

 Что нового?

- Поддерживает экспорт в файл xml Aspose.Pdf
 Поддерживает копирование комментариев и именованных диапазонов при копировании листа.
 Поддерживает интерфейс ICustomFunction расчета формул
Поддерживает отслеживание предшествующих и зависимых элементов ячейки

 Что улучшено?

- Поддерживает только расчет всех формул на листе
 Поддерживает POJO и POJO со смарт-маркерами

 Что исправлено?

- Исправлена ошибка получения именованных диапазонов рабочего листа
 Исправлена ошибка создания сводной таблицы
 Исправлена ошибка защиты файла
 Исправлена ошибка вычисления формул
 Исправлена ошибка сохранения файла.
