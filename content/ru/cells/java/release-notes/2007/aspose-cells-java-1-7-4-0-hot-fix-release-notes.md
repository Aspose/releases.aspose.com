 Дорогие клиенты,

 Мы выпустили Aspose.Cells for Java V1.7.4! Включены следующие улучшения и исправления:

- Добавляет методы Column.applyStyle и Row.applyStyle.
 Исправлены некоторые ошибки установки и вычисления формул.
 Исправлена ошибка копирования ряда.
 Исправлена ошибка с замораживанием окон.
 Исправлена ошибка сохранения графиков.
 Исправлена ошибка сохранения сводных таблиц.
 Исправлена ошибка метода Cells.importResultSet().

Примечание. Метод getNumerOfSheets Workbook/Worksheets был переименован в getNumberOfSheets. И лучше вызывать Worksheets.size(), чтобы получить количество листов.
