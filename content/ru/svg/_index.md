## Зачем скачивать Aspose.SVG для .NET?

Этот локальный API .NET поможет вам использовать [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) функциональность в ваших C#, VB.NET, ASP.NET и других приложениях на основе .NET.

## Особенности обработки файлов SVG

- [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/) файлы формата.
- Преобразование SVG в другие [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- Управление деревом DOM в соответствии с официальными спецификациями SVG.
- Поддержка навигации по содержимому с помощью [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), селекторов CSS, элементов и функций обхода документа.
- Поддержка качественного рендеринга.
