## Зачем скачивать Aspose.PUB для .NET?

Aspose.PUB для .NET — это простой API, который позволяет читать и преобразовывать файлы Microsoft Publisher® (PUB) в формат PDF. Он также предоставляет простые для понимания интерфейсы для редактирования метаданных файлов PUB.

## Особенности обработки файлов PUB

- Чтение файлов Microsoft Publisher® (PUB) для преобразования в формат PDF.
- Чтение и запись метаданных файлов PUB через API.
