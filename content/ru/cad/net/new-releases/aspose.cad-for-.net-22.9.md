# Экспорт венгерских специальных символов

Ранее в некоторых случаях венгерские специальные символы ошибочно экспортировались из DWG в PDF. В этом выпуске API вышеупомянутая проблема была решена.

# Улучшено преобразование DWG в PDF

Исправлена ошибка «OutOfMemoryError» при преобразовании формата «DWG» в формат «PDF».

# Улучшенная чувствительность к движущимся сценам

Улучшена чувствительность движущейся сцены.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.CAD for .NET 22.9 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-9-release-notes/).
