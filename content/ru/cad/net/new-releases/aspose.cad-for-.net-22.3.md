# Поддержка большого шрифта

Добавлена поддержка большого шрифта в API.

# Преобразование DWG в PDF с эффективным использованием памяти

Раньше, когда DWG конвертировался в формат PDF, он потреблял много памяти. Сейчас эта проблема решена. Теперь вы можете конвертировать DWG в PDF с улучшенными результатами.

# Улучшен экспорт DAE в OBJ

В некоторых предыдущих случаях экспорт DAE в OBJ завершался сбоем. Теперь эта проблема решена в API.
