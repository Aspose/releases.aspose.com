# Лучшее управление памятью

Утечка памяти наблюдалась при преобразовании AutoCAD® DXF в формат TIFF (растровая графика). Таким образом, этот выпуск API предлагает лучшее управление памятью для преобразования DXF в TIFF.

# Улучшенный экспорт DWG в WMF

Улучшен экспорт AutoCAD® `DWG` в `WMF` (метафайл Windows).

# Экспорт встроенного изображения

Возможность экспорта изображения из встроенных данных изображения.

# Улучшенный рендеринг `DWG`

Было замечено, что `DWG` визуализировался без цветов. Теперь эта проблема решена в этом выпуске API.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.CAD for .NET 22.4 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-4-release-notes/).
