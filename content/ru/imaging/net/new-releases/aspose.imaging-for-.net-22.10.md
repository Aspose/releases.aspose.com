# Поддержка 16-битного/64-битного PNG канала

Добавлена поддержка 16-битных/64-битных изображений PNG.

```csharp
using (RasterImage image = (RasterImage)Image.Load("image0.png"))
{
    ImageOptionsBase options = image.GetOriginalOptions();
    image.Save("result.png", options);
}
```

# Улучшенная обработка цветов CMYK при экспорте в PSD

Раньше экспортер PSD API обработки изображений сохранял цвета CMYK как RGB. Теперь эта проблема решена.

```csharp
using (JpegImage image = (JpegImage)Image.Load("cat.jpg"))
{
    image.Save(
        "output_CMYK_8.psd",
        new PsdOptions()
        {
            ColorMode = ColorModes.Cmyk
        });
}
```

# Поддержка альтернативного графического движка

Добавлена поддержка альтернативного графического движка Aspose.Drawing вместо System.Drawng.Common/GDI+ для конфигураций .NET Standard 2.0 и выше.

# Улучшено преобразование DNG в JPG

`DNG` в `JPG` изменяет цвет изображения.

```csharp
using (Image image = Image.Load("input.dng", new DngLoadOptions() { AdjustWhiteBalance = true }))
{
    image.Save("input.dng.jpg", new JpegOptions());
}
```
