# Поддержка резервного копирования шрифтов в Linux

Добавлена поддержка резервного шрифта в Linux. Кроме того, если шрифт имеет только курсив, также был добавлен запасной вариант для такого шрифта.

# Нарисовать сглаженную линию

Улучшены возможности рисования API для рисования линии со сглаживанием.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Drawing for .NET 22.5 Release Notes](https://releases.aspose.com/drawing/net/release-notes/2022/aspose-drawing-for-net-22-5-release-notes/).
