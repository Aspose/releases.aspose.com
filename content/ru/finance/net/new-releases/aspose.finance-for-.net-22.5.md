# Схема обработки с сайта xbrl.ifrs.org

Возможность работы со схемой с сайта xbrl.ifrs.org (Таксономия бухгалтерского учета по МСФО).

# Поддержка вложенных нечисловых фактов

Добавлена поддержка обработки вложенных нечисловых фактов.

# Поддержка встроенного XBRL исключить элемент

Добавлена поддержка элемента «исключить», который можно использовать для инкапсуляции данных, которые не должны включаться при обработке сноски или нечисловых элементов. Этот элемент служит для предотвращения включения текстового содержимого в свойства {value} сноски или нечислового значения.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Finance for .NET 22.5 Release Notes](https://releases.aspose.com/finance/net/release-notes/2022/aspose-finance-for-net-22-5-release-notes/).
