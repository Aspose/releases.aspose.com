### Завершение проекта % для форматов Primavera

Этот релиз вводит функцию для доступа к значениям дробного числа физического процента и процентного уровня, чтобы прочитать процентное (%) завершение Primavera & Reg; проекты

### Столбец приоритета задачи для прочтения для файлов MPX

Усовершенствование было сделано в чтении столбца приоритета задачи для Microsoft Project & Reg; `Файлы MPX`, где столбец приоритета был сохранен с использованием значений` enum`.

### Доступ к значениям труда для Primavera & Reg; форматы

Теперь через API вы можете получить доступ к значениям трудовых единиц для Primavera & Ref; Форматы проекта.
