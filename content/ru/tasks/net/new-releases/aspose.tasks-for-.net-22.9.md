# Улучшено преобразование XML в MPP

Исправлено исключение, возникающее при сохранении определенного XML-файла в формате MPP. В следующем фрагменте кода C# показано, как XML-файл Primavera P6 можно преобразовать в файл Microsoft Project® с помощью API:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Улучшенная функция чтения скорости ресурсов Primavera XER

Исправлена проблема чтения скорости ресурсов из файлов Primavera XER.

# Улучшено преобразование из различных форматов в MPP

Исправлены ошибки, возникающие при сохранении определенных файлов в формате MPP.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Tasks for .NET 22.9 Release Notes](https://releases.aspose.com/tasks/net/release-notes/2022/aspose-tasks-for-net-22-9-release-notes/).
