# Импорт мультипроектных файлов XER

Расширен импорт `XER` Primavera P6, и теперь он поддерживает импорт файлов `XER` с более чем одним (1) проектом.

# Улучшенный импорт ограничений задачи для XER

Добавлена возможность указать поведение импорта ограничений задачи для формата `XER`.

# Чтение пользовательских полей для формата XER

Добавлена поддержка чтения пользовательских полей для формата Primavera `XER`.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Tasks for .NET 22.4 Release Notes](https://releases.aspose.com/tasks/net/release-notes/2022/aspose-tasks-for-net-22-4-release-notes/).
