# Решение проблемы преобразования изображения

Преобразование изображения не работало при преобразовании Microsoft Visio® VSDX в PDF. В текущем выпуске эта проблема устранена.

# Преобразование цвета фона фигуры Visio®

Раньше в некоторых случаях цвет фона фигур неправильно преобразовывался во время процесса преобразования VSDX в PDF. Теперь эта проблема решена.

# Обработка позиции текста

Улучшена обработка положения текста во время преобразования формата Visio в SVG.

# Улучшенная визуализация таблиц в Visio для преобразования PNG

В этом выпуске API устранены проблемы с форматированием при отображении таблицы и переключателей в процессе преобразования VSDX в PNG.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Diagram for .NET 22.4 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-4-release-notes/).
