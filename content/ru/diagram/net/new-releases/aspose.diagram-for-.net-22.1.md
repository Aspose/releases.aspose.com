# Улучшено встраивание внешних ресурсов

Раньше в некоторых случаях при сохранении диаграмм Microsoft Visio® в HTML-потоке внешние ресурсы не внедрялись. Однако в настоящее время этот вопрос решен.

# Улучшенная обработка затенения формы

При преобразовании Visio® VSDX в PDF возникла проблема, из-за которой отсутствовало затенение фигур. В этом выпуске эта проблема решена и улучшена обработка оттенков формы.

# Сохраняем HTML в один поток

Добавлена поддержка сохранения HTML в один поток.

# Сохранить диаграмму в HTML со встроенными ресурсами

Возможность сохранять диаграммы Visio® в HTML со встроенными ресурсами или без них.

> Полный список функций, улучшений и исправлений ошибок в этом выпуске можно найти на странице [Aspose.Diagram for .NET 22.1 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-1-release-notes/).
