 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.8.1 freigegeben!

 Folgende Verbesserungen und Fehlerbehebungen sind enthalten:

Fehler beim Kopieren von Zeilen behoben. Fehler beim Kopieren von Arbeitsmappen behoben. Fehler beim Hinzufügen von Bildern behoben. Fehler beim Einstellen von printArea behoben. Fehler bei AutoFit behoben. Fehler beim Erstellen von PivotTables behoben. Fehler beim automatischen Filter behoben. a Fehler beim Einstellen der Formel. Ein Fehler beim Berechnen der Formel wurde behoben.
