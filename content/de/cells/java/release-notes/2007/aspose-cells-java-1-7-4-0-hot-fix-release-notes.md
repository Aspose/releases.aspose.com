 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.7.4 veröffentlicht! Folgende Verbesserungen und Fehlerbehebungen sind enthalten:

- Fügt die Methoden Column.applyStyle und Row.applyStyle hinzu.
 Behebt einige Fehler beim Setzen und Berechnen von Formeln.
 Behebt einen Fehler beim Coping Row.
 Behebt einen Fehler beim Einfrieren von Fenstern.
 Behebt einen Fehler beim Speichern von Diagrammen.
 Behebt einen Fehler beim Speichern von Pivot-Tabellen.
 Behebt einen Fehler in der Methode cells.importResultSet().

Hinweis: Die Methode getNumerOfSheets von Workbook/Worksheets wurde in getNumberOfSheets umbenannt. Und es ist besser, Worksheets.size() aufzurufen, um die Anzahl der Blätter zu erhalten.
