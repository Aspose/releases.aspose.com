 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.4.3.0 veröffentlicht! Folgende Verbesserungen und Fehlerbehebungen sind enthalten:

- Unterstützt das Hinzufügen von Textfeldern und Kommentaren zur Laufzeit
- Unterstützung für Dokumenteigenschaften
- Kartenpunkt einstellen
- Gliederung auf einem Arbeitsblatt festlegen
- Festlegen von Position und Größe des Diagrammbereichs und des Diagrammplotbereichs
- Einstellung von Position, Füllung, Farbe und Größe der Form
- Importieren von Daten aus Array und Sammlung
- Fehler in den Autofilter-Einstellungen behoben
- Fehler im benutzerdefinierten Zahlenformat behoben
- Fehler beim Einfügen/Löschen von Zeilen und Spalten behoben
