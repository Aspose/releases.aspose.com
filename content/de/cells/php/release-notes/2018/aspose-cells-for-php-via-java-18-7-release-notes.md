Wir freuen uns, die erste öffentliche Version von Aspose.Cells for PHP via Java bekannt zu geben. Die APIs beinhalten MS Excel-Tabellenverarbeitungsfunktionen in PHP (einer weit verbreiteten Open-Source-Skriptsprache). Aspose.Cells for PHP via Java ist eine Subset-Komponente, die alle wichtigen und nützlichen Funktionen enthält, die in ihrer nativen Version vorhanden sind[Aspose.Cells for Java](https://products.aspose.com/cells/java/)Das neue API ist ebenso robust und funktionsreich. Es unterstützt High-Fidelity-Dateiformatkonvertierungen zu und von XLS, XLSX, XLSM, SpreadsheetML, CSV, Tabulatorgetrennt, HTML, MHTML und OpenDocument Spreadsheet in PHP. Bitte besuche[Vergleichsseite der Produkteigenschaften](https://docs.aspose.com/cells/de/java/aspose-cells-for-php-via-java-features/)für ihre referenz. Lesen Sie außerdem das Dokument zur Vorgehensweise[Installationsumgebung und andere Installationsrichtlinien](https://docs.aspose.com/cells/de/java/setup-and-installation-guidelines/)für ihre referenz.
