### **Öffentliche API und rückwärts inkompatible Änderungen**
Im Folgenden finden Sie eine Liste aller Änderungen, die an der öffentlichen API vorgenommen wurden, z. B. hinzugefügte, umbenannte, entfernte oder veraltete Mitglieder, sowie alle nicht abwärtskompatiblen Änderungen, die an Aspose.Cells for JasperReports vorgenommen wurden das Aspose.Cells Support-Forum.
#### **Unterstützt JasperServer V6.3.0**
Wir haben die Versionen entfernt, die alte Versionen von JasperReports unterstützen, und behalten nur diejenige, die JasperServer v6.3.0 im Release-Archiv unterstützt. Wenn Benutzer mit alten Versionen von JasperReports arbeiten müssen, laden Sie bitte alte Versionen von Aspose.Cells for JasperReports herunter.
