 Diese Seite enthält Versionshinweise für[Aspose.Cells for JasperReports 1.1.0](https://releases.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.1.0/)

Wir freuen uns, Aspose.Cells for JasperReports v1.1.0 für die Benutzer ankündigen zu können! Was ist neu:

- Unterstützt die Formate Microsoft Excel 97 bis 2003 und Microsoft Excel 2007. Weitere Details finden Sie im Dokument „Konfigurationsparameter“ in der Produktdokumentation.
- Behebt mehrere Fehler bei der Berechnung der Berichtselementgröße.
- Unterstützt JasperReports 3.7.x.
