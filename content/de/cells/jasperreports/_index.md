### Spreadsheet JasperReports®-Integrationsfunktionen

- Einfache Integration mit JasperReports® und JasperServer®.
- Exportieren Sie Berichte in den verschiedenen unterstützten Dateiformaten.
- Kein Verlust von Layout oder Formatierung nach der Konvertierung.
- Funktioniert ohne Installation von Microsoft Excel®.
- Diagramme werden anstelle von Bildern in einem bearbeitbaren Format exportiert.

#### Unterstützte Ausgabeformate

**Microsoft Excel®:** XLS, XLSX\
**OpenDocument:** ODS\
**Festes Layout:** PDF\
**Auszeichnung:** XML \
**Andere:** CSV, TSV, TabDelimited

#### System Anforderungen

- Java 2 Platform Standard Edition 1.6 oder höher.
- JasperReports 6.3.0 oder höher
