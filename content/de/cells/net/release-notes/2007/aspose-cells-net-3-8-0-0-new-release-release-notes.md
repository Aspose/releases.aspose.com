 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for .NET v3.8 veröffentlicht!

- Unterstützt das Erstellen von Kontrollkästchen zur Laufzeit
- Unterstützt das Erstellen von OLE-Objekten zur Laufzeit
- Unterstützt das Schützen/Entschützen von Arbeitsmappen
- Ermöglicht das Ändern der Datenquelle externer Links
- Verbesserte Smart-Marker-Verarbeitung
- Verbesserte Formelberechnungs-Engine
- Fehler in DeleteRows und InsertRows behoben
- Fehler beim wiederholten Öffnen und Speichern einer Datei mit Testversion behoben
- AddCopy-Fehler beim Kopieren einiger Formeln behoben
- Weitere Verbesserungen und Korrekturen seit v3.7
