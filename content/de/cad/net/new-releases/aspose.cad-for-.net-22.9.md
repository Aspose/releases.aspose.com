# Ungarische Sonderzeichen exportieren

Bisher wurden in einigen Fällen die ungarischen Sonderzeichen fälschlicherweise von DWG nach PDF exportiert. In dieser Version der API wurde das oben genannte Problem behoben.

# Verbesserte DWG-zu-PDF-Konvertierung

„OutOfMemoryError“ beim Konvertieren von „DWG“ in das „PDF“-Format behoben.

# Verbesserte Szenenbewegungsempfindlichkeit

Die Empfindlichkeit der bewegten Szene wurde verbessert.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.CAD for .NET 22.9 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-9-release-notes/).
