# Unterstützung für große Schriftarten

Unterstützung für Big Font in der API hinzugefügt.

# Speichereffiziente Konvertierung von DWG in PDF

Wenn früher DWG in das PDF-Format konvertiert wurde, verbrauchte es viel Speicherplatz. Dieses Problem wurde jetzt behoben. Jetzt können Sie DWG mit verbesserten Ergebnissen in PDF konvertieren.

# Verbesserter DAE-zu-OBJ-Export

In einigen früheren Fällen schlug der Export von DAE nach OBJ fehl. Dieses Problem wurde nun in der API behoben.
