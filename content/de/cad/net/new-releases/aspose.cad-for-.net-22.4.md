# Bessere Speicherverwaltung

Beim Konvertieren von AutoCAD® DXF in das TIFF-Format (Rastergrafik) wurde ein Speicherverlust beobachtet. Daher bietet diese Version der API eine bessere Speicherverwaltung für die DXF-zu-TIFF-Konvertierung.

# Verbesserter DWG-zu-WMF-Export

Der Export von AutoCAD® `DWG` nach `WMF` (Windows Metafile) wurde verbessert.

# Eingebettetes Bild exportieren

Möglichkeit zum Exportieren von Bildern aus eingebetteten Bilddaten.

# Verbessertes `DWG`-Rendering

Es wurde beobachtet, dass die „DWG“ ohne Farben gerendert wurde. Dieses Problem wurde jetzt in dieser Version der API behoben.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.CAD for .NET 22.4 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-4-release-notes/).
