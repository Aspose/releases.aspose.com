## Warum Aspose.SVG für .NET herunterladen?

Diese .NET-On-Premise-API unterstützt Sie bei der [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/)-Funktionalität in Ihren C#-, VB.NET-, ASP.NET- und anderen .NET-basierten Apps.

## Funktionen zur Verarbeitung von SVG-Dateien

- Dateien im Format [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/).
- Konvertiere SVG in andere [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- Manipulation des DOM-Baums gemäß den offiziellen SVG-Spezifikationen.
- Unterstützung für die Inhaltsnavigation über [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), CSS-Selektoren, Element- und Dokumentdurchlauffunktionen.
- Unterstützung für Qualitäts-Rendering.

### Unterstützte Formate lesen

SVG

### SVG speichern unter

**Festes Layout:** PDF, XPS\
**Bild:** TIFF, BMP, PNG, JPEG, GIF
