### Barcode-Verarbeitung Python über Java On-Premise-API-Funktionen

- Es bietet Unterstützung für die meisten gängigen Barcode-Spezifikationen und -Standards.
- Sie können Barcodes in mehreren Bildformaten lesen und exportieren, z. B. JPEG, PNG, GIF und BMP.
- Fähigkeit, mit Barcode-Etiketten zu arbeiten.
- Unterstützung der X-Dimension sowie der Y-Dimension für 2D-Barcodes.
