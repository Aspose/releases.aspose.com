## Erstellen einer Szene mit primitiven 3D‑Modellen mittels C#‑Code

Sie können das folgende Code‑Snippet ausführen, um zu sehen, wie Aspose.3D in Ihrer Umgebung funktioniert, oder das [GitHub‑Repository](https://github.com/aspose-3d/Aspose.3D-for-.NET) für weitere Anwendungsbeispiele prüfen.

## Export einer 3D‑Szene zu komprimiertem AMF via C#‑Code

Aspose.3D für .NET ermöglicht das Speichern von 3D‑Meshes in einem benutzerdefinierten Binärformat, das Abrufen aller Eigenschaftswerte von 3D‑Szenen sowie das Umkehren ihres Koordinatensystems. Das folgende Beispiel demonstriert die Konvertierung einer 3D‑Szene ins AMF‑Format bei gleichzeitiger Komprimierung.

```csharp
// lade eine Szene
Scene scene = new Scene();
var box = new Box();
var tr = scene.RootNode.CreateChildNode(box).Transform;
tr.Scale = new Vector3(12, 12, 12);
tr.Translation = new Vector3(10, 0, 0);
tr = scene.RootNode.CreateChildNode(box).Transform;
// Skalierungstransformation
tr.Scale = new Vector3(5, 5, 5);
// setze Euler‑Winkel
tr.EulerAngles = new Vector3(50, 10, 0);
scene.RootNode.CreateChildNode();
scene.RootNode.CreateChildNode().CreateChildNode(box);
scene.RootNode.CreateChildNode().CreateChildNode(box);
// speichere komprimierte AMF‑Datei
scene.Save(dir + "output.amf", new AMFSaveOptions() {
  EnableCompression = true
});
```
