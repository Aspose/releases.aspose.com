# Verbesserte Beladung von 3D -Formaten

Die 3D -API warf eine Ausnahme beim Laden von `gltf`-,` x`- und `vrml` -Dateien. Diese 3D -Formate sind jetzt vollständig kompatibel und werden ohne Probleme geladen.

# Tangent/Binormale Erkennung in GLB

Zuvor wurde in einigen Fällen die in "GLB" mit aktivierte Tangente/Binormal definiert. Dieses Problem wurde innerhalb dieser Veröffentlichung der API gelöst.

# Verbesserte Umwandlung von 3D -Formaten

Verbesserungen wurden an den folgenden 3D -Formatkonvertierungen vorgenommen:

- ** ase to 3ds ** Autodesk & reg; ASCII -Szenenexportdatei -Dateiformat zum 3D Studio Format Conversion.

- ** ase to j ** Autodesk & reg; ASCII -Szenenexportdatei -Dateiformat zur Umwandlung von OBJ -Format.

- ** USD an DRC ** Pixars Universal Szenenbeschreibung (USD) Dateiformat zum Google Draco (DRC) -Format (DRC).

> Für eine vollständige Liste von Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version besuchen Sie bitte [ASSON3D für NET 231 -Versionsnotizen](https://releases.aspose.com/3d/net/release-notes/2023/aspose-3d-for-net-23-1-release-notes/).
