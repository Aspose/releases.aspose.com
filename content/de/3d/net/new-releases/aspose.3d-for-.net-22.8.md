# Benutzerdefinierte `XFORM`-Operatorunterstützung

Unterstützung des benutzerdefinierten „xform“-Operators im USD/USDZ-Importer hinzugefügt.

# Verbesserte Konvertierung des 3D-Modells FBX in USDZ

Zuvor waren die von FBX auf USDZ umgestellten Modelle komplett schwarz. Dieses Problem wurde in der aktuellen Version der API behoben.

# Verbesserter Export von Materialattributen

Materialattribute wurden nicht exportiert, wenn keine Textur angehängt war. Dieses Problem wurde nun behoben.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.3D for .NET 22.8 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-8-release-notes/).
