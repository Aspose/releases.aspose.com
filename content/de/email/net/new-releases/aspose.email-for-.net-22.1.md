# Bessere Handhabung des E-Mail-Versands über SMTP

Beim Arbeiten mit der Version 21.12 der API trat in einigen Fällen beim Senden von E-Mails über SMTP das Problem „Operation wurde abgebrochen“ auf. Dieses Problem wurde nun behoben.

# Verbesserte Behandlung von IMAP-Seen-Flags

Zuvor wurde das Flag Aspose IMAP Seen immer als "false" zurückgegeben. Dieses Problem wurde nun behoben.

# Verbesserungen in der MBOX-Posteingangsverwaltung

In einigen Fällen wurden die E-Mail-Nachrichten entfernt, nachdem wir der vorhandenen MBOX eine neue Nachricht hinzugefügt hatten. Dieses Problem wurde nun behoben.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.Email for .NET 22.1 Release Notes](https://releases.aspose.com/email/net/release-notes/2022/aspose-email-for-net-22-1-release-notes/).
