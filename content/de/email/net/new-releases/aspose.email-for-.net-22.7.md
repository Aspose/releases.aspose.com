Es wurde beobachtet, dass bei EML-zu-PDF- oder EML-zu-MHTML-Konvertierungen der Zeilenabstand verloren ging. Dieses Problem wurde nun behoben.

# E-Mail-UID vom Postfach auf dem Server abrufen

Fähigkeit, die Nachrichtenidentifikationsinformationen (z. B. UID oder Sequenznummer) für die von einer Mailbox auf dem Server empfangenen Nachrichten abzurufen.

```csharp
using (var client = new ImapClient(imapHost, port, emailAddress, password, securityOption))
{
    var msgs = client.ListMessages("INBOX").Take(5);
    var seqIds = msgs.Select(t => t.SequenceNumber);
    var msgsViaFetch = client.FetchMessages(seqIds);
	
    for (var i = 0; i < 5; i++)
    {
        var thisMsg = msgsViaFetch[i];
        Console.WriteLine($"Message ID:{seqIds.ElementAt(i)} SequenceNumber: {thisMsg.ItemId.SequenceNumber} Subject:{thisMsg.Subject}");
    }
}
```
