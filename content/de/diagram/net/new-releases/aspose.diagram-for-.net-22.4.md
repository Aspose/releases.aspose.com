# Problemlösung bei der Bildkonvertierung

Die Bildkonvertierung funktionierte nicht, wenn Microsoft Visio® VSDX in PDF konvertiert wurde. Die aktuelle Version hat dieses Problem behoben.

# Konvertierung der Hintergrundfarbe von Visio®-Formen

Zuvor wurde in einigen Fällen die Hintergrundfarbe der Formen während des Prozesses zum Konvertieren von VSDX in PDF falsch konvertiert. Dieses Problem wurde nun behoben.

# Handhabung der Textposition

Bessere Handhabung der Textposition während der Konvertierung von Visio in das SVG-Format.

# Besseres Rendern von Tabellen bei der Visio-zu-PNG-Konvertierung

In dieser Version der API wurden die Formatierungsprobleme beim Rendern von Tabellen und Optionsfeldern während des VSDX-zu-PNG-Konvertierungsprozesses behoben.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.Diagram for .NET 22.4 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-4-release-notes/).
