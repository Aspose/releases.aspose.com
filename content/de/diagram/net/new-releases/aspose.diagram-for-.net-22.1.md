# Verbesserte Einbettung externer Ressourcen

Früher wurden beim Speichern der Microsoft Visio®-Diagramme im HTML-Stream in einigen Fällen die externen Ressourcen nicht eingebettet. Dieses Problem wurde nun jedoch behoben.

# Bessere Handhabung der Formschattierung

Beim Konvertieren von Visio® VSDX in PDF trat ein Problem auf, bei dem die Schattierung von Formen fehlte. Diese Version behebt dieses Problem und bietet eine bessere Handhabung von Formschattierungen.

# Speichern Sie HTML in einem einzelnen Stream

Unterstützung für das Speichern von HTML in einem einzelnen Stream hinzugefügt.

# Diagramm mit eingebetteten Ressourcen in HTML speichern

Möglichkeit, Visio®-Diagramme mit oder ohne eingebettete Ressourcen im HTML-Format zu speichern.
