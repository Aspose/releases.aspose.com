# Font-Fallback-Unterstützung unter Linux

Font-Fallback-Unterstützung unter Linux hinzugefügt. Wenn eine Schriftart nur kursiv ist, wurde außerdem ein Fallback für diese Schriftart hinzugefügt.

# Anti-Aliasing-Linie zeichnen

Die Zeichenfunktionen der API zum Zeichnen einer Anti-Aliasing-Linie wurden verbessert.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.Drawing for .NET 22.5 Release Notes](https://docs.aspose.com/drawing/net/aspose-drawing-for-net-22-5-release-notes/).
