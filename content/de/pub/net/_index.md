## Warum Aspose.PUB für .NET herunterladen?

Aspose.PUB für .NET ist eine einfache API, mit der Sie Microsoft Publisher®-Dateien (PUB) lesen und in das PDF-Format konvertieren können. Es bietet auch leicht verständliche Schnittstellen zum Bearbeiten der Metadaten von PUB-Dateien.

## PUB-Dateiverarbeitungsfunktionen

- Lesen Sie Microsoft Publisher®-Dateien (PUB) zur Konvertierung in das PDF-Format.
- Metadaten von PUB-Dateien über API lesen und schreiben.

### PUB-Dateien lesen

**Microsoft Publisher®:** PUB

### PUB speichern unter

**Festes Layout:** PDF
