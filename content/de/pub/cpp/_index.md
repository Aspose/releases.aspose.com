## Warum Aspose.PUB für C++ herunterladen?

Aspose.PUB für C++ ist eine einfache API, mit der Sie Microsoft Publisher®-Dateien (PUB) programmgesteuert in Ihren C++-Apps lesen und in das PDF-Format konvertieren können. Es bietet auch leicht verständliche Schnittstellen zum Bearbeiten der Metadaten von PUB-Dateien.

## PUB-Dateiverarbeitungsfunktionen

- Lesen Sie Microsoft Publisher®-Dateien (PUB) zur Konvertierung in das PDF-Format.
- Metadaten von PUB-Dateien über API lesen und bearbeiten.

### PUB-Dateien lesen

**Microsoft Publisher®:** PUB

### PUB speichern unter

**Festes Layout:** PDF

### Plattformunabhängigkeit

Sie können Aspose.PUB für C++ verwenden, um jede Art von 32-Bit- und 64-Bit-C++-Anwendungen zu erstellen. Sie können es server- und clientseitig verwenden, indem Sie einfach die Assembly kopieren, ohne sich um andere Dienste oder Module kümmern zu müssen.
