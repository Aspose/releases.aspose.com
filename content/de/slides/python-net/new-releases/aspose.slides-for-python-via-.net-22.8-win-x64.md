# Umfassende Unterstützung für Diashow -Einstellungen

Diese API -Version führt die detaillierte Unterstützung für die Diashow -Einstellungen der Präsentation vor. Einige der neuen Klassen und Eigenschaften sind wie folgt:

- [`Präsentation"] (https://reference.aspsis.com/slides/python-net/asips.slides/presentation/) .`Sslide_show_Settings` Eigenschaft: Gibt die Diashow-Einstellungen für die Präsentation zurück.
- [`SlideSettingS`] (https://reference.aspsis.com/slides/python-net/asipsis.slides/slideshowSets/) Klasse: Repräsentiert die Diashow-Einstellungen für die Präsentation. Einige der verfügbaren Eigenschaften sind;
  - Diashow in einer Schleife laufen lassen
  - Fähigkeit, Diashow -Erzählungen zu zeigen / zu verbergen
  - Animation in der Diashow aktivieren / deaktivieren
  - Stiftfarbe für die Diashow einstellen

! [SlideSetup.png] (https://docs.asips.com/slides/python-net/asipsis-lides-for-python-net-22-8-release-notes/slideSetup.png)

> Für eine vollständige Liste von Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version besuchen Sie bitte [ASSONS.SLIDES FÜR PYTHON 22.8 RESUNGS-ANSUGEN] (https://docs.aspsis.com/slides/python-net/ase-slides- For-Python-NET-22-8-Release-Notes/).
