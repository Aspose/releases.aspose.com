# Support für Apple & reg; s macOS & reg;

Diese Präsentation Manipulation Python API -Veröffentlichung führt die Unterstützung für die MacOS & Reg;.

# Information mit Schriftart Substitution erhalten

Fähigkeit, die Informationen über die Schriftarten zu erhalten, die während des Präsentationsrenders ersetzt werden. Eine neue Methode, "get_substitutions ()", wurde hinzugefügt, um dies zu erreichen.

# Timing für die Folienanimation festlegen

Jetzt können Sie das Timing festlegen, um die Präsentationsleitschichtanimation über die folgenden zwei neuen Eigenschaften zu wiederholen, die dem [`Timing`] (https://reference.aspsis.com/slides/python-net/asips.slides.animation/timing hinzugefügt wurden /) Klasse:

- `repep_until_next_click`: Diese Bool -Eigenschaft setzt, wenn der Animationseffekt wiederholt wird, bis die Maustaste als nächstes geklickt wird.
- `repep_until_end_slide`: Diese BOOL -Eigenschaft setzt, wenn der Folienanimationseffekt bis zum Ende der aktuellen Folie wiederholt wird.

> Für eine vollständige Liste von Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version besuchen Sie bitte [ASSONS.SLIDES FÜR PYTHON 22.9 Releases] (https://docs.aspsis.com/slides/python-net/ase-slides- For-Python-NET-22-9-Release-Notes/).
