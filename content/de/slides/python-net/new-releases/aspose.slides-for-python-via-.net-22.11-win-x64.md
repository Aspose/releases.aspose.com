# PDF 1.6 & PDF 1.7 Support exportieren

Die Unterstützung wurde implementiert, damit PDF in die Formate PDF 1.6 und PDF 1.7 exportiert werden soll.

# PDF mit A2A, A2B Compliance -Niveaus exportieren

Diese Python -API -Veröffentlichung führt die Unterstützung der PDF -Exportfunktion mit zahlreichen Konformitätsniveaus vor, d. H. A2A, A2B, A2U, A3A und A3B.

# Hyperlink Sound in Präsentationen verwalten

Es wurde eine neue Eigenschaft ([`hyperlink.sound`] (https://reference.asispsis.com/slides/python-net/asips.slides/hyperlink/)) hinzugefügt, die den gespielten Klang eines Hyperlinks darstellt.

# Animationseffekt stoppt den vorherigen Sound

Eine neue Eigenschaft ([`effect.stop_previous_sound`] (https://reference.aspsis.com/slides/python-net/asiple.slides.animation/effect/)) wurde eingeführt, die festgelegt, ob der Animationseffekt die vorherige Stopps stoppt Klang.

> Für eine vollständige Liste von Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version besuchen Sie bitte [ASSONS.SLIDES FÜR PYTHON 22.11 RESUNGSHINWEISE] (https://docs.aspsis.com/slides/python-net/ase-slides- For-Python-NET-22-11-Release-Notes/).
