# Rahmen- und Rotationsverhalten einstellen

Fähigkeit, das Verhalten der Rahmenrotation und Rahmengröße zu spezifizieren und zu steuern und gleichzeitig Formen aus `pptx` und anderen unterstützten Präsentationsformaten zu rendern.

# Eingebettete Schriftarten komprimieren, um die Präsentationsgröße zu verringern

Wenn Ihre Präsentation die eingebetteten Schriftarten enthält, bietet diese Veröffentlichung unserer Python -API der Präsentation Processing die Möglichkeit, die Komprimierung aller dieser eingebetteten Schriftarten durchzuführen und wiederum die Größe der Präsentationsdateien wie `pptx` zu reduzieren.

# Präsentationsleiter Linien Farbe festlegen

Diese API -Version führt die Fähigkeit ein, die Farbe von Leader -Linien in Ihre Präsentationsfolien festzulegen.

> Für eine vollständige Liste von Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version besuchen Sie bitte [ASSONS.SLIDES FÜR PYTHON 22.10 RESUNGS-ANMERKUNGEN] (https://docs.aspsis.com/slides/python-net/ase-slides- For-Python-NET-22-10-Release-Notes/).
