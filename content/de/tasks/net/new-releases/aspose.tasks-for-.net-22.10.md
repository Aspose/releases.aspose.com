### Primavera UIDs erhalten

Es wurde eine neue Einstellung hinzugefügt, die es ermöglicht, die Primavera & reg zu erhalten; UIDs beim Importieren von Projekten aus XER- oder P6xML -Formaten.

### .NET Framework 2.0 Support

Die Unterstützung für .NET Framework 2.0 wurde entfernt.

### Zugriffsfeldern zu Zugangsentitäten

Die API wurde verbessert, um mithilfe von .NET -Eigenschaften auf die Felder der Entitäten zuzugreifen.
