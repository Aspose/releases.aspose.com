# Verbesserte XML-zu-MPP-Konvertierung

Die Ausnahme beim Speichern einer bestimmten XML-Datei im MPP-Format wurde behoben. Das folgende C#-Code-Snippet zeigt, wie eine Primavera P6-XML-Datei mithilfe der API in eine Microsoft Project®-Datei konvertiert werden kann:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Verbesserte Lesefunktion von Primavera XER Resource Rates

Das Problem des Lesens von Ressourcenraten aus den Primavera XER-Dateien wurde behoben.

# Verbesserte Konvertierungen von verschiedenen Formaten in MPP

Die Fehler behoben, die beim Speichern bestimmter Dateien im MPP-Format ausgelöst wurden.
