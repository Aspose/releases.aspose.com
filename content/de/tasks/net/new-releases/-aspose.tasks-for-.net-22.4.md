# Importieren Sie Multi-Projekt-XER-Dateien

Der `XER`-Import von Primavera P6 wurde erweitert und unterstützt jetzt den Import von `XER`-Dateien mit mehr als einem (1) Projekt.

# Verbesserter Import von Task Constraints für XER

Möglichkeit hinzugefügt, das Verhalten des Aufgabeneinschränkungsimports für das `XER`-Format festzulegen.

# Benutzerdefinierte Felder für das XER-Format lesen

Unterstützung für das Lesen von benutzerdefinierten Feldern für das Primavera-XER-Format hinzugefügt.

> Eine vollständige Liste der Funktionen, Verbesserungen und Fehlerbehebungen in dieser Version finden Sie unter [Aspose.Tasks for .NET 22.4 Release Notes](https://releases.aspose.com/tasks/net/release-notes/2022/aspose-tasks-for-net-22-4-release-notes/).
