### バーコード処理 Python 経由 Java オンプレミス API 機能

- 一般的なバーコードの仕様と規格のほとんどをサポートしています。
- JPEG、PNG、GIF、BMP などの複数の画像形式でバーコードを読み取ってエクスポートできます。
- バーコード ラベルを操作する機能。
- 2D バーコードの X 次元と Y 次元のサポート。
