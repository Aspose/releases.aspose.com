### PHP BarCode オンプレミス API 機能

- ほとんどの確立されたバーコード規格とバーコード仕様をサポートします。
- BMP、GIF、JPEG、PNG などの複数の画像形式でバーコードを読み取り、エクスポートする機能。
- バーコード ラベルを完全に制御できます。
- 2D バーコードの X 寸法と Y 寸法をサポートします。
