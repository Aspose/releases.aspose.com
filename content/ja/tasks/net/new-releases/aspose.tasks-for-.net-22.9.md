# XML から MPP への変換の改善

特定の XML ファイルを MPP 形式で保存するときに例外が発生する問題を修正しました。次の C# コード スニペットは、API を使用して Primavera P6 XML ファイルを Microsoft Project® ファイルに変換する方法を示しています。

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Primavera XER リソース レートの読み取り機能の改善

Primavera XER ファイルから読み取るリソース レートの問題を修正しました。

# さまざまなフォーマットから MPP への変換の改善

特定のファイルを MPP 形式で保存する際に発生するエラーを修正しました。
