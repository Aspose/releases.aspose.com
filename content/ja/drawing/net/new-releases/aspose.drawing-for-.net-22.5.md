# Linux でのフォント フォールバックのサポート

Linux でのフォント フォールバック サポートが追加されました。さらに、フォントがイタリック スタイルのみの場合、そのようなフォントのフォールバックも追加されました。

# アンチエイリアス線を引く

アンチエイリアス ラインを描画する API の描画機能を改善しました。
