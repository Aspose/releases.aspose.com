## Aspose.OMR for .NET をダウンロードする理由

Aspose.OMR for .NET は、光学式マーク認識エンジンとして機能するだけではありません。また、しきい値とマークアップを手動で調整してリアルタイムで変更を確認するために使用できる便利なグラフィカル コントロールも提供します。

## OMR API の機能

- スキャンした画像と写真の認識。
- 回転および遠近法 (側面図) の画像を処理する機能。
- テスト、試験、アンケート、調査などからのデータを認識します。
- 精度が高く、結果を CSV 形式でエクスポートできます。
- OMR テンプレートを作成します。

### OMR のイメージを読み取る

JPEG、PNG、GIF、TIFF、BMP

### OMR結果を名前を付けて保存

CSV、JSON、XML
