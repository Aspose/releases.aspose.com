Aspose.Grid 1.5.3 をリリースしました！

このリリースの機能強化とバグ修正:

 Aspose.Grid.デスクトップ

強化:1.列の検証をサポート。2.列の制御をサポート。3.R1C1 参照スタイルをサポート。4.並べ替えをサポート。

Aspose.Grid.ウェブ

バグ修正:1.行数が約100を超えるとFreezePanes(Fixed Rows/Columns)が消えることがある.2.セルの範囲を複数選択してデータを貼り付けると、読み取り専用のセルが変更される.3.DropDownList型のセルが変更される. 4.一部の地域設定で列の幅が適切でない。

強化:1.カスタムの列/行ヘッダー キャプションの設定をサポートします (WebCells.SetRowCaption、WebCells.SetColumnCaption)。2.上から下および左から右の方向のセル範囲の並べ替えをサポートします (WebCells.Sort)。3.エクスポートをサポートします。セル データを DataTable オブジェクト (WebCells.Export).4.Aspose.Grid.Web UI デザイナーに変換します。システムのスタート メニューまたは Visual Studio 環境から実行します (これまでのところ、.net 1.0/1.1 のみをサポートしています)。
