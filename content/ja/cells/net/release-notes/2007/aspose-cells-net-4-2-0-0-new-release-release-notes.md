親愛なる顧客:

 Aspose.Cells for .NET v4.2 をリリースしました！

この新しいバージョンでは、いくつかの重要な新機能が追加されています。

- 現在、Aspose.Cells は、テンプレート ファイル内のチャート、画像、およびその他の描画オブジェクトの操作を完全にサポートしています。
- Chart2Image 機能がサポートされています。現在、縦棒グラフ、折れ線グラフ、円グラフ、棒グラフのみがサポートされています。また、2D グラフのみがサポートされています。将来のバージョンでは、より多くのチャート タイプをサポートする予定です。
- Xls2Pdf 変換が強化され、最適化されました。
- 数式計算エンジンが強化され、最適化されました。
