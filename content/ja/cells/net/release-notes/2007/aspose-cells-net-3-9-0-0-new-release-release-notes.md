お客様各位

Aspose.Cells for .NET v3.9 をリリースしました！

- いくつかの外部的な理由により、クラス Excel の名前が Workbook に変更され、クラス ExcelDesigner の名前が WorkbookDesigner に変更されました。これら 2 つのクラスのすべてのプロパティとメソッドは変更されていません。
- EMF 画像を raw 形式で追加することをサポート
- 新しいスマートマーカーが追加されました
- カスタムセパレーター付きのテキストファイルの読み込みをサポート
- ハイパーリンクをローカル ファイルに設定する際のバグを修正しました
- InsertRows メソッドのバグを修正
- データ検証設定のバグを修正
- 一部の Excel ファイルを開く際のバグを修正しました
