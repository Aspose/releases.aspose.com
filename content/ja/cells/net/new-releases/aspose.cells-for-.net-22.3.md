# スプレッドシート テンプレート形式のサポート

この API リリースでは、Excel&reg と Excel&reg の両方のスプレッドシート テンプレート形式のサポートが導入されました。 (XLT) & OpenOffice Calc&reg; (OTS) プログラム。これらは、`SaveFormat.Ots` および `SaveFormat.Xlt` 経由の出力形式としてサポートされています。

# 内側の境界線を設定する

内側の境界線を設定する機能。

# 選択した行または列ごとにピボットフィールドを並べ替える

ピボット フィールド値の並べ替えサポートが追加されました。ピボット フィールドの並べ替えは、目的の行または列に基づいて実行できます。

 > このリリースの機能、拡張機能、バグ修正の完全なリストについては、次のサイトをご覧ください。[Aspose.Cells for .NET 22.3 リリースノート](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-3-release-notes/).
