#  `.NUMBERS` データ ストレージ 5.0 のサポート

`.numbers` のデータ ストレージ 5.0 のサポートを追加しました。

#  XLSX から PDF のレンダリングを改善しました

XLSX を PDF に変換するときに、棒グラフが正しくレンダリングされないことが観察されました。この問題は現在解決されています。

#  Excel&reg の改良グラフのラベルの処理

この API リリースでは、Excel&reg; に関するいくつかの修正が提供されています。グラフのラベル。それらは次のとおりです。

- グラフのラベルがボックスに適合していませんでした。
- グラフのラベルが水平ではなく垂直に配置されていました。

# 表の数式を XLSB に保存

テーブル数式を `XLSB` に保存するためのサポートが追加されました。

#  Linux でのサポートの向上

Linux では、図形やテキスト ボックスなどのオブジェクトに書かれた文字やキャプションが文字化けして表示されることが確認されました。この問題は現在解決されています。
