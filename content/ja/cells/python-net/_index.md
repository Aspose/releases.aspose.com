### .NETオンプレミスAPI機能を介したスプレッドシートPython

- API を介したスプレッドシートの生成と操作。
- 高品質のファイル形式の変換とレンダリング。
- Microsoft Excel® ファイルを物理プリンターまたは仮想プリンターに印刷します。
- Excel® シートを結合、変更、保護、または解析します。
- ワークシートの書式設定を適用します。
- ワークシートのページ設定を構成して適用します。
- Excel® チャート、ピボット テーブル、条件付きの作成とカスタマイズ
  フォーマット ルール、スライサー、テーブル、スパーク ライン。
- Excel® チャートを画像と PDF に変換します。
- Excel® ファイルを他のさまざまな形式に変換します。
- すべての基本および高度な Excel® 機能をサポートする数式計算エンジン。

機能の詳細なリストについては、[official documentation](https://docs.aspose.com/cells/python-net/) にアクセスしてください。

### 読み取りと書き込みのスプレッドシート ファイル形式

**Microsoft Excel®:** XLS、XLSX、XLSB、XLSM、XLT、XLTX、XLTM、CSV、TSV、TabDelimited、SpreadsheetML\
**OpenOffice:** ODS、SXC、FODS\
**テキスト:** TXT\
**Web:** HTML、MHTML\
**iWork®:** 数字\
**その他:** SXC、FODS

### スプレッドシート ファイルを名前を付けて保存

**Microsoft Word®:** DOCX\
**Microsoft PowerPoint®:** PPTX\
**Microsoft Excel®:** XLAM\
**固定レイアウト:** PDF、XPS\
**データ交換:** DIF\
**ベクターグラフィック:** SVG\
**画像:** TIFF、PNG、BMP、JPEG、GIF\
**メタ ファイル:** EMF\
**マークダウン:** MD

詳細については、[Supported File Formats](https://docs.aspose.com/cells/python-net/supported-file-formats/) にアクセスしてください。

＃＃＃ システム要求

マシンに Microsoft Excel® または OpenOffice® ソフトウェアがインストールされている必要はありません。

### サポートされているオペレーティング システム

**Microsoft Windows®:** Windows デスクトップ & サーバー (`x64`、`x86`)\
**Linux:** Ubuntu、OpenSUSE、CentOS など\
**その他:** Mono (.NET 4.0 Framework サポート) をインストールできる、または .NET コアを使用できる任意のオペレーティング システム (OS)。
