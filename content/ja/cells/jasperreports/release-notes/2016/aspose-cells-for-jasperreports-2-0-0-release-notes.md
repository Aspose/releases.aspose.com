### **Public API および下位互換性のない変更**
以下は、Aspose.Cells for JasperReports に対して行われた下位互換性のない変更と同様に、追加、名前変更、削除、または廃止されたメンバーなど、パブリック API に対して行われた変更のリストです。リストされている変更について懸念がある場合は、 Aspose.Cells サポート フォーラム。
#### **JasperServer V6.3.0 をサポート**
古いバージョンの JasperReports をサポートするバージョンを削除し、JasperServer v6.3.0 をサポートするバージョンのみをリリース アーカイブに残しました。古いバージョンの JasperReports を使用する必要がある場合は、古いバージョンの Aspose.Cells for JasperReports をダウンロードしてください。
