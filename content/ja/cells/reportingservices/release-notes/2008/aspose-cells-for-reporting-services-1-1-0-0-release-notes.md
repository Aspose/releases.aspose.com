新着情報？



- ` `クライアント アドイン ツール - Aspose Cells Report Designer を提供します。これにより、ユーザーは MS Excel をレポート デザイン ツールとして使用できます。 Aspose Cells Report Designer ユーザーは、Visual Studio やその他のレポート ビルダー ツールの代わりに、Excel でレポート テンプレートをデザインできます。レポート テンプレートは Reporting Services RDL ファイルに埋め込まれており、レポート サーバーに発行できます。レポートのデザインを簡単かつ迅速に行うことができます。
-  Aspose.Cells for Reporting Services サーバーのレンダリングが大幅に強化されました。レポート テンプレートを使用すると、ユーザーは、ネイティブの Excel グラフ、高度な数式、ピボット テーブルなど、Reporting Services のすべての高度な Excel 機能を利用できます。
