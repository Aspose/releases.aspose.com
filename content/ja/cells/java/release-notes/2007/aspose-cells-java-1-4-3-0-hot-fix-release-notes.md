Aspose.Cells for Java V1.4.3.0 をリリースしました!次の機能強化と修正が含まれています。

- 実行時のテキスト ボックス、コメントの追加をサポート
- ドキュメント プロパティのサポート
- チャートポイントの設定
- ワークシートにアウトラインを設定する
- チャートエリアとチャートプロットエリアの位置とサイズの設定
- シェイプの位置、塗りつぶし、色、サイズの設定
- 配列とコレクションからのデータのインポート
- 自動フィルター設定のバグを修正
- カスタム数値フォーマットのバグを修正
- 行と列の挿入/削除のバグを修正しました
