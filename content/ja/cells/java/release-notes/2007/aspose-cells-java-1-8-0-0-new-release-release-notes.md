お客様各位

Aspose.Cells for Java V1.8.0 をリリースしました！

以下の機能強化と修正が含まれています。

アクティブシートとセルの設定をサポート。数式設定の不具合を修正。計算式のいくつかのバグを修正します。ファイルを開く際の不具合を修正。ピボットテーブルの不具合を修正。 validtion のバグを修正します。autofilters のバグを修正します。freezePanes のバグを修正します。

注: Aspose.Cells for Java 1.4 の場合、retrotranslator-runtime と backport-util-concurrent のすべてのクラスが組み込まれているため、2 つの jar を参照する必要はありません。
