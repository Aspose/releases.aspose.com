このページには、[Aspose.Cells for Java 8.1.0](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-8.1.0/)

{{% /alert %}} 

Aspose.Cells for Java がバージョン 8.1.0 に更新されました。このリリースでは、10 の新しい便利な改善点が追加されたことをお知らせします。
Aspose.Cells for Java を使用すると、アプリケーションで XLS、SpreadsheetML、OOXML、XLSB、CSV、HTML、ODS、PDF、XPS およびその他の形式で作業できます。 Microsoft Excel を使用せずに、ワークブックを生成、変更、変換、レンダリング、および印刷することもできます。
ドキュメントを参照して、Aspose.Cells for Java の使用を開始する方法を確認してください。
このダウンロードには、製品の完全に機能するバージョンが含まれていますが、ライセンス セットがないと、いくつかの制限付きで評価モードで実行されます。これらの評価制限なしで Aspose.Cells をテストするには、無料の 30 日間の一時ライセンスをリクエストできます。
以下は、このバージョンの Aspose.Cells for Java での変更点のリストです。

その他の改善と変更

機能強化

(CELLSJAVA-40823) - CellsHelper.setFontDir メソッドを使用して指定されたフォント ディレクトリを使用するように API を制限します。
(CELLSJAVA-40716) - ダイアグラムの線がくっきり/鋭くない
(CELLSJAVA-40827) - カスタム数値フォーマットで定義された表示色を取得

バグ

(CELLSJAVA-40809) - 一部の色が 1 つのテーブルの列の直前に表示される
(CELLSJAVA-40814) - Ubuntu でスプレッドシートを変換すると、結果の PDF に画像が表示されない
(CELLSJAVA-40826) - グリッド線とフォントの設定が出力にない HTML
(CELLSJAVA-40829) - ワークシートの印刷品質を設定できない
(CELLSJAVA-40838) - PrintCopies は XLS 形式では保持されますが、XLSX 形式では保持されません

例外

(CELLSJAVA-40739) - ピボットテーブルを mht として保存
(CELLSJAVA-40531) - CellsException: マップサイズ (0) は >= 1 でなければなりません


Public API および下位互換性のない変更

以下は、Aspose.Cells for Java に対して行われた下位互換性のない変更と同様に、追加、名前変更、削除、または廃止されたメンバーなど、パブリック API に対して行われた変更のリストです。リストされている変更について懸念がある場合は、 Aspose.Cells サポート フォーラム。

BuiltInDocumentPropertyCollection.Bytes/Characters/CharactersWithSpaces/Lines/Paragraphs プロパティを廃止
これらのプロパティは、Word および PowerPoint 用に記述されています。Excel では省略されます。

Cell.StringValueWithoutFormat プロパティを追加
セルの値を書式なしの文字列として取得します。

HtmlSaveOptions.ExportHiddenWorksheet プロパティを追加します
html ファイルの保存時に非表示のワークシート コンテンツをエクスポートするかどうかを示します。デフォルト値は true です。

ノート
Aspose.Cells for Java のコード ベースは、関連する .NET バージョンのコードと一致するため、Aspose.Cells for .NET v8.1.0 に含まれるほとんどの変更、機能強化、および修正は、この Aspose.Cells for Java v8.1.0 にも含まれています。
