# より良いメモリ管理

AutoCAD® DXF を TIFF (ラスター グラフィックス) 形式に変換しているときに、メモリ リークが発生していました。そのため、API のこのリリースでは、DXF から TIFF への変換のメモリ管理が改善されています。

# DWG から WMF へのエクスポートの改善

AutoCAD® `DWG` の `WMF` (Windows メタファイル) へのエクスポートが改善されました。

# 埋め込み画像のエクスポート

埋め込まれた画像データから画像をエクスポートする機能。

# `DWG` レンダリングの改善

「DWG」が色なしでレンダリングされていることが確認されました。この問題は、API のこのリリースで解決されました。

> このリリースの機能、拡張機能、バグ修正の完全なリストについては、[Aspose.CAD for .NET 22.4 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-4-release-notes/) にアクセスしてください。
