# 3D モデルの保存 (有線)

3D ワイヤード モデルを保持する機能。

# 両面レンダリング

両面プレーンをレンダリングする機能。

# 透かしを拡大

透かしの拡大のサポートが追加されました。

# DWG から PDF への変換の改善

場合によっては、「DWG」が「PDF」形式に変換されたときに完全にレンダリングされないことが確認されました。 API のこのリリースでは、前述の問題が解決されています。
