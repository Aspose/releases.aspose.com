# ハンガリーの特殊文字をエクスポート

以前は、ハンガリー語の特殊文字が DWG から PDF に誤ってエクスポートされる場合がありました。 API のこのリリースでは、前述の問題が解決されています。

# DWG から PDF への変換の改善

`DWG` を `PDF` 形式に変換する際の `OutOfMemoryError` を修正しました。

# シーン移動感度の向上

動くシーンの感度が向上しました。
