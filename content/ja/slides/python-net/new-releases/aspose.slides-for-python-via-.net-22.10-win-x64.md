# フレームレンダリングと回転動作を設定します

`PPTX` およびその他のサポートされているプレゼンテーション形式から形状をレンダリングしながら、フレーム回転とフレームサイズの動作を指定および制御する機能。

# プレゼンテーションサイズを小さくするために、埋め込みフォントを圧縮します

プレゼンテーションに組み込みフォントが含まれている場合、プレゼンテーション処理Python APIのこのリリースは、そのようなすべての埋め込みフォントの圧縮を実行する機能を提供し、`PPTX` などのプレゼンテーションファイルのサイズを縮小します。

# プレゼンテーションリーダーラインの色を設定します

このAPIリリースでは、プレゼンテーションスライド内にリーダーラインの色を設定する機能が紹介されます。

>このリリースの機能、機能強化、およびバグ修正の完全なリストについては、[Aspose.slides for Python 22.10リリースノート]（https://docsops.com/slides/python-net/aspose-slides- for-python-net-22-10-release-notes/）。
