# 更新されたプレゼンテーションソースロック動作

時代遅れのプレゼンテーションロッキングBehavior `LOAD_AND_RELEASE_LEGACY_TEMPORARY_DEFAULT` enumメンバーが削除されました。 `LOAD_AND_RELEASE` は、デフォルトのプレゼンテーションロック動作です。

このデフォルト設定は、「ipresentation」コンストラクターの実行時のみソースをロックします。

>このリリースの機能、機能強化、およびバグ修正の完全なリストについては、[Asoppes.slides for Python 22.6リリースノート]（https://docsops.com/slides/python-net/aspose-slides- for-python-net-22-6-release-notes/）。
