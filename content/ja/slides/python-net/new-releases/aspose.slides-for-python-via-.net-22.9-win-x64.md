# Apple＆Reg;のMacos＆Reg;のサポート

このプレゼンテーション操作Python APIリリースでは、Macos＆Reg;のサポートが紹介されています。

# フォント代替情報を取得します

プレゼンテーションレンダリング中にフォントが交換される情報を取得する機能。これを達成するために、新しい方法 `get_substitutions()`が追加されました。

# スライドアニメーションのタイミングを設定します

これで、[タイミング]（https://reference.aspose.com/slides/python-net/aspose.slides.animation/timingに追加された次の2つの新しいプロパティを介してプレゼンテーションスライドアニメーションを繰り返すタイミングを設定することができます。 /） クラス：

- `repeat_until_next_click`:マウスボタンが次にクリックされるまでアニメーション効果が繰り返されると、このブールプロパティが設定されます。
- `repeat_until_end_slide`:：このブールプロパティは、スライドアニメーション効果が現在のスライドの終了まで繰り返されると設定されます。

>このリリースの機能、機能強化、およびバグ修正の完全なリストについては、[Aspose.slides for Python 22.9リリースノート]（https://docsops.com/slides/python-net/aspose-slides- for-python-net-22-9-release-notes/）。
