# スライドショー設定の包括的なサポート

このAPIリリースでは、プレゼンテーションスライドショーの設定の詳細なサポートが紹介されます。新しいクラスとプロパティの一部は次のとおりです。

 -  [`プレゼンテーション`]（https://reference.aspose.com/slides/python-net/aspose.slides/presentation/）.`slide_show_settings`プロパティ：プレゼンテーションのスライドショーの設定を返します。
 -  [`SlideShowSettings`]（https://reference.aspose.com/slides/python-net/aspose.slides/slideshowsettings/）クラス：プレゼンテーションのスライドショーの設定を表します。利用可能なプロパティの一部は次のとおりです。
   - スライドショーをループで実行します
   - スライドショーのナレーションを表示 /非表示にする機能
   - スライドショーでアニメーションを有効 /無効にします
   - スライドショーのペンカラーを調整します
