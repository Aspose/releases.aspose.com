# PDF 1.6およびPDF 1.7のエクスポートサポート

PDFがPDF 1.6およびPDF 1.7形式にエクスポートされるためのサポートが実装されています。

# A2A、A2BコンプライアンスレベルでPDFをエクスポートします

このPython APIリリースでは、多数のコンプライアンスレベル、つまりA2A、A2B、A2U、A3A、およびA3BのPDFエクスポート機能のサポートを紹介します。

# プレゼンテーションでハイパーリンクサウンドを管理します

新しいプロパティ（[`Hyperlink.sound`]（https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/）が追加されています。
