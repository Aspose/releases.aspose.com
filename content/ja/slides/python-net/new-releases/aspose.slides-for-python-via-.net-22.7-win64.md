# Ubuntuフォーカルの改善されたサポート

Microsoft PowerPoint＆Reg;の変換中に観察されました。 `PPTX` から` PDF` 形式は、ubuntu focalに表示されました。これらの問題は現在解決されています。

# 3D列チャートサポート

このAPIリリースでは、3D列チャートのサポートが導入されています。これで、デフォルトデータを備えたチャートを追加してから、 `Rotation3D`プロパティ（直角軸、深さパーセントなど）を設定できます。

# ヘブライ語のテキスト効果のサポートが改善されました

以前は、`PPTX` プレゼンテーションが `PDF` 形式に変換されると、ヘブライ語のテキストシャドウが間違っていることが観察されていました。この問題は現在解決されています。

>このリリースの機能、機能強化、およびバグ修正の完全なリストについては、[Aspose.slides for Python 22.7リリースノート]（https://docsops.com/slides/python-net/aspose-slides- for-python-net-22-7-release-notes/）。
