# 外部リソースの埋め込みの改善

以前は、一部のインスタンスで、Microsoft Visio® ダイアグラムを HTML ストリームに保存すると、外部リソースが埋め込まれませんでした。ただし、この問題は解決されました。

# シェイプ シェーディングのより良い処理

Visio® VSDX を PDF に変換すると、図形のシェーディングが失われるという問題がありました。このリリースでは、この問題が解決され、シェープ シェードの処理が改善されています。

# HTML を単一のストリームに保存する

HTML を単一のストリームに保存するためのサポートが追加されました。

# リソースが埋め込まれた HTML にダイアグラムを保存

埋め込みリソースの有無にかかわらず、Visio® ダイアグラムを HTML に保存する機能。

> このリリースの機能、拡張機能、バグ修正の完全なリストについては、[Aspose.Diagram for .NET 22.1 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-1-release-notes/) にアクセスしてください。
