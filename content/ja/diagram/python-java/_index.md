### Java オンプレミス API 機能によるダイアグラム処理 Python

- API を介してゼロから Microsoft Visio® ダイアグラムを作成します。
- Microsoft Visio® ファイルの読み取りまたは書き込み。
- Visio® の図を、PDF、画像、HTML などのさまざまな一般的な形式にエクスポートします。
- ダイアグラムの要素とプロパティにアクセスして、操作または検査のみを行います。
