[Aspose.Diagram for Java](https://products.aspose.com/diagram/java) APIは、Microsoft Visioファイルの操作要件のソリューションです。これにより、Javaアプリケーションは、シェイプ、ページ、画像、形状マスター、テキスト、レイヤー、ヘッダー、フッター、ユーザー定義セル、ユーザー定義セル、ユーザー定義セルなど、すべてのVisioオブジェクトとプロパティをサポートしながら、Microsoft Visioダイアグを読み取り、書き込み、エクスポート、および処理できます。ファイル保護、ジオメトリ、テキストボックス、およびコメント。

Java用のAsopse.Diagramは、偶発的な修正を避けるために、背景、ステンシル（マスター）、形状とスタイルをロックすることで[Microsoft Visio図面に保護を適用します](https://docs.aspose.com/diagram/java/working-with-protection/)にも使用できます。

## Visioファイル処理機能
-Microsoft Visioダイアグラムをロード、作成、保存、変換します。

-[Sctrachからマスターを作成します](https://docs.aspose.com/diagram/java/working-with-masters/)または形状のステンシルから追加します。
 -  Visioコネクタとフォント情報を取得するだけでなく、シェイプを作成、レイアウト、自動化します。
 - コネクタ形状のジオメトリセクションを変更します。
 -  Visio図にActiveXコントロールを挿入します。

-[特定のプリンターへの印刷](https://docs.aspose.com/diagram/java/working-with-print/)またはプリンターとドキュメント名を設定します。

## Visioフォーマットの読み取りと書き込み
** Microsoft Visio：** VSDX、VSX、VTX、VDX、VSSX、VSTX、VSDM、VSSM、VSTM

## bisioダイアグラムを保存します
**修正レイアウト：** PDF、XPS \
**画像：** jpeg、png、bmp、tiff、svg、emf \
** web：** html \
**その他：** Xaml、SWF

## Visioフォーマットを読みます
** Microsoft Visio：** VDW、VSD、VSS、VST

## サポートされている環境
 -  ** Microsoft Windows：** Windowsデスクトップ＆サーバー（x86、x64）
 -  ** macOS：** Mac OS X
 -  ** linux：** ubuntu、opensuse、centosなど
 -  ** Javaバージョン：** `J2SE 7.0（1.7）`、 `J2SE 8.0（1.8）`

## 始めましょう

Aspose.Diagram Java APIは[Aspose Repository](https://releases.aspose.com/diagram/java/)でホストされています。簡単な構成を使用して、MavenプロジェクトでJava APIにAspose.Diagramを簡単に使用できます。詳細な手順については、[Installing Aspose.Diagram for Java from Aspose Repository](https://docs.aspose.com/diagram/java/installation/)ドキュメントページをご覧ください。

## Javaを使用した自動フィットVisioダイアグラム

```java
// For complete examples and data files, please go to https://github.com/aspose-diagram/Aspose.Diagram-for-Java
// The path to the documents directory.
String dataDir = Utils.getDataDir(AutoFitShapesInVisio.class);

// load a Visio diagram
Diagram diagram = new Diagram(dataDir + "BFlowcht.vsdx");

// use saving options
DiagramSaveOptions options = new DiagramSaveOptions(SaveFileFormat.VSDX);

// set Auto fit page property
options.setAutoFitPageToDrawingContent(true);

// save Visio diagram
diagram.save(dataDir + "AutoFitShapesInVisio_Out.vsdx", options);
```
