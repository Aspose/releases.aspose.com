## **改善点と変更点**

|**Key**|**Summary**|**Category**|
| :- | :- | :- |
| THREEDNET-1432 | USDA ファイルのサポートを追加 | タスク |
| THREEDNET-1435 | SkiaSharp 統合により、画像が正しくレンダリングされない | サポート |


### API の変更点


クラス **com.aspose.threed.FileFormat** へのメンバー追加：

{{< highlight java >}}

    /**
     * Universal Scene Description in ASCII format.
     */
    public static final FileFormat USDA;


{{</highlight>}}

Aspose.3D 23.11 以降、Aspose.3D は USDA ファイルまたは USDZ ファイルとしてパッケージ化された USDA ファイルをインポートできるようになりました。
