# FBX インポーターの一時ファイルのサポート

FBX インポーターの内部一時ファイル システム サポートが追加されました。

# GLTF エクスポートの改善

`GLTF` のエクスポートは、以前のリリースではあまり良くありませんでした。ただし、3D API の現在のバージョン リリースでは、`GLTF` 形式のファイルのエクスポート機能が改善されています。

> このリリースの機能、拡張機能、バグ修正の完全なリストについては、[Aspose.3D for .NET 22.9 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-9-release-notes/) にアクセスしてください。
