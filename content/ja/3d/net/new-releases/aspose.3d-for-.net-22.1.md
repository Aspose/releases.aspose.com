# .Net 2.0 をサポート

.NET Standard 2.0 のサポートを復元しました。

# USDZ から GLB への変換の改善

以前は、一部のインスタンスで、API が「USDZ」ファイルを開いて「GLB」形式に変換できませんでした。このバージョンの API では、この問題が解決されています。

# プリミティブ エンティティのエンコードとデコード

USD エクスポーターにプリミティブ エンティティのエンコードとデコードのサポートを追加しました。

> このリリースの機能、拡張機能、バグ修正の完全なリストについては、[Aspose.3D for .NET 22.1 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-1-release-notes/) にアクセスしてください。
