## **改善点と変更点**

|**Key**|**Summary**|**Category**|
| :- | :- | :- |
| THREEDNET-1601 | サポートされていない bGPdata と、blender ファイル内の未解決配列。 | タスク |
| THREEDNET-1594 | センターラインプロファイルサポート | 新機能 |

## API の変更点 ##

### 追加されたクラス **com.aspose.threed.CenterLineProfile**

このプロファイルを使用すると、曲面に厚さを与えることができ、LinearExtrusion のような手続き型モデリング機能で使用できます。
