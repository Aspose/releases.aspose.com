### API の変更点


クラス **aspose.threed.FileFormat** に追加されたメンバー：

{{< highlight csharp >}}

        # ASCII 形式のユニバーサル シーン記述。
        USDA : FileFormat

{{</highlight>}}


Aspose.3D 23.11 以降、Aspose.3D は USDA ファイルをインポートしたり、USDZ ファイルとして USDA をパッケージ化したりできるようになりました。
