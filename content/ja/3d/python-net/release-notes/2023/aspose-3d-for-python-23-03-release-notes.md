## **改善点と変更点**

|**Key**|**Summary**|**Category**|
| :- | :- | :- |
| THREEDPYTHON-72 | PYI 生成を追加するために PyRefGenerator を改善 | Improvement |
| THREEDNET-1340 | fbx から usd へのシーンの保存で例外が発生する可能性があります | Bug fixing |
| THREEDNET-1341 | アプリのバグ - html へのファイル保存で ArgumentNullException が発生する可能性があります | Bug fixing |
| THREEDNET-1343 | null ラベルのメタデータが FBX ファイルに保存されると失敗します | Bug fixing |
| THREEDNET-1350 | FBX エクスポートでは、short int 型のプロパティはサポートされていません | Bug fixing |
| THREEDNET-1352 | バイナリ maya ファイルのサポートされていない maya セクション VEC# | Bug fixing |
| THREEDNET-1353 | バイナリ maya ファイルのサポートされていない maya セクション SBDS | Bug fixing |
| THREEDNET-1354 | バイナリ maya ファイル形式で不明な UV セット | Bug fixing |
| THREEDNET-1355 | GLB ファイルのロード中に NullReferenceException | Bug fixing |
| THREEDNET-1356 | サポートされていない maya コマンド: instanceable | Bug fixing |


## API の変更点 ##

23.3 では、FBX と Maya のいくつかのバグを修正しましたが、API の変更はありません。
