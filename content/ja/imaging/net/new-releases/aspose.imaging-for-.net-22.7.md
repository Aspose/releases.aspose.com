# `CDR` フォーマットのメモリ管理を最適化

CorelDRAW® `CDR` ベクター グラフィックスの読み込みと処理を、メモリ管理やラスタライズなどのさまざまな方法で高速化する機能。

```csharp
var baseFolder = @"D:\";
var fileName = "6õ4 ÎÑÍÎÂÀ.cdr";
var inputFilePath = Path.Combine(baseFolder, fileName);
var outputFilePath = inputFilePath + ".png";
using (var image = (CdrImage)Image.Load(inputFilePath))
{
    image.Save(outputFilePath, new PngOptions()
    {
        VectorRasterizationOptions = new CdrRasterizationOptions
        {
             Positioning = PositioningTypes.DefinedByDocument
        }
    });
}
```

# `EPS` から `PNG` への変換

高品質の `EPS` PostScript ベクター形式ファイルを `PNG` 形式のラスター イメージに変換する機能。

```csharp
using (var image = Image.Load("3_a_1_2.eps"))
{
   image.Save("output.png", new PngOptions());
}
```
