# Melhor gerenciamento de memória

O vazamento de memória foi observado ao converter o formato AutoCAD® DXF para TIFF (gráficos raster). Portanto, esta versão da API oferece um melhor gerenciamento de memória para conversão de DXF para TIFF.

# Exportação de DWG para WMF aprimorada

Melhorada a exportação de AutoCAD® `DWG` para `WMF` (Metarquivo do Windows).

# Exportar imagem incorporada

Capacidade de exportar imagem de dados de imagem incorporados.

# Melhoria na renderização de `DWG`

Observou-se que o `DWG` estava sendo renderizado sem cores. Esse problema já foi resolvido nesta versão da API.

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite [Aspose.CAD for .NET 22.4 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-4-release-notes/).
