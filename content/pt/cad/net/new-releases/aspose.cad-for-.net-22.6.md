# Melhor renderização de linha do Acrobat Reader® e Foxit®

Anteriormente, em alguns casos, as linhas estavam faltando no Acrobat Reader®, mas estavam presentes no Foxit®. Este problema já foi resolvido.

# Suporte para o formato AutoCAD® DXB

Implementou e introduziu o suporte para o formato de arquivo AutoCAD® DXB. `DXB` normalmente armazena os desenhos CAD 3D. `DXB` é a versão binária do arquivo `DXF`.

# Conversão aprimorada de DWF para PDF

Anteriormente, em alguns casos, a conversão de DWF para PDF não converteu todas as páginas corretamente. Este problema já foi resolvido.

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite [Aspose.CAD for .NET 22.6 - Release Notes](https://releases.aspose.com/cad/net/release-notes/2022/aspose-cad-for-net-22-6-release-notes/).
