Recursos da API de imagem##

Os seguintes são os principais recursos do Inimaging.

- Crie imagens
- Leia imagens
- Escreva imagens
- Desenhe imagens
- Processar imagens (incluindo modificações por pixel)
- converter entre formatos
- Corte, gire e redimensione imagens via API.
- De-Skew & Transform Images.
- Defina propriedades de imagem.

Visite a [documentação oficial](https://docs.aspose.com/imaging/python-net/) para uma lista mais detalhada de recursos.

### Leia e escreva formatos de imagem

** Formatos de raster: ** JPEG2000, JPEG, BMP, TIFF, BIGTIFF, GIF, PNG, APNG \
** Imagem médica: ** dicom \
** Metafiles: ** EMF, WMF, EMZ, WMZ \
** TrueVision: ** TGA \
** Gráficos vetoriais: ** SVG, SVGZ

### salvar imagens como

** Corrigido: ** PDF \
** Photoshop: ** PSD \
** Web: ** HTML5 Canvas

### leia formatos de imagem

** Ebook: ** DJVU \
** Câmera digital crua: ** DNG \
** OpenOffice: ** ODG, OTG \
** Bitmap: ** DIB \
** Imagem da Web: ** Webp \
** CORELDRAW: ** CDR (x6, x7), CMX (v2.0 32 bits) \
** PostScript: ** EPS

Visite [Formatos de arquivo suportado](https://docs.aspose.com/imaging/python-nalt/supported-file-formats/) para obter mais detalhes.

### Requisitos de sistema

Aspom.Imageing for Python via .NET suporta sistemas operacionais Windows de 64 bits, Linux de 64 bits, onde o Python 3.5 ou posterior é instalado.

### Sistemas operacionais suportados

** Microsoft Windows: ** Windows Desktop & Server (`x64`) \
** Linux: ** Ubuntu, OpenSuse, Centos e outros.
