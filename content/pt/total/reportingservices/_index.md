O Aspose.Total para Reporting Services contém os seguintes componentes:

* Aspose.BarCode para Reporting Services: Aprimora seus relatórios SQL para exibir mais de 60 tipos de códigos de barras em alta resolução.
* Aspose.Cells para Reporting Services: Projete e entregue relatórios do Microsoft Excel no Microsoft SQL Server Reporting Services.
* Aspose.PDF para Reporting Services: Gere relatórios em PDF, PDF/A e XPS de alta qualidade a partir do MS SQL Server Reporting Services.
* Aspose.Slides para Reporting Services: Gere facilmente relatórios em formato PPT e PPS dentro do Microsoft SQL Server Reporting Services.
* Aspose.Words para Reporting Services: Manipule e exporte relatórios RDL e RDLC para formatos de arquivo suportados de dentro do MS SQL Server Reporting Services.
