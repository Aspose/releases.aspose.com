### Recursos de integração do PowerPoint® SharePoint®

- Suporte para formatos de apresentação populares.
- Converta apresentações do Microsoft PowerPoint diretamente no servidor.
- Conversão rápida de formato através da conveniente opção Converter via Aspose.Slides no menu.
- Não é necessário instalar o Microsoft PowerPoint.
- API pública que você pode usar para desenvolver seus próprios aplicativos do SharePoint.

### Ler e gravar arquivos do PowerPoint

**Microsoft PowerPoint®:** PPT, PPTX, PPS, PPSX, PPTM, PPSM, POTX, POTM

### Salvar apresentação como

- **Layout fixo:** PDF, XPS
- **Imagem:** SVG, TIFF
- **Web:** HTML, SWF

### Requisitos de sistema

#### Requisitos do servidor

- Windows SharePoint Services 3.0 (WSS)
- Padrão Microsoft Office SharePoint Server 2007 (MOSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
-Microsoft Office SharePoint Server 2013
-Microsoft Office SharePoint Server 2019
- Recomenda-se a instalação do Service Pack 2 ou posterior para esses produtos.

#### Requisitos do cliente

- Microsoft Internet Explorer 6, 7, 8
-Mozilla Firefox 3.5

#### Versões suportadas do Microsoft PowerPoint

-Microsoft PowerPoint 97
-Microsoft PowerPoint 2000
-Microsoft PowerPointXP
-Microsoft PowerPoint 2003
-Microsoft PowerPoint 2007
-Microsoft PowerPoint 2010
-Microsoft PowerPoint 2013
-Microsoft PowerPoint 2016
-Microsoft PowerPoint 2019
- Microsoft PowerPoint para MAC
