# Suporte para MacOS & Reg;

Esta apresentação Manipulação Python API Lançamento apresenta o suporte para o MacOS & Reg;.

# Obtenha informações de substituição de fontes

Capacidade de obter as informações sobre as fontes que estão sendo substituídas durante a renderização da apresentação. Um novo método, `get_substitutions ()`, foi adicionado para conseguir isso.

# Defina o tempo para a animação de slide

Agora você pode definir o tempo para repetir a animação de slide de apresentação através das duas novas propriedades a seguir adicionadas ao ['timing`] (https://reference.aspose.com/slides/python-nalt/aspose.slides.animation/timing /) classe:

- `Repety_until_next_click`: esta propriedade BOOL define se o efeito de animação for repetido até que o botão do mouse seja o próximo clicado.
- `Repatious_until_end_slide`: esta propriedade BOOL define se o efeito de animação do slide for repetido até o final do slide atual.
