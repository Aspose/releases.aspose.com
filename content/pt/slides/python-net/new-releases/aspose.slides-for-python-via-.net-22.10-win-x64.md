# Definir comportamento de renderização e rotação do quadro

Capacidade de especificar e controlar o comportamento da rotação do quadro e do tamanho do quadro enquanto renderiza formas de `pptx` e outros formatos de apresentação suportados.

# Comprimir fontes incorporadas para reduzir o tamanho da apresentação

Se sua apresentação contiver as fontes incorporadas, esta versão da nossa API de processamento de processamento de apresentação oferece a capacidade de executar a compressão de todas essas fontes incorporadas e, por sua vez, reduz o tamanho dos arquivos de apresentação como o `pptx`.

# Definir linhas de apresentação linhas de cor

Este lançamento da API apresenta a capacidade de definir a cor das linhas líderes em seus slides de apresentação.

> Para uma lista completa de recursos, aprimoramentos e correções de bugs neste lançamento, visite [Aspose.slides para Python 22.10 Notas de lançamento] (https://docs.aspose.com/slides/python-nalt/aspose-slides- para-python-Net-22-10-lançamento-nota/).
