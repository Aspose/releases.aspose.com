# Suporte abrangente para configurações de apresentação de slides

Esta versão da API apresenta o suporte detalhado para as configurações de apresentação de slides de apresentação. Algumas das novas classes e propriedades são as seguintes:

- [`Apresentation`] (https://reference.aspose.com/slides/python-nalt/aspose.slides/presentation/) .lide_show_settings` Propriedade: Retorna as configurações da apresentação de slides para a apresentação.
- [`slides-shoutstings`] (https://reference.aspose.com/slides/python-nalt/aspose.slides/slideshowsettings/) classe: representa as configurações da apresentação de slides para a apresentação. Algumas das propriedades disponíveis são;
  - Faça a apresentação de slides funcionar em um loop
  - Capacidade de mostrar / ocultar narração de apresentação de slides
  - Ativar / desativar a animação na apresentação de slides
  - Ajustar a cor da caneta para a apresentação de slides
