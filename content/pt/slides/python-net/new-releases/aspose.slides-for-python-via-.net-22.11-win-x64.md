# Exportar PDF 1.6 e PDF 1.7 Suporte

O suporte foi implementado para que o PDF seja exportado para os formatos PDF 1.6 e PDF 1.7.

# Exportar PDF com A2A, níveis de conformidade A2B

Esta versão da API do Python apresenta o suporte ao recurso de exportação em PDF com vários níveis de conformidade, isto é, A2A, A2B, A2U, A3A e A3B.

# Gerencie o som do hiperlink em apresentações

Foi adicionado uma nova propriedade ([`hyperlink.sound`] (https://reference.aspose.com/slides/python-nalt/aspose.slides/hyperlink/)) foi adicionado que representa o som tocado de um hiperlink.

# Efeito de animação para o som anterior

Uma nova propriedade ([`efet.stop_previous_sound`] (https://reference.aspose.com/slides/python-nalt/aspose.slides.animation/effect/) foi introduzida que especifica se o efeito da animação para o anterior som.
