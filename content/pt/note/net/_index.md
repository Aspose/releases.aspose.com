> É uma biblioteca de classes autônoma que permite interagir com documentos do Microsoft OneNote® para processamento e conversão.

## Por que baixar o Aspose.Note para .NET?

O Aspose.Note para .NET pode ser usado para imprimir documentos ONE, bem como manipular páginas, imagens, texto, tabelas, anexos, tags, tarefas, estilos de texto e hiperlinks, sem precisar do Microsoft OneNote®.

## Recursos de processamento de arquivos do Microsoft OneNote®

- Carregue, edite e salve documentos do Microsoft OneNote® via API.
- Navegue pelo Modelo de Objeto de Documento (DOM) do OneNote®.
- Insira uma imagem em um arquivo do OneNote®.
- Analise e exporte vários formatos de lista numerada.
- Extraia texto de qualquer parte de um documento do OneNote®.
- Exporte documentos do OneNote® como outros formatos populares.

### Ler e escrever no formato OneNote®

**Microsoft OneNote®:** UM

### Salvar arquivos do OneNote® como

**Layout fixo:** PDF\
**Imagens:** PNG, GIF, JPEG, BMP, TIFF

### Ler Formatos

**Microsoft OneNote®:** ONETOC2
