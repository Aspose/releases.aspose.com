É uma API independente no local que ajuda a reconhecer os dados marcados por humanos a partir de imagens de documentos digitalizados, pesquisas, questionários, questionários e provas.

## Por que baixar o Aspose.OMR para .NET?

O Aspose.OMR para .NET não atua apenas como um mecanismo de Reconhecimento de Marca Óptica. Ele também fornece um controle gráfico útil que pode ser usado para ajuste manual do limite e marcação para ver as alterações em tempo real.

## Recursos da API OMR

- Reconhecimento de imagens e fotos digitalizadas.
- Capacidade de processar imagens giradas e em perspectiva (vistas de lado).
- Reconhecer dados de testes, exames, questionários, pesquisas, etc.
- Alta taxa de precisão e capacidade de exportar os resultados no formato CSV.
- Criar modelos OMR.

### Leia imagens para OMR

JPEG, PNG, GIF, TIFF, BMP

### Salvar resultados de OMR como

CSV, JSON, XML
