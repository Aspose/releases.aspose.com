# Suporte de fallback de fonte no Linux

Adicionado o suporte de fallback de fonte no Linux. Além disso, se uma fonte tiver apenas o estilo itálico, o substituto para essa fonte também foi adicionado.

# Desenha uma linha com suavização de serrilhado

Melhorou os recursos de desenho da API para desenhar uma linha com suavização de serrilhado.

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite [Aspose.Drawing for .NET 22.5 Release Notes](https://docs.aspose.com/drawing/net/aspose-drawing-for-net-22-5-release-notes/).
