## Por que baixar o Aspose.OCR para C++?

Aspose.OCR para C++ é uma API de OCR independente que aprimora seus aplicativos C++ para executar a operação de OCR em imagens JPEG, PNG e BMP para extração de conteúdo em inglês.

## Recursos da API de OCR de imagem

- Reconhecer caracteres de imagens.
- Calcule o ângulo de inclinação das imagens.
- Atualmente, ele suporta o idioma inglês.
- Alinhar automaticamente a imagem e detectar automaticamente as áreas de texto.
- Obtenha uma única linha e reconheça-a.

### Ler formatos de imagem para OCR

**Formatos de varredura:** JPEG, BMP (imagens de 32 e 16 bits não são suportadas), PNG

### Charset suportado para reconhecimento

PT

### Pré-requisitos

A biblioteca Aspose OCR requer `onnxruntime.dll` no caminho do sistema.

### Limitação da versão de avaliação

A versão de avaliação do Aspose.OCR para C++ limita o número de caracteres extraídos de uma imagem a 300.
