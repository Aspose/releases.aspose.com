## Por que baixar o Aspose.SVG para .NET?

Essa API .NET on-premise ajuda você a integrar perfeitamente a funcionalidade de processamento e manipulação de arquivos SVG em seus aplicativos C#, VB.NET, ASP.NET e outros aplicativos baseados em .NET.

## Recursos de processamento de arquivos SVG

- Criar, ler e escrever arquivos no formato SVG.
- Converta SVG para outros formatos de arquivo suportados.
- Manipulação da árvore DOM de acordo com as especificações oficiais do SVG.
- Suporte para navegação de conteúdo via XPath Query, CSS Selectors, Element e Document Traversal.
- Suporte para renderização de qualidade.

### Leia os formatos suportados

SVG

### Salvar SVG como

**Layout fixo:** PDF, XPS\
**Imagem:** TIFF, BMP, PNG, JPEG, GIF
