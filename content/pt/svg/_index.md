## Por que baixar o Aspose.SVG para .NET?

Esta API .NET no local ajuda você a [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) funcionalidade em seu C#, VB.NET, ASP.NET e outros aplicativos baseados em .NET.

## Recursos de processamento de arquivos SVG

- arquivos de formato [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/).
- Converta SVG para outro [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- Manipulação da árvore DOM de acordo com as especificações oficiais do SVG.
- Suporte para navegação de conteúdo por meio de [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), seletores CSS, elementos e recursos de travessia de documentos.
- Suporte para renderização de qualidade.
