# Carregamento aprimorado dos formatos 3D

A API 3D estava lançando exceção ao carregar arquivos `Gltf`,` x` e `VRML`. Esses formatos 3D agora são totalmente compatíveis e são carregados sem problemas.

# Reconhecimento Tangente/Binormal em Glb

Anteriormente, em alguns casos, a tangente/binormal definida em `glb 'com Draco habilitada não estava sendo reconhecida. Este problema foi resolvido dentro desta versão da API.

# Conversão aprimorada de formatos 3D

Foram feitas melhorias nas seguintes conversões de formato 3D:

- ** ASE para 3DS ** Autodesk & Reg; Formato de arquivo ASCII Scene Export (ASE) para conversão de formato de estúdio 3D.

- ** ASE TO OBJ ** AUTODESK & REG; Formato de arquivo ASCII Scene Export (ASE) para o formato OBJ Conversão.

- ** USD para DRC ** Formato de arquivo da cena universal da Pixar (USD) para o formato do Google Draco (DRC).

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite, [Aspose3d para Not 231 Notas de lançamento](https://releases.aspose.com/3d/net/release-notes/2023/aspose-3d-for-net-23-1-release-notes/).
