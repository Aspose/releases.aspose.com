# Suporte .Net 2.0

Restaurado o suporte do .NET Standard 2.0.

# Conversão melhorada de USDZ para GLB

Anteriormente, em alguns casos, a API não conseguia abrir arquivos `USDZ` para converter para o formato `GLB`. Esta versão da API resolveu esse problema.

# Codificação e Decodificação de Entidades Primitivas

Adicionado o suporte de codificação e decodificação de entidades primitivas no exportador de USD.

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite [Aspose.3D for .NET 22.1 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-1-release-notes/).
