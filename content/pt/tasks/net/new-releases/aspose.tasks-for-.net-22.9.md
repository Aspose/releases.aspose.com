# Melhoria na conversão de XML para MPP

Corrigida a exceção que ocorria ao salvar um arquivo XML específico no formato MPP. O trecho de código C# a seguir demonstra como um arquivo XML do Primavera P6 pode ser convertido em um arquivo do Microsoft Project® usando a API:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Recurso de leitura aprimorado das taxas de recursos do Primavera XER

Corrigido o problema de leitura de taxas de recursos dos arquivos Primavera XER.

# Conversões melhoradas de vários formatos para MPP

Corrigidos os erros que eram acionados ao salvar arquivos específicos no formato MPP.
