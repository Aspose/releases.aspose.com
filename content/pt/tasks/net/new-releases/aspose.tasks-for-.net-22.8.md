# Processamento aprimorado de arquivos Primavera P6 XER

Esta versão da API oferece muitas melhorias no processamento dos arquivos do projeto Primavera P6 XER. Isso inclui a capacidade de ler o calendário de tarefas, ler a moeda do projeto e ler o LinkLag, etc.

> **Mudanças importantes:** Agora, por padrão, o projeto será renderizado com a granularidade da escala de tempo especificada no MS Project®.

> Para obter uma lista completa de recursos, aprimoramentos e correções de bugs nesta versão, visite [Aspose.Tasks for .NET 22.8 Release Notes](https://docs.aspose.com/tasks/net/aspose-tasks-for-net-22-8-release-notes/).
