### Conclusão do projeto % para formatos primavera

Esta versão apresenta o recurso para acessar os valores do número fracionário de porcentagem física completa e porcentagem de completa para ler a porcentagem (%) de conclusão da primavera & reg; projetos.

### Leia a coluna de prioridade da tarefa para arquivos MPX

Foi feito um aprimoramento na leitura da coluna de prioridade da tarefa para o Microsoft Project & Reg; `MPX` Arquivos onde a coluna prioritária foi salva usando os valores` enum`.

### Acesso Valores da unidade de trabalho para primavera & reg; formatos

Agora, via API, você pode acessar os valores das unidades trabalhistas para primavera & ref; Formatos de projeto.
