### Processamento de código de barras Python via Java Recursos da API On-premise

- Oferece suporte para a maioria das especificações e padrões de código de barras populares.
- Você pode ler e exportar códigos de barras em vários formatos de imagem, por exemplo, JPEG, PNG, GIF e BMP.
- Capacidade de trabalhar com etiquetas de código de barras.
- Suporte para a dimensão X, bem como a dimensão Y para códigos de barras 2D.
