### Recursos da API local do código de barras PHP

- Suporta padrões de código de barras e especificações de código de barras mais estabelecidos.
- Capacidade de ler e exportar códigos de barras em vários formatos de imagem, incluindo BMP, GIF, JPEG e PNG.
- Fornece controle total sobre etiquetas de código de barras.
- Suporta dimensão X e dimensão Y para códigos de barras 2D.
