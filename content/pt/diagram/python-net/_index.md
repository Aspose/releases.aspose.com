### Visio Python via .NET On-premise API Features

- Crie arquivos do visio via API.
- Conversão e renderização de formato de arquivo de alta qualidade.
- Imprima arquivos do Microsoft Visio® em impressoras físicas ou virtuais.
- Combine, modifique, proteja ou analise as páginas do Visio®.
- Converta formas em imagens ou PDF.
- Configure e aplique a configuração de página para as páginas.
- Converta arquivos do Visio® para vários outros formatos.

Visite o [official documentation](https://docs.aspose.com/diagram/python-net/) para obter uma lista mais detalhada de recursos.

### Ler formatos de arquivo do Visio

**Microsoft Visio**: VSD, VDW, VSX, VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM

### Salvar arquivos do Visio AS

**Microsoft Visio**: VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM
**Layout fixo**: PDF, XPS
**Imagens**: JPEG, PNG, BMP, SVG, TIFF, GIF, EMF
**Web**: HTML
### Requisitos de sistema

Sua máquina não precisa ter o software Microsoft Visio® instalado.

### Sistemas operacionais suportados

**Microsoft Windows®:** Windows Desktop & Server (`x64`, `x86`)\
**Linux:** Ubuntu, OpenSUSE, CentOS e outros\
**Outros:** Qualquer sistema operacional (SO) que possa instalar o Mono (suporte a .NET 4.0 Framework) ou usar o .NET core.
