Aspose.Diagram for .NET API é uma solução para os requisitos de manipulação de arquivos do Microsoft Visio®. Ele permite que os aplicativos .NET leiam, escrevam, exportem e processem diagramas do Microsoft Visio® enquanto suportam todos os objetos e propriedades do Visio®, incluindo formas, páginas, imagens, mestres de formas, estênceis, texto, camadas, cabeçalho, rodapés, células, hiperlinks, proteção de arquivos, geometrias, caixas de texto e comentários.

O Aspose.Diagram for .NET também pode ser usado para aplicar proteção aos desenhos do Microsoft Visio® bloqueando fundos, estênceis (mestre), bem como formas e estilos para evitar alterações acidentais.

## Por que baixar Aspose.Diagram para .NET?

Aspose.Diagram for .NET on-premise API permite que seus aplicativos .NET interajam facilmente com o Microsoft Visio® Object Model. Isso permite que seus aplicativos criem, leiam, escrevam, exportem, imprimam e convertam VSD, VSS, VDW, VST e outros formatos de arquivo do Visio® para PDF, XPS, SVG, HTML, PNG, JPEG, EMF, TIFF, SWF, XAML , etc, sem instalar o Microsoft Visio® ou qualquer outro software de terceiros.

> Aspose.Diagram for .NET é escrito inteiramente em C# e contém apenas código gerenciado. Aspose.Diagram.dll é compatível com CLS. Qualquer tipo de aplicativo .NET de 32 ou 64 bits, incluindo ASP.NET, Web Services e WinForms pode ser desenvolvido usando Aspose.Diagram para classes .NET.

Aspose.Diagram for .NET API suporta todos os objetos e propriedades do Visio®, incluindo formas, páginas, imagens, mestres de formas, estênceis, texto, camadas, cabeçalho, rodapés, células definidas pelo usuário, hiperlinks, proteção de arquivos, geometrias, caixas de texto e comentários. Os desenvolvedores também podem aplicar proteção aos desenhos do Microsoft Visio® bloqueando planos de fundo, estênceis (mestre), bem como formas e estilos para evitar alterações acidentais.

## Recursos de processamento de arquivos do Visio®

- Crie diagramas do Microsoft Visio® do zero via API.
- Ler ou escrever desenhos do Microsoft Visio®.
- Exporte diagramas do Visio® para vários formatos populares, incluindo PDF, imagens, HTML e muito mais.
- Imprima diagramas do Visio® em uma impressora física ou virtual.
- Acesse as propriedades do diagrama do Visio® para manipulação ou apenas inspeção.
- Proteja os diagramas do Visio® através da aplicação de bloqueios em vários níveis.
- Manipule os objetos OLE incorporados nos diagramas do Visio®.

### Formatos de leitura e gravação suportados do Visio®

**Microsoft Visio®:** VSDX, VDX, VSX, VTX, VSSX, VSTX, VSDM, VSSM, VSTM

### Salvar diagramas do Visio® como

**Layout fixo:** PDF, XPS\
**Vetor:** SVG\
**Metarquivo:** EMF\
**Imagem:** JPEG, PNG, BMP, TIFF\
**Marcação:** HTML, XAML\
**Outro:** SWF

### Formatos de leitura do Visio® suportados

**Microsoft Visio®:** VSD, VDW, VSS, VST
