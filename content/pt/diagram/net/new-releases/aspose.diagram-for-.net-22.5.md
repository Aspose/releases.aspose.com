# Execução de método assíncrono melhorada

Ao converter o Visio® VSDX para PDF, o arquivo de saída não estava sendo salvo no método Async até que o programa fosse totalmente executado. A versão atual da API resolveu esse problema.

# Resolução de problema de fórmula/valor

Os campos de fórmula/valor não estavam sendo atualizados corretamente. Esse problema foi resolvido na API.

# Melhor manipulação de texto de tabela na conversão de VSDX para PDF

Durante o processo de conversão de VSDX para PDF, em alguns casos o texto presente na tabela estava sendo removido. Este problema foi corrigido agora.
