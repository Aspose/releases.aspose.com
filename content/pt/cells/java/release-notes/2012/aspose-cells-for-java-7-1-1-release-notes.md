Nós somos
 tenho o prazer de anunciar Aspose.Cells for Java v7.1.1!

 Novas características

- Verifique se uma coluna está em branco

 Melhorias

- Salvando em XLSX com LightCells API renderizando diretamente em OutputStream

 Exceções

- O método Cell.setHtmlString() lança NullPointerException
- Lendo um arquivo de modelo XLSX lança NullPointerException

Insetos

- Eixo de categoria secundária do gráfico SCATTER torna-se eixo de texto
- O conteúdo da caixa de texto não foi renderizado corretamente
- A função ROUND não pode arredondar um valor maior que 922337,20
- gráfico não foi lido corretamente no arquivo ODStemplate
- Formato de número personalizado: "dd.MM.yyyy" não foi salvo corretamente para o arquivo ODS
- LineShape fica mais curto no documento PDF gerado
