 Esta página contém notas de lançamento para[Aspose.Cells for Java 7.2.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.2/)

{{% /alert %}} 

Nós somos
tenho o prazer de anunciar Aspose.Cells for Java v7.2.2!

 Novas características

- Pesquisa de texto RegEx para o método Cells.Find()

 Melhorias

- Torne Aspose.Cells compatível com a versão mais antiga dos potes Woodstox
- Arquivos OOXML incorporados OLE em XLS estão saindo como arquivos compactados OLE em vez de arquivos descompactados
- Suporte ExportObjectListener para salvar arquivos HTML
- Copiar formatação condicional ao copiar linhas

 Exceções

- Picture.isPrintable() para Picture inPageSetup causa NullPointerException
- Salvar pasta de trabalho criptografada com tabela dinâmica causa java.lang.NegativeArraySizeException

Insetos

- Cells.importCustomObjects() com formato DateTime especificado não funciona
- Tipo de gráfico de dispersão incorreto
- O valor duplo perde precisão ao ler o arquivo de modelo CSV
- A série de gráficos virou de cabeça para baixo ao convertê-la em uma imagem
- Arquivo XLSX salvo novamente causa erro "Conteúdo ilegível do Excel..."
- Tabela dinâmica salva causou "ProtectionView" quando aberta no MS Excel
- Usando validação de lista por Aspose.Cellscreates um arquivo XLS que não funciona após alterar o listseparator do sistema
