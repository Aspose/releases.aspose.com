Nós somos
 tenho o prazer de anunciar Aspose.Cells for Java v7.3.4!

 Novas características

- Suporte para formato TIFF no recurso Sheet-to-Image

 Melhorias

- O rodapé direito não está acima do rodapé central como no MS Excel

 Exceções

- Exceção de falta de memória ao converter Excel para PDF
- Definir "+100" como fórmula condicional causou uma exceção
- Exceção: “StackOverflowError” ao calcular fórmulas
- “IllegalArgumentException” é lançada quandoWorkbook.copy() é chamado

Insetos

- O texto em árabe tornou-se caracteres inúteis no arquivo CSV salvo com UTF-8
- Erro "Os dados podem ter sido perdidos" para o arquivo XLS salvo novamente por Aspose.Cells
- "Excel encontrou conteúdo ilegível….."erro para relatório gerado Aspose.Cells'
- Cell.getFormula() não distinguiu intervalos com nomes diferentes com o mesmo nome, mas escopo diferente
- Título automático para emissão de gráfico PIE
- Cell.getR1C1Formula() não distinguiu intervalos nomeados diferentes com o mesmo nome, mas escopo diferente
