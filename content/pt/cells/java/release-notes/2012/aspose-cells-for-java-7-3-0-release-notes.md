Nós somos
 tenho o prazer de anunciar Aspose.Cells for Java v7.3.0!

 O que há de novo

- Ler
 e escrever arquivos MHT
- Apoia
 Mapas XML
- Aplicar
 Temas do MS Excel 2007/2010 para gráficos



 Outros recursos, melhorias e correções notáveis são fornecidos abaixo.

 Novas características

 – Suporta configurações para avaliação de fórmula de transição

 Melhorias

- Formate os valores das células com o local especificado pelo usuário

 Exceções

- Arquivos de fonte não suportados podem fazer com que o salvamento em PDF falhe com exceção

Insetos

- O método Cell.setR1C1Formula() não funcionou corretamente sem deslocamento de coluna
- O intervalo nomeado foi criado duas vezes
- O método sortNames() causou exceção ao salvar um arquivo XLSM
- O número não foi formatado corretamente para fração
- Em branco PDF gerado paraPrintOrderType.OVER_THEN_DOWN
- cor de fundo e as grades estão incorretas no PDF gerado
- As funções de interceptação e inclinação não foram calculadas corretamente
- Remova o limite de 33k de itens de campo dinâmico para formatos de arquivo do Excel 2007
- A notação 1:1 não é suportada na função IF
- A fórmula DATEDIF foi calculada incorretamente
- Pesquisa incorreta de células no caso de linha nomeada
