 Esta página contém notas de lançamento para[Aspose.Cells for Java 7.2.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.1/)

Nós somos
 tenho o prazer de anunciar Aspose.Cells for Java v7.2.1!

 Novas características

- Formate a tabela dinâmica com estilos personalizados para diferentes categorias (Modificar estilo rápido da tabela dinâmica)

 Melhorias

- Cells.findString()/find() suporta pesquisa RegExand em intervalo específico
- Suporte Picture.setTitle()/getTitle()
- Salve gráficos do MS Excel no arquivo ODS
- Torne o arquivo Aspose.Cells criado XLS compatível com POI

 Exceções

- A leitura do arquivo XLSX produz: “java.lang.ClassCastException:org.dom4j.Namespace”

Insetos

- Arquivo salvo XLSX apresenta erro: "Datamay foi perdido"
- O número formatado estava incorreto no gerado PDF (milhares de caracteres do grupo foram perdidos)
- O gráfico de barras não estava aparecendo no PDF gerado para a versão JDK6
- As referências não são atualizadas ao expandir um intervalo
