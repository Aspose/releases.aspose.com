 Esta página contém notas de lançamento para[Aspose.Cells for Java 7.0.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.0.1/)

{{% /alert %}} 

 Temos o prazer de anunciar Aspose.Cells for Java v7.0.1!

 Melhorias

 30556 Melhorar o recurso Planilha para Imagem

30557 Aprimorar conversão de Excel para PDF

 30458 Adicionar novas APIs: Worksheet.calculateFormula(), Row.getFirstCell(), Row.getLastCell()

 30487 Suporte Aspose.Total licença

 29838 Renderizar imagem para gráficos personalizados do usuário

 30558 Inclui método FormatCondition.getPriority()

 30505 Inclui método Cells.checkColumn(int)

 30504 Inclui getters/setters para atributos Cells.StandardHeightPixels/StandardWidthPixels

 30459 Incluir métodos WorkbookSettings.setLocale(Locale) e Protection.getPasswordHash()

 30469 Inclui getters/setters para WorkbookSettings.WindowHeightInch/WindowWidthInch

 Exceções

 29285 Obtenha uma exceção ao salvar PDF com localidade francesa

 30459 Obtenha uma exceção ao ler o arquivo XLS com a localidade padrão que não possui código de país

 30489 Worksheet.autoFitColumn() e Chart.toImage() podem causar “java.awt.HeadlessException”

 30587 Obtém uma exceção ao ler o arquivo de modelo XLS

Insetos

 30103 As fontes do gráfico foram alteradas após salvar novamente o arquivo XLSX

 28858 A classificação de nomes faz com que a célula vinculada do controle seja alterada

 30486 O alinhamento vertical foi alterado no arquivo XLSX salvo novamente
