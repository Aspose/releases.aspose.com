{{% alert color="primary" %}} 

 Esta página contém notas de lançamento para[Aspose.Cells for Java 7.7.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.7.2/)

{{% /alert %}} 

 Aspose.Cells for Java foi atualizado para a versão 7.7.2 e temos o prazer de anunciar que esta versão traz a adição de cerca de 20 novas melhorias úteis.
Usando Aspose.Cells for Java Você pode trabalhar com XLS, SpreadsheetML, ooxml, XLSB, CSV, HTML, ODS, PDF, Você também pode gerar, modificar, converter, renderizar e imprimir pastas de trabalho sem usar o Microsoft Excel.
Visite a documentação para saber como começar a usar Aspose.Cells for Java.
Observe que este download contém uma versão totalmente funcional do produto; no entanto, sem um conjunto de licenças, ele será executado em modo de avaliação com algumas limitações. Para testar o Aspose.Cells sem essas limitações de avaliação você pode solicitar uma licença temporária gratuita de 30 dias.
A seguir está uma lista de alterações nesta versão de Aspose.Cells for Java.

Principais características
Suporte para abertura de arquivos MS Excel 2013 protegidos por senha

Outras melhorias e mudanças

Novas características

(CELLSJAVA-40475) – Imprimir comentários em um documento Excel
(CELLSJAVA-40732) - Suporte para configuração de alinhamento distribuído justificado

Melhorias

(CELLSJAVA-40719) – Remover uso de código rígido de com.sun.image.codec.jpeg.JPEGCodec
(CELLSJAVA-40708) – Reexibir linha com altura atribuída anteriormente

Insetos

(CELLSJAVA-40711) - Axis.setTickLabelSpacing() não está funcionando para a versão XLS
(CELLSJAVA-40714) - Cell bordas foram renderizadas em células opacas
(CELLSJAVA-40710) - Menor que igual ao erro de cálculo
(CELLSJAVA-40715) – Problema com análise de curinga no cálculo de fórmula
(CELLSJAVA-40707) – Algumas colunas são estreitas para conter o texto corretamente
(CELLSJAVA-40720) - IStreamProvider.setCustomPath não afeta o arquivo tabstrip.htm
(CELLSJAVA-40729) – Extraindo problema de data chinesa formatada
(CELLSJAVA-40728) - getStringValue() retorna #VALUE!
(CELLSJAVA-40722) – A validação não funciona após a conversão para o formato XLS
(CELLSJAVA-40690) - XLSX a PDF: O sublinhado abaixo do hiperlink está se espalhando para posições de caracteres adjacentes

Exceções

(CELLSJAVA-40629) – Exceção ao abrir xlsx convertido para xls
(CELLSJAVA-40630) – Exceção ao abrir xlsx convertido para xls
(CELLSJAVA-40717) – CellsException: nome de célula inválido no construtor Workbook
(CELLSJAVA-40723) - Renderização causada por com.aspose.cells.CellsException: java.lang.String não pode ser convertido em java.lang.Double
(CELLSJAVA-40724) - com.aspose.cells.CellsException em SheetRender
(CELLSJAVA-40727) – Exceção: java.lang.NullPointerException na renderização de gráficos em uma planilha

Observação
Como a base de código Aspose.Cells for Java corresponde ao código da versão .NET relevante, a maioria das alterações, aprimoramentos e correções incluídas no Aspose.Cells for .NET v7.7.2 também estão incluídas neste Aspose.Cells for Java v7.7.2.
