 Queridos clientes,

 Lançamos Aspose.Cells for Java V1.9.0!

 O que há de novo?

- Suporta exportação para arquivo xml Aspose.Pdf
 Suporta copiar comentários e intervalos nomeados ao copiar planilha
 Suporta interface ICustomFunction de cálculo de fórmulas
 Suporta rastrear predentes e dependentes de uma célula

 O que melhorou?

- Suporta apenas calcular todas as fórmulas em uma planilha
 Suporta POJO e POJOs com marcadores inteligentes

 O que foi consertado?

- Corrigido um bug ao obter intervalos nomeados da planilha
 Corrigido um bug na criação da Tabela Dinâmica
 Corrigido um bug de proteção de arquivo
 Corrigido um bug de cálculo de fórmulas
 Corrigido um bug ao salvar o arquivo.
