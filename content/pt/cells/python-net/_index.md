### Planilha Python via .NET Recursos de API no local

- Geração e manipulação de planilhas via API.
- Conversão e renderização de formato de arquivo de alta qualidade.
- Imprima arquivos do Microsoft Excel® em impressoras físicas ou virtuais.
- Combine, modifique, proteja ou analise planilhas do Excel®.
- Aplicar formatação de planilha.
- Configure e aplique a configuração de página para as planilhas.
- Crie e personalize gráficos do Excel®, tabelas dinâmicas, condicionais
  regras de formatação, slicers, tabelas e spark-lines.
- Converta gráficos do Excel® em imagens e PDF.
- Converta arquivos Excel® para vários outros formatos.
- Motor de cálculo de fórmulas que suporta todas as funções básicas e avançadas do Excel®.

Visite o [official documentation](https://docs.aspose.com/cells/python-net/) para obter uma lista mais detalhada de recursos.

### Ler e gravar formatos de arquivo de planilha

**Microsoft Excel®:** XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM, CSV, TSV, TabDelimited, SpreadsheetML\
**OpenOffice:** ODS, SXC, FODS\
**Texto:** TXT\
**Web:** HTML, MHTML\
**iWork®:** Números\
**Outros:** SXC, FODS

### Salvar arquivos de planilha AS

**Microsoft Word®:** DOCX\
**Microsoft PowerPoint®:** PPTX\
**Microsoft Excel®:** XLAM\
**Layout fixo:** PDF, XPS\
**Intercâmbio de dados:** DIF\
**Gráficos vetoriais:** SVG\
**Imagem:** TIFF,PNG, BMP, JPEG, GIF\
**Metaarquivo:** EMF\
**Remarcação:** MD

Visite [Supported File Formats](https://docs.aspose.com/cells/python-net/supported-file-formats/) para obter mais detalhes.

### Requisitos de sistema

Sua máquina não precisa ter o software Microsoft Excel® ou OpenOffice® instalado.

### Sistemas operacionais suportados

**Microsoft Windows®:** Windows Desktop & Server (`x64`, `x86`)\
**Linux:** Ubuntu, OpenSUSE, CentOS e outros\
**Outros:** Qualquer sistema operacional (SO) que possa instalar o Mono (suporte a .NET 4.0 Framework) ou usar o .NET core.
