Temos o prazer de anunciar o primeiro lançamento público de Aspose.Cells for PHP via Java. As APIs incorporam funcionalidades de processamento de planilhas do MS Excel em PHP (uma linguagem de script de código aberto amplamente utilizada). Aspose.Cells for PHP via Java é um componente de subconjunto que inclui todos os recursos importantes e úteis presentes em seu nativo[Aspose.Cells for Java](https://products.aspose.com/cells/java/). O novo API é igualmente robusto e rico em recursos. Ele suporta conversões de formato de arquivo de alta fidelidade de e para XLS, XLSX, XLSM, SpreadsheetML, CSV, delimitado por tabulação, HTML, MHTML e planilha OpenDocument em PHP. Por favor visite[Página de comparação de recursos do produto](https://docs.aspose.com/cells/java/aspose-cells-for-php-via-java-features/)para sua referência. Além disso, consulte o documento sobre como[Ambiente de configuração e outras diretrizes de instalação](https://docs.aspose.com/cells/java/setup-and-installation-guidelines/)para sua referência.
