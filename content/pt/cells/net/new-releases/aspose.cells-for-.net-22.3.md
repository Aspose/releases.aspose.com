#  Suporte para formatos de modelo de planilha

Esta versão API introduziu suporte para formatos de modelo de planilha para Excel&reg; (XLT) & OpenOffice Calc&reg; (OTS). Eles são suportados como formatos de saída por meio de `SaveFormat.Ots` e `SaveFormat.Xlt`.

#  Definir bordas internas

Capacidade de definir as bordas internas.

#  Classificar campo dinâmico por linha ou coluna selecionada

O suporte de classificação para valores de campo dinâmico foi adicionado. A classificação do campo dinâmico pode ser feita com base na linha ou coluna desejada.
