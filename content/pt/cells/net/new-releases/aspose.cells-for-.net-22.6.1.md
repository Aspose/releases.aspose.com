#  `.NUMBERS` Suporte para armazenamento de dados 5.0

Adicionado suporte para armazenamento de dados 5.0 de `.numbers`.

#  Renderização de XLSX a PDF aprimorada

Foi observado que durante a conversão de XLSX em PDF, os gráficos de barras não eram renderizados corretamente. Este problema já foi resolvido.

#  Excel® aprimorado; Tratamento de rótulos de gráfico

Esta versão API oferece algumas correções relacionadas ao Excel®. rótulos do gráfico. São os seguintes:

- Os rótulos do gráfico não cabiam na caixa.
- Os rótulos dos gráficos estavam sendo alinhados verticalmente em vez de horizontalmente.

#  Salvar fórmulas de tabela em XLSB

Foi adicionado suporte para salvar fórmulas de tabela em `XLSB`.

#  Suporte aprimorado no Linux

Observou-se que caracteres ou legendas escritas em objetos como formas e caixas de texto apareciam distorcidas no Linux. Este problema já foi resolvido.
