 Queridos clientes,

 Lançamos Aspose.Cells for .NET V4.0!

- Adicionadas novas demonstrações de gráficos
- Corrigido um bug na configuração da fórmula
- Corrigido um bug ao salvar arquivo em stream com formato CSV
