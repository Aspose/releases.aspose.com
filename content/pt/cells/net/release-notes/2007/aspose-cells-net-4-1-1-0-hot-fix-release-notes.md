 Queridos clientes,

 Lançamos Aspose.Cells for .NET v4.1.1!

- Demonstrações aprimoradas do WebForm
- F0007: Função VPL suportada no mecanismo de cálculo de fórmulas
- F0009: Suportado para leitura de strings de caixas de texto em arquivos de template
- F0015: Criptografia BIFF7 suportada em alguns arquivos protegidos
- F0016: Função INDIRETO, ISBLANK, ISERROR, ISNA, ISNUMBER suportada no mecanismo de cálculo de fórmula
- F0017: String de formato de data e hora em alemão suportada
- F0019: Suportado para definir célula ativa em planilha
- B0001: Corrigido um bug na propriedade Cells.MaxDataRowInColumn
- B0002: Corrigido um bug ao carregar arquivos csv com strings em branco
- B0003: Corrigido um bug ao carregar comentários com mais de 8k de conteúdo de string
- B0004: Corrigido um bug para carregar um arquivo com proteção de pasta de trabalho
- B0005: Corrigido um bug no método InsertRows que pode causar o erro "Fórmula inválida..." para alguns arquivos
- B0006: Corrigido um bug de distorção de imagem ao executar com resolução de tela de 120 DPI.
- B0007: Corrigido um bug no método CalculaFormula ao comparar dois valores DateTime
