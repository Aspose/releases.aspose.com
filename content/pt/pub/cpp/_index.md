## Por que baixar o Aspose.PUB para C++?

Aspose.PUB para C++ é uma API simples que permite que você leia e converta arquivos do Microsoft Publisher® (PUB) para o formato PDF programaticamente em seus aplicativos C++. Ele também fornece interfaces fáceis de entender para editar os metadados de arquivos PUB.

## Recursos de processamento de arquivos PUB

- Leia arquivos do Microsoft Publisher® (PUB) para conversão para o formato PDF.
- Leia e edite metadados de arquivos PUB via API.

### Ler arquivos PUB

**Microsoft Publisher®:** PUB

### Salvar PUB como

**Layout fixo:** PDF

### Independência de plataforma

Você pode usar o Aspose.PUB para C++ para construir qualquer tipo de aplicativos C++ de 32 bits e 64 bits. Você pode usá-lo no servidor e no cliente simplesmente copiando o assembly sem se preocupar com outros serviços ou módulos.
