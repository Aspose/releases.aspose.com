## Por que baixar o Aspose.PUB para .NET?

Aspose.PUB para .NET é uma API simples que permite ler e converter arquivos do Microsoft Publisher® (PUB) para o formato PDF. Ele também fornece interfaces fáceis de entender para editar os metadados de arquivos PUB.

## Recursos de processamento de arquivos PUB

- Leia arquivos do Microsoft Publisher® (PUB) para conversão para o formato PDF.
- Ler e gravar metadados de arquivos PUB via API.
