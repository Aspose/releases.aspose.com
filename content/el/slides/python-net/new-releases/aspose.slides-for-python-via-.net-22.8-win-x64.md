# Περιεκτική υποστήριξη για ρυθμίσεις διαφανειών

Αυτή η απελευθέρωση API εισάγει τη λεπτομερή υποστήριξη για τις ρυθμίσεις εμφάνισης διαφανειών παρουσίασης. Ορισμένες από τις νέες κατηγορίες και ιδιότητες έχουν ως εξής:

- [`Παρουσίαση`] (https://reference.aspose.com/slides/python-net/aspose.slides/presentation/).
- [`slideshowsettings`] (https://reference.aspose.com/slides/python-net/aspose.slides/slideshowsettings/): αντιπροσωπεύει τις ρυθμίσεις slide show για την παρουσίαση. Ορισμένες από τις διαθέσιμες ιδιότητες είναι:
  - Κάντε την παρουσίαση σε ένα βρόχο
  - Δυνατότητα εμφάνισης / απόκρυψης της αφήγησης slideshow
  - Ενεργοποίηση / απενεργοποίηση κινούμενων σχεδίων στην παρουσίαση
  - Ρυθμίστε το χρώμα της στυλό για την παρουσίαση διαφανειών

!

> Για μια πλήρη λίστα χαρακτηριστικών, βελτιώσεων και διορθώσεων σφαλμάτων σε αυτή την έκδοση, παρακαλώ επισκεφθείτε, [aspose.slides for python 22.8 σημειώσεις απελευθέρωσης] (https://docs.aspose.com/slides/python-net/aspose-slides- για-python-net-22-8-απελευθέρωση-notes/).
