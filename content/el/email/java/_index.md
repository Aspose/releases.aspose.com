Το [Aspose.Email for Java](https://products.aspose.com/email/java) είναι ένα πλήρες σύνολο API επεξεργασίας ηλεκτρονικού ταχυδρομείου για να δημιουργήσετε, να διαβάσετε και να χειριστείτε τα μηνύματα ηλεκτρονικού ταχυδρομείου από τις εφαρμογές σας. Διευκολύνει την εργασία με πολλές μορφές μηνυμάτων ηλεκτρονικού ταχυδρομείου προοπτικών, όπως αρχεία MSG, EML, EMLX και MHT χωρίς την ανάγκη εγκατάστασης του Microsoft Outlook. Σας δίνει επίσης τη δυνατότητα να διαχειριστείτε αρχεία αποθήκευσης μηνυμάτων - Αρχεία προσωπικών αποθήκευσης (PST), αρχεία αποθήκευσης εκτός σύνδεσης (OST) μαζί με δυνατότητες αποστολής και λήψης μηνυμάτων. Μπορείτε επίσης να διαβάσετε και να εξαγάγετε αρχείο PST Outlook που μπορεί να αποθηκευτεί στο δίσκο σε μορφή MSG.

## Χαρακτηριστικά API Email API
- Δημιουργήστε μηνύματα από το μηδέν ή φορτώστε τα υπάρχοντα αρχεία email για επεξεργασία.
- Δημιουργήστε και ορίστε περιεχόμενα μηνυμάτων MIME.
- Εξαγάγετε περιεχόμενα από μηνύματα ηλεκτρονικού ταχυδρομείου.

- Φορτώστε και αποθηκεύστε το [ραντεβού σε μορφή ICS](https://docs.aspose.com/email/java/working-with-appointments/).
- Δυνατότητα σύνδεσης σε SMTP, POP3, IMAP, Exchange Server.
- Λειτουργεί με τις σημειώσεις Thunderbird, Zimbra και IBM.

## Διαβάστε και γράψτε μορφές email
** Microsoft Outlook: ** MSG, PST, OST, OFT \
** Email: ** EML, EMLX, MBOX \
** Άλλοι: ** ICS, VCF, HTML, MHTML

## Διαβάστε μορφές email

Για τις λεπτομερείς οδηγίες, επισκεφτείτε τη σελίδα τεκμηρίωσης [Installing Aspose.Email for Java from Maven Repository](https://docs.aspose.com/email/java/installation/).

## Εκτελέστε λειτουργία αντιγράφων ασφαλείας μηνυμάτων IMAP χρησιμοποιώντας Java

```java
ImapClient imapClient = new ImapClient();
imapClient.setHost("<HOST>");
imapClient.setPort(993);
imapClient.setUsername("<USERNAME>");
imapClient.setPassword("<PASSWORD>");
imapClient.setSupportedEncryption(EncryptionProtocols.Tls);
imapClient.setSecurityOptions(SecurityOptions.SSLImplicit);

ImapMailboxInfo mailboxInfo = imapClient.getMailboxInfo();

ImapFolderInfo info = imapClient.getFolderInfo(mailboxInfo.getInbox().getName());
ImapFolderInfoCollection infos = new ImapFolderInfoCollection();
infos.add(info);

imapClient.backup(infos, dataDir + "\\ImapBackup.pst", BackupOptions.None);
```
