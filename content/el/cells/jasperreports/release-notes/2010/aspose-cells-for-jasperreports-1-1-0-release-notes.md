 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for JasperReports 1.1.0](https://releases.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.1.0/)

{{% /alert %}} 

 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε το Aspose.Cells for JasperReports v1.1.0 για τους χρήστες! Τι νέο υπάρχει:

- Υποστηρίζει μορφές Microsoft Excel 97 έως 2003 και Microsoft Excel 2007. Περισσότερες λεπτομέρειες θα βρείτε στο έγγραφο «Παράμετροι διαμόρφωσης» στην τεκμηρίωση του προϊόντος.
- Διορθώνει πολλά σφάλματα υπολογισμού στο μέγεθος του στοιχείου αναφοράς.
- Υποστηρίζει JasperReports 3.7.x.
