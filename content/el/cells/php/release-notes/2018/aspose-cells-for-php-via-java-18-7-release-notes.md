Είμαστε στην ευχάριστη θέση να ανακοινώσουμε την πρώτη δημόσια έκδοση του Aspose.Cells for PHP via Java. Τα API ενσωματώνουν λειτουργίες επεξεργασίας υπολογιστικών φύλλων MS Excel στην PHP (μια ευρέως χρησιμοποιούμενη γλώσσα προγραμματισμού ανοιχτού κώδικα). Aspose.Cells for PHP via Java είναι ένα στοιχείο υποσυνόλου που περιλαμβάνει όλα τα σημαντικά και χρήσιμα χαρακτηριστικά που υπάρχουν στην αρχική του[Aspose.Cells for Java](https://products.aspose.com/cells/java/). Το νέο API είναι εξίσου στιβαρό και πλούσιο σε χαρακτηριστικά. Υποστηρίζει μετατροπές μορφών αρχείου υψηλής πιστότητας προς και από XLS, XLSX, XLSM, SpreadsheetML, CSV, Tab Delimited, HTML, MHTML, MHTML. Παρακαλούμε επισκεφθείτε[Σελίδα σύγκρισης χαρακτηριστικών προϊόντων](https://docs.aspose.com/cells/java/aspose-cells-for-php-via-java-features/)για την αναφορά σας. Επιπλέον, δείτε το έγγραφο σχετικά με τον τρόπο[Περιβάλλον εγκατάστασης και άλλες Οδηγίες εγκατάστασης](https://docs.aspose.com/cells/java/setup-and-installation-guidelines/)για την αναφορά σας.
