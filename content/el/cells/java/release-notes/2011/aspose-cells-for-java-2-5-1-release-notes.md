 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 2.5.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.5.1/)

{{% /alert %}} 

 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε Aspose.Cells for Java 2.5.1!

 Τι έχει αλλάξει:

 · Υποστηρίζει τη διακοπή της διαδικασίας ανοίγματος/αποθήκευσης κατά την εργασία με μεγάλα αρχεία.

 · Παρέχει υποστήριξη για τον ορισμό και τον υπολογισμό περισσότερων τύπων.

 · Υποστηρίζει για λήψη/ρυθμισμό της απόλυτης θέσης του σχήματος.

 · Διορθώνει σφάλματα ανάγνωσης/αποθήκευσης αρχείων Excel 2007.

· Διορθώνει σφάλματα ανάγνωσης/αποθήκευσης ODS αρχείων.

 · Διορθώνει σφάλματα ανάγνωσης/αποθήκευσης αρχείων Excel 97-2003.

 · Η βελτίωση γίνεται για τις λειτουργίες Chart to Image, Sheet to Image και Excel to PDF.

 · Γίνεται βελτίωση για την ανάλυση τύπων.

 · Γίνεται βελτίωση για τον υπολογισμό των τύπων.

 · 22 διορθώσεις και βελτιώσεις.
