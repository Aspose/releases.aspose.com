Ακολουθεί μια λίστα με βελτιώσεις και αλλαγές σε αυτήν την έκδοση του Aspose.Cells

##  Aspose.Cells

###  **Κύρια χαρακτηριστικά**

Η κύρια βάση κωδικών μεταφέρθηκε στο Java 6 (υποστηρίζονται επίσης τα Java 7 και 8). Διακοπή υποστήριξης for Java 5 και 1.4.

###  **Άλλες Βελτιώσεις και Αλλαγές**

###  **Νέα χαρακτηριστικά**

(CELLSJAVA-41235) - Υποστήριξη RenderToSize API για εικόνα φύλλου εργασίας

(CELLSJAVA-41234) - Υποστήριξη Κουκκίδων κατά τη χρήση SmartMarkers ή μεθόδου Cell.setHtmlString

###  **Σφάλματα**

(CELLSJAVA-41229) - Aspose.Cells δεν δημιουργεί μεμονωμένα αρχεία HTM και CSS για τα φύλλα στο Excel σε μετατροπή HTML

(CELLSJAVA-41170) - Το SheetRender.toImage αποδίδει την εικόνα με ετικέτες "(κενό)" στον άξονα x του γραφήματος

(CELLSJAVA-41270) - Πρόβλημα με Cells.insertRange() καθώς η συγχωνευμένη περιοχή δεν μετατοπίζεται σωστά

(CELLSJAVA-41240) - Το κείμενο με γραμματοσειρά Arial περικόπτεται από την κορυφή κατά την απόδοση του υπολογιστικού φύλλου στο PDF

(CELLSJAVA-41238) - Το PAPER_A_2 δεν λειτουργεί όπως αναμένεται κατά την αποθήκευση ως PDF

(CELLSJAVA-41217) - Όταν τα δεδομένα κατηγορίας σειράς έχουν κόμμα, τότε οι μύθοι του γραφήματος PIE δεν εμφανίζονται σωστά

(CELLSJAVA-41194) - Επικάλυψη των καταχωρήσεων Legend κατά τη μετατροπή του γραφήματος σε εικόνα

(CELLSJAVA-41002) - Η διακεκομμένη γραμμή λείπει στο Διάγραμμα 1

(CELLSJAVA-40993) - Οι οριζόντιες γραμμές πλέγματος λείπουν στο γράφημα ανάπτυξης

(CELLSJAVA-41259) - Η ρύθμιση του Name.setRefersTo και ο εκ νέου υπολογισμός των τύπων έχουν ως αποτέλεσμα λανθασμένη τιμή κατά τη μετατροπή του υπολογιστικού φύλλου σε HTML

(CELLSJAVA-41258) - Η φόρτωση και η αποθήκευση του XLSX με Aspose.Cells καθιστούν το υπολογιστικό φύλλο που προκύπτει κατεστραμμένο

(CELLSJAVA-41255) - Το προσαρμοσμένο κουμπί γίνεται εικόνα και η λεζάντα εξαφανίζεται στην έξοδο XLSX

(CELLSJAVA-41254) - Microsoft Το Excel διακόπτεται όταν ανοίγει το αρχείο εξόδου XLSX

(CELLSJAVA-41253) - Το αναπτυσσόμενο μενού εξαφανίζεται στο αρχείο εξόδου XLSX

###  **Εξαιρέσεις**

(CELLSJAVA-41266) - java.lang.NumberFormatException συνέβη κατά το άνοιγμα του αρχείου προτύπου XLSX

(CELLSJAVA-41248) - Εξαίρεση μηδενικού δείκτη στο αρχείο πηγής ανοίγματος XLSX

(CELLSJAVA-41265) - Εξαίρεση: "java.lang.NullPointerException" κατά το άνοιγμα ενός αρχείου SpreadsheetML

(CELLSJAVA-41264) - Εξαίρεση κατά τη χρήση Cell.getStringValueWithoutFormat

(CELLSJAVA-41246) - Εξαίρεση: Μη έγκυρος δυναμικός τύπος... που περιλαμβάνει συνάρτηση ευρετηρίου κατά τη χρήση δυναμικών τύπων Smart Markers

##  **Δημόσιο API και Μη συμβατές αλλαγές προς τα πίσω**

Ακολουθεί μια λίστα με τυχόν αλλαγές που έγιναν στο κοινό API, όπως προστέθηκαν, μετονομάστηκαν, αφαιρέθηκαν ή καταργήθηκαν μέλη, καθώς και οποιαδήποτε μη συμβατή αλλαγή που έγινε στο Aspose.Cells for Java. Εάν έχετε ανησυχίες σχετικά με οποιαδήποτε αλλαγή που παρατίθεται, θέστε την στο το φόρουμ υποστήριξης Aspose.Cells.

Προσθέτει την ιδιοκτησία Workbook.IsLicensed.

Υποδεικνύει εάν η άδεια έχει οριστεί.

Απαρχαιωμένο βιβλίο εργασίας.Μέθοδος ValidateFormula.

Χρησιμοποιήστε αντί αυτού την ιδιότητα Cell.Formula.

Προσθέτει την ιδιότητα ImageOrPrintOptions.SVGFitToViewPort.

Υποδεικνύει εάν η εικόνα SVG που δημιουργήθηκε είναι κατάλληλη για προβολή θύρας.

Προσθέτει τη μέθοδο ImageOrPrintOptions.SetDesiredSize.

Ορίζει το επιθυμητό πλάτος και ύψος της εικόνας.

Προσθέτει τη μέθοδο Aspose.Cells.GridDesktop.WorksheetCollection.MoveTo

Μετακινεί το φύλλο εργασίας στο καθορισμένο ευρετήριο σε άλλο ευρετήριο.
