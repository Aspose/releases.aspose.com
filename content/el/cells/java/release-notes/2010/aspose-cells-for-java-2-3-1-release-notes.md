 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 2.3.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.3.1/)

{{% /alert %}} 

 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε Aspose.Cells for Java 2.3.1!

 Τι έχει αλλάξει:

- Παρέχει υποστήριξη για τύπο κρυπτογράφησης AES για αρχεία Excel 2007.
- Υποστηρίζει την ανάγνωση/εγγραφή Συγκεντρωτικών Πινάκων για αρχεία Excel 2007.
- Προσθέτει υποστήριξη μορφής εικόνας svg/svgz για τη λειτουργία Chart to Image.
- Υποστηρίζει πολλαπλές περιοχές εκτύπωσης για επιλογές PageSetup.
- Ορίζει προσαρμοσμένες μορφές αριθμών για το PivotField.
- Περιλαμβάνει νέες ενσωματωμένες ιδιότητες εγγράφου για αρχείο Excel 2007.
- Συνδυάζει πολλά βιβλία εργασίας.
- Υποστηρίζει την αποθήκευση PDF στη ροή για την άμεση δυνατότητα Excel σε PDF.
- Έγινε μια βελτίωση για την ανάγνωση/εγγραφή αρχείων Excel 2007.
- Έγινε μια βελτίωση για την ανάγνωση/εγγραφή αρχείων Excel 2003.
- Έγινε μια βελτίωση για την ανάγνωση αρχείων προτύπων XLS.
- 25 διορθώσεις και άλλες βελτιώσεις.
