 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε Aspose.Cells for Java 2.2.1!

 Τι έχει αλλάξει:

-  Περιλαμβάνει υποστήριξη για μορφοποίηση για προχωρημένους υπό όρους που παρέχεται από το MS Excel 2007
- Υποστηρίζει εκτεταμένες εγγραφές μορφοποίησης υπό όρους που αποθηκεύτηκαν από το MS Excel 2007 για το αρχείο XLS
-  Διατηρεί μη υποστηριζόμενους τύπους κατά την ανάγνωση αρχείων MS Excel 2007
-  Διαβάζει εικόνες από την Κεφαλίδα/Υποσέλιδο στο αρχείο προτύπου MS Excel 2007
-  Γίνονται βελτιώσεις για την ανάγνωση αρχείων προτύπων
-  17 διορθώσεις και άλλες βελτιώσεις.
