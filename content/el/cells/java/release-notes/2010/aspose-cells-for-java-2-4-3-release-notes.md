 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 2.4.3](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.3/)

{{% /alert %}} 

 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε Aspose.Cells for Java 2.4.3!

 Τι έχει αλλάξει:

- Υποστηρίζει τη λήψη χρωμάτων θέματος Excel 2007.
- Εξάγει γραφήματα/σχήματα ως εικόνες για μετατροπή αρχείου HTML.
- Υποστηρίζει την ανάγνωση/εγγραφή διαφορετικών κεφαλίδων/υποσελίδων για πρώτες/ζυγές/μονές σελίδες.
- Διορθώνει ορισμένα εσωτερικά σφάλματα ανάγνωσης/αποθήκευσης αρχείων Excel 2007.
- Διορθώνει ορισμένα εσωτερικά σφάλματα ανάγνωσης/αποθήκευσης αρχείων ODS.
- Διορθώνει ορισμένα εσωτερικά σφάλματα ανάγνωσης/αποθήκευσης αρχείων Excel 97-2003.
- Η βελτίωση έχει γίνει για τη δυνατότητα Chart to Image.
- Η βελτίωση πραγματοποιείται για τη δυνατότητα Direct Excel σε PDF.
- 14 διορθώσεις και βελτιώσεις.
