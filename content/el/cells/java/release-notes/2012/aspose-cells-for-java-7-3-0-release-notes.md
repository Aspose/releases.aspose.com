 Τι νέα

 και Γράψτε αρχεία MHT
- Υποστηρίζει
 Χάρτες XML
- Ισχύουν
 Θέματα MS Excel 2007/2010 σε γραφήματα



 Άλλα αξιοσημείωτα χαρακτηριστικά, βελτιώσεις και διορθώσεις δίνονται παρακάτω.

 Νέα χαρακτηριστικά

 – Υποστηρίζει ρυθμίσεις για την αξιολόγηση τύπων μετάβασης

 Βελτιώσεις

- Μορφοποιήστε τις τιμές των κελιών με το user specifiedlocale

 Εξαιρέσεις

- Τα μη υποστηριζόμενα αρχεία γραμματοσειράς ενδέχεται να προκαλέσουν αποτυχία αποθήκευσης σε αρχείο PDF με εξαίρεση

Σφάλματα

- Η μέθοδος Cell.setR1C1Formula() δεν λειτούργησε σωστά χωρίς μετατόπιση στήλης
- Η επώνυμη περιοχή δημιουργήθηκε δύο φορές
- Η μέθοδος sortNames() προκάλεσε εξαίρεση κατά την αποθήκευση του αρχείου anXLSM
- Ο αριθμός δεν είχε μορφοποιηθεί σωστά διάσπαση
- Το κενό PDF δημιουργήθηκε για τοPrintOrderType.OVER_THEN_DOWN
- Το χρώμα φόντου και τα πλέγματα είναι λανθασμένα με το PDF που δημιουργήθηκε
- Οι συναρτήσεις Intercept και Slope δεν υπολογίστηκαν σωστά
- Καταργήστε το όριο των 33 χιλιάδων στοιχείων του πεδίου Συγκεντρωτική για τις μορφές αρχείων του Excel 2007
- Το Notation 1:1 δεν υποστηρίζεται στο IFfunction
- Ο τύπος DATEDIF υπολογίστηκε εσφαλμένα
- Εσφαλμένη αναζήτηση κελιών σε περίπτωση ονομασμένης σειράς
