 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 7.3.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.3.2/)

Είμαστε
 ευχαρίστως να ανακοινώσω Aspose.Cells for Java v7.3.2!

 Νέα χαρακτηριστικά

- Shape.getRight()/getBottom() για να πάρετε μια μετατόπιση του σχήματος από την κάτω δεξιά γωνία στο κελί
- Ορίστε το χρώμα της καρτέλας του φύλλου εργασίας στο προεπιλεγμένο χρώμα

 Βελτιώσεις

- Κλείσιμο εσωτερικών ροών για την απελευθέρωση πόρων μνήμης σε μετατροπή PDF
- Αντιγραφή Sparklines κατά την αντιγραφή φύλλου εργασίας

 Εξαιρέσεις

- Το άνοιγμα ενός αρχείου XLS προκάλεσε Σφάλμα StackOverflow
- Η αποθήκευση του PDF προκάλεσε Εξαίρεση
- Το φύλλο εργασίας.getFreezedPanes() προκάλεσε NullPointerException
- Το άνοιγμα ενός κενού αρχείου XML προκάλεσε εξαίρεση
- Η αποθήκευση στο PDF προκάλεσε εξαίρεση -I
- Η αποθήκευση στο PDF προκάλεσε εξαίρεση -II
- Η αποθήκευση στο PDF προκάλεσε εξαίρεση -III
- Η αποθήκευση στο PDF προκάλεσε εξαίρεση -IV
- Εξαίρεση όταν είναι ανοιχτό αρχείο προτύπου HTM
- IllegalArgumentException κατά το άνοιγμα του αρχείου anXLS
- NullPointerException κατά το άνοιγμα ενός αρχείου XLS
- Εξαίρεση ArrayIndexOutOfBounds κατά το άνοιγμα ενός αρχείου XLS
- Η αποθήκευση του Συγκεντρωτικού Πίνακα προκάλεσε τοClassCastException μετά τη ρύθμιση της αυτόματης ταξινόμησης
- Σφάλμα σχήματος εικόνας κατά την αποθήκευση PDF χρησιμοποιώνταςAspose Cells
- CellsException κατά την εκτύπωση γραμμών πλέγματος σε PDF

Σφάλματα

- Ορισμένες περιοχές που είναι αποθηκευμένες στα πρότυπα XLS δεν μπορούν να αναγνωριστούν κατά την ανάγνωση
- Το διάγραμμα σε εικόνα αποτυγχάνει να αποδώσει την κλίμακα για γραφήματα ράβδων
- Απώλεια δεδομένων μετά από μετατροπή από XLSX σε PDF
- Τα γραφήματα Pie και Scatter δεν λειτουργούν εάν η Cellcollection διαγραφεί
