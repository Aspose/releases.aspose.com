 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 7.1.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.1/)

{{% /alert %}} 

Είμαστε
 ευχαρίστως να ανακοινώσω Aspose.Cells for Java v7.1.1!

 Νέα χαρακτηριστικά

- Ελέγξτε εάν μια στήλη είναι κενή

 Βελτιώσεις

- Αποθήκευση στο XLSX με LightCells API με απευθείας απόδοση στο OutputStream

 Εξαιρέσεις

- Η μέθοδος Cell.setHtmlString() ρίχνει το NullPointerException
- Το αρχείο προτύπου Readingan XLSX ρίχνει το NullPointerException

Σφάλματα

- Ο άξονας δευτερεύουσας κατηγορίας του γραφήματος SCATTER γίνεται άξονας κειμένου
- Τα περιεχόμενα στο πλαίσιο κειμένου δεν έχουν αποδοθεί σωστά
- Η συνάρτηση ROUND δεν μπορεί να στρογγυλοποιήσει μια τιμή μεγαλύτερη από 922337,20
- Το γράφημα δεν διαβάστηκε σωστά από το αρχείο ODStemplate
- Μορφή προσαρμοσμένου αριθμού: "ηη.ΜΜ.εεεε" δεν αποθηκεύτηκε σωστά για το αρχείο ODS
- Το LineShape γίνεται μικρότερο στο παραγόμενο έγγραφο PDF
