 Είμαστε στην ευχάριστη θέση να ανακοινώσουμε το Aspose.Cells for Java v7.1.0!

 Βελτιώσεις

- Διαγραφή αρχείων προτύπου μετά την ανάγνωση αρχείων XLSX/ODS με το εργαλείο zip του Apache

 Εξαιρέσεις

 -Cell.setHtmlString() ρίχνει NullPointerException

Σφάλματα

 -Cell τα περιεχόμενα εξαφανίστηκαν από το αρχείο PDF που δημιουργήθηκε

 -Μήνυμα σφάλματος: "Ο συγκεντρωτικός πίνακας δημιουργήθηκε σε νεότερη έκδοση του Excel..."κατά το άνοιγμα του αρχείου στο MS Excel

` `- Το ζουμ του φύλλου αλλάζει κατά το άνοιγμα ενός αρχείου προτύπου XLS

 -Cell.getStringValue() δίνει λανθασμένη τιμή για διπλούς τύπους

 -Κατά την αποθήκευση στο PDF, το δεξί μέρος του κειμένου του σχήματος περικόπτεται και μέρος της εικόνας χάνεται

 -Η μορφή αριθμού χάθηκε κατά την αποθήκευση στη μορφή αρχείου ODS
