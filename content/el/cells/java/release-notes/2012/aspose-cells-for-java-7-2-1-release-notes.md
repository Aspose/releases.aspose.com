 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 7.2.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.1/)

{{% /alert %}} 

Είμαστε
 ευχαρίστως να ανακοινώσω Aspose.Cells for Java v7.2.1!

 Νέα χαρακτηριστικά

- Μορφοποίηση Συγκεντρωτικού Πίνακα με προσαρμοσμένα στυλ για διαφορετικές κατηγορίες (Τροποποίηση του Γρήγορου Στυλ του Συγκεντρωτικού Πίνακα)

 Βελτιώσεις

- Cells.findString()/find() υποστηρίζει την αναζήτηση RegExand σε συγκεκριμένο εύρος
- Υποστήριξη Picture.setTitle()/getTitle()
- Αποθηκεύστε γραφήματα MS Excel στο αρχείο ODS
- Κάντε το αρχείο Aspose.Cells που δημιουργήθηκε XLS συμβατό με POI

 Εξαιρέσεις

- Η ανάγνωση του αρχείου XLSX παράγει: "java.lang.ClassCastException:org.dom4j.Namespace"

Σφάλματα

- Το αποθηκευμένο αρχείο XLSX δίνει σφάλμα: "Το Datamay έχει χαθεί"
- Ο μορφοποιημένος αριθμός ήταν εσφαλμένος στο PDF που δημιουργήθηκε (Χάθηκαν χιλιάδες χαρακτήρες ομάδας)
- Το γράφημα ράβδων δεν εμφανιζόταν στο δημιουργημένο PDF για την έκδοση JDK6
- Οι αναφορές δεν ενημερώνονται κατά την επέκταση ενός εύρους
