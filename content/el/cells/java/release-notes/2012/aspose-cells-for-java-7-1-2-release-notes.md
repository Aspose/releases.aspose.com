 Αυτή η σελίδα περιέχει σημειώσεις έκδοσης για[Aspose.Cells for Java 7.1.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.2/)

Είμαστε
 ευχαρίστως να ανακοινώσω Aspose.Cells for Java v7.1.2!

 Νέα χαρακτηριστικά

 – Υποστήριξη μεθόδου Cells.importCustomObjects() για εισαγωγή POJO

 Βελτιώσεις

- Γίνονται βελτιώσεις για τις μεθόδους autoFitColumns()
- Απόδοση για τη λειτουργία φύλλο σε εικόνα με ενεργοποιημένες γραμμές πλέγματος

 Εξαιρέσεις

- Παρουσιάστηκε NullPointerException κατά την κλήση της μεθόδου Cell.getDisplayStyle()

Σφάλματα

- Η μέγιστη τιμή για το γράφημα Scatter δεν μπορεί να καθοριστεί σωστά
- Το κείμενο σε σχήματα χάθηκε ή μετακινήθηκε σε εκτός εύρους σχημάτων στο αρχείο PDF που δημιουργήθηκε
- Το όνομα της σειράς δεν εμφανίστηκε για το Date Label στο γράφημα
