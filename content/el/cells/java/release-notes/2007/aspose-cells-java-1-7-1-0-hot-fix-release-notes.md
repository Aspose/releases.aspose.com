 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει Aspose.Cells for Java V1.7.1! Περιλαμβάνονται οι ακόλουθες βελτιώσεις και διορθώσεις:

- Υποστηρίζει την εισαγωγή περιεχομένου και στυλ των κελιών από το αρχείο SpreadsheetML
 Υποστηρίζει την εξαγωγή περιεχομένου και στυλ των κελιών στο αρχείο SpreadsheetML
 Υποστηρίζει τη μηχανή υπολογισμού του τύπου
 Υποστηρίζει τη ρύθμιση του τύπου R1C1
Υποστηρίζει άλλους δύο τύπους πηγών δεδομένων Συγκεντρωτικού Πίνακα
 Διορθώνει ορισμένα σφάλματα της προσθήκης Συγκεντρωτικού Πίνακα .

Σημείωση: Καταργήσαμε τη μέθοδο const CellValueType.FORMULA και προσθέσαμε τη μέθοδο Cell.isFormula() για να αποφασίσουμε εάν το κελί περιέχει τύπο.
