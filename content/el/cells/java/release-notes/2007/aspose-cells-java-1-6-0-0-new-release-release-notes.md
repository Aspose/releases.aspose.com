 Αγαπητοί πελάτες,

Έχουμε κυκλοφορήσει Aspose.Cells for Java V1.6.0! Συμπεριλαμβάνονται οι ακόλουθες βελτιώσεις και διορθώσεις:

- Υποστηρίζει Smart Marker Designer
 Υποστηρίζει την εισαγωγή εικόνας σε μορφή EMF/WMF
 Υποστηρίζει τη ρύθμιση ομαλής γραμμής στο γράφημα
 Προσθέτει τη μέθοδο Area.setVisible() για απόκρυψη/αποκάλυψη της περιοχής
 Ενημερώνει τις επιδείξεις
 Διορθώνει το σφάλμα της μεθόδου Cells.clearContent().
 Άλλες βελτιώσεις και διορθώσεις από την έκδοση 1.5

 Για κάποιους εξωτερικούς λόγους, ορισμένα ονόματα κλάσης έχουν μετονομαστεί:
