 Αγαπητοί πελάτες,

Έχουμε κυκλοφορήσει Aspose.Cells for Java V1.7.4! Περιλαμβάνονται οι ακόλουθες βελτιώσεις και διορθώσεις:

- Προσθέτει μεθόδους Column.applyStyle και Row.applyStyle.
 Διορθώνει ορισμένα σφάλματα ρύθμισης και υπολογισμού τύπων.
 Διορθώνει ένα σφάλμα της σειράς αντιμετώπισης.
 Διορθώνει ένα σφάλμα παγώματος των υαλοπινάκων.
 Διορθώνει ένα σφάλμα αποθήκευσης γραφημάτων.
 Διορθώνει ένα σφάλμα αποθήκευσης συγκεντρωτικών πινάκων.
 Διορθώνει ένα σφάλμα της μεθόδουcell.importResultSet().

Σημείωση: Η μέθοδος getNumerOfSheets of Workbook/Worksheets έχει μετονομαστεί σε getNumberOfSheets. Και είναι καλύτερο να καλέσετε το Worksheets.size() για να λάβετε τον αριθμό των φύλλων.
