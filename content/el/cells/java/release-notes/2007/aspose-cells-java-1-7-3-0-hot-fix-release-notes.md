 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει Aspose.Cells for Java V1.7.3! Περιλαμβάνονται οι ακόλουθες βελτιώσεις και διορθώσεις:

- Υποστηρίζει την εισαγωγή/διαγραφή εύρους.
 Υποστηρίζει το άνοιγμα και την αποθήκευση κρυπτογραφημένου αρχείου Excel.
 Βελτιώνει τη μέθοδο Cell.getStringValue() για την υποστήριξη περισσότερων μορφών ημερομηνίας/αριθμού.
 Ενημερώθηκαν οι επιδείξεις ιστού.
 Υποστηρίζει τον υπολογισμό τύπων με JDK1.4.
 Διορθώνει ένα σφάλμα της μεθόδου License.setLicense().
 Διορθώνει ένα σφάλμα ρύθμισης προσαρμοσμένης μορφής αριθμού.
 Διορθώνει ένα σφάλμα κατά την αφαίρεση ενός φύλλου εργασίας από το βιβλίο εργασίας.
