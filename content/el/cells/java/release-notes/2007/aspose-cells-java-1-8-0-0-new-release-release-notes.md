 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει Aspose.Cells for Java V1.8.0!

 Περιλαμβάνονται οι ακόλουθες βελτιώσεις και διορθώσεις:

 Υποστηρίζει τον ορισμό ενεργών φύλλων και κελιών. Διορθώνει ορισμένα σφάλματα των τύπων ρύθμισης. Διορθώνει ορισμένα σφάλματα υπολογισμού τύπων. Διορθώνει ένα σφάλμα του ανοίγματος του αρχείου. Διορθώνει ένα σφάλμα του φύλλου εργασίας αντιμετώπισης. Διορθώνει ένα σφάλμα του συγκεντρωτικού πίνακα. Διορθώνει ένα σφάλμα επικύρωσης. Διορθώνει ένα σφάλμα των αυτόματων φίλτρων. Διορθώνει ένα σφάλμα του freezePanes.

Σημείωση: Για το Aspose.Cells for Java 1.4, έχουμε ενσωματώσει όλες τις κατηγορίες ρετρομεταφραστή-runtime και backport-util-concurrent. Επομένως, δεν χρειάζεται να αναφερθείτε στα δύο βάζα.
