 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει το Aspose.Cells for .NET V4.0!

- Αλλαγή ονόματος τάξης: ExcelHelper->CellsHelper, ExcelException->CellsException
- Σχηματίστηκε η μέθοδος AutoFitRow για να γίνει πιο ακριβής
- Προστέθηκε η μέθοδος Worksheets.SortNames για την επίλυση ενός προβλήματος κατά την προσθήκη μεγάλου αριθμού ονομασμένων περιοχών.
- Προστέθηκαν νέες επιδείξεις γραφημάτων
- Διορθώθηκε ένα σφάλμα στη ρύθμιση του τύπου
- Διορθώθηκε ένα σφάλμα κατά την αποθήκευση του αρχείου σε ροή με μορφή CSV
