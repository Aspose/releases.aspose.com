 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει το Aspose.Cells for .NET v3.9!

- Για ορισμένους εξωτερικούς λόγους, τώρα η κλάση Excel μετονομάζεται σε Βιβλίο εργασίας και η κλάση ExcelDesigner μετονομάζεται σε WorkbookDesigner. Όλες οι ιδιότητες και οι μέθοδοι αυτών των δύο κλάσεων δεν έχουν αλλάξει.
- Υποστηρίζει την προσθήκη εικόνων EMF στην ακατέργαστη μορφή τους
- Προστέθηκε νέος έξυπνος δείκτης
- Υποστηρίζει τη φόρτωση αρχείων κειμένου με προσαρμοσμένο διαχωριστικό
- Διορθώθηκε ένα σφάλμα στη ρύθμιση υπερσυνδέσμων σε τοπικά αρχεία
- Διορθώθηκαν σφάλματα στη μέθοδο InsertRows
- Διορθώθηκε ένα σφάλμα στις ρυθμίσεις επικύρωσης δεδομένων
- Διορθώθηκε ένα σφάλμα στο άνοιγμα ορισμένων αρχείων Excel
