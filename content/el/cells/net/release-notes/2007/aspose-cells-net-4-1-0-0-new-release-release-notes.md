 Αγαπητοί πελάτες,

 Έχουμε κυκλοφορήσει το Aspose.Cells for .NET v4.1!

- Παρέχονται API για τη δημιουργία μορφοποιήσεων υπό όρους από την αρχή
- Παρέχονται API για τη δημιουργία συγκεντρωτικών πινάκων από την αρχή
- Προστέθηκε η ιδιότητα LinkedCell για την κατηγορία CheckBox
- Προστέθηκαν νέες επιδείξεις γρήγορης εκκίνησης C#
- Διορθώθηκαν 2 μικρά σφάλματα στη ρύθμιση τύπων
- Διορθώθηκε ένα σφάλμα στον υπολογισμό των τύπων
- Διορθώθηκε το πρόβλημα της χρήσης Aspose.Cells στο Windows Vista RC2. Αυτό το πρόβλημα προκαλείται από συσκότιση.
- Διορθώθηκε ένα σφάλμα προσανατολισμού σελίδας κατά τη μετατροπή σε αρχείο pdf
- Διορθώθηκε ένα σφάλμα αντιγραφής σειρών με κοινόχρηστους τύπους
