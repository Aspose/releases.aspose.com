 Αγαπητοί πελάτες:

 Έχουμε κυκλοφορήσει το Aspose.Cells for .NET v4.2!

 Σε αυτή τη νέα έκδοση, προστίθενται ορισμένες σημαντικές νέες δυνατότητες:

- Τώρα το Aspose.Cells υποστηρίζει πλήρως το χειρισμό γραφημάτων, εικόνων και άλλων αντικειμένων σχεδίασης σε αρχεία προτύπων
- Υποστηρίζεται η δυνατότητα Chart2Image. Προς το παρόν υποστηρίζονται μόνο γραφήματα στήλης, γραμμής, πίτας και ράβδου. Και υποστηρίζονται μόνο 2D γραφήματα. Θα υποστηρίξουμε περισσότερους τύπους γραφημάτων στις μελλοντικές εκδόσεις.
- Η μετατροπή Xls2Pdf βελτιώνεται και βελτιστοποιείται.
- Η μηχανή υπολογισμού τύπου είναι βελτιωμένη και βελτιστοποιημένη.
