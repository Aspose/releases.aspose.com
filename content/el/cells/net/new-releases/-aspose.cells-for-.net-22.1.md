#  Ορισμός ονόματος εργασίας εκτυπωτή

Προστέθηκε η υποστήριξη για την παροχή και τον ορισμό του ονόματος εργασίας εκτύπωσης με `PrinterSettings` κατά την απόδοση υπολογιστικών φύλλων στον εκτυπωτή.

#  Αυτόματος εντοπισμός `CSV` και `TSV` Μορφή αρχείου

Αυτή η έκδοση API εισάγει τη δυνατότητα αυτόματης ανίχνευσης αρχείων κειμένου ως μορφών αρχείων διαχωρισμένων με κόμματα `CSV` και διαχωρισμού καρτελών `TSV` χρησιμοποιώντας API.

#  Εξαίρεση κρυφών σειρών / στηλών από το εύρος

Η ιδιότητα εμβέλειας κάλυπτε όλα τα κελιά στη συλλογή κελιών. Από αυτήν την έκδοση, αυτή η λειτουργικότητα έχει βελτιωθεί για να αγνοούνται / εξαιρούνται οι κρυφές σειρές και στήλες κατά τη λήψη του `Cells.MaxDisplayRange`.

 > Για μια πλήρη λίστα λειτουργιών, βελτιώσεων και διορθώσεων σφαλμάτων σε αυτήν την έκδοση, επισκεφθείτε το[Aspose.Cells for .NET 22.1 Σημειώσεις έκδοσης](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-1-release-notes/).
