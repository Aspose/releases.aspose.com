#  Υποστήριξη για μορφές προτύπων υπολογιστικών φύλλων

Αυτή η έκδοση API εισήγαγε υποστήριξη για μορφές προτύπων υπολογιστικών φύλλων και για το Excel&reg; (XLT) & OpenOffice Calc&reg; (OTS) προγράμματα. Αυτά υποστηρίζονται ως μορφές εξόδου μέσω των `SaveFormat.Ots` & `SaveFormat.Xlt`.

#  Ορισμός εσωτερικών συνόρων

Δυνατότητα ορισμού των εσωτερικών ορίων.

#  Ταξινόμηση Συγκεντρωτικού Πεδίου ανά επιλεγμένη σειρά ή στήλη

Προστέθηκε υποστήριξη ταξινόμησης για τιμές Συγκεντρωτικού Πεδίου. Η ταξινόμηση του Pivot Field μπορεί να γίνει με βάση την επιθυμητή γραμμή ή στήλη.

 > Για μια πλήρη λίστα λειτουργιών, βελτιώσεων και διορθώσεων σφαλμάτων σε αυτήν την έκδοση, επισκεφθείτε το[Aspose.Cells for .NET 22.3 Σημειώσεις έκδοσης](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-3-release-notes/).
