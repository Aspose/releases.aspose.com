#  `.NUMBERS` Υποστήριξη Data Storage 5.0

Προστέθηκε η υποστήριξη για αποθήκευση δεδομένων 5.0 από `.numbers`.

#  Βελτιωμένη απόδοση XLSX έως PDF

Παρατηρήθηκε ότι κατά τη μετατροπή του XLSX σε PDF, τα γραφήματα ράβδων δεν αποδίδονταν σωστά. Αυτό το ζήτημα έχει πλέον επιλυθεί.

#  Βελτιωμένο Excel&reg; Χειρισμός ετικετών γραφήματος

Αυτή η έκδοση API προσφέρει μερικές επιδιορθώσεις σχετικά με το Excel&reg; ετικέτες γραφημάτων. Αυτά είναι τα εξής:

- Οι ετικέτες των γραφημάτων δεν ταίριαζαν στο πλαίσιο.
- Οι ετικέτες γραφημάτων ευθυγραμμίζονταν κατακόρυφα αντί για οριζόντια.

#  Αποθηκεύστε τους τύπους πίνακα στο XLSB

Προστέθηκε υποστήριξη για αποθήκευση τύπων πινάκων στο `XLSB`.

#  Βελτιωμένη υποστήριξη στο Linux

Παρατηρήθηκε ότι χαρακτήρες ή λεζάντες γραμμένοι σε αντικείμενα όπως σχήματα και πλαίσια κειμένου εμφανίζονταν αλλοιωμένοι στο Linux. Αυτό το ζήτημα έχει πλέον επιλυθεί.
