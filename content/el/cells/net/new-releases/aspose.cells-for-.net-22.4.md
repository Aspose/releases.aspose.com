# Βελτιωμένη απόδοση σχημάτων WordArt

Παρατηρήθηκε ότι Microsoft Excel&reg; Τα εφέ WordArt και τα σχήματα δεν αποδίδονταν σωστά κατά τη μετατροπή του Excel σε PDF. Αυτό το ζήτημα έχει πλέον επιλυθεί.

#  Βελτιωμένη μετατροπή XLS σε PDF

Προηγουμένως, σε ορισμένες περιπτώσεις, όταν μετατρέπαμε το `XLS` σε `PDF`, το μέγεθος της κενής γραμμής κειμένου στο πλαίσιο κειμένου δεν αποδόθηκε σωστά. Αυτό το ζήτημα έχει πλέον επιλυθεί.

#  Κατάργηση Trailing Blank Cells από CSV

Αυτή η έκδοση API εισάγει τη δυνατότητα αφαίρεσης κενών κενών κελιών κατά την αποθήκευση του αρχείου τιμών διαχωρισμένων με κόμματα (CSV).
