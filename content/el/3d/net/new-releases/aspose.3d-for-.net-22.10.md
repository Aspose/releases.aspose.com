# Εισαγωγή αρχείου Maya ASCII

Προστέθηκε η δυνατότητα στο API χειρισμού 3D για εισαγωγή του αρχείου Autodesk® Maya ASCII. Τώρα το 3D API μας υποστηρίζει τη μορφή Maya ASCII ως μορφή εισόδου (.ma).
