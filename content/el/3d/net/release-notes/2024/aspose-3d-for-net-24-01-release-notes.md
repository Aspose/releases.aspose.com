## Αλλαγές API ##

### Προστέθηκαν μέλη στην κλάση **Aspose.ThreeD.Animation.AnimationChannel**:

{{< highlight csharp >}}
        /// <summary>
        /// Λαμβάνει την σχετική ακολουθία καρέ κλειδιά μέσα σε αυτό το κανάλι
        /// </summary>
        Aspose.ThreeD.Animation.KeyframeSequence KeyframeSequence{ get;set;}
{{< /highlight >}}

Οι παλαιότερες διεπαφές στην *AnimationChannel* που έχουν σχεδιαστεί για την πρόσβαση σε ακολουθίες καρέ κλειδιά θα καταργηθούν στο μέλλον. Αυτή η νέα ιδιότητα θα χρησιμεύσει ως αντικατάσταση. Επί του παρόντος, δεν υπάρχει ένδειξη ότι χρησιμοποιείται περισσότερο από μία ακολουθία καρέ κλειδιά για ένα μόνο κανάλι κίνησης.


### Προστέθηκαν μέλη στην κλάση **Aspose.ThreeD.Utilities.Quaternion**:

{{< highlight csharp >}}
        /// <summary>
        /// Εκτελέστε σφαιρική γραμμική παρεμβολή μεταξύ δύο τιμών
        /// </summary>
        /// <param name="t">t είναι μεταξύ 0 και 1</param>
        /// <param name="v1">Πρώτη τιμή</param>
        /// <param name="v2">Δεύτερη τιμή</param>
        public static Aspose.ThreeD.Utilities.Quaternion Slerp(double t, Aspose.ThreeD.Utilities.Quaternion v1, Aspose.ThreeD.Utilities.Quaternion v2)
{{< /highlight >}}

Μια βοηθητική μέθοδος για τον υπολογισμό σφαιρικής γραμμικής παρεμβολής μεταξύ δύο τετραγωνικών μορφών.
