## Δημιουργία Σκηνής με Πρωτότυπα 3D Μοντέλα χρησιμοποιώντας C# Κώδικα

Μπορείτε να εκτελέσετε το παρακάτω απόσπασμα κώδικα για να δείτε πώς λειτουργεί το Aspose.3D στο περιβάλλον σας ή να επισκεφθείτε το [GitHub Repository](https://github.com/aspose-3d/Aspose.3D-for-.NET) για άλλα κοινά σενάρια χρήσης.

```csharp
// αρχικοποίηση αντικειμένου Scene
Scene scene = new Scene();
// δημιουργία μοντέλου Box
scene.RootNode.CreateChildNode("box", new Box());
// δημιουργία μοντέλου Cylinder
scene.RootNode.CreateChildNode("cylinder", new Cylinder());
// αποθήκευση σε μορφή FBX
scene.Save(dir + "output.fbx", FileFormat.FBX7500ASCII);
```

## Εξαγωγή 3D Σκηνής σε Συμπιεσμένο AMF μέσω C# Κώδικα

Aspose.3D για .NET σας επιτρέπει να αποθηκεύετε 3D πλέγματα σε προσαρμοσμένη δυαδική μορφή, να λαμβάνετε όλες τις τιμές ιδιοτήτων των 3D σκηνών καθώς και να αντιστρέφετε το σύστημα συντεταγμένων τους. Το παρακάτω παράδειγμα δείχνει τη μετατροπή μιας 3D σκηνής σε μορφή AMF με εφαρμογή συμπίεσης.

```csharp
// φόρτωση σκηνής
Scene scene = new Scene();
var box = new Box();
var tr = scene.RootNode.CreateChildNode(box).Transform;
tr.Scale = new Vector3(12, 12, 12);
tr.Translation = new Vector3(10, 0, 0);
tr = scene.RootNode.CreateChildNode(box).Transform;
// κλιμάκωση
tr.Scale = new Vector3(5, 5, 5);
// ορισμός γωνιών Euler
tr.EulerAngles = new Vector3(50, 10, 0);
scene.RootNode.CreateChildNode();
scene.RootNode.CreateChildNode().CreateChildNode(box);
scene.RootNode.CreateChildNode().CreateChildNode(box);
// αποθήκευση συμπιεσμένου αρχείου AMF
scene.Save(dir + "output.amf", new AMFSaveOptions() {
  EnableCompression = true
});
```
