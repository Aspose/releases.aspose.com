# Java API για επεξεργασία εικόνας 3D

Το [Aspose.3D for Java API](https://products.aspose.com/3d/java) είναι κατασκευασμένο για να δημιουργήσει, να επεξεργαστεί, να χειριστεί και να αποθηκεύσει 3D μορφές. Δείχνει τις εφαρμογές Java να συνδεθούν με έγγραφα 3D χωρίς να εγκαθιστούν οποιοδήποτε πακέτο λογισμικού στον υπολογιστή. Aspose.3D Για τους προγραμματιστές Java API βοηθούν τους προγραμματιστές να μοντελοποιήσουν και να δημιουργήσουν μαζικούς κόσμους σε παιχνίδια, υπέροχες σκηνές για απεικόνιση σχεδιασμού και να εμπλέκουν εμπειρίες εικονικής πραγματικότητας. Το API είναι φιλικό προς το χρήστη και εξοικονομεί χρόνο και χρήμα από τη δημιουργία μιας παρόμοιας λύσης από το μηδέν.

## 3D λειτουργίες επεξεργασίας αρχείων

- [Δημιουργώ](https://docs.aspose.com/3d/java/create-an-empty-3d-document/), φορτώστε και αποθηκεύστε τη σκηνή 3D.
- Εργαστείτε με μοντέλα 3D και 3D αντικείμενα.
- Ρύθμιση κάμερας στόχου σε 3D έγγραφο.
- Προσθέστε ιεραρχία κόμβων και μοιραστείτε γεωμετρικά δεδομένα πλέγματος μεταξύ πολλαπλών κόμβων 3D σκηνής.
- Δημιουργήστε τρισδιάστατο πλέγμα και εκθέστε γεωμετρικό μετασχηματισμό.
- Μετατρέψτε όλα τα πολύγωνα σε τρίγωνα σε μοντέλο 3D.

- Εργαστείτε με [γραμμική εξώθηση](https://docs.aspose.com/3d/java/working-with-linear-extrusion/), [κύλινδρος](https://docs.aspose.com/3d/java/working-with-κύλινδρος/) και [Σημείο](https://docs.aspose.com/3d/java/working-with-pointcloud/).

## Εκτελέστε γραμμική εξώθηση χρησιμοποιώντας το Java API

``` java
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
// perform Linear extrusion by passing a 2D shape as input and extend the shape in the 3rd dimension
LinearExtrusion extrusion = new LinearExtrusion(profile, 10) {{ setSlices(100); setCenter(true); setTwist(360); setTwistOffset(new Vector3(10, 0, 0));}};
extrusion.setSlices(100);
extrusion.setCenter(true);
extrusion.setTwist(360);
extrusion.setTwistOffset(new Vector3(10, 0, 0));
// create a scene
Scene scene = new Scene();
// create a child node by passing extrusion
scene.getRootNode().createChildNode(extrusion);
// save 3D scene
scene.save("output.obj", FileFormat.WAVEFRONTOBJ);
```
