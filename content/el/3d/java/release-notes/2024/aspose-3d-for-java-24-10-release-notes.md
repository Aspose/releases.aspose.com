## Αλλαγές API ##

### Προστέθηκε κλάση **com.aspose.threed.CenterLineProfile**

Αυτό το προφίλ σάς επιτρέπει να δώσετε ένα πάχος σε μια καμπύλη, το οποίο μπορεί να χρησιμοποιηθεί με εγκαταστάσεις διαδικαστικής μοντελοποίησης όπως το LinearExtrusion.
