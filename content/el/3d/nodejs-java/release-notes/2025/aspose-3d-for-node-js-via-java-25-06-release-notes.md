## Αλλαγές API ##

Αυτή η έκδοση είναι κυρίως έκδοση διόρθωσης σφαλμάτων, βελτίωσε τη σταθερότητα της διαδικαστικής μοντελοποίησης όπως η εξώθηση, οι Boolean λειτουργίες.


### Προστέθηκαν μέλη στην κλάση **com.aspose.threed.BooleanOperand**:

{{< highlight java >}}
        public com.aspose.threed.A3DObject getOperand()
{{< /highlight >}}

Η νέα ιδιότητα που προστέθηκε επιτρέπει την πρόσβαση στην παρουσία Node/Entity στο BooleanOperand.
