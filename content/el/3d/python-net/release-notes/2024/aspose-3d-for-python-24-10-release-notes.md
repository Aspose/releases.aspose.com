## Αλλαγές API ##

### Προστέθηκε κλάση **aspose.threed.profiles.CenterLineProfile**

Αυτό το προφίλ σας επιτρέπει να δώσετε ένα πάχος σε μια καμπύλη, η οποία μπορεί να χρησιμοποιηθεί με εγκαταστάσεις διαδικαστικής μοντελοποίησης όπως το LinearExtrusion.
