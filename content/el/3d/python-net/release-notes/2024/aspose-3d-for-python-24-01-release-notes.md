## Αλλαγές API ##

### Προστέθηκαν μέλη στην κλάση **aspose.threeD.animation.AnimationChannel**:

{{< highlight python >}}
        @property
        def keyframe_sequence(self : "aspose.threed.animation.AnimationChannel") -> "aspose.threed.animation.KeyframeSequence"
        """
            Λαμβάνει την σχετιζόμενη ακολουθία βασικών καρέ μέσα σε αυτό το κανάλι
        """

        @keyframe_sequence.setter()
        def keyframe_sequence(self : "aspose.threed.animation.AnimationChannel", v : "aspose.threed.animation.KeyframeSequence")
        """
            Ορίζει την σχετιζόμενη ακολουθία βασικών καρέ μέσα σε αυτό το κανάλι
        """

{{< /highlight >}}

Οι παλαιότερες διεπαφές στην *AnimationChannel* που έχουν σχεδιαστεί για την πρόσβαση σε ακολουθίες βασικών καρέ θα καταργηθούν στο μέλλον. Αυτή η νέα ιδιότητα θα χρησιμεύσει ως αντικατάσταση. Επί του παρόντος, δεν υπάρχει ένδειξη ότι χρησιμοποιείται περισσότερο από μία ακολουθία βασικών καρέ για ένα μόνο κανάλι κίνησης.

### Προστέθηκαν μέλη στην κλάση **aspose.threeD.utilities.Quaternion**:

{{< highlight python >}}
        def slerp(t : float, v1 : Quaternion, v2 : Quaternion) -> Quaternion
        """
                Εκτελέστε σφαιρική γραμμική παρεμβολή μεταξύ δύο τιμών
                t : float
                        Το t είναι μεταξύ 0 και 1
                v1 : Quaternion
                        Πρώτη τιμή
                v2 : Quaternion
                        Δεύτερη τιμή
        """
{{< /highlight >}}

Μια βοηθητική μέθοδος για τον υπολογισμό σφαιρικής γραμμικής παρεμβολής μεταξύ δύο τετραγωνικών φορμών.
