## Αλλαγές API ##

Αυτή η έκδοση είναι κυρίως έκδοση διόρθωσης σφαλμάτων, βελτίωσε τη σταθερότητα της διαδικαστικής μοντελοποίησης όπως η εξώθηση, οι boolean λειτουργίες.

### Προστέθηκαν μέλη στην κλάση **aspose.threed.entities.BooleanOperand**:

{{< highlight python >}}
        @property
        def operand(self) -> aspose.threed.A3DObject
{{< /highlight >}}

Προστέθηκε νέα ιδιότητα που σας επιτρέπει να έχετε πρόσβαση στην παρουσία κόμβου/οντότητας στο BooleanOperand.
