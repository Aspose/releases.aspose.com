## .NET API για επεξεργασία αρχείων Photoshop&reg;

Μια αυτόνομη API .NET για ανάγνωση, εγγραφή, επεξεργασία και μετατροπή των μορφών Adobe Photoshop&reg; PSD και PSB χωρίς να απαιτείται η εγκατάσταση του Adobe Photoshop&reg;. Η API Aspose.PSD για .NET σας επιτρέπει να δημιουργείτε και να επεξεργάζεστε αρχεία Photoshop&reg; και παρέχει τη δυνατότητα ενημέρωσης των ιδιοτήτων των επιπέδων, προσθήκης υδατογραφημάτων, εκτέλεσης γραφικών λειτουργιών και μετατροπής αρχείων μεταξύ διαφορετικών μορφών.

## Γιατί να επιλέξετε το Aspose.PSD για .NET

- **Δεν απαιτείται το Adobe Photoshop®:** Το Aspose.PSD για .NET λειτουργεί ανεξάρτητα από το Adobe Photoshop&reg;, επιτρέποντάς σας να εργάζεστε με αρχεία PSD και PSB απευθείας στις εφαρμογές .NET σας.
- **Υψηλή απόδοση:** Σχεδιασμένη για ταχύτητα και αποτελεσματικότητα, η API επεξεργάζεται γρήγορα μεγάλα αρχεία PSD, ακόμα και με σύνθετα επίπεδα και εφέ.
- **Συμβατότητα πολλαπλών πλατφορμών:** Υποστηρίζει .NET Framework, .NET Core και Mono, καθιστώντας το κατάλληλο για περιβάλλοντα Windows, macOS και Linux.
- **Πλήρης υποστήριξη μορφών:** Ανάγνωση, εγγραφή και μετατροπή των μορφών Photoshop&reg; και Illustrator&reg; χωρίς προβλήματα.
- **Εύκολη ενσωμάτωση:** Απλή ενσωμάτωση στις υπάρχουσες εφαρμογές .NET σας με πλήρη τεκμηρίωση και υποστήριξη.

## Λειτουργίες επεξεργασίας αρχείων Photoshop&reg;

- **Δημιουργία και επεξεργασία αρχείων Photoshop&reg;:** Δημιουργήστε νέα αρχεία PSD και PSB ή επεξεργαστείτε υπάρχοντα προγραμματιστικά.
- **Προχωρημένη διαχείριση επιπέδων:** Ενημερώστε τις ιδιότητες των επιπέδων, εφαρμόστε λειτουργίες ανάμειξης, προσθέστε μάσκες και διαχειριστείτε τα έξυπνα αντικείμενα.
- **Εφαρμογή φίλτρων και εφέ:** Υλοποιήστε φίλτρα Median και Wiener, εφαρμόστε ισορροπία χρωμάτων, αντιστρέψτε επίπεδα προσαρμογής και πολλά άλλα.
- **Υποστήριξη γραφικών raster και vector:** Σχεδιάστε βασικά σχήματα, διαχειριστείτε διανυσματικά μονοπάτια και εφαρμόστε μετασχηματισμούς.
- **Διαχείριση δεδομένων XMP:** Δημιουργία, ανάγνωση και εγγραφή μεταδεδομένων XMP σε αρχεία PSD.
- **Ολοκληρωμένες λειτουργίες εικόνας:** Αλλάξτε το μέγεθος, περικόψτε, περιστρέψτε και προσαρμόστε εικόνες με ακρίβεια.
- **Εξαγωγή σε διάφορες μορφές:** Αποθηκεύστε τη δουλειά σας σε πολλαπλές μορφές raster και vector, όπως TIFF, JPEG, PNG, GIF, BMP, JPEG2000 και PDF.
- **Επεξεργασία παρτίδας:** Επεξεργαστείτε αποτελεσματικά πολλά αρχεία σε μια μόνο λειτουργία παρτίδας.
- **Βελτιστοποίηση μνήμης:** Διαχειριστείτε μεγάλα αρχεία με βελτιστοποιημένη χρήση μνήμης, εξασφαλίζοντας ότι οι εφαρμογές σας παραμένουν γρήγορες ακόμα και με σύνθετα PSD.
- **Διαχείριση έξυπνων φίλτρων:** Εφαρμόστε και διαχειριστείτε μη καταστρεπτικά έξυπνα φίλτρα, διατηρώντας την ακεραιότητα των αρχικών δεδομένων της εικόνας.
- **Διαχείριση χρωμάτων:** Προσαρμόστε τη φωτεινότητα, την αντίθεση, το gamma και εφαρμόστε διάφορες τεχνικές διόρθωσης χρωμάτων.
- **Διαχείριση κειμένου:** Εργαστείτε με επίπεδα κειμένου, συμπεριλαμβανομένης της διαχείρισης γραμματοσειρών και των εφέ κειμένου.
- **Προσθήκη υδατογραφημάτων:** Προσθέστε εύκολα υδατογραφήματα στις εικόνες για προστασία πνευματικών δικαιωμάτων.

## Περικοπή ενός PSD Photoshop&reg; για αποθήκευση του αποτελέσματος σε μορφή PNG

Μπορείτε να εκτελέσετε το παρακάτω απόσπασμα κώδικα για να δείτε πώς λειτουργεί η API Aspose.PSD στο δικό σας περιβάλλον ανάπτυξης ή να ελέγξετε το [αποθετήριο GitHub](https://github.com/aspose) για άλλες κοινές περιπτώσεις χρήσης.

```csharp
// Εφαρμόστε τη σωστή μέθοδο περικοπής για αρχεία PSD.
using (RasterImage image = Image.Load(dir + "template.psd") as RasterImage)
{
    image.Crop(new Rectangle(10, 30, 100, 100));
    image.Save(dir + "output.png", new PngOptions() { ColorType = PngColorType.TruecolorWithAlpha });
}
```

## Σχεδιάστε ορθογώνια σε μια εικόνα PSD

```csharp
// Δημιουργήστε ένα στιγμιότυπο της εικόνας
using (Image image = new PsdImage(100, 100))
{
    Graphics graphic = new Graphics(image);
    graphic.Clear(Color.Yellow);
    // Σχεδιάστε ένα ορθογώνιο με το εργαλείο Pen
    graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
    // Σχεδιάστε ένα άλλο ορθογώνιο με ένα Solid Brush σε μπλε χρώμα
    graphic.DrawRectangle(new Pen(new SolidBrush(Color.Blue)), new Rectangle(10, 30, 80, 40));
}
```
