# Βελτιωμένη επεξεργασία αρχείων Primavera P6 XER

Αυτή η έκδοση API προσφέρει πολλές βελτιώσεις στην επεξεργασία των αρχείων έργου Primavera P6 XER. Αυτά περιλαμβάνουν τη δυνατότητα ανάγνωσης ημερολογίου εργασιών, ανάγνωσης του νομίσματος του έργου και ανάγνωσης του LinkLag κ.λπ.

> **Εξάλειψη αλλαγών:** Τώρα, από προεπιλογή, το έργο θα αποδοθεί με την αναλυτικότητα της κλίμακας χρόνου που καθορίζεται στο MS Project®.

> Για μια πλήρη λίστα λειτουργιών, βελτιώσεων και διορθώσεων σφαλμάτων σε αυτήν την έκδοση, επισκεφτείτε τη διεύθυνση [Aspose.Tasks for .NET 22.8 Release Notes](https://docs.aspose.com/tasks/net/aspose-tasks-for-net-22-8-release-notes/).
