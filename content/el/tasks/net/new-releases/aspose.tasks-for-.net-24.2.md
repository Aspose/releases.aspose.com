Aspose.Tasks για .NET 24.2 παρέχει στους προγραμματιστές βελτιωμένες δυνατότητες διαχείρισης έργων! Αυτή η ενημέρωση εισάγει μια πολύ απαιτητική δυνατότητα για τον έλεγχο του προσανατολισμού της σελίδας κατά την αποθήκευση δεδομένων έργου σε διάφορες οπτικές μορφές. Επιπλέον, διορθώνει σφάλματα για να εξασφαλίσει απρόσκοπτο χειρισμό έργων χρησιμοποιώντας το MSI στις εφαρμογές σας .NET στα Windows.

## Βασικά Σημεία

- Έλεγχος Προσανατολισμού Σελίδας: Αποκτήστε έλεγχο του προσανατολισμού της σελίδας (κατακόρυφο/οριζόντιο) κατά την εξαγωγή αρχείων Microsoft Project (MPP) σε μορφές PDF, HTML και εικόνας χρησιμοποιώντας την νέα ιδιότητα IsPortrait στις SaveOptions.
- Αυξημένη Ευελιξία Αναφοράς: Αυτή η προσθήκη επιτρέπει στους προγραμματιστές να προσαρμόζουν τις διατάξεις των αναφορών σε συγκεκριμένες απαιτήσεις, μεγιστοποιώντας την αναγνωσιμότητα και την οπτική εντύπωση.
- Απλοποιημένη Ενσωμάτωση: Το διαισθητικό API διαχείρισης έργων ενσωματώνεται απρόσκοπτα στις υπάρχουσες ροές εργασίας σας, επιτρέποντας εύκολο έλεγχο της παρουσίασης των αναφορών.
- Βελτιωμένη Εμπειρία Προγραμματιστή: Οι προγραμματιστές μπορούν να αξιοποιήσουν νέους κατασκευαστές για την κλάση WeekDay για έναν πιο διαισθητικό και αποδοτικό τρόπο ορισμού των εργάσιμων ημερών και περιόδων εντός των χρονοδιαγραμμάτων του έργου σας.
- Διορθώσεις Σφαλμάτων και Βελτιώσεις: Αυτή η έκδοση αντιμετωπίζει μικρά σφάλματα σχετικά με την ανάγνωση δεδομένων χρονικής φάσης, τη γραφή κριτηρίων φίλτρου και τον χειρισμό της σημαίας Filter.ShowRelatedSummaryRows για μια πιο ισχυρή εμπειρία ανάπτυξης.

## Παράδειγμα Κώδικα

Το ακόλουθο παράδειγμα κώδικα C# δείχνει πώς να εφαρμόσετε την ιδιότητα IsPortrait των SaveOptions και να εξάγετε ένα έγγραφο MPP ως PDF σε κατακόρυφο προσανατολισμό.

```c#
Project project = new Project("test.mpp");
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.PageSize = PageSize.A4;
saveOptions.IsPortrait = true;
project.Save("output.pdf", saveOptions);
```
