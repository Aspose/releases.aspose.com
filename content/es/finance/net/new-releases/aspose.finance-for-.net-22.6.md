# Cargar datos desde un archivo XBRL existente

Anteriormente, en algunos casos, existía la siguiente excepción:

> “El protocolo de seguridad solicitado no es compatible”

Este problema se solucionó ahora y los datos de un archivo XBRL existente se pueden cargar correctamente a través de la API.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.Finance for .NET 22.6 Release Notes](https://releases.aspose.com/finance/net/release-notes/2022/aspose-finance-for-net-22-6-release-notes/).
