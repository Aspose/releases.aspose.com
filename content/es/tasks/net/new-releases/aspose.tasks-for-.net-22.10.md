### Preserve primavera uids

Se ha agregado una nueva configuración que permite preservar la Primavera y Reg; UID mientras importan proyectos de formatos XER o P6XML.

Se ha eliminado el soporte para .NET Framework 2.0.

La API se ha mejorado para acceder a los campos de las entidades utilizando las propiedades .NET.
