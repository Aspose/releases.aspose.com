# Importar archivos XER de proyectos múltiples

Extendió la importación Primavera P6 `XER` y ahora admite la importación de archivos `XER` con más de un (1) proyecto.

# Importación mejorada de restricciones de tareas para XER

Se agregó la capacidad de especificar el comportamiento de la importación de restricciones de tareas para el formato `XER`.

# Leer campos definidos por el usuario para el formato XER

Se agregó el soporte de lectura de campos definidos por el usuario para el formato Primavera `XER`.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.Tasks for .NET 22.4 Release Notes](https://docs.aspose.com/tasks/net/aspose-tasks-for-net-22-4-release-notes/).
