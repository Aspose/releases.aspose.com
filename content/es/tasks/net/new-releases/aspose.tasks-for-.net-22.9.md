# Conversión de XML a MPP mejorada

Se corrigió la excepción que ocurría al guardar un archivo XML específico en formato MPP. El siguiente fragmento de código de C# demuestra cómo un archivo XML de Primavera P6 se puede convertir en un archivo de Microsoft Project® mediante la API:

```csharp
Project project = new Project("PrimaveraProject.xml");
project.Save("Output.mpp", SaveFileFormat.MPP);
```

# Función de lectura mejorada de las tasas de recursos de Primavera XER

Se solucionó el problema de la lectura de tasas de recursos de los archivos Primavera XER.

# Conversiones mejoradas de varios formatos a MPP

Se corrigieron los errores que se activaron al guardar archivos específicos en formato MPP.
