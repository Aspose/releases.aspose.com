### Procesamiento de código de barras Python a través de las características de la API local de Java

- Ofrece soporte para la mayoría de las especificaciones y estándares de códigos de barras populares.
- Puede leer y exportar códigos de barras en múltiples formatos de imagen, por ejemplo, JPEG, PNG, GIF y BMP.
- Capacidad para trabajar con etiquetas de código de barras.
- Compatibilidad con la dimensión X y la dimensión Y para códigos de barras 2D.
