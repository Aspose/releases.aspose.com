### Funciones de integración de código de barras JasperReports®

- Generar códigos de barras para informes.
- Imprima códigos de barras basados en una fuente de datos externa
- Aplicar subtítulos arriba y debajo de la imagen del código de barras
- Establecer la altura de las barras de código de barras
- Configure las dimensiones X e Y del código de barras y la relación de aspecto
- Compatibilidad con códigos de barras complementarios de 2 y 5 dígitos
- Establecer el espacio de suplemento
- Habilite sumas de verificación para un código de barras y hágalo oculto o visible
- Controle la proporción de elementos de código de barras anchos a estrechos
- Capacidad para establecer bordes de imagen de código de barras, márgenes de borde y ancho de borde
- Generar códigos de barras de colores
- Aplique color al código de barras, al texto del código, al fondo, al borde y al título
- Personalice la rotación y la resolución de la imagen del código de barras
- Compatibilidad con suavizado para mejorar la calidad de la imagen del código de barras
- Posibilidad de elegir unidades de tamaño de imagen de código de barras, como pulgadas, milímetros, píxeles, puntos, etc.

#### Formatos de entrada/salida admitidos

**Imágenes:** JPEG, TIFF, PNG, BMP, GIF, EXIF, EMF, SVG

#### Simbologías admitidas para código de barras lineal

- Código128
- Código39
- Código93
-EAN13
- EAN14 (SSCC14)
-EAN8
-UPCA
- UPCE
- BooklandEAN
- Intercalado 2 de 5
- Estándar2de5
-MSI
- Código11
- Codabar
- Postnet
- Planeta
- SSCC18

#### Simbologías admitidas para código de barras 2D

- PDF417
- QR
- Matriz de datos
- SwissQR

#### Requisitos admitidos

- Java 2 Platform Standard Edition 5.0 o superior.
- JasperReports 2.0, 3.0 o superior.
