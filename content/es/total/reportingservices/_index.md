Aspose.Total for Reporting Services es un conjunto de extensiones de representación para MS SQL Server Reporting Services que admite muchos formatos populares.

Aspose.Total para Reporting Services contiene los siguientes componentes:

* Aspose.BarCode for Reporting Services: mejora sus informes SQL para mostrar más de 60 tipos de códigos de barras en alta resolución.
* Aspose.Cells for Reporting Services: Diseñe y entregue informes de Microsoft Excel en Microsoft SQL Server Reporting Services.
* Aspose.PDF para Reporting Services: genere informes en PDF, PDF/A y XPS de alta calidad desde MS SQL Server Reporting Services.
* Aspose.Slides para Reporting Services: genere fácilmente informes en formato PPT y PPS dentro de Microsoft SQL Server Reporting Services.
* Aspose.Words para Reporting Services: manipule y exporte informes RDL y RDLC a formatos de archivo compatibles desde MS SQL Server Reporting Services.
