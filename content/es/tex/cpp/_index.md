## ¿Por qué descargar Aspose.TeX para C++?

Aspose.TeX para C++ es una biblioteca que proporciona una extensión de motor TeX llamada ObjectTeX. Se puede usar para componer documentos descritos por archivos TeX. "Objeto" significa que el resultado de la composición tipográfica intermedia es un modelo de objeto específico que luego se puede convertir uniformemente en una serie de formatos finales.

## TipoConfiguración de funciones de procesamiento de archivos

- Composición tipográfica de archivos Texas
- Crear formatos TeX personalizados
- Proporcionar datos de entrada de varias maneras.
- Obtener datos de salida de varias maneras

### Formatos de entrada admitidos

Texas

### Guardar TeX como

XPS, PDF, JPEG, PNG, TIFF y BMP

### Plataformas compatibles
