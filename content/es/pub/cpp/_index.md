## ¿Por qué descargar Aspose.PUB para C++?

Aspose.PUB para C++ es una API simple que le permite leer y convertir archivos de Microsoft Publisher® (PUB) a formato PDF mediante programación en sus aplicaciones de C++. También proporciona interfaces fáciles de entender para editar los metadatos de los archivos PUB.

## Funciones de procesamiento de archivos PUB

- Lea archivos de Microsoft Publisher® (PUB) para convertirlos a formato PDF.
