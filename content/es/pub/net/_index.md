## ¿Por qué descargar Aspose.PUB para .NET?

Aspose.PUB para .NET es una API simple que le permite leer y convertir archivos de Microsoft Publisher® (PUB) a formato PDF. También proporciona interfaces fáciles de entender para editar los metadatos de los archivos PUB.

## Funciones de procesamiento de archivos PUB

- Lea archivos de Microsoft Publisher® (PUB) para convertirlos a formato PDF.
- Leer y escribir metadatos de archivos PUB a través de API.
