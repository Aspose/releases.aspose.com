# Soporte para fuente grande

Se agregó el soporte para Big Font en la API.

# Conversión eficiente de memoria DWG a PDF

Anteriormente, cuando DWG se convertía al formato PDF, solía consumir mucha memoria. Este problema se ha resuelto ahora. Ahora puede convertir DWG a PDF con mejores resultados.

# Exportación mejorada de DAE a OBJ

En algunos casos anteriores, la exportación de DAE a OBJ falló. Este problema ahora se ha resuelto en la API.
