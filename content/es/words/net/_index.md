Aspose.Words para .NET es una biblioteca de clases que pueden usar los desarrolladores de C#, F# y VB.NET para una variedad de tareas de procesamiento de documentos, incluida la generación, modificación, conversión y representación de documentos. Nuestra biblioteca es autosuficiente y no depende de ningún software de terceros, como Microsoft Word®, OpenOffice® y suites ofimáticas similares.

## ¿Por qué descargar Aspose.Words para .NET?

Este paquete se puede utilizar para desarrollar aplicaciones para una amplia gama de sistemas operativos (Windows, Linux, macOS, iOS, Android) y plataformas como Windows Azure, Xamarin.Android, Xamarin.iOS, Xamarin.Mac. Puede compilar software de 32 y 64 bits, incluidos ASP.NET, WCF y WinForms. Además, puede usar nuestra biblioteca a través de COM Interop desde los lenguajes de programación ASP, PHP, Perl y Python.

> *Tenga en cuenta*: nuestra biblioteca implica el uso de lenguajes de programación .NET, compatibles con la infraestructura CLI. Si necesita una biblioteca nativa correspondiente para C++, puede descargarla desde [here](https://www.nuget.org/packages/Aspose.Words.Cpp/).

## Funcionalidad

- Proporciona importación y exportación de documentos integrales con más de 35 formatos de archivo compatibles. Esto permite a los desarrolladores convertir documentos de un formato de archivo a otro. Por ejemplo, puede convertir documentos PDF a Word y Word a PDF con calidad profesional.
- Brinda acceso completo a todos los elementos del documento de Word y OpenOffice, incluidas las propiedades de formato y el estilo.
- Proporciona representación de alta fidelidad de documentos de Word a PDF, JPG, PNG y otros formatos de imagen.
- Brinda la capacidad de imprimir documentos de OpenOffice y Word mediante programación.
- Admite la potente función de generación de informes con combinación de correspondencia, que permite crear documentos de forma dinámica utilizando plantillas y fuentes de datos.
- Contiene un motor de informes LINQ flexible, diseñado para obtener datos de bases de datos, XML, JSON, OData, documentos externos.
- Proporciona un amplio conjunto de funciones de utilidad que puede utilizar para dividir un documento en partes, unir documentos, comparar documentos y mucho más.

Para familiarizarse con la funcionalidad más popular de Aspose.Words, consulte nuestro [free online applications](https://products.aspose.app/words/family).
