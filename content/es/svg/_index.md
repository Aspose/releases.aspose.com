## ¿Por qué descargar Aspose.SVG para .NET?

Esta API local de .NET lo ayuda con la funcionalidad de [seamlessly integrate SVG file processing & manipulation](https://products.aspose.com/svg/net/) en su C#, VB.NET, ASP.NET y otras aplicaciones basadas en .NET.

## Funciones de procesamiento de archivos SVG

- [Create, read and write SVG](https://docs.aspose.com/svg/net/drawing-basics/) archivos de formato.
- Convertir SVG a otro [supported file formats](https://docs.aspose.com/svg/net/getting-started/supported-file-formats/).
- Manipulación del árbol DOM según las especificaciones oficiales de SVG.
- Compatibilidad con la navegación de contenido a través de [XPath Query](https://docs.aspose.com/svg/net/how-to-work-with-aspose-svg-api/navigation-inspection/), selectores de CSS, elementos y funciones transversales de documentos.
- Soporte para renderizado de calidad.
