### Visio Python a través de .NET Características de la API local

- Crear archivos de visio a través de API.
- Conversión y renderizado de formato de archivo de alta calidad.
- Imprima archivos de Microsoft Visio® en impresoras físicas o virtuales.
- Combinar, modificar, proteger o analizar páginas de Visio®.
- Convierte formas a imágenes o PDF.
- Configurar y aplicar la configuración de página para las páginas.
- Convierta archivos de Visio® a varios otros formatos.

Visite [official documentation](https://docs.aspose.com/diagram/python-net/) para obtener una lista más detallada de funciones.

### Leer formatos de archivo de Visio

**Microsoft Visio**: VSD, VDW, VSX, VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM

### Guardar archivos de Visio como

**Microsoft Visio**: VSX, VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM
**Diseño fijo**: PDF, XPS
**Imágenes**: JPEG, PNG, BMP, SVG, TIFF, GIF, EMF
**Web**: HTML


### Requisitos del sistema

Su máquina no necesita tener instalado el software Microsoft Visio®.

### Sistemas operativos compatibles

**Microsoft Windows®:** Escritorio y servidor de Windows (`x64`, `x86`)\
**Linux:** Ubuntu, OpenSUSE, CentOS y otros\
**Otro:** Cualquier sistema operativo (SO) que pueda instalar Mono (compatible con .NET 4.0 Framework) o usar .NET core.
