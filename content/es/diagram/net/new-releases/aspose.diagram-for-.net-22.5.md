# Ejecución mejorada del método asíncrono

Al convertir Visio® VSDX a PDF, el archivo de salida no se guardaba en el método asíncrono hasta que el programa se ejecutaba por completo. La versión actual de la API ha resuelto este problema.

# Resolución de problema de fórmula/valor

Los campos de fórmula/valor no se actualizaban correctamente. Este problema se ha resuelto en la API.

# Mejor manejo del texto de la tabla en la conversión de VSDX a PDF

Durante el proceso de conversión de VSDX a PDF, en algunos casos se eliminaba el texto presente en la tabla. Este problema está solucionado ahora.
