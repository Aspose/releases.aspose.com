# Incrustación de recursos externos mejorada

Anteriormente, en algunos casos, al guardar los diagramas de Microsoft Visio® en la secuencia HTML, no se incrustaban los recursos externos. Sin embargo, este problema ya se ha resuelto.

# Mejor manejo del sombreado de formas

Al convertir un Visio® VSDX a PDF, se produjo un problema en el que faltaba el sombreado de las formas. Esta versión resuelve ese problema y ofrece un mejor manejo de los tonos de forma.

# Guardar HTML en una sola transmisión

Se agregó el soporte para guardar HTML en una sola secuencia.

# Guardar diagrama en HTML con recursos integrados

Capacidad para guardar diagramas de Visio® en HTML con o sin los recursos integrados.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.Diagram for .NET 22.1 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-1-release-notes/).
