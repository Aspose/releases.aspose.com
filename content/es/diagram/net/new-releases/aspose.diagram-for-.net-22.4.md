# Resolución de problemas de conversión de imágenes

La conversión de imágenes no funcionaba al convertir Microsoft Visio® VSDX a PDF. La versión actual ha solucionado este problema.

# Conversión de color de fondo de forma de Visio®

Anteriormente, en algunos casos, el color de fondo de las formas se convertía incorrectamente durante el proceso de conversión de VSDX a PDF. Este problema ya se ha resuelto.

# Manejo de la posición del texto

Mejor manejo de la posición del texto durante la conversión de formato Visio a SVG.

# Mejor representación de tablas en conversión de Visio a PNG

En esta versión de la API, se han resuelto los problemas de formato en la representación de tablas y botones de opción durante el proceso de conversión de VSDX a PNG.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.Diagram for .NET 22.4 Release Notes](https://releases.aspose.com/diagram/net/release-notes/2022/aspose-diagram-for-net-22-4-release-notes/).
