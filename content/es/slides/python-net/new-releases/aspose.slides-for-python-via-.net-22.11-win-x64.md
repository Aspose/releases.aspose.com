# Exportar PDF 1.6 y PDF 1.7 Soporte

Se ha implementado el soporte para que PDF se exporte a los formatos PDF 1.6 y PDF 1.7.

# Exportar PDF con A2A, niveles de cumplimiento de A2B

Este lanzamiento de la API de Python presenta el soporte de la función de exportación PDF con numerosos niveles de cumplimiento, es decir, A2A, A2B, A2U, A3A y A3B.

# Administrar el sonido del hipervínculo en las presentaciones

Se ha agregado una nueva propiedad ([`Hyperlink.Sound`] (https://reference.aspose.com/slides/python-net/aspose.slides/hyperlink/)) que representa el sonido reproducido de un hiperlink.

# El efecto de animación detiene el sonido anterior

Se ha introducido una nueva propiedad ([`Effect.stop_previous_sound`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/effect/)) que especifica si el efecto de animación detiene el anterior sonido.
