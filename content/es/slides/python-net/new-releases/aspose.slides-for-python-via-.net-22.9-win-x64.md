# Soporte para Apple & Reg; MacOS & Reg;

Este lanzamiento de la API de Python de manipulación de la presentación introduce el soporte para MacOS & Reg;.

# Obtener información de sustitución de fuentes

Capacidad para obtener la información sobre las fuentes que se reemplazan durante la representación de presentación. Se ha agregado un nuevo método, `get_substitutions ()`, para lograrlo.

# Establezca el tiempo para la animación de diapositivas

Ahora puede establecer el tiempo para repetir la animación de diapositivas de presentación a través de las siguientes dos nuevas propiedades agregadas a [`Timing`] (https://reference.aspose.com/slides/python-net/aspose.slides.animation/timing /) clase:

- `Repet_until_next_click`: esta propiedad Bool se establece si el efecto de animación se repite hasta que se haga clic en el botón del mouse.
- `Repet_until_end_slide`: esta propiedad Bool se establece si el efecto de animación de diapositivas se repite hasta el final de la diapositiva actual.
