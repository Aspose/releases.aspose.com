# Soporte mejorado para Ubuntu Focal

Se observó que al convertir Microsoft PowerPoint & Reg; `PPTX` a` PDF` formatearon pocos problemas aparecieron en Ubuntu Focal. Esos problemas ahora se han resuelto.

# Soporte de cuadros de columna 3D

Esta versión de API presenta el soporte de gráficos de columna 3D. Ahora puede agregar gráfico con datos predeterminados y luego establecer las propiedades `rotación3D` (como el eje del ángulo recto, el porcentaje de profundidad, etc.).

# Soporte de efecto de texto hebreo mejorado

Anteriormente se observó que en algunos casos la sombra de texto hebreo estaba siendo incorrecta cuando las presentaciones de `pptx` se convierten en formato` pdf`. Este problema ahora se ha resuelto.

> Para obtener una lista completa de características, mejoras y correcciones de errores en esta versión, visite, [Aspose.Slides para Python 22.7 Notas de versión] (https://docs.aspose.com/slides/python-net/aspose-slides-- para python-net-22-7-libere-notas/).
