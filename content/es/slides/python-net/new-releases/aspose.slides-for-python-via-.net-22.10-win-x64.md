# Establecer el comportamiento de renderización y rotación del marco

Capacidad para especificar y controlar el comportamiento de la rotación del marco y el tamaño del marco al tiempo que representa formas de `PPTX` y otros formatos de presentación compatibles.

# Comprimir fuentes integradas para reducir el tamaño de la presentación

Si su presentación contiene las fuentes incrustadas, esta versión de nuestra API Python de procesamiento de presentación ofrece la capacidad de realizar la compresión de todas esas fuentes incrustadas y, a su vez, reduce el tamaño de los archivos de presentación como `pptx`.

# Establecer Presentación Líder Líneas Color

Este lanzamiento de API introduce la capacidad de establecer el color de las líneas de líder dentro de sus diapositivas de presentación.

> Para obtener una lista completa de características, mejoras y correcciones de errores en esta versión, visite, [Aspose.Slides para Python 22.10 Notas de versión] (https://docs.aspose.com/slides/python-net/aspose-slides-- para python-net-22-10-libere-notas/).
