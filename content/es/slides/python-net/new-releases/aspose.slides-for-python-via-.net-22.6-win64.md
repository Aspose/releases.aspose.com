# Comportamiento de bloqueo de fuente de presentación actualizada

La presentación obsoleta se ha eliminado el miembro de enum `load_and_release_legacy_tempuary_default` enum. `Load_and_release` es el comportamiento de bloqueo de presentación predeterminado ahora.

Esta configuración predeterminada bloquea la fuente solo para el momento de la ejecución del constructor `Ipresentation`.

> Para obtener una lista completa de características, mejoras y correcciones de errores en esta versión, visite, [Aspose.Slides para Python 22.6 Notas de versión] (https://docs.aspose.com/slides/python-net/aspose-slides-- para python-net-22-6-libere-notas/).
