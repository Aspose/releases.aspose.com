Nos complace anunciar el primer lanzamiento público de Aspose.Cells for PHP via Java. Las API incorporan funcionalidades de procesamiento de hojas de cálculo de MS Excel en PHP (un lenguaje de secuencias de comandos de código abierto ampliamente utilizado). Aspose.Cells for PHP via Java es un componente de subconjunto que incluye todas las funciones importantes y útiles presentes en su versión nativa[Aspose.Cells for Java](https://products.aspose.com/cells/java/)El nuevo API es un componente igualmente robusto y rico en funciones. Admite conversiones de formato de archivo de alta fidelidad hacia y desde XLS, XLSX, XLSM, SpreadsheetML, CSV, delimitado por tabulaciones, HTML, MHTML y hoja de cálculo OpenDocument en PHP. Por favor visita[Página de comparación de características del producto](https://docs.aspose.com/cells/es/java/aspose-cells-for-php-via-java-features/)para tu referencia. Además, consulte el documento sobre cómo[Entorno de configuración y otras pautas de instalación](https://docs.aspose.com/cells/es/java/setup-and-installation-guidelines/)para tu referencia.
