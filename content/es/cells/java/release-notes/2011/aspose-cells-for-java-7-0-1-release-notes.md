 ¡Nos complace anunciar Aspose.Cells for Java v7.0.1!

 Mejoras

 30556 Mejora de la función Hoja de cálculo a imagen

 30557 Mejorar la conversión de Excel a PDF

30458 Agregar nuevas API: Worksheet.calculateFormula(), Row.getFirstCell(), Row.getLastCell()

 30487 Soporte Aspose.Total licencia

 29838 Renderizar imagen a gráficos personalizados del usuario

 30558 Incluir el método FormatCondition.getPriority()

 30505 Incluye el método Cells.checkColumn(int)

 30504 Incluir getters/setters para Cells. Atributos StandardHeightPixels/StandardWidthPixels

 30459 Incluye los métodos WorkbookSettings.setLocale(Locale) y Protection.getPasswordHash()

 30469 Incluir getters/setters para WorkbookSettings.WindowHeightInch/WindowWidthInch

 Excepciones

 29285 Obtenga una excepción al guardar PDF con configuración regional francesa

 30459 Obtiene una excepción al leer el archivo XLS con la configuración regional predeterminada que no tiene código de país

 30489 Worksheet.autoFitColumn() y Chart.toImage() pueden causar "java.awt.HeadlessException"

 30587 Obtiene una excepción al leer el archivo de plantilla XLS

30103 Las fuentes del gráfico cambiaron después de volver a guardar el archivo XLSX

 28858 Ordenar nombres hace que la celda vinculada del control cambie
