 Esta página contiene notas de la versión para[Aspose.Cells for Java 7.1.1](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.1/)

Estamos
 feliz de anunciar Aspose.Cells for Java v7.1.1!

 Nuevas características

- Comprobar si una columna está en blanco

 Mejoras

- Guardando en XLSX con LightCells API al renderizar directamente a OutputStream

 Excepciones

- El método Cell.setHtmlString() arroja NullPointerException
- Al leer un archivo de plantilla XLSX, se genera NullPointerException

 Insectos

- El eje de categoría secundario del gráfico SCATTER se convierte en eje de texto
- Los contenidos en el cuadro de texto no se han renderizado correctamente
- La función ROUND no puede redondear un valor superior a 922337,20
- El gráfico no se leyó correctamente del archivo ODStemplate
- El formato de número personalizado: "dd.MM.yyyy" no se guardó correctamente para el archivo ODS
- LineShape se acorta en el documento PDF generado
