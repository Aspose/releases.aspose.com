 Esta página contiene notas de la versión para[Aspose.Cells for Java 7.3.2](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.3.2/)

{{% /alert %}} 

Estamos
 feliz de anunciar Aspose.Cells for Java v7.3.2!

 Nuevas características

- Shape.getRight()/getBottom() para obtener un desplazamiento de la forma desde la esquina inferior derecha hasta la celda
- Establecer el color de la pestaña de la hoja de trabajo en el color predeterminado

 Mejoras

- Cerrar flujos internos para liberar recursos de memoria en la conversión PDF
- Copie Sparklines al copiar la hoja de trabajo

 Excepciones

- Abrir un archivo XLS provocó un error de desbordamiento de pila
- Guardar PDF provocó una excepción
- Worksheet.getFreezedPanes() causó NullPointerException
- Abrir un archivo XML vacío provocó una excepción
- Guardar en PDF provocó una excepción -I
- Guardar en PDF provocó una excepción -II
- Guardar en PDF provocó una excepción -III
- Guardar en PDF provocó una excepción -IV
- Excepción al abrir el archivo de plantilla HTM
- IllegalArgumentException al abrir un archivo XLS
- NullPointerException al abrir un archivo XLSfile
- Excepción ArrayIndexOutOfBounds al abrir un archivo XLS
- Guardar la tabla dinámica causó ClassCastException después de configurar la ordenación automática
- Error de forma a imagen al guardar PDF usando Aspose Cells
- CellsException al imprimir líneas de cuadrícula en PDF

 Insectos

- Algunas regiones guardadas en las plantillas XLS no se pudieron reconocer al leer
- Chart-to-Image falla al representar la escala para los gráficos de barras
- Pérdida de datos después de la conversión de XLSX a PDF
- Los gráficos circulares y de dispersión no funcionan si se borra Cellcollection
