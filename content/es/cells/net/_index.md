Aspose.Cells para .NET es una API de programación de hojas de cálculo de Excel® para acelerar las tareas de gestión y procesamiento de hojas de cálculo. Compatibilidad con la API de Excel® .NET para crear aplicaciones multiplataforma que tengan la capacidad de generar, modificar, convertir, representar e imprimir hojas de cálculo. Permite a los desarrolladores administrar hojas de trabajo, filas, columnas y celdas, crear contenidos y estilos de hojas de cálculo desde cero, importar datos a las hojas de trabajo desde diferentes fuentes de datos, agregar fórmulas matemáticas, financieras y de texto comunes y complejas, crear y manipular tablas dinámicas, gráficos , hipervínculos, comentarios, objetos de dibujo y mucho más.

Aspose.Cells para .NET proporciona una solución Grid con dos controles .NET basados en GUI. Aspose.Cells.GridDesktop admite aplicaciones de escritorio y Aspose.Cells.GridWeb está diseñado específicamente para aplicaciones web basadas en .NET. Ambos controles de Gird le permiten importar/exportar archivos de Excel®, manipular datos y formatear, personalizar el diseño y la distribución de cuadrículas, administrar varias hojas de trabajo, crear y calcular fórmulas de Excel® y muchas otras operaciones similares a Excel®.

## ¿Por qué descargar Aspose.Cells para .NET?

Aspose.Cells para .NET permite que sus aplicaciones .NET, C#, ASP.NET y VB.NET creen, formateen, representen, impriman y conviertan hojas de cálculo de Microsoft Excel® y OpenDocument (XLS, XLSX, ODS, etc.) a varios formatos compatibles, sin necesidad de instalar Microsoft Excel® o cualquier software de terceros. Su potente motor de cálculo de fórmulas admite varias fórmulas/funciones estándar y avanzadas de MS Excel®.

> Aspose.Cells para .NET está escrito en C# administrado y se puede usar en cualquier tipo de aplicación de escritorio de Windows basada en .NET o aplicación web ASP.NET. Nuestra biblioteca también se puede utilizar con ASP clásico y PHP.

Su modelo de objetos enriquecido consta de más de 100 clases públicas que ofrecen a los desarrolladores de .NET integrar funciones interesantes de manipulación de hojas de cálculo en sus aplicaciones, que van desde la creación de gráficos dinámicos, la combinación de hojas de cálculo, el análisis de datos hasta el cifrado y la exportación de archivos de hojas de cálculo. Renderice páginas de hojas de cálculo a imágenes de trama (TIFF, TIFF de varias páginas, PNG, JPEG, BMP) o vectoriales (EMF). Aspose.Cells para .NET también admite trabajar con XLSB, XLT, XLTX, XLTM, XLSM, XML, HTML, CSV, TSV, HTML/MHTML, etc. Es capaz de convertir hojas de cálculo a formatos PDF y XPS manteniendo la máxima calidad visual. fidelidad. Puede agregar gráficos estándar y personalizados y establecer datos dinámicamente para actualizar los gráficos. Los gráficos también se pueden convertir a formatos de imagen ráster y PDF de alta calidad.

## Funciones de procesamiento de archivos de hoja de cálculo

- Generación y manipulación de hojas de cálculo a través de API.
- Conversión y renderizado de formato de archivo de alta calidad.
- Imprima archivos de Microsoft Excel® en impresoras físicas o virtuales.
- Combine, modifique, proteja o analice hojas de Excel®.
- Aplicar formato de hoja de trabajo.
- Configurar y aplicar configuración de página para las hojas de trabajo.
- Cree y personalice gráficos de Excel®, tablas dinámicas, reglas de formato condicional, segmentaciones, tablas y minigráficos.
- Convierta gráficos de Excel® en imágenes y PDF.
- Convierta archivos de Excel® a varios otros formatos.
- Motor de cálculo de fórmulas que admite todas las funciones básicas y avanzadas de Excel®.

### Formatos de lectura y escritura admitidos

**Microsoft Excel®:** XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM, CSV, TSV, TabDelimited, SpreadsheetML\
**OpenOffice®:** SAO, SXC, FODS\
**Texto:** TXT\
**Web:** HTML, MHTML\
**iWork®:** Números\
**Otros:** SXC, FODS

### Guardar archivos de Excel® como

**Microsoft Word®:** DOCX\
**Microsoft PowerPoint®:** PPTX\
**Microsoft Excel®:** XLAM\
**Diseño fijo:** PDF, XPS\
**Intercambio de datos:** DIF\
**Gráficos vectoriales:** SVG\
**Imagen:** TIFF, PNG, BMP, JPEG, GIF\
**Metarchivo:** EMF\
**Rebaja:** MD
