 Queridos clientes,

 ¡Hemos lanzado Aspose.Grid.Web v1.5.1!

 En esta nueva corrección, realizamos las siguientes mejoras y correcciones:

 Aspose.Grid.Escritorio: optimizar el rendimiento y el uso de la memoria Compatible con filas, estilos de columnas Corregido: error de valor de la barra de desplazamiento al congelar filas o columnas Corregido: hipervínculo, comentario, índice de imagen no se actualizan al insertar, eliminar fila o columna

Aspose.Grid.Web: cambie el nombre de clase del control de "ExcelWeb" a "GridWeb". Admite ocultar la barra de encabezado. Admite ocultar la barra de desplazamiento. Admite la ruta relativa para los archivos de uso del lado del cliente. Admite el estilo de imagen de fondo de celda/tabla. AGWF-Aspose Formato web de cuadrícula). Se agregó la nueva propiedad "AGWClientPath", puede usar esta propiedad para establecer la ruta para los archivos de uso del lado del cliente (scripts, imágenes, etc.). Se corrigió el error de color al exportar a archivos de Excel. Se corrigió el error de fuente predeterminada .
