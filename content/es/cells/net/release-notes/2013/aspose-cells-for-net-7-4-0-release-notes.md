 Esta página contiene notas de la versión para[Aspose.Cells for .NET 7.4.0](https://releases.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.4.0/)

 ¡Nos complace anunciar Aspose.Cells para .NETv7.4.0!



\1) Aspose.Cells 



 Mejoras

- Representación de formato condicional avanzado en la función Hoja a imagen
- La regla es invisible en la vista Diseño de página



 Rendimiento

- Colgado al guardar en formato PDF



 Excepciones

- Excepción al abrir un archivo Open XML



 Insectos

- Bugin usando la fórmula como parámetro de una función personalizada
- Problema de actualización y cálculo de datos de la tabla dinámica
- HLOOKUP con error de fórmulas INDIRECTAS
- La desprotección de la hoja de trabajo falla para los archivos guardados con Excel 2013
- Problema con el número de página del pie de página en la exportación a PDF
- AcceptAllRevisions no se aplica correctamente al guardar Workbook en secuencias

 -Casillas de verificación corruptas renderizadas al guardar

- Descifrar un archivo protegido con contraseña corrompe el archivo

 -Rangos de nombres dañados al guardar

- La configuración regional no funciona para una celda con formato DataTime
- Error al formatear un documento con comillas simples en la columna de la pestaña Encabezado
- El formato de caracteres específicos en una cadena no funciona para un archivo XLSX



 \2) Aspose.Cells.GridWeb



 Nuevas características

 41269 - Cargar archivos de Excel con menús desplegables: validación de datos de lista.

 41312 - Evento de clic de soporte para dibujar objetos/formas



 Mejoras

 41301 - El valor de escritura directa en una celda no es compatible con Chrome, FireFox y Safari



 Insectos

41279 - La lista del menú de validación de GridWeb fallará en los elementos del menú que contengan el signo de dólar - "$"

 41280- La lista del menú de validación de GridWeb se actualiza después de actualizar el valor de la celda a la que se hace referencia

 41294 - GridWeb no funcionará en FireFox para los archivos que contienen celdas con lista de validación

 41308 - Al editar un valor de celda, al presionar la tecla Intro se perderá el salto de línea original en IE

- Problema de copiar y pegar en la cuadrícula web que no funciona para Chrome, FireFox, etc.



\3) Aspose.Cells.GridEscritorio



 Mejoras

 41291 - Problema al usar el teclado para navegar a cualquier celda para ingresar un valor
