#  Soporte para formatos de plantillas de hojas de cálculo

Esta versión API ha introducido compatibilidad con formatos de plantillas de hojas de cálculo tanto para Excel&reg como para Excel&reg; (XLT) & OpenOffice Calc® (OTS) programas. Estos son compatibles como formatos de salida a través de `SaveFormat.Ots` y `SaveFormat.Xlt`.

#  Establecer bordes interiores

Posibilidad de establecer los límites interiores.

#  Ordenar campo dinámico por fila o columna seleccionada

Se ha agregado compatibilidad con la clasificación de los valores del campo dinámico. La clasificación del campo dinámico se puede realizar según la fila o columna deseada.

 > Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite[Aspose.Cells for .NET 22.3 Notas de la versión](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-3-release-notes/).
