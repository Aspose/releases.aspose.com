#  `.NUMBERS` Soporte de almacenamiento de datos 5.0

Se agregó el soporte para almacenamiento de datos 5.0 de `.numbers`.

#  Representación mejorada de XLSX a PDF

Se observó que al convertir XLSX a PDF, los gráficos de barras no se representaban correctamente. Este problema ya se ha resuelto.

#  Excel® mejorado Manejo de etiquetas de gráficos

Esta versión API ofrece un par de correcciones relacionadas con Excel&reg; etiquetas de gráficos. Son los siguientes:

- Las etiquetas de los gráficos no encajaban en la caja.
- Las etiquetas de los gráficos se estaban alineando verticalmente en lugar de horizontalmente.

#  Guardar fórmulas de tabla en XLSB

Se ha agregado soporte para guardar fórmulas de tablas en `XLSB`.

#  Soporte mejorado en Linux

Se observó que los caracteres o leyendas escritas en objetos como formas y cuadros de texto aparecían confusos en Linux. Este problema ya se ha resuelto.
