#  Establecer nombre del trabajo de la impresora

Se agregó soporte para proporcionar y configurar el nombre del trabajo de impresión con `PrinterSettings` mientras se procesan hojas de cálculo en la impresora.

#  Detección automática de formatos de archivo `CSV` y `TSV`

Esta versión API presenta la capacidad de detectar automáticamente archivos de texto como formatos de archivo separados por comas `CSV` y tabuladores `TSV` utilizando API.

#  Excluir filas/columnas ocultas del rango

La propiedad de rango cubría todas las celdas de la colección de celdas. A partir de esta versión, esta funcionalidad se ha mejorado para ignorar/excluir las filas y columnas ocultas al obtener `Cells.MaxDisplayRange`.

 > Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite[Aspose.Cells for .NET 22.1 Notas de la versión](https://docs.aspose.com/cells/net/aspose-cells-for-net-22-1-release-notes/).
