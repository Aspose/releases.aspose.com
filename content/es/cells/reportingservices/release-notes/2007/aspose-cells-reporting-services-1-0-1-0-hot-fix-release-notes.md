 Introducción

 Aspose.Cells para Reporting Service v1.0.1 incluye mejoras de funciones y correcciones de errores desde

 el primer lanzamiento.

 ¿Qué hay de nuevo?

- Compatible con matriz de exportación
- Admitido para exportar archivos CSV y TabDelimited
- Apoyó la imagen de fondo.

 ¿Qué está arreglado?

- Se corrigió el error cuando image.ImageData es nulo.
- Se corrigió el error cuando el valor del cuadro de texto contiene formato de número.
- Se corrigió el error cuando se agrupa la matriz.
