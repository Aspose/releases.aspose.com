# Compatibilidad de compresión de archivo CPIO mejorada

Soporte para comprimir archivos 'CPIO' a compresión de archivos 'GZIP', 'BZIP2' y 'XZ'. Mientras que los formatos `TAR` y `CPIO` también se respetan al guardar en `BZIP2`.

# Soporte para el algoritmo Snappy

Se agregó el soporte para el algoritmo ágil que ofrece alta velocidad y buena compresión.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.ZIP for .NET 22.7 Release Notes](https://docs.aspose.com/zip/net/aspose-zip-for-net-22-7-release-notes/).
