# Soporte .Net 2.0

Restaurado el soporte de .NET Standard 2.0.

# Conversión mejorada de USDZ a GLB

Anteriormente, en algunos casos, la API no podía abrir los archivos 'USDZ' para convertirlos al formato 'GLB'. Esta versión de la API ha resuelto ese problema.

# Codificación y decodificación de entidades primitivas

Se agregó el soporte de codificación y decodificación de entidades primitivas en el exportador USD.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.3D for .NET 22.1 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-1-release-notes/).
