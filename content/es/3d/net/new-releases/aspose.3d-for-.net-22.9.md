# Compatibilidad con archivos temporales para el importador FBX

Se agregó el soporte del sistema de archivos temporal interno para el importador FBX.

# Exportación GLTF mejorada

La exportación `GLTF` no fue tan buena en las versiones anteriores. Sin embargo, con el lanzamiento de la versión actual de la API 3D, se mejoró la función de exportación de archivos con formato `GLTF`.

> Para obtener una lista completa de funciones, mejoras y correcciones de errores en esta versión, visite [Aspose.3D for .NET 22.9 Release Notes](https://releases.aspose.com/3d/net/release-notes/2022/aspose-3d-for-net-22-9-release-notes/).
