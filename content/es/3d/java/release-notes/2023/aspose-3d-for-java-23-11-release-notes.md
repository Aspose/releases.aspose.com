### Cambios de API


### Se agregaron miembros a la clase **com.aspose.threed.FileFormat**:

{{< highlight java >}}

    /**
     * Universal Scene Description in ASCII format.
     */
    public static final FileFormat USDA;

Desde Aspose.3D 23.11, Aspose.3D puede importar archivos USDA o USDA empaquetados como archivos USDZ ahora.
