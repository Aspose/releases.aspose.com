Developers can format pages to the most granular level, create and manipulate shape, render pages, shapes to PDF and images, and much more - all without any dependency on Microsoft Office or Visio application. Visio Python via .NET On-premise API Features

  • Create visio files via API.
  • High-quality file format conversion & rendering.
  • Print Microsoft Visio® files to physical or virtual printers.
  • Combine, modify, protect, or parse Visio® pages.
  • Convert shapes to images or PDF.
  • Configure and apply page setup for the pages.
  • Convert Visio® files to various other formats.

Please visit the official documentation for a more detailed list of features.

Read Visio File Formats

Microsoft Visio: VSD, VDW,VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM

Save Visio Files AS

Microsoft Visio: VSX,VTX, VDX, VSDX, VSTX, VSSX, VSTM, VSSM Fixed Layout: PDF, XPS Images: JPEG, PNG, BMP, SVG, TIFF, GIF, EMF Web: HTML

System Requirements

Your machine does not need to have Microsoft Visio® software installed.

Supported Operating Systems

Microsoft Windows®: Windows Desktop & Server (x64, x86)
Linux: Ubuntu, OpenSUSE, CentOS, and others
Other: Any operating system (OS) that can install Mono(.NET 4.0 Framework support) or use .NET core.
