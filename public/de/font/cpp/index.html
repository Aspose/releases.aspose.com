Warum Aspose.Font für C++ herunterladen?

Aspose.Font für C++ ist eine C++-API, die Ihnen eine Bibliothek mit leicht verfügbarem C++-Code für die Verarbeitung und Bearbeitung von Schriftarten bietet. Unsere API ermöglicht es Ihren C++-Anwendungen, Schriftdaten zu laden, zu bearbeiten und zu speichern sowie beliebige Glyphen oder Texte zu rendern.

Funktionen zur Schriftverarbeitung

  • Laden Sie Schriftdateien sowohl von der Disc als auch vom Stream.
  • Schriftinformationen lesen und aktualisierte Schriftdateien auf Disc speichern.
  • Unterstützung für TrueType- und OpenType-Schriftarten.
  • Arbeiten Sie mit CFF- und Type1-Schriftarten in Ihren C++-Apps.
  • Glyphen- und Metrikinformationen aus Schriftdateien lesen.
  • Erkennen Sie lateinische Symbole in Schriftarten.
  • Extrahieren Sie eingebettete Lizenzinformationen aus Schriftartdateien.
  • Rendern Sie Text mit Schriftart-Glyphen.

Schriftformate lesen und schreiben

TTF

Schriftformate lesen

TTC, OpenType, CFF, Type1, WOFF/WOFF2

Unterstützte Plattformen

Aspose.Font für C++ ist eine native Bibliothek. Es unterstützt sowohl 32-Bit- als auch 64-Bit-Betriebssysteme. Außerdem können Sie Aspose.Font für C++ verwenden, um Anwendungen in jeder Entwicklungsumgebung zu entwickeln, die C++ unterstützt, aber die aufgeführten Umgebungen werden ausdrücklich unterstützt.

Microsoft Windows®: Microsoft Windows Desktop (XP, Vista, 7, 8, 10) und Serverbetriebssysteme (2003, 2008, 2012), etc. Entwicklungsumgebungen: Microsoft Visual C++ Version 2017 oder höher.
