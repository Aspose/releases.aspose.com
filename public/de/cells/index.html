Mit Aspose.Cells für C++ können Sie leistungsstarke C++-Anwendungen erstellen, die Excel-Tabellenkalkulationen erstellen, bearbeiten und konvertieren können, ohne auf Office-Automatisierung oder Microsoft Excel-Anwendungen angewiesen zu sein.

Aspose.Cells für JasperReports

Aspose.Cells for JasperReports ist eine flexible JasperReports- und JasperServer-Erweiterung, die es ermöglicht, Berichte in Microsoft Excel XLS-, XLSX-, PDF-, ODS-, TXT- und SpreadsheetML-Formaten zu exportieren, ohne Microsoft Excel zu verwenden. Es fungiert als Alternative zum ursprünglichen Microsoft Excel-Exporter von JasperReports und eliminiert die Abhängigkeit von der POI-Bibliothek.

Aspose.Cells für Python über Java
