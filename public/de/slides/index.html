Support Forum

Aspose.Slides für .NET

Aspose.Slides for .NET ist eine einzigartige Präsentationsverarbeitungs-API, die es jeder .NET-Anwendung ermöglicht, PowerPoint-Präsentationen zu lesen, zu schreiben, zu ändern und zu konvertieren. Als eigenständige API benötigt Aspose.Slides für .NET weder Microsoft PowerPoint noch Office Automation

ASSOSH.Slides für Java

ASSON.Slides für Java ist die Manipulation und Verarbeitungs -API für PowerPoint -Präsentation zum Lesen, Schreiben, Manipulieren und Konvertieren von PowerPoint -Dateien in jeder Java -basierten Anwendung. API hat keine externen Abhängigkeiten, daher kann sie ohne Microsoft PowerPoint verwendet werden.

Aspose.Slides für C++

Aspose.Slides for C++ ist eine Präsentationsverarbeitungsbibliothek, mit der C++-Anwendungen PowerPoint-Präsentationen bearbeiten, rendern und konvertieren können, ohne dass eine Microsoft PowerPoint-Anwendung erforderlich ist.

Aspose.Slides für Android über Java

Aspose.Slides für Android über Java ist eine API zur Manipulation von PowerPoint-Dateien für Programmierer mobiler Anwendungen. Android-Entwickler können eine Vielzahl von Aufgaben zur Präsentationsverarbeitung ausführen, darunter Lesen, Schreiben, Bearbeiten und Konvertieren von PowerPoint-Präsentationen. Es ist eine eigenständige Präsentations-Android-API, daher besteht keine Abhängigkeit von anderen Produkten, einschließlich Microsoft PowerPoint.

Aspose.Slides für Reporting Services

Aspose.Slides for Reporting Services ist die einzige Lösung auf dem Markt, die es ermöglicht, Präsentationen in den nativen Formaten von PowerPoint aus den Microsoft SQL Server Reporting Services heraus zu erstellen.

Aspose.Folien für SharePoint

Aspose.Slides for SharePoint ist eine SharePoint-App, die Microsoft PowerPoint-Dateien in andere Formate konvertieren kann. Aspose.Slides for SharePoint kann problemlos gängige Microsoft-Präsentationsformate wie PPT, POT, PPS, PPTX, POTX und PPSX importieren und diese Formate in PDF-, TIFF- und XPS-Dateien exportieren.

Aspose.Slides für JasperReports

Aspose.Slides for JasperReports ist eine flexible JasperReports- und JasperServer-Erweiterung, mit der Sie Berichte nach Microsoft PowerPoint exportieren können. JasperServer verfügt nicht über die integrierte Funktion zum Exportieren von Berichten als PowerPoint-Präsentationen, aber mit dieser Erweiterung können die JasperServer-Berichte in die Präsentationsformate PPT, PPS, PPTX und PPSX exportiert werden, ohne Microsoft PowerPoint zu verwenden.

Aspose.Slides für Python über .NET

Aspose.Slides für Python über .NET ist eine fortschrittliche Powerpoint-Präsentationsverarbeitungs-API, mit der Sie eine Vielzahl von Dokumentenverarbeitungsaufgaben direkt in Ihren Anwendungen ausführen können.

Aspose.Slides für PHP über Java

Eine Präsentationsverarbeitungs-API, die es den PHP-Anwendungen ermöglicht, PowerPoint-Präsentationen zu manipulieren, zu rendern und zu konvertieren, ohne dass eine Microsoft PowerPoint-Anwendung erforderlich ist.
