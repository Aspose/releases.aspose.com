Aspose.CAD for Java 1.1.0 Release notes

We are pleased to announce the release of Aspose.CAD 1.1.0 for Java. The following is a list of changes in this version of Aspose.CAD.

Features and Improvements

KeySummaryCategory
CADJAVA-48Find or search text in the DWG fileNew Feature
CADJAVA-47Support to retrieve block attribute values from within a dwg and its external referencesNew Feature
CADJAVA-42Support for DGN file format for possible conversion to PDF & raster image formatsNew Feature
CADJAVA-46Cad export to Bmp fails with out of memory exceptionEnhancement
CADJAVA-44Unable to access Face3D Objects in DXF file [.Net]Enhancement
CADJAVA-40Update Java packaging to conform to Aspose guidelinesEnhancement
