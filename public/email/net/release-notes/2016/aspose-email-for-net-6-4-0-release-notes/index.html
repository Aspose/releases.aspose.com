Aspose.Email for .NET 6.4.0 Release Notes

Aspose.Email for .NET is a class library that enables applications to manipulate popular message formats including Microsoft Outlook messages. It also supports communication protocols such as IMAP, SMTP, POP3, and Microsoft Exchange Server. In addition, the API supports working with PST as well as OST file formats.

Major Features

  • Sending Calendar invitation
  • Moving messages using IMAP
  • Paging Support in Email clients (IMAP, EWS Listing Messages, Listing Folders Listing Appointments)
  • Saving messages as Outlook Template (.oft)
  • Context Timezone support in EWS Client
  • Using search criteria and paging simultaneously for messages listing using the IMAP client
  • Getting the division marker when working with MBOX format
