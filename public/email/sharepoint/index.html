Email SharePoint® Integration Features

  • Convert EML to MSG and MSG to EML email formats.
  • Extract all or selected email attachments.
  • Synchronize emails with a POP3, IMAP, or Exchange server.
  • Synchronize files between SharePoint® document library and FTP server.
  • Synchronize files between SharePoint® document library and Dropbox®.
  • Aspose Tools Ribbon within your SharePoint® UI.
  • Transfer and synchronize files between the SharePoint® document library and supported file hosting/sharing services.

Supported Message File Formats
