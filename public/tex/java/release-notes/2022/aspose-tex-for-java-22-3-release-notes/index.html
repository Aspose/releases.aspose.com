Aspose.TeX for Java 22.3 Release Notes

Aspose.TeX 22.3 for Java has been released!

  • Support for SVG as an output format.
  • Support for PS/EPS and XPS/OXPS documents inclusion with graphics(x) package.
  • Support for tikz package.
KeySummaryCategory
TEXNET-190Implement TeX to SVG conversionEnhancement
TEXNET-193Implement TeX math formula rendering to SVGEnhancement
TEXNET-194Implement PS/EPS graphics inclusionEnhancement
TEXNET-196Implement XPS/OXPS graphics inclusionEnhancement
TEXNET-197Adapt tikz packageFeature
