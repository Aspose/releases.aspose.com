Aspose.Slides for Reporting Services

Aspose.Slides for Reporting Services is a Microsoft PowerPoint (PPT and PPS-based) report generation extension with support for all popular Microsoft SQL Server Reporting Services (32-bit & 64-bit) versions.

Features

  • Accurate and high-fidelity rendering of reports.
  • Ability to export reports in the MS PowerPoint Presentation supported formats.
  • Supports embedding video and audio within the presentation (PPTX export only).
  • MS PowerPoint installation is not required.

Supported Input Formats

  • RPL (Report Page Layout)
  • RDL (Report Definition Language)

Supported Output Formats

  • Microsoft PowerPoint: PPT, PPS, PPTX, PPSX
  • Fixed Layout: XPS

Supported RDL Elements

  • Page, headers, footers
  • Textboxes
  • Images
  • Sub-reports
  • Charts
  • Lists
  • Tables
  • Matrices
  • Styles
  • Rectangles

Supported Platforms

  • Microsoft SQL Server 2005 Reporting Services (32-bit & 64-bit)
  • Microsoft SQL Server 2008 Reporting Services (32-bit & 64-bit)
  • .NET Framework 2.0

Supported Microsoft PowerPoint Versions

  • Microsoft PowerPoint 97
  • Microsoft PowerPoint 2000
  • Microsoft PowerPoint XP
  • Microsoft PowerPoint 2003
  • Microsoft PowerPoint 2007
  • Microsoft PowerPoint 2010
  • Microsoft PowerPoint 2013
  • Microsoft PowerPoint 2016
  • Microsoft PowerPoint 2019
  • Microsoft PowerPoint for MAC
