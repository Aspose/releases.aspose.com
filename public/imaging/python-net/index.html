Also Aspose.Imaging library supports drawing and work with graphic primitives. Imaging API Features

The following are Aspose.Imaging’s core features:

  • Create images
  • Read images
  • Write images
  • Draw images
  • Process images (including per-pixel modifications)
  • Convert between formats
  • Crop, rotate & resize images via API.
  • De-skew & transform images.
  • Set image properties.

Please visit the official documentation for a more detailed list of features.
