Aspose.Tasks for Java 18.3 Release Notes

This page contains release notes information about Aspose.Tasks for Java 18.3.

Major Features

  • Renumbering of WBS Code
  • Eliminating the gap between Tasks list and page footer
  • Get page count depending upon start and end date time
