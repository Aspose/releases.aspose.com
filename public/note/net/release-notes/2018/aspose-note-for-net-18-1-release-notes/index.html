Aspose.Note for .NET 18.1 Release Notes

This page contains release notes information of Aspose.Note for .NET 18.1.

Major Features

This version includes new feature of saving document to stream in HTML format. This includes:

  • Saving to Stream with Embedded Resources
  • Saving to File with Resources In Separate Files
  • Saving to Stream with Callbacks for Saving Resources
