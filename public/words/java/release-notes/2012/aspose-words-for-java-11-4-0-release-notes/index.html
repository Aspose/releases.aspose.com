Aspose.Words for Java 11.4.0 Release Notes

This page contains release notes for Aspose.Words for Java 11.4.0

What’s new There are 133 improvements and fixes in this regular monthly release. Most notable are: Implemented embedded EMF+ metafiles rendering;Implemented font style simulation while rendering to fixed page formats;Improved rendering of gray colors to PDF for proper printing;Improved PDF A1/B compatibility;Improved EMF rendering.
