Aspose.Words for Java 14.3 Release Notes

Major Features

There are 95 improvements and fixes in this regular monthly release. The most notable are:

  • Public API for insertion of OLE objects both linked and embedded Outline, Shadow, Reflection, Glow and Fill text effects for rendering text inside DrawingML shapes EUDC fonts rendering partially supported “PDF Logical Structure” export reworked, significantly improving memory usage
