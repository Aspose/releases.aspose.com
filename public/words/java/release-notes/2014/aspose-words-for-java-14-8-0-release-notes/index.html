Aspose.Words for Java 14.8 Release Notes

Major Features

There are 79 improvements and fixes in this regular monthly release. The most notable are:

  • Improved how TOC field culture and formatting is handled Timestamping of PDF digital signature using external timestamp server implemented Improvements to advanced rendering features e.g. pre-blending semi-transparent images Horizontal rotation of characters having HorizontalRotatedFarEast orientation supported Improved page layout for additional compatibility modes Improved floater positioning in complex cases
